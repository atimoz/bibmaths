Suites recurrentes et point fixe

Analyse -- 5 questions
Standard
Enonce type Soit \(f(x) = \frac{1}{2}\left(x + \frac{2}{x}\right)\) definie sur \(]0, +\infty[\). On considere la suite \((u_n)\) definie par \(u_0 > 0\) et \(u_{n+1} = f(u_n)\). Etudier la convergence de \((u_n)\) et determiner sa limite.
Question 1 -- Point fixe de \(f\)

Determiner les points fixes de \(f\) sur \(]0, +\infty[\).

Rappel de cours
Un point fixe de \(f\) est une valeur \(\ell\) telle que \(f(\ell) = \ell\). Si la suite \((u_n)\) converge vers \(\ell\) et \(f\) est continue, alors \(\ell\) est un point fixe. C'est le theoreme du point fixe : on cherche la limite AVANT de prouver la convergence.

On resout \(f(x) = x\) :

$$\frac{1}{2}\left(x + \frac{2}{x}\right) = x \implies \frac{2}{x} = x \implies x^2 = 2$$
Resultat
$$\boxed{\ell = \sqrt{2} \quad (\text{unique point fixe sur } ]0, +\infty[)}$$
Question 2 -- Monotonie et minorant

Montrer que pour tout \(n \geq 1\), \(u_n \geq \sqrt{2}\).

Pourquoi on fait ca ?
C'est la methode classique pour les suites recurrentes : on montre d'abord que la suite reste dans un intervalle (ici \([\sqrt{2}, +\infty[\)) puis on montre la monotonie. L'inegalite AM-GM (moyenne arithmetique / geometrique) est l'outil parfait ici.

Par l'inegalite AM-GM, pour tout \(x > 0\) :

$$f(x) = \frac{1}{2}\left(x + \frac{2}{x}\right) \geq \sqrt{x \cdot \frac{2}{x}} = \sqrt{2}$$

Egalite si et seulement si \(x = \frac{2}{x}\), soit \(x = \sqrt{2}\). Donc pour tout \(n \geq 1\), \(u_n = f(u_{n-1}) \geq \sqrt{2}\).

Resultat
$$\boxed{\forall n \geq 1, \quad u_n \geq \sqrt{2}}$$
Piege classique
Attention : le minorant vaut a partir de \(n = 1\), pas \(n = 0\). Si \(u_0 = 0.5\), alors \(u_0 < \sqrt{2}\) mais \(u_1 = \frac{1}{2}(0.5 + 4) = 2.25 > \sqrt{2}\). Ne pas oublier ce decalage d'indice.
Question 3 -- Decroissance

Montrer que pour \(n \geq 1\), la suite est decroissante.

Pour \(x \geq \sqrt{2}\), on calcule :

$$f(x) - x = \frac{1}{2}\left(\frac{2}{x} - x\right) = \frac{2 - x^2}{2x} \leq 0$$

car \(x \geq \sqrt{2}\) implique \(x^2 \geq 2\). Donc \(u_{n+1} = f(u_n) \leq u_n\) pour \(n \geq 1\).

Resultat
$$\boxed{(u_n)_{n \geq 1} \text{ est decroissante et minoree par } \sqrt{2}, \text{ donc convergente vers } \sqrt{2}}$$
Methode -- Etude de suites \(u_{n+1} = f(u_n)\)
La recette standard : (1) trouver les points fixes, (2) etudier le signe de \(f(x) - x\) pour determiner la monotonie, (3) montrer que la suite est bornee, (4) conclure par convergence monotone. Pour la vitesse de convergence, etudier \(|f'(\ell)|\).
Question 4 -- Vitesse de convergence

Etudier la vitesse de convergence de \((u_n)\) vers \(\sqrt{2}\).

Pourquoi on fait ca ?
La methode de Newton (dont cette suite est un cas particulier pour \(\sqrt{2}\)) a une convergence quadratique, c'est-a-dire que l'erreur est mise au carre a chaque iteration. Concretement, si on a 3 decimales justes a l'etape \(n\), on en a 6 a l'etape \(n+1\).

Posons \(\varepsilon_n = u_n - \sqrt{2}\). Alors :

$$\varepsilon_{n+1} = u_{n+1} - \sqrt{2} = \frac{1}{2}\left(u_n + \frac{2}{u_n}\right) - \sqrt{2} = \frac{u_n^2 - 2\sqrt{2}\,u_n + 2}{2u_n} = \frac{(u_n - \sqrt{2})^2}{2u_n} = \frac{\varepsilon_n^2}{2u_n}$$

Comme \(u_n \geq \sqrt{2}\), on obtient \(\varepsilon_{n+1} \leq \frac{\varepsilon_n^2}{2\sqrt{2}}\).

Resultat
$$\boxed{\varepsilon_{n+1} = \frac{\varepsilon_n^2}{2u_n} \leq \frac{\varepsilon_n^2}{2\sqrt{2}} \quad \text{(convergence quadratique)}}$$
Question 5 -- Application numerique

Calculer \(u_1, u_2, u_3\) pour \(u_0 = 2\).

Endomorphismes et polynome minimal

Algebre lineaire -- 5 questions
Intermediaire
Enonce type Soit \(E\) un espace vectoriel de dimension 3 et \(u \in \mathcal{L}(E)\) tel que \(u^3 = u\). Determiner les valeurs propres possibles de \(u\), montrer que \(u\) est diagonalisable, et decrire les formes possibles de la matrice de \(u\) dans une base adaptee.
Question 1 -- Valeurs propres possibles

Determiner les valeurs propres possibles de \(u\).

Rappel de cours
Si \(P(u) = 0\) (polynome annulateur), alors toute valeur propre de \(u\) est racine de \(P\). C'est parce que si \(u(x) = \lambda x\), alors \(P(u)(x) = P(\lambda)x = 0\), donc \(P(\lambda) = 0\) (car \(x \neq 0\)).

\(u^3 = u\) signifie que \(P(X) = X^3 - X = X(X-1)(X+1)\) annule \(u\). Les valeurs propres de \(u\) sont donc parmi les racines de \(P\) :

Resultat
$$\boxed{\text{Valeurs propres possibles : } \lambda \in \{-1, 0, 1\}}$$
Question 2 -- Diagonalisabilite

Montrer que \(u\) est diagonalisable.

Pourquoi on fait ca ?
Le critere est le suivant : si un endomorphisme est annule par un polynome scinde a racines simples, alors il est diagonalisable. C'est un des criteres les plus puissants et les plus rapides. Ici, \(X^3 - X = X(X-1)(X+1)\) est scinde a racines simples.

Le polynome \(P(X) = X(X-1)(X+1)\) est scinde a racines simples et annule \(u\). Donc le polynome minimal de \(u\) divise \(P\) et est aussi scinde a racines simples.

Resultat
$$\boxed{u \text{ est diagonalisable}}$$
Methode -- Critere de diagonalisabilite par polynome annulateur
Le schema : (1) trouver un polynome \(P\) qui annule \(u\), (2) verifier qu'il est scinde a racines simples, (3) conclure. C'est le critere le plus pratique au concours, bien plus rapide que de calculer tous les sous-espaces propres.
Question 3 -- Decomposition de \(E\)

Ecrire la decomposition de \(E\) en somme directe de sous-espaces propres.

Rappel de cours
Le lemme des noyaux dit que si \(P = P_1 \cdots P_k\) avec les \(P_i\) premiers entre eux et \(P(u) = 0\), alors \(E = \ker(P_1(u)) \oplus \cdots \oplus \ker(P_k(u))\).

Par le lemme des noyaux applique a \(P(X) = X(X-1)(X+1)\) :

$$E = \ker(u) \oplus \ker(u - \text{Id}) \oplus \ker(u + \text{Id})$$

Notons \(p = \dim(\ker u)\), \(q = \dim(\ker(u - \text{Id}))\), \(r = \dim(\ker(u + \text{Id}))\) avec \(p + q + r = 3\).

Resultat
$$\boxed{E = E_0 \oplus E_1 \oplus E_{-1} \quad \text{avec } \dim E_0 + \dim E_1 + \dim E_{-1} = 3}$$
Question 4 -- Formes possibles

Lister toutes les formes diagonales possibles pour la matrice de \(u\).

Les triplets \((p, q, r)\) possibles avec \(p + q + r = 3\) et \(p, q, r \geq 0\) sont :

Cas\(\dim E_0\)\(\dim E_1\)\(\dim E_{-1}\)Matrice diagonale
1300\(\text{diag}(0,0,0)\)
2030\(\text{diag}(1,1,1)\)
3003\(\text{diag}(-1,-1,-1)\)
4210\(\text{diag}(0,0,1)\)
5111\(\text{diag}(0,1,-1)\)
...et les autres permutations
Piege classique
Ne pas oublier que certaines valeurs propres peuvent ne pas etre presentes ! Le fait que \(-1, 0, 1\) soient valeurs propres possibles ne signifie pas qu'elles sont toutes valeurs propres. Par exemple, \(u = \text{Id}\) verifie \(u^3 = u\) et n'a que la valeur propre 1.
Question 5 -- Projecteurs associes

Exprimer les projecteurs sur les sous-espaces propres en fonction de \(u\).

Pourquoi on fait ca ?
Les projecteurs spectraux permettent de decomposer n'importe quel vecteur sur les sous-espaces propres sans calculer de base. C'est un outil tres puissant pour les applications (systemes differentiels, puissances de matrices...).

Par les interpolateurs de Lagrange appliques aux polynomes d'endomorphismes :

$$p_0 = \frac{(u - \text{Id})(u + \text{Id})}{(0 - 1)(0 + 1)} = -(u - \text{Id})(u + \text{Id}) = \text{Id} - u^2$$ $$p_1 = \frac{u(u + \text{Id})}{(1)(2)} = \frac{u^2 + u}{2}$$ $$p_{-1} = \frac{u(u - \text{Id})}{(-1)(-2)} = \frac{u^2 - u}{2}$$
Verification : \(p_0 + p_1 + p_{-1} = \text{Id}\)

\((\text{Id} - u^2) + \frac{u^2 + u}{2} + \frac{u^2 - u}{2} = \text{Id} - u^2 + \frac{2u^2}{2} = \text{Id} - u^2 + u^2 = \text{Id}\). OK.

Resultat
$$\boxed{p_0 = \text{Id} - u^2, \quad p_1 = \frac{u^2 + u}{2}, \quad p_{-1} = \frac{u^2 - u}{2}}$$

Integrales a parametre et fonction Gamma

Analyse -- 5 questions
Intermediaire
Enonce type On considere la fonction \(\Gamma\) definie pour \(x > 0\) par \(\Gamma(x) = \int_0^{+\infty} t^{x-1} e^{-t}\,\mathrm{d}t\). Montrer que \(\Gamma\) est bien definie, calculer \(\Gamma(1)\) et \(\Gamma(1/2)\), et etablir la relation \(\Gamma(x+1) = x\,\Gamma(x)\).
Question 1 -- Convergence de l'integrale

Montrer que \(\Gamma(x)\) est bien definie pour tout \(x > 0\).

Rappel de cours
L'integrale \(\int_0^{+\infty}\) est impropre en 0 et en \(+\infty\). Il faut etudier la convergence aux deux bornes separement. En 0, on compare a \(t^\alpha\) ; en \(+\infty\), l'exponentielle l'emporte sur toute puissance.
Pourquoi on fait ca ?
Avant de manipuler une integrale (deriver, integrer par parties...), il faut d'abord verifier qu'elle existe ! C'est une erreur classique au concours de faire des calculs sur une integrale divergente.

En \(+\infty\) : pour tout \(N > 0\), \(t^{x-1}e^{-t} = o(t^{-N})\) quand \(t \to +\infty\). En prenant \(N = 2\), on a convergence.

En 0 : \(t^{x-1}e^{-t} \sim t^{x-1}\) quand \(t \to 0^+\). Or \(\int_0^1 t^{x-1}\,\mathrm{d}t\) converge si et seulement si \(x - 1 > -1\), soit \(x > 0\).

Resultat
$$\boxed{\Gamma(x) \text{ est bien definie pour tout } x > 0}$$
Question 2 -- Calcul de \(\Gamma(1)\)

Calculer \(\Gamma(1)\).

$$\Gamma(1) = \int_0^{+\infty} t^0 e^{-t}\,\mathrm{d}t = \int_0^{+\infty} e^{-t}\,\mathrm{d}t = \left[-e^{-t}\right]_0^{+\infty} = 1$$
Resultat
$$\boxed{\Gamma(1) = 1}$$
Question 3 -- Relation fonctionnelle

Montrer que \(\Gamma(x+1) = x\,\Gamma(x)\) pour tout \(x > 0\).

Pourquoi on fait ca ?
Cette relation est l'analogue continu de \(n! = n \times (n-1)!\). C'est elle qui fait de \(\Gamma\) la "generalisation de la factorielle" aux reels positifs. Combinee avec \(\Gamma(1) = 1\), on obtient \(\Gamma(n+1) = n!\) pour tout entier \(n\).

Integration par parties avec \(u = t^x\) et \(\mathrm{d}v = e^{-t}\,\mathrm{d}t\) :

$$\Gamma(x+1) = \int_0^{+\infty} t^x e^{-t}\,\mathrm{d}t = \underbrace{\left[-t^x e^{-t}\right]_0^{+\infty}}_{= 0} + x\int_0^{+\infty} t^{x-1} e^{-t}\,\mathrm{d}t = x\,\Gamma(x)$$
Resultat
$$\boxed{\Gamma(x+1) = x\,\Gamma(x) \quad \text{et en particulier} \quad \Gamma(n+1) = n! \;\text{ pour } n \in \mathbb{N}}$$
Piege classique
Le terme entre crochets \([-t^x e^{-t}]_0^{+\infty}\) doit bien etre nul. En \(+\infty\), c'est parce que \(e^{-t}\) l'emporte sur toute puissance. En 0, c'est parce que \(x > 0\). Si \(x \leq 0\), l'IPP ne s'applique plus directement.
Question 4 -- Calcul de \(\Gamma(1/2)\)

Calculer \(\Gamma(1/2)\).

Methode -- Changement de variable dans les integrales Gamma
Pour calculer \(\Gamma(1/2)\), on fait le changement de variable \(t = s^2\) (donc \(\mathrm{d}t = 2s\,\mathrm{d}s\)) pour se ramener a l'integrale de Gauss \(\int_0^{+\infty} e^{-s^2}\,\mathrm{d}s = \frac{\sqrt{\pi}}{2}\). Retenir ce lien Gamma-Gauss.
$$\Gamma(1/2) = \int_0^{+\infty} t^{-1/2} e^{-t}\,\mathrm{d}t$$

Changement de variable \(t = s^2\), \(\mathrm{d}t = 2s\,\mathrm{d}s\) :

$$\Gamma(1/2) = \int_0^{+\infty} \frac{1}{s}\,e^{-s^2}\,2s\,\mathrm{d}s = 2\int_0^{+\infty} e^{-s^2}\,\mathrm{d}s = 2 \cdot \frac{\sqrt{\pi}}{2} = \sqrt{\pi}$$
Resultat
$$\boxed{\Gamma(1/2) = \sqrt{\pi}}$$
Question 5 -- Continuite et classe \(\mathcal{C}^1\) de \(\Gamma\)

Montrer que \(\Gamma\) est de classe \(\mathcal{C}^1\) sur \(]0, +\infty[\).

Rappel de cours
Le theoreme de derivation sous le signe integrale : si \(f(x,t)\) et \(\partial_x f(x,t)\) sont continues et si on a une domination \(|\partial_x f(x,t)| \leq g(t)\) avec \(g\) integrable independante de \(x\) (au moins localement), alors \(F(x) = \int f(x,t)\,\mathrm{d}t\) est \(\mathcal{C}^1\) et \(F'(x) = \int \partial_x f(x,t)\,\mathrm{d}t\).

On applique le theoreme de derivation sous le signe integrale. L'integrande est \(f(x,t) = t^{x-1}e^{-t}\).

Sa derivee partielle : \(\frac{\partial f}{\partial x}(x,t) = t^{x-1}\ln(t)\,e^{-t}\).

Pour \(x \in [a, b] \subset\, ]0, +\infty[\), on trouve une domination :

  • En \(+\infty\) : \(|t^{x-1}\ln(t)\,e^{-t}| \leq t^{b-1}|\ln t|\,e^{-t}\), integrable.
  • En 0 : \(|t^{x-1}\ln(t)\,e^{-t}| \leq t^{a-1}|\ln t|\), integrable car \(a > 0\).
Resultat
$$\boxed{\Gamma \in \mathcal{C}^1(]0,+\infty[) \quad \text{et} \quad \Gamma'(x) = \int_0^{+\infty} t^{x-1}\ln(t)\,e^{-t}\,\mathrm{d}t}$$
Methode -- Derivation sous le signe integrale
Le schema est toujours le meme : (1) identifier l'integrande \(f(x,t)\), (2) calculer \(\partial_x f\), (3) trouver une fonction dominante \(g(t)\) integrable independante de \(x\) sur un compact de \(x\), (4) conclure par le theoreme. Ce schema tombe chaque annee aux concours.

Probabilites : couples de variables aleatoires

Probabilites -- 5 questions
Avance
Enonce type Soit \((X, Y)\) un couple de variables aleatoires de densite jointe \(f_{X,Y}(x,y) = c\,e^{-(x+y)}\,\mathbf{1}_{0 < x < y}\). Determiner la constante \(c\), les lois marginales, l'independance, et calculer \(\text{Cov}(X, Y)\).
Question 1 -- Constante de normalisation

Determiner \(c\) pour que \(f_{X,Y}\) soit une densite.

Rappel de cours
Pour qu'une fonction \(f\) soit une densite jointe, il faut : \(f \geq 0\) partout, et \(\iint_{\mathbb{R}^2} f(x,y)\,\mathrm{d}x\,\mathrm{d}y = 1\). Le domaine d'integration est ici \(\{(x,y) : 0 < x < y\}\).
Pourquoi on fait ca ?
Le domaine \(0 < x < y\) est un triangle dans le plan. Pour integrer sur ce domaine, on peut parcourir \(x\) de 0 a \(+\infty\), et pour chaque \(x\), \(y\) va de \(x\) a \(+\infty\). Ou bien \(y\) de 0 a \(+\infty\) et \(x\) de 0 a \(y\). Les deux marchent.
$$\int_0^{+\infty}\!\!\int_x^{+\infty} c\,e^{-(x+y)}\,\mathrm{d}y\,\mathrm{d}x = c\int_0^{+\infty} e^{-x}\left(\int_x^{+\infty} e^{-y}\,\mathrm{d}y\right)\mathrm{d}x = c\int_0^{+\infty} e^{-x} \cdot e^{-x}\,\mathrm{d}x = c\int_0^{+\infty} e^{-2x}\,\mathrm{d}x = \frac{c}{2}$$
Resultat
$$\boxed{c = 2}$$
Question 2 -- Lois marginales

Determiner les densites marginales de \(X\) et de \(Y\).

Loi marginale de \(X\) : on integre en \(y\) sur \([x, +\infty[\) pour \(x > 0\) :

$$f_X(x) = \int_x^{+\infty} 2\,e^{-(x+y)}\,\mathrm{d}y = 2\,e^{-x}\,e^{-x} = 2\,e^{-2x}$$

Loi marginale de \(Y\) : on integre en \(x\) sur \([0, y]\) pour \(y > 0\) :

$$f_Y(y) = \int_0^{y} 2\,e^{-(x+y)}\,\mathrm{d}x = 2\,e^{-y}(1 - e^{-y})$$
Resultat
$$\boxed{f_X(x) = 2\,e^{-2x}\,\mathbf{1}_{x > 0} \quad (\text{loi } \mathcal{E}(2))}$$ $$\boxed{f_Y(y) = 2\,e^{-y}(1 - e^{-y})\,\mathbf{1}_{y > 0}}$$
Question 3 -- Independance

\(X\) et \(Y\) sont-elles independantes ?

On verifie si \(f_{X,Y}(x,y) = f_X(x) \cdot f_Y(y)\) :

$$f_X(x) \cdot f_Y(y) = 2e^{-2x} \cdot 2e^{-y}(1 - e^{-y}) = 4\,e^{-2x-y}(1 - e^{-y})$$

Or \(f_{X,Y}(x,y) = 2\,e^{-(x+y)}\,\mathbf{1}_{0 < x < y}\). Ce n'est clairement pas egal : la densite jointe a la contrainte \(x < y\), ce qui couple les deux variables.

Resultat
$$\boxed{X \text{ et } Y \text{ ne sont pas independantes}}$$
Piege classique
L'argument rapide est que le support de la loi jointe n'est pas un produit cartesien. Le domaine \(\{0 < x < y\}\) est un triangle, pas un rectangle. Des que le support n'est pas un produit, les variables ne sont pas independantes. C'est un critere visuel tres rapide.
Question 4 -- Esperances et moments

Calculer \(E(X)\), \(E(Y)\) et \(E(XY)\).

\(X \sim \mathcal{E}(2)\), donc \(E(X) = \frac{1}{2}\).

Pour \(E(Y)\) :

$$E(Y) = \int_0^{+\infty} y \cdot 2e^{-y}(1 - e^{-y})\,\mathrm{d}y = 2\int_0^{+\infty} ye^{-y}\,\mathrm{d}y - 2\int_0^{+\infty} ye^{-2y}\,\mathrm{d}y = 2 \cdot 1 - 2 \cdot \frac{1}{4} = \frac{3}{2}$$

Pour \(E(XY)\), on integre sur le domaine \(0 < x < y\) :

$$E(XY) = \int_0^{+\infty}\!\!\int_x^{+\infty} 2xy\,e^{-(x+y)}\,\mathrm{d}y\,\mathrm{d}x$$
Calcul detaille de \(E(XY)\)

L'integrale interieure : \(\int_x^{+\infty} ye^{-y}\,\mathrm{d}y = (x+1)e^{-x}\) (par IPP).

Donc \(E(XY) = 2\int_0^{+\infty} x(x+1)e^{-2x}\,\mathrm{d}x = 2\int_0^{+\infty} (x^2 + x)e^{-2x}\,\mathrm{d}x = 2\left(\frac{1}{4} + \frac{1}{4}\right) = 1\).

Resultat
$$\boxed{E(X) = \frac{1}{2}, \quad E(Y) = \frac{3}{2}, \quad E(XY) = 1}$$
Question 5 -- Covariance et coefficient de correlation

Calculer \(\text{Cov}(X, Y)\) et le coefficient de correlation.

$$\text{Cov}(X, Y) = E(XY) - E(X)E(Y) = 1 - \frac{1}{2} \cdot \frac{3}{2} = 1 - \frac{3}{4} = \frac{1}{4}$$

Pour le coefficient de correlation, il nous faut \(V(X)\) et \(V(Y)\) :

  • \(V(X) = \frac{1}{4}\) (car \(X \sim \mathcal{E}(2)\), \(V = \frac{1}{\lambda^2}\))
  • \(V(Y) = E(Y^2) - (E(Y))^2\). On calcule \(E(Y^2) = \frac{7}{2}\) d'ou \(V(Y) = \frac{7}{2} - \frac{9}{4} = \frac{5}{4}\).
$$\rho(X,Y) = \frac{\text{Cov}(X,Y)}{\sqrt{V(X)V(Y)}} = \frac{1/4}{\sqrt{1/4 \cdot 5/4}} = \frac{1/4}{\sqrt{5}/4} = \frac{1}{\sqrt{5}}$$
Resultat
$$\boxed{\text{Cov}(X,Y) = \frac{1}{4}, \quad \rho(X,Y) = \frac{1}{\sqrt{5}} \approx 0.447}$$
Methode -- Calcul de covariance
Toujours utiliser \(\text{Cov}(X,Y) = E(XY) - E(X)E(Y)\). Ne jamais oublier que covariance nulle n'implique pas independance (sauf pour des gaussiennes). Ici, \(\rho > 0\) confirme la correlation positive attendue : comme \(X < Y\), quand \(X\) est grand, \(Y\) tend a etre encore plus grand.
