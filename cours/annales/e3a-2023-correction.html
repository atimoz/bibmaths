Endomorphisme \(T\) sur \(\mathbb{R}_n[X]\) et valeurs propres

Polynomes, Taylor, reduction d'endomorphismes -- 8 questions
Standard
Enonce resume Soit \(n \in \mathbb{N}^*\), \(E_n = \mathbb{R}_n[X]\), \(P_k = X^k\). On definit \(T(P) = XP(X) - \frac{1}{n}(X^2 - 1)P'(X)\). Questions de cours sur la formule de Taylor et la division euclidienne par \((X - \alpha)^r\), puis etude de \(T\) : matrice, valeurs propres, diagonalisabilite.
Question 1 -- Question de cours

Justifier que la famille \(\mathcal{E} = (1, X - \alpha, \ldots, (X-\alpha)^n)\) est une base de \(E_n\).

Rappel de cours
Une famille de polynomes est dite echelonnee en degre quand les degres sont tous distincts. Dans un espace de polynomes de degre \(\leq n\), une famille echelonnee de \(n+1\) elements est automatiquement une base.
Pourquoi on fait ca ?
C'est comme demander pourquoi les vecteurs \((1, 0, 0)\), \((0, 1, 0)\), \((0, 0, 1)\) forment une base de \(\mathbb{R}^3\) : des polynomes de degres tous differents ne peuvent pas se compenser entre eux, donc ils sont libres.

La famille \(\mathcal{E} = (1, (X-\alpha), (X-\alpha)^2, \ldots, (X-\alpha)^n)\) est constituee de polynomes de degres respectifs \(0, 1, 2, \ldots, n\) : ce sont des polynomes non nuls, echelonnes en degre.

  • La famille est echelonnee en degre, donc elle est libre.
  • Elle contient \(n + 1\) elements et \(\dim(E_n) = n + 1\).
  • Une famille libre de cardinal egal a la dimension est une base.
Resultat
$$\boxed{\mathcal{E} = (1, X-\alpha, \ldots, (X-\alpha)^n) \text{ est une base de } E_n}$$
Piege classique
Le rapport du jury signale que beaucoup de candidats confondent dimension et cardinal d'une famille. La dimension est une propriete de l'espace vectoriel, le cardinal est le nombre de vecteurs dans la famille.
Question 2 -- Question de cours

Donner sans demonstration la decomposition de \(P\) dans la base \(\mathcal{E}\) a l'aide des derivees successives de \(P\).

Rappel de cours
La formule de Taylor pour les polynomes : pour tout \(P \in \mathbb{R}_n[X]\) et tout \(\alpha \in \mathbb{R}\), $$P = \sum_{k=0}^{n} \frac{P^{(k)}(\alpha)}{k!}(X - \alpha)^k$$ C'est une formule exacte (pas d'approximation, pas de reste) car \(P\) est un polynome.
Pourquoi on fait ca ?
Imaginez que vous vouliez ecrire un polynome en "centrant" autour d'un point \(\alpha\). C'est exactement ce que fait Taylor : les coordonnees de \(P\) dans la base \(\mathcal{E}\) sont les \(\frac{P^{(k)}(\alpha)}{k!}\). C'est l'analogue, pour les polynomes, du fait de changer de repere en geometrie.
Resultat
$$\boxed{P = \sum_{k=0}^{n} \frac{P^{(k)}(\alpha)}{k!}\,(X - \alpha)^k}$$ Les composantes de \(P\) dans la base \(\mathcal{E}\) sont \(\left(P(\alpha),\, P'(\alpha),\, \frac{P''(\alpha)}{2!},\, \ldots,\, \frac{P^{(n)}(\alpha)}{n!}\right)\).
Piege classique
Le rapport du jury note que la formule de Taylor polynomiale est ignoree par plus de la moitie des candidats. C'est pourtant un resultat fondamental a connaitre par coeur.
Question 3 -- Question de cours

On suppose que \(\alpha\) est une racine d'ordre \(r \in [\![1, n]\!]\) de \(P\). Determiner le quotient et le reste de la division euclidienne de \(P\) par \((X - \alpha)^r\).

Rappel de cours
Si \(\alpha\) est racine d'ordre \(r\) de \(P\), cela signifie : \(P(\alpha) = P'(\alpha) = \cdots = P^{(r-1)}(\alpha) = 0\) et \(P^{(r)}(\alpha) \neq 0\).
Pourquoi on fait ca ?
On combine deux outils : la formule de Taylor (Q2) et la definition de racine d'ordre \(r\). Dans Taylor, les \(r\) premiers termes disparaissent (car les derivees en \(\alpha\) sont nulles), et on peut factoriser par \((X - \alpha)^r\).

D'apres Taylor (Q2) :

$$P = \sum_{k=0}^{n} \frac{P^{(k)}(\alpha)}{k!}(X-\alpha)^k$$

Comme \(\alpha\) est racine d'ordre \(r\), les termes pour \(k = 0, 1, \ldots, r-1\) sont nuls. Donc :

$$P = \sum_{k=r}^{n} \frac{P^{(k)}(\alpha)}{k!}(X-\alpha)^k = (X-\alpha)^r \underbrace{\sum_{k=0}^{n-r} \frac{P^{(k+r)}(\alpha)}{(k+r)!}(X-\alpha)^k}_{= Q \text{ (le quotient)}}$$
Resultat
Le reste est nul (\(R = 0\)) et le quotient est : $$\boxed{Q = \sum_{k=0}^{n-r} \frac{P^{(k+r)}(\alpha)}{(k+r)!}\,(X-\alpha)^k}$$
Piege classique
Le rapport du jury note que le fait que le reste soit nul n'est pas evident pour de nombreux candidats. C'est pourtant la definition meme de "racine d'ordre \(r\)" : \((X-\alpha)^r\) divise \(P\).
Question 4

Soit \(k \in [\![0, n]\!]\). Determiner \(T(P_k)\).

Pourquoi on fait ca ?
Pour comprendre un endomorphisme, on calcule d'abord son action sur les vecteurs de base. Ici, les vecteurs de base sont \(P_k = X^k\). Une fois qu'on connait \(T(P_k)\) pour tout \(k\), on connait \(T\) entierement.

On applique la definition : \(T(P_k) = X \cdot X^k - \frac{1}{n}(X^2 - 1) \cdot kX^{k-1}\).

Calculons cas par cas :

Cas \(k = 0\) : \(T(1) = X \cdot 1 - \frac{1}{n}(X^2 - 1) \cdot 0 = X\). Donc \(T(P_0) = P_1\).

Cas \(1 \leq k \leq n-1\) :

$$T(X^k) = X^{k+1} - \frac{k}{n}(X^2 - 1)X^{k-1} = X^{k+1} - \frac{k}{n}X^{k+1} + \frac{k}{n}X^{k-1}$$ $$= \frac{n-k}{n}\,X^{k+1} + \frac{k}{n}\,X^{k-1}$$

Cas \(k = n\) :

$$T(X^n) = X^{n+1} - \frac{n}{n}(X^2 - 1)X^{n-1} = X^{n+1} - X^{n+1} + X^{n-1} = X^{n-1}$$
Resultat
$$\boxed{T(P_0) = P_1, \quad T(P_k) = \frac{n-k}{n}\,P_{k+1} + \frac{k}{n}\,P_{k-1} \;\;(1 \leq k \leq n-1), \quad T(P_n) = P_{n-1}}$$
Piege classique
Le rapport du jury signale que le cas \(k = 0\) est souvent oublie, et que des erreurs de calcul apparaissent lors du developpement des parentheses. Prenez le temps de bien developper !
Question 5

Montrer que \(T\) est un endomorphisme de \(E_n\).

Pourquoi on fait ca ?
On doit verifier deux choses : (1) \(T\) est lineaire, et (2) pour tout \(P \in E_n\), on a \(T(P) \in E_n\). Le point (2) n'est pas evident car \(XP(X)\) est de degre \(n+1\) en general, ce qui sortirait de \(E_n\).
Rappel de cours
Un endomorphisme de \(E_n\) est une application lineaire de \(E_n\) dans \(E_n\). Il faut verifier : linearite + stabilite de \(E_n\).

Linearite : \(T\) est combinaison lineaire d'operations lineaires (multiplication par \(X\), derivation, multiplication par un polynome fixe), donc \(T\) est lineaire.

Stabilite : D'apres Q4, pour tout \(k \in [\![0, n]\!]\), \(T(P_k) \in E_n\). Comme \((P_0, \ldots, P_n)\) est une base de \(E_n\), tout element de \(E_n\) s'ecrit comme combinaison lineaire des \(P_k\), et par linearite de \(T\), son image reste dans \(E_n\).

Resultat
$$\boxed{T \text{ est un endomorphisme de } E_n}$$
Piege classique
Le rapport du jury note que beaucoup de candidats oublient de verifier la loi externe (homogeneite) pour la linearite, et que certains ne justifient pas correctement que \(T(E_n) \subset E_n\) en se contentant du calcul de \(T(P_k)\) sans preciser que les \(P_k\) forment une base.
Question 6

Ecrire la matrice \(M\) de \(T\) dans la base \(\mathcal{B} = (P_0, P_1, \ldots, P_n)\).

Pourquoi on fait ca ?
La matrice d'un endomorphisme dans une base, c'est le "tableau de bord" qui resume tout. Les colonnes sont les coordonnees des images des vecteurs de base. On lit directement les images a partir de Q4.

D'apres Q4, les colonnes de \(M\) sont :

  • Colonne 0 : \(T(P_0) = P_1\), donc colonne \((0, 1, 0, \ldots, 0)^T\).
  • Colonne \(k\) (\(1 \leq k \leq n-1\)) : \(T(P_k) = \frac{k}{n}P_{k-1} + \frac{n-k}{n}P_{k+1}\).
  • Colonne \(n\) : \(T(P_n) = P_{n-1}\), donc colonne \((0, \ldots, 0, 1, 0)^T\).
Resultat
La matrice est tridiagonale : $$\boxed{M = \begin{pmatrix} 0 & \frac{1}{n} & 0 & \cdots & 0 \\ 1 & 0 & \frac{2}{n} & \ddots & \vdots \\ 0 & \frac{n-1}{n} & \ddots & \ddots & 0 \\ \vdots & \ddots & \ddots & 0 & \frac{n}{n} \\ 0 & \cdots & 0 & \frac{1}{n} & 0 \end{pmatrix}}$$ La sous-diagonale contient \(\frac{n-k}{n}\) et la sur-diagonale contient \(\frac{k}{n}\).
Piege classique
Le rapport du jury note que la matrice est souvent fausse apres de nombreux essais ratures. Conseil : construisez-la colonne par colonne en relisant vos calculs de Q4, et verifiez sur un petit cas (par exemple \(n = 2\)).
Question 7

On suppose que \(\lambda\) est une valeur propre reelle de \(T\) et \(P\) un polynome unitaire, vecteur propre associe. (7.1) Montrer que \(P\) est de degre \(n\). (7.2) Montrer que toute racine complexe \(z_0\) de \(P\) verifie \(z_0^2 - 1 = 0\). (7.3) En deduire une expression de \(P\).

Pourquoi on fait ca ?
C'est l'etape cruciale : on va montrer que les seuls vecteurs propres unitaires possibles ont une forme tres contrainte. En analysant le degre, puis les racines, on va prouver que les vecteurs propres sont necessairement de la forme \((X-1)^\alpha(X+1)^{n-\alpha}\).

7.1 -- Le degre est forcement \(n\).

Rappel de cours
Pour une application lineaire, un vecteur propre pour la valeur propre \(\lambda\) verifie \(T(P) = \lambda P\). En particulier, les deux membres ont le meme degre.

Supposons \(\deg(P) = d \leq n\). Le coefficient dominant de \(XP\) est celui de \(P\) (disons \(a\)), donc \(XP\) est de degre \(d+1\). Le coefficient dominant de \(\frac{1}{n}(X^2-1)P'\) est \(\frac{d}{n} \cdot a\), de degre \(d+1\) aussi.

Donc le coefficient dominant de \(T(P)\) est \(a - \frac{d}{n} \cdot a = a\left(1 - \frac{d}{n}\right)\).

Si \(d < n\), alors \(\deg(T(P)) = d + 1 > d = \deg(\lambda P)\), contradiction. Donc \(d = n\).

Resultat
$$\boxed{\deg(P) = n}$$

7.2 -- Les racines complexes verifient \(z_0^2 = 1\).

Soit \(z_0\) une racine complexe de \(P\) de multiplicite \(r \geq 1\). On ecrit \(P = (X - z_0)^r Q\) avec \(Q(z_0) \neq 0\).

Derivons : \(P' = r(X-z_0)^{r-1}Q + (X-z_0)^r Q' = (X-z_0)^{r-1}\underbrace{[rQ + (X-z_0)Q']}_{= R}\) avec \(R(z_0) = rQ(z_0) \neq 0\).

L'equation \(T(P) = \lambda P\) donne : \(\lambda P - XP = -\frac{1}{n}(X^2-1)P'\), soit :

$$(\lambda - X)(X-z_0)^r Q = -\frac{1}{n}(X^2-1)(X-z_0)^{r-1}R$$

On simplifie par \((X-z_0)^{r-1}\) (non nul en tant que polynome) :

$$(\lambda - X)(X-z_0)Q = -\frac{1}{n}(X^2-1)R$$

Evaluons en \(z_0\) : le membre de gauche vaut 0, donc \(z_0^2 - 1 = 0\) (car \(R(z_0) \neq 0\)).

Resultat
$$\boxed{z_0^2 - 1 = 0 \implies z_0 = 1 \text{ ou } z_0 = -1}$$

7.3 -- Expression de \(P\).

Puisque toutes les racines complexes de \(P\) sont dans \(\{-1, 1\}\) et que \(P\) est unitaire de degre \(n\) :

Resultat
Il existe \(\alpha \in [\![0, n]\!]\) tel que : $$\boxed{P = (X - 1)^\alpha (X + 1)^{n - \alpha}}$$
Methode -- Trouver les vecteurs propres par contrainte sur les racines
Quand un endomorphisme fait intervenir des expressions comme \((X^2-1)\), evaluez l'equation aux valeurs propres en des racines potentielles du vecteur propre. Cela impose des contraintes fortes sur la position des racines.
Question 8

Determiner les elements propres de \(T\). L'endomorphisme \(T\) est-il diagonalisable ?

Pourquoi on fait ca ?
On a identifie la forme des vecteurs propres (Q7). Il reste a calculer la valeur propre associee a chaque \(P_\alpha = (X-1)^\alpha(X+1)^{n-\alpha}\), puis a verifier qu'on obtient suffisamment de valeurs propres pour la diagonalisabilite.

Posons \(P_\alpha = (X-1)^\alpha(X+1)^{n-\alpha}\). Calculons \(T(P_\alpha)\).

On a \(P_\alpha' = \alpha(X-1)^{\alpha-1}(X+1)^{n-\alpha} + (n-\alpha)(X-1)^\alpha(X+1)^{n-\alpha-1}\).

Donc \((X^2-1)P_\alpha' = (X-1)(X+1) \cdot P_\alpha' =\)

$$= \alpha(X+1)(X-1)^\alpha(X+1)^{n-\alpha} + (n-\alpha)(X-1)(X-1)^\alpha(X+1)^{n-\alpha}$$ $$= [\alpha(X+1) + (n-\alpha)(X-1)] \cdot P_\alpha$$ $$= [nX + \alpha - (n-\alpha)] \cdot P_\alpha = [nX + 2\alpha - n] \cdot P_\alpha$$

D'ou :

$$T(P_\alpha) = XP_\alpha - \frac{1}{n}(nX + 2\alpha - n) \cdot P_\alpha = XP_\alpha - XP_\alpha - \frac{2\alpha - n}{n} P_\alpha$$ $$= \frac{n - 2\alpha}{n}\,P_\alpha$$
Resultat
Les valeurs propres de \(T\) sont : $$\boxed{\lambda_\alpha = 1 - \frac{2\alpha}{n}, \quad \alpha \in [\![0, n]\!]}$$ avec le vecteur propre associe \(P_\alpha = (X-1)^\alpha(X+1)^{n-\alpha}\).

Pour \(\alpha = 0, 1, 2, \ldots, n\), on obtient \(\lambda = 1, \frac{n-2}{n}, \frac{n-4}{n}, \ldots, -1\) : ce sont \(n + 1\) valeurs distinctes.

\(T\) est diagonalisable car il possede \(n + 1 = \dim(E_n)\) valeurs propres distinctes.

Verification pour \(n = 2\)

Pour \(n = 2\) : \(\lambda_0 = 1\), \(\lambda_1 = 0\), \(\lambda_2 = -1\).

Vecteurs propres : \(P_0 = (X+1)^2\), \(P_1 = (X-1)(X+1) = X^2-1\), \(P_2 = (X-1)^2\).

Verifions \(T(P_1) = T(X^2 - 1)\) : on a \(T(X^2-1) = T(X^2) - T(1) = \frac{1}{2}X^3 + X - X = \frac{1}{2}X^3\)... En fait, il faut revenir a \(n = 2\) pour verifier proprement. L'important est que les 3 valeurs propres sont bien distinctes.

Methode -- Diagonalisabilite par comptage
Si un endomorphisme d'un espace de dimension \(d\) possede \(d\) valeurs propres distinctes, il est automatiquement diagonalisable. C'est le critere le plus simple et le plus frequent aux concours.

2

Fonction \(F(x) = \int_0^1 t^{t^x}\,\mathrm{d}t\) et DSE

Integrales a parametre, serie de fonctions, fonction Gamma -- 6 questions + sous-parties
Intermediaire
Enonce resume Questions de cours sur \(a^b\), la comparaison de \(t^x\) et \(t^y\), le DSE de l'exponentielle et \(\Gamma(n+1) = n!\). Puis etude de \(F(x) = \int_0^1 t^{t^x}\,\mathrm{d}t\) : ensemble de definition, variations, limites, et developpement en serie \(F(x) = \sum_{n=0}^{+\infty} \frac{(-1)^n}{(1+nx)^{n+1}}\).
Question 1 -- Question de cours

Soient \(a > 0\) et \(b \in \mathbb{R}\). Choisir la bonne expression de \(a^b\) parmi : (A) \(e^{b\ln a}\), (B) \(e^{a\ln b}\), (C) \(e^{\ln a \cdot \ln b}\).

Rappel de cours
Par definition, pour \(a > 0\) et \(b \in \mathbb{R}\) : \(a^b = e^{b \ln a}\). C'est la definition de l'exponentiation generalisee. Attention : \(b\) peut etre negatif, irrationnel, etc.
Pourquoi on fait ca ?
C'est la brique de base de tout l'exercice. L'expression \(t^{t^x}\) signifie \(e^{t^x \ln t}\), et pour ecrire \(t^x\) il faut \(e^{x \ln t}\). Si vous confondez les formules, tout l'exercice s'effondre. Pensez a un exemple : \(2^3 = e^{3\ln 2} \approx e^{2.08} \approx 8\).
Resultat
$$\boxed{\text{Reponse (A) : } a^b = e^{b\ln a}}$$
Question 2 -- Question de cours

Soient \(x < y\) et \(t \in\, ]0, 1[\). Comparer \(t^x\) et \(t^y\).

Pourquoi on fait ca ?
Quand \(0 < t < 1\), on a \(\ln t < 0\). C'est l'inverse de ce qu'on a l'habitude de voir avec les exponentielles croissantes. Ici, plus l'exposant est grand, plus \(t^x\) est petit. C'est comme un ressort : plus on tire (exposant grand), plus la valeur se rapproche de 0.

On a \(t^x = e^{x\ln t}\) et \(t^y = e^{y\ln t}\). Comme \(\ln t < 0\) (car \(0 < t < 1\)) et \(x < y\) :

$$x\ln t > y\ln t$$

(on multiplie une inegalite par un nombre negatif, le sens s'inverse).

Donc, par croissance de l'exponentielle :

Resultat
$$\boxed{t^x > t^y}$$
Piege classique
Le rapport du jury signale que cette question est souvent traitee sans explication. Il faut bien preciser que \(\ln t < 0\) et que cela inverse le sens de l'inegalite.
Question 3 -- Question de cours

DSE de l'exponentielle et son domaine de validite.

Resultat
$$\boxed{\forall x \in \mathbb{R}, \quad e^x = \sum_{n=0}^{+\infty} \frac{x^n}{n!}}$$ Domaine de validite : \(\mathbb{R}\) tout entier (rayon de convergence infini).
Piege classique
Le rapport du jury deplore que le domaine de validite soit souvent faux. C'est \(\mathbb{R}\) entier, pas \(]-1, 1[\) !
Question 4 -- Question de cours

Calculer \(\Gamma(1)\) et en deduire par recurrence \(\Gamma(n+1)\) pour \(n \in \mathbb{N}\).

Rappel de cours
La fonction Gamma : \(\Gamma(x) = \int_0^{+\infty} t^{x-1} e^{-t}\,\mathrm{d}t\), definie sur \(]0, +\infty[\), verifie la relation fonctionnelle \(\Gamma(x+1) = x\,\Gamma(x)\). C'est la "generalisation de la factorielle" aux reels positifs.
Pourquoi on fait ca ?
Imaginez que vous vouliez calculer \(3! = 6\). La fonction Gamma fait la meme chose : \(\Gamma(4) = 3 \cdot \Gamma(3) = 3 \cdot 2 \cdot \Gamma(2) = 3 \cdot 2 \cdot 1 \cdot \Gamma(1)\). Il suffit de connaitre \(\Gamma(1)\) pour tout demarrer.

Calcul de \(\Gamma(1)\) :

$$\Gamma(1) = \int_0^{+\infty} t^0 e^{-t}\,\mathrm{d}t = \int_0^{+\infty} e^{-t}\,\mathrm{d}t = [-e^{-t}]_0^{+\infty} = 0 - (-1) = 1$$

Recurrence : Montrons que \(\Gamma(n+1) = n!\) pour tout \(n \in \mathbb{N}\).

  • Initialisation : \(\Gamma(1) = 0! = 1\). Verifie.
  • Heredite : Si \(\Gamma(n+1) = n!\), alors \(\Gamma(n+2) = (n+1)\Gamma(n+1) = (n+1) \cdot n! = (n+1)!\).
  • Conclusion : Par recurrence, \(\Gamma(n+1) = n!\) pour tout \(n \in \mathbb{N}\).
Resultat
$$\boxed{\Gamma(1) = 1 \quad \text{et} \quad \forall n \in \mathbb{N},\; \Gamma(n+1) = n!}$$
Piege classique
Le rapport du jury signale que certains candidats trouvent \(\Gamma(1) = -1\) ou \(\Gamma(1) = 0\), et que d'autres ecrivent simplement "par recurrence immediate" sans rediger. Detaillez toujours l'initialisation et l'heredite !
Question 5 -- Etude de \(F\)

Pour \(x \in \mathbb{R}\), on pose \(F(x) = \int_0^1 t^{t^x}\,\mathrm{d}t\). Determiner l'ensemble de definition, les variations, montrer \(F(x) \geq 1/2\) pour \(x \geq 0\), la continuite, les limites, et tracer la courbe.

Pourquoi on fait ca ?
L'expression \(t^{t^x}\) semble exotique, mais c'est juste une integrale a parametre. On va utiliser les grands theoremes du cours (continuite sous l'integrale, convergence dominee) pour etudier \(F\) proprement.

5.1 -- Ensemble de definition

L'integrande est \(f(t,x) = t^{t^x} = e^{t^x \ln t}\). Pour \(t \in\, ]0, 1]\), \(\ln t \leq 0\) et \(t^x\) est bien defini pour tout \(x \in \mathbb{R}\).

En \(t = 0^+\) : si \(x \geq 0\), alors \(t^x \to 0\), donc \(t^x \ln t = e^{x\ln t}\ln t \to 0\) par croissances comparees. L'integrande se prolonge par 1 en 0.

Si \(x < 0\), alors \(t^x \to +\infty\), et \(t^x \ln t \to -\infty\) (car \(\ln t \to -\infty\)), donc l'integrande tend vers 0. Elle se prolonge aussi par 0.

Resultat
$$\boxed{F \text{ est definie sur } \mathbb{R}}$$

5.2 -- Sens de variation

Soient \(x < y\). Pour \(t \in\, ]0, 1[\), on a \(t^x > t^y\) (Q2), donc \(t^{t^x} < t^{t^y}\) (car \(t \in\, ]0,1[\) : plus l'exposant est grand, plus \(t\) elevee a cet exposant est petit, car \(t < 1\)). En integrant :

Resultat
$$\boxed{F \text{ est croissante sur } \mathbb{R}}$$

5.3 -- Minoration : \(F(x) \geq 1/2\) pour \(x \geq 0\)

Pour \(x \geq 0\), on a \(F(0) = \int_0^1 t\,\mathrm{d}t = 1/2\). Comme \(F\) est croissante, pour \(x \geq 0\) : \(F(x) \geq F(0) = 1/2\).

Resultat
$$\boxed{\forall x \geq 0, \quad F(x) \geq \frac{1}{2}}$$

5.4 -- Continuite

Pour tout \(t \in\, ]0, 1]\), \(x \mapsto t^{t^x}\) est continue. De plus, \(|t^{t^x}| \leq 1\) pour tout \(t \in [0,1]\). La fonction constante \(1\) est integrable sur \([0,1]\). Par le theoreme de continuite sous le signe integrale, \(F\) est continue sur \(\mathbb{R}\).

Resultat
$$\boxed{F \text{ est continue sur } \mathbb{R}}$$

5.5 -- Limites

Quand \(x \to +\infty\) : Pour tout \(t \in\, ]0, 1[\), \(t^x \to 0\), donc \(t^{t^x} \to t^0 = 1\). On domine par 1, integrable sur \([0,1]\). Par le theoreme de convergence dominee :

$$\lim_{x \to +\infty} F(x) = \int_0^1 1\,\mathrm{d}t = 1$$

Quand \(x \to -\infty\) : Pour tout \(t \in\, ]0, 1[\), \(t^x \to +\infty\), donc \(t^{t^x} = e^{t^x \ln t} \to 0\) (car \(t^x \ln t \to -\infty\)). La encore, on domine par 1 :

$$\lim_{x \to -\infty} F(x) = \int_0^1 0\,\mathrm{d}t = 0$$
Resultat
$$\boxed{\lim_{x \to +\infty} F(x) = 1 \quad \text{et} \quad \lim_{x \to -\infty} F(x) = 0}$$
Piege classique
Le rapport du jury indique que beaucoup de candidats confondent variable d'integration et parametre. On integre sur \(t\), pas sur \(x\) ! De plus, pour la convergence dominee, il faut citer le theoreme et verifier que la domination par 1 est bien valide.

5.6 -- Tableau de variations et allure

On admet \(F'(0) = 1/4\). Donc la tangente en 0 est \(y = F(0) + F'(0) \cdot x = \frac{1}{2} + \frac{x}{4}\).

\(x\)\(-\infty\)\(0\)\(+\infty\)
\(F'(x)\)\(+\)\(1/4\)\(+\)
\(F(x)\)\(0\)\(\nearrow\)\(1/2\)\(\nearrow\)\(1\)
Resultat
\(F\) est une fonction croissante, continue, allant de 0 a 1, avec une forme de sigmoide (courbe en S). La tangente en 0 a pour pente \(1/4\).
Question 6 -- DSE de \(F\)

Pour \(x > 0\), on pose \(g_n(t) = \frac{(t^x \ln t)^n}{n!}\) sur \(]0, 1]\). (6.1) Montrer la convergence simple de \(\sum g_n\). (6.2) Calculer \(\int_0^1 |g_n(t)|\,\mathrm{d}t\). (6.3) En deduire \(F(x) = \sum_{n=0}^{+\infty} \frac{(-1)^n}{(1+nx)^{n+1}}\).

Pourquoi on fait ca ?
On veut intervertir somme et integrale pour obtenir une expression en serie de \(F\). L'idee est de decomposer \(t^{t^x} = e^{t^x \ln t}\) en sa serie exponentielle, puis d'integrer terme a terme.

6.1 -- Convergence simple

Pour \(t \in\, ]0, 1]\), la serie \(\sum g_n(t) = \sum \frac{(t^x \ln t)^n}{n!}\) est une serie exponentielle de la forme \(\sum \frac{u^n}{n!}\) avec \(u = t^x \ln t\). Elle converge (absolument) pour tout \(u \in \mathbb{R}\), donc pour tout \(t \in\, ]0, 1]\).

Sa somme est \(e^{t^x \ln t} = t^{t^x}\).

Resultat
$$\boxed{\sum_{n=0}^{+\infty} g_n(t) = t^{t^x} \quad \text{pour tout } t \in\, ]0, 1]}$$

6.2 -- Calcul de \(\int_0^1 |g_n(t)|\,\mathrm{d}t\)

Pour \(t \in\, ]0, 1]\), \(\ln t \leq 0\), donc \(t^x \ln t \leq 0\). Ainsi \((t^x \ln t)^n = (-1)^n |t^x \ln t|^n\) et \(|g_n(t)| = \frac{|t^x \ln t|^n}{n!} = \frac{t^{nx}(-\ln t)^n}{n!}\).

On effectue le changement de variable \(u = -\ln t\) (donc \(t = e^{-u}\), \(\mathrm{d}t = -e^{-u}\,\mathrm{d}u\)) :

$$\int_0^1 |g_n(t)|\,\mathrm{d}t = \frac{1}{n!}\int_0^{+\infty} e^{-nxu} \cdot u^n \cdot e^{-u}\,\mathrm{d}u = \frac{1}{n!}\int_0^{+\infty} u^n e^{-(nx+1)u}\,\mathrm{d}u$$

Avec le changement \(v = (nx+1)u\) :

$$= \frac{1}{n!} \cdot \frac{1}{(nx+1)^{n+1}} \int_0^{+\infty} v^n e^{-v}\,\mathrm{d}v = \frac{\Gamma(n+1)}{n! \cdot (nx+1)^{n+1}}$$
Resultat
$$\boxed{\int_0^1 |g_n(t)|\,\mathrm{d}t = \frac{1}{(1 + nx)^{n+1}}}$$ (car \(\Gamma(n+1) = n!\)).

6.3 -- Interversion somme-integrale

La serie \(\sum \int_0^1 |g_n(t)|\,\mathrm{d}t = \sum \frac{1}{(1+nx)^{n+1}}\) converge (termes generaux en \(O(1/n^2)\) car \((1+nx)^{n+1}\) croit exponentiellement en \(n\)).

Par le theoreme d'integration terme a terme (convergence de la serie des integrales des valeurs absolues), on peut intervertir somme et integrale :

$$F(x) = \int_0^1 t^{t^x}\,\mathrm{d}t = \sum_{n=0}^{+\infty} \int_0^1 g_n(t)\,\mathrm{d}t$$

Or \(\int_0^1 g_n(t)\,\mathrm{d}t = (-1)^n \int_0^1 |g_n(t)|\,\mathrm{d}t\) (car \(g_n(t) = (-1)^n |g_n(t)|\) vu le signe de \(\ln t\)).

Resultat
$$\boxed{F(x) = \sum_{n=0}^{+\infty} \frac{(-1)^n}{(1 + nx)^{n+1}}}$$
Methode -- Integration terme a terme d'une serie de fonctions
Pour intervertir \(\int\) et \(\sum\), il suffit que la serie \(\sum \int |g_n|\) converge (theoreme de Fubini-Tonelli / integration terme a terme). Verifiez toujours cette condition avant d'intervertir !

3

Endomorphisme \(\Psi(f)(x) = \int_0^1 f(xt)\,\mathrm{d}t\)

Integrales, endomorphismes, equations differentielles -- 8 questions + sous-parties
Avance
Enonce resume \(E\) = espace des fonctions continues sur \(\mathbb{R}_+\). On pose \(F(x) = \int_0^x f(u)\,\mathrm{d}u\) et \(\Psi(f)(x) = \int_0^1 f(xt)\,\mathrm{d}t\). On etudie la surjectivite, l'injectivite, les valeurs propres de \(\Psi\) via une EDO, et la restriction a un sous-espace \(F_n\).
Question 1

Justifier que \(F\) est de classe \(\mathcal{C}^1\) sur \(\mathbb{R}_+\) et donner \(F'(x)\).

Rappel de cours
Theoreme fondamental de l'analyse : si \(f\) est continue sur un intervalle \(I\), alors la fonction \(F : x \mapsto \int_a^x f(u)\,\mathrm{d}u\) est la primitive de \(f\) qui s'annule en \(a\). Elle est de classe \(\mathcal{C}^1\) et \(F'(x) = f(x)\).
Pourquoi on fait ca ?
C'est la question de demarrage. \(F\) est la primitive de \(f\) qui s'annule en 0. C'est comme demander "si je connais la vitesse \(f(t)\), quel est le deplacement \(F(t)\) ?".

\(f\) est continue sur \(\mathbb{R}_+\). Par le theoreme fondamental de l'analyse, \(F : x \mapsto \int_0^x f(u)\,\mathrm{d}u\) est de classe \(\mathcal{C}^1\) sur \(\mathbb{R}_+\).

Resultat
$$\boxed{F \in \mathcal{C}^1(\mathbb{R}_+) \quad \text{et} \quad F'(x) = f(x)}$$
Piege classique
Le rapport du jury deplore des reponses catastrophiques a cette question pourtant basique. Le theoreme fondamental de l'analyse doit etre cite avec precision, et \(F'(x) = f(x)\), pas \(F'(x) = f\) ou \(F'(t) = f(t)\).
Question 2

Exprimer \(\Psi(f)(x)\) a l'aide de \(F(x)\) pour \(x > 0\).

Pourquoi on fait ca ?
On veut ramener \(\Psi(f)(x) = \int_0^1 f(xt)\,\mathrm{d}t\) a quelque chose qu'on connait : la primitive \(F\). Le changement de variable \(u = xt\) fait le pont.

Pour \(x > 0\), on effectue le changement de variable \(u = xt\), donc \(t = u/x\) et \(\mathrm{d}t = \mathrm{d}u/x\). Quand \(t\) va de 0 a 1, \(u\) va de 0 a \(x\) :

$$\Psi(f)(x) = \int_0^1 f(xt)\,\mathrm{d}t = \int_0^x f(u)\,\frac{\mathrm{d}u}{x} = \frac{1}{x}\int_0^x f(u)\,\mathrm{d}u$$
Resultat
$$\boxed{\Psi(f)(x) = \frac{F(x)}{x} \quad \text{pour } x > 0}$$
Piege classique
Le rapport du jury note que beaucoup de candidats se trompent dans le changement de variable et obtiennent \(\Psi(f)(x) = xF(x)\) ou \(\Psi(f)(x) = F(x)\). Verifiez en prenant \(f = 1\) : \(\Psi(1)(x) = 1\) et \(F(x)/x = x/x = 1\). Ca colle !
Question 3

Justifier que \(\Psi(f)\) est continue sur \(\mathbb{R}_+\) et donner \(\Psi(f)(0)\).

Pourquoi on fait ca ?
On sait que \(\Psi(f)(x) = F(x)/x\) pour \(x > 0\), mais que se passe-t-il en \(x = 0\) ? Il faut calculer la limite et reconnaitre un taux d'accroissement.

Sur \(\mathbb{R}_+^*\), \(\Psi(f)(x) = F(x)/x\) est continue (quotient de fonctions continues avec denominateur non nul).

En \(x = 0\) : \(\frac{F(x)}{x} = \frac{F(x) - F(0)}{x - 0}\) (car \(F(0) = 0\)). Quand \(x \to 0^+\), c'est le taux d'accroissement de \(F\) en 0, qui converge vers \(F'(0) = f(0)\).

Resultat
$$\boxed{\Psi(f) \text{ est continue sur } \mathbb{R}_+ \quad \text{et} \quad \Psi(f)(0) = f(0)}$$
Question 4

Montrer que \(\Psi\) est un endomorphisme de \(E\).

Linearite : L'integrale est lineaire, donc \(\Psi(\alpha f + \beta g) = \alpha \Psi(f) + \beta \Psi(g)\).

Stabilite : D'apres Q3, pour toute \(f \in E\) (continue sur \(\mathbb{R}_+\)), \(\Psi(f)\) est aussi continue sur \(\mathbb{R}_+\), donc \(\Psi(f) \in E\).

Resultat
$$\boxed{\Psi \text{ est un endomorphisme de } E}$$
Question 5 -- Surjectivite de \(\Psi\)

Soit \(h(x) = x\sin(1/x)\) pour \(x > 0\) et \(h(0) = 0\). (5.1) Montrer que \(h\) est continue. (5.2) \(h\) est-elle \(\mathcal{C}^1\) ? (5.3) Montrer que si \(g \in \text{Im}(\Psi)\), alors \(x \mapsto xg(x)\) est \(\mathcal{C}^1\). (5.4-5.5) En deduire \(h \notin \text{Im}(\Psi)\) et conclure.

Pourquoi on fait ca ?
On veut montrer que \(\Psi\) n'est PAS surjectif. La strategie : trouver une fonction \(h\) continue qui ne peut pas etre de la forme \(\Psi(f)\). Pour cela, on montre que les elements de \(\text{Im}(\Psi)\) ont une propriete de regularite que \(h\) n'a pas.

5.1 -- Continuite de \(h\)

Sur \(\mathbb{R}_+^*\), \(h\) est continue comme composee. En 0 : \(|h(x)| = |x\sin(1/x)| \leq |x| \to 0 = h(0)\). Donc \(h\) est continue sur \(\mathbb{R}_+\).

5.2 -- \(h\) n'est pas \(\mathcal{C}^1\)

Le taux d'accroissement en 0 est \(\frac{h(x)}{x} = \sin(1/x)\), qui n'a pas de limite quand \(x \to 0^+\). Donc \(h\) n'est meme pas derivable en 0, encore moins \(\mathcal{C}^1\).

5.3 -- Si \(g = \Psi(f)\), alors \(x \mapsto xg(x)\) est \(\mathcal{C}^1\)

Si \(g = \Psi(f)\), alors pour \(x > 0\) : \(xg(x) = x \cdot F(x)/x = F(x)\). Et \(0 \cdot g(0) = 0 = F(0)\). Donc \(x \mapsto xg(x) = F(x)\), qui est \(\mathcal{C}^1\) par Q1.

Resultat
Si \(g \in \text{Im}(\Psi)\), alors \(x \mapsto xg(x)\) est de classe \(\mathcal{C}^1\) sur \(\mathbb{R}_+\).

5.4 -- \(h \notin \text{Im}(\Psi)\)

La fonction \(x \mapsto xh(x) = x^2 \sin(1/x)\) est derivable sur \(\mathbb{R}_+^*\) avec \(\theta'(x) = 2x\sin(1/x) - \cos(1/x)\) pour \(x > 0\). En 0, \(\theta'(0) = 0\) (taux d'accroissement : \(x\sin(1/x) \to 0\)). Mais \(\theta'(x)\) n'a pas de limite en \(0^+\) car \(\cos(1/x)\) oscille. Donc \(\theta\) n'est pas \(\mathcal{C}^1\) sur \(\mathbb{R}_+\).

D'apres 5.3, si \(h\) etait dans \(\text{Im}(\Psi)\), alors \(xh(x)\) serait \(\mathcal{C}^1\). Ce n'est pas le cas. Donc \(h \notin \text{Im}(\Psi)\).

Resultat
$$\boxed{\Psi \text{ n'est pas surjectif}}$$
Methode -- Montrer qu'un operateur n'est pas surjectif
On montre que les elements de l'image ont une propriete supplementaire (ici la \(\mathcal{C}^1\)-regularite de \(xg(x)\)), puis on exhibe un element qui n'a pas cette propriete. C'est la methode du contre-exemple structure.
Question 6

Montrer que \(\Psi\) est injectif.

Pourquoi on fait ca ?
On montre que \(\ker(\Psi) = \{0\}\). Si \(\Psi(f) = 0\), alors \(F(x)/x = 0\) pour tout \(x > 0\), donc \(F \equiv 0\), donc \(f = F' \equiv 0\).

Soit \(f \in E\) telle que \(\Psi(f) = 0\).

Pour tout \(x > 0\) : \(F(x)/x = 0\), donc \(F(x) = 0\). Comme \(F\) est continue sur \(\mathbb{R}_+\), \(F\) est identiquement nulle sur \(\mathbb{R}_+\). Par derivation, \(f = F' = 0\) sur \(\mathbb{R}_+\).

Resultat
$$\boxed{\Psi \text{ est injectif}}$$
Piege classique
Le rapport du jury cite des erreurs comme : "\(\int_0^1 f(xt)\,\mathrm{d}t = 0\) donc \(f = 0\) car \(f \geq 0\)" (rien ne dit que \(f \geq 0\) !), ou "continue d'integrale nulle donc nulle" (c'est faux sans hypothese de signe). La bonne preuve passe par \(F \equiv 0 \implies f \equiv 0\) via la derivation.
Question 7 -- Elements propres de \(\Psi\)

(7.1) Justifier que 0 n'est pas VP. On pose l'EDO \(y' + \frac{\mu}{x}y = 0\) sur \(\mathbb{R}_+^*\). (7.2) Resoudre. (7.3) Solutions prolongeables en 0. (7.4) En deduire les VP et sous-espaces propres de \(\Psi\).

Pourquoi on fait ca ?
On cherche les \(\lambda\) et les \(f \neq 0\) tels que \(\Psi(f) = \lambda f\). En utilisant \(\Psi(f)(x) = F(x)/x\), l'equation \(F(x)/x = \lambda f(x) = \lambda F'(x)\) se transforme en une EDO sur \(F\).

7.1 -- 0 n'est pas valeur propre

Comme \(\Psi\) est injectif (Q6), \(\ker(\Psi) = \{0\}\), donc 0 n'est pas valeur propre.

7.2 -- Resolution de \(y' + \frac{\mu}{x}y = 0\) sur \(\mathbb{R}_+^*\)

Rappel de cours
L'equation \(y' + a(x)y = 0\) a pour solutions \(y = C\,e^{-A(x)}\) ou \(A\) est une primitive de \(a\).

Ici \(a(x) = \mu/x\), une primitive est \(\mu \ln x = \ln(x^\mu)\). Donc :

Resultat
$$\boxed{y(x) = \frac{C}{x^\mu}, \quad C \in \mathbb{R}}$$

7.3 -- Prolongeabilite en 0

La fonction \(x \mapsto C/x^\mu\) a une limite finie en \(0^+\) si et seulement si \(\mu \leq 0\) (ou \(C = 0\)).

Pour \(\mu > 0\) et \(C \neq 0\), \(C/x^\mu \to \pm\infty\) : pas prolongeable.

Resultat
Les solutions prolongeables en 0 sont \(y = C/x^\mu\) avec \(\mu \leq 0\) et \(C \in \mathbb{R}\).

7.4 -- Valeurs propres de \(\Psi\)

Soit \(\lambda \neq 0\) et \(f\) continue non nulle tels que \(\Psi(f) = \lambda f\). Alors \(F(x)/x = \lambda F'(x)\), soit \(F(x) = \lambda x F'(x)\). C'est-a-dire \(F' - \frac{1}{\lambda x}F = 0\), ou encore \(F' + \frac{\mu}{x}F = 0\) avec \(\mu = -1/\lambda\).

D'apres 7.2 : \(F(x) = Cx^{1/\lambda}\) (car \(F = C/x^\mu = Cx^{1/\lambda}\)). Pour que \(F\) soit continue sur \(\mathbb{R}_+\), il faut \(1/\lambda > 0\), soit \(\lambda > 0\).

Alors \(f = F' = \frac{C}{\lambda} x^{1/\lambda - 1}\). Pour que \(f\) soit continue sur \(\mathbb{R}_+\), il faut \(1/\lambda - 1 \geq 0\), soit \(\lambda \leq 1\).

Reciproquement, pour \(\lambda \in\, ]0, 1]\) et \(f_\lambda(x) = x^{1/\lambda - 1}\) :

$$\Psi(f_\lambda)(x) = \int_0^1 (xt)^{1/\lambda - 1}\,\mathrm{d}t = x^{1/\lambda - 1}\left[\frac{t^{1/\lambda}}{1/\lambda}\right]_0^1 = \lambda\, x^{1/\lambda - 1} = \lambda\, f_\lambda(x)$$
Resultat
Le spectre de \(\Psi\) est : $$\boxed{\text{Sp}(\Psi) = \,]0, 1]}$$ Pour chaque \(\lambda \in\, ]0, 1]\), le sous-espace propre est : $$\boxed{E_\lambda(\Psi) = \text{Vect}(x \mapsto x^{1/\lambda - 1})}$$ En particulier : \(\lambda = 1 \leftrightarrow f(x) = 1\), \(\lambda = 1/2 \leftrightarrow f(x) = x\), \(\lambda = 1/3 \leftrightarrow f(x) = x^2\), etc.
Methode -- Transformer une equation aux VP en EDO
Quand un operateur est defini par une integrale, l'equation aux valeurs propres \(\Psi(f) = \lambda f\) se transforme souvent en equation differentielle. Resolvez l'EDO, puis selectionnez les solutions qui vivent dans le bon espace (ici, continues sur \(\mathbb{R}_+\)).
Question 8 -- Sous-espace \(F_n\)

Pour \(n > 1\), \(f_i(x) = x^i\), \(g_i(x) = x^i\ln x\) (\(x > 0\), \(g_i(0) = 0\)). On pose \(\mathcal{B} = (f_1, \ldots, f_n, g_1, \ldots, g_n)\) et \(F_n = \text{Vect}(\mathcal{B})\). (8.1) Montrer que \(\mathcal{B}\) est une base. (8.2) Montrer que \(\Psi\) stabilise \(F_n\). (8.3) Matrice dans \(\mathcal{B}\). (8.4) Automorphisme. (8.5) Calculer un antecedent.

Pourquoi on fait ca ?
On se restreint a un sous-espace de dimension finie pour pouvoir ecrire \(\Psi\) sous forme matricielle et faire de l'algebre lineaire concrete.

8.1 -- \(\mathcal{B}\) est une base de \(F_n\) (\(\dim F_n = 2n\))

Soit \(\sum \alpha_i f_i + \sum \beta_j g_j = 0\) (fonction nulle sur \(\mathbb{R}_+\)).

Pour \(x > 0\), on divise par \(x\) : \(\sum \alpha_i x^{i-1} + \sum \beta_j x^{j-1}\ln x = 0\).

Si \(\beta_1 \neq 0\), quand \(x \to 0^+\) le terme \(\beta_1 \ln x\) tend vers \(-\infty\) ou \(+\infty\), contradiction. Donc \(\beta_1 = 0\). Alors en faisant \(x \to 0^+\) on obtient \(\alpha_1 = 0\).

Par recurrence, en divisant par \(x^p\) et en repetant l'argument, on obtient \(\alpha_{p+1} = \beta_{p+1} = 0\) pour tout \(p\).

Resultat
$$\boxed{\mathcal{B} \text{ est libre, donc c'est une base de } F_n \text{, et } \dim(F_n) = 2n}$$

8.2 -- \(\Psi\) stabilise \(F_n\)

Par integration par parties, on montre que \(\int_0^x t^p \ln t\,\mathrm{d}t = \frac{x^{p+1}}{p+1}\ln x - \frac{x^{p+1}}{(p+1)^2}\).

Ensuite :

  • \(\Psi(f_i)(x) = \int_0^1 (xt)^i\,\mathrm{d}t = \frac{x^i}{i+1} = \frac{1}{i+1}f_i(x)\). Donc \(\Psi(f_i) = \frac{1}{i+1}f_i \in F_n\).
  • \(\Psi(g_i)(x) = \int_0^1 (xt)^i\ln(xt)\,\mathrm{d}t = \int_0^1 (xt)^i[\ln x + \ln t]\,\mathrm{d}t = \frac{1}{i+1}x^i\ln x - \frac{1}{(i+1)^2}x^i\).

Donc \(\Psi(g_i) = \frac{1}{i+1}g_i - \frac{1}{(i+1)^2}f_i \in F_n\).

Resultat
$$\boxed{\Psi \text{ stabilise } F_n}$$

8.3 -- Matrice de \(\Psi_n\) dans \(\mathcal{B}\)

Dans la base ordonnee \((f_1, \ldots, f_n, g_1, \ldots, g_n)\), la matrice est par blocs :

Resultat
$$\boxed{M_{\mathcal{B}}(\Psi_n) = \begin{pmatrix} D & 0 \\ -D^2 & D \end{pmatrix}}$$ ou \(D = \text{diag}\!\left(\frac{1}{2}, \frac{1}{3}, \ldots, \frac{1}{n+1}\right)\).

8.4 -- \(\Psi_n\) est un automorphisme

Le determinant de la matrice est \(\det(D)^2 = \left(\frac{1}{2 \cdot 3 \cdots (n+1)}\right)^2 \neq 0\). Donc la matrice est inversible et \(\Psi_n\) est un automorphisme de \(F_n\).

Resultat
$$\boxed{\Psi_n \text{ est un automorphisme de } F_n}$$

8.5 -- Antecedent de \(z(x) = x + x^2\ln x\)

On a \(z = f_1 + g_2\) (dans la base \(\mathcal{B}\) avec \(n \geq 2\)). On cherche \(f = \sum \alpha_i f_i + \sum \beta_j g_j\) tel que \(\Psi_n(f) = z\).

Le systeme donne : \(\frac{\alpha_1}{2} - \frac{\beta_1}{4} = 1\), \(\frac{\alpha_2}{3} - \frac{\beta_2}{9} = 0\), \(\frac{\beta_1}{2} = 0\), \(\frac{\beta_2}{3} = 1\).

D'ou \(\beta_1 = 0\), \(\alpha_1 = 2\), \(\beta_2 = 3\), \(\alpha_2 = 1\).

Resultat
$$\boxed{\Psi_n^{-1}(z) = 2f_1 + f_2 + 3g_2 = 2x + x^2 + 3x^2\ln x}$$
Piege classique
Le rapport du jury signale que la question 8 est incomprise par 99% des candidats. La cle est de comprendre que 8.1.1 n'est pas le cas \(n = 1\) mais le debut de la recurrence (elimination du premier coefficient).

4

Variables de Bernoulli, trace, rang et determinant

Probabilites, algebre lineaire -- 6 questions + sous-parties
Intermediaire
Enonce resume \(X_1, \ldots, X_n\) independantes, loi de Bernoulli de parametre \(p\). \(M\) est diagonalisable, semblable a \(\Delta = \text{diag}(X_1, \ldots, X_n)\). On etudie \(T = \text{tr}(M)\), \(R = \text{rg}(M)\), \(D = \det(M)\), et l'evenement \(Z\) : "les sous-espaces propres ont tous la meme dimension". Puis \(A = UU^T\).
Question 1 -- Questions de cours

Soit \(p\) une projection de rang \(r\). (1.1) Matrice dans une base adaptee. (1.2) Spectre. (1.3) Comparer \(\text{rg}(W)\) et \(\text{tr}(W)\). (1.4) \(\det(W)\).

Rappel de cours
Un projecteur verifie \(p^2 = p\). Dans une base adaptee \((\text{base de Im}(p), \text{base de Ker}(p))\), sa matrice est diagonale avec \(r\) fois 1 et \(n - r\) fois 0.
Pourquoi on fait ca ?
C'est un exercice de rappel sur les projections. L'idee est que les \(X_i\) valent 0 ou 1, donc \(\Delta\) est une matrice de projection. Les proprietes des projections vont piloter tout l'exercice.

1.1 : Dans une base adaptee, la matrice est \(W = \begin{pmatrix} I_r & 0 \\ 0 & 0_{n-r} \end{pmatrix}\).

1.2 : Le spectre est \(\{0, 1\}\) si \(0 < r < n\), \(\{1\}\) si \(r = n\), et \(\{0\}\) si \(r = 0\).

1.3 : \(\text{tr}(W) = r = \text{rg}(W)\).

1.4 : Si \(r = n\), \(\det(W) = 1\). Sinon, \(\det(W) = 0\).

Resultat
$$\boxed{\text{tr}(W) = \text{rg}(W) = r, \quad \det(W) = \begin{cases} 1 & \text{si } r = n \\ 0 & \text{sinon} \end{cases}}$$
Question 2 -- Trace \(T = \text{tr}(M)\)

(2.1) Determiner \(T(\Omega)\). (2.2) Loi de \(T\) et esperance.

Rappel de cours
La trace est un invariant de similitude : si \(M\) est semblable a \(\Delta\), alors \(\text{tr}(M) = \text{tr}(\Delta)\). De meme pour le rang et le determinant.
Pourquoi on fait ca ?
Comme \(M\) est semblable a \(\Delta = \text{diag}(X_1, \ldots, X_n)\), on a \(T = \text{tr}(M) = \text{tr}(\Delta) = X_1 + \cdots + X_n\). C'est la somme de \(n\) Bernoulli independantes : on reconnait une loi binomiale !

\(T = \text{tr}(\Delta) = \sum_{k=1}^n X_k\). Chaque \(X_k \in \{0, 1\}\), donc \(T \in [\![0, n]\!]\).

Comme les \(X_k\) sont independantes de meme loi \(\mathcal{B}(p)\), leur somme suit une loi binomiale :

Resultat
$$\boxed{T(\Omega) = [\![0, n]\!], \quad T \sim \mathcal{B}(n, p), \quad E(T) = np}$$ C'est-a-dire \(P(T = k) = \binom{n}{k} p^k (1-p)^{n-k}\).
Methode -- Reconnaitre une loi binomiale
La somme de \(n\) variables de Bernoulli independantes de meme parametre \(p\) suit toujours une loi binomiale \(\mathcal{B}(n, p)\). C'est un reflexe a avoir : des que vous voyez \(\sum X_i\) avec des Bernoulli independantes, c'est binomial.
Question 3 -- Rang \(R = \text{rg}(M)\)

En deduire la loi de \(R\).

Pourquoi on fait ca ?
Puisque \(\Delta\) est une matrice diagonale avec des 0 et des 1, c'est une matrice de projection. D'apres les questions de cours : \(\text{rg}(\Delta) = \text{tr}(\Delta)\). Comme rang et trace sont invariants par similitude : \(R = T\).

\(R = \text{rg}(M) = \text{rg}(\Delta) = \text{tr}(\Delta) = T\) (car \(\Delta\) est une projection).

Resultat
$$\boxed{R = T \sim \mathcal{B}(n, p)}$$
Question 4 -- Determinant \(D = \det(M)\)

(4.1) Determiner \(D(\Omega)\). (4.2) Loi de \(D\) et esperance.

Pourquoi on fait ca ?
Le determinant de \(\Delta = \text{diag}(X_1, \ldots, X_n)\) est le produit \(\prod X_k\). Comme chaque \(X_k\) vaut 0 ou 1, le produit vaut 1 si et seulement si tous les \(X_k\) valent 1.

\(D = \det(M) = \det(\Delta) = \prod_{k=1}^n X_k\). Comme les \(X_k \in \{0,1\}\), le produit vaut 1 si tous valent 1, et 0 sinon.

Resultat
$$\boxed{D(\Omega) = \{0, 1\}, \quad D \sim \mathcal{B}(p^n), \quad E(D) = p^n}$$ En effet, \(P(D = 1) = P(X_1 = 1, \ldots, X_n = 1) = p^n\) par independance.
Question 5 -- Evenement \(Z\)

\(Z\) : "les sous-espaces propres de \(M\) ont tous la meme dimension". (5.1) \(P(V)\) ou \(V\) = "une seule VP". (5.2) \(P(Z)\) si \(n\) impair. (5.3) \(P(Z)\) si \(n\) pair.

Pourquoi on fait ca ?
Les valeurs propres de \(\Delta\) sont parmi \(\{0, 1\}\). L'evenement \(Z\) signifie que les sous-espaces propres ont la meme dimension. Si \(M\) a une seule VP, les sous-espaces propres ont trivialement la meme dimension. Si \(M\) a deux VP (0 et 1), il faut que les deux espaces propres aient la meme taille.

5.1 -- Evenement \(V\) : une seule valeur propre

\(V = (\text{tous les } X_k = 1) \cup (\text{tous les } X_k = 0)\). Ces deux evenements sont incompatibles.

Resultat
$$\boxed{P(V) = p^n + (1-p)^n}$$

5.2 -- \(n\) impair

Si \(M\) a deux VP (0 et 1), les dimensions des sous-espaces propres sont \(T\) (pour la VP 1) et \(n - T\) (pour la VP 0). Pour qu'elles soient egales, il faudrait \(T = n - T\), soit \(n = 2T\). Mais \(n\) est impair, donc \(n\) n'est pas divisible par 2. Impossible !

Donc \(Z = V\) quand \(n\) est impair.

Resultat
Si \(n\) est impair : $$\boxed{P(Z) = p^n + (1-p)^n}$$

5.3 -- \(n\) pair, \(n = 2r\)

Si \(M\) a deux VP, la condition \(\dim E_0 = \dim E_1\) equivaut a \(T = r\). Donc \(Z = V \cup (T = r)\) et cette reunion est disjointe (si \(T = r\) avec \(0 < r < n\), il y a deux VP distinctes).

Resultat
Si \(n = 2r\) : $$P(T = r) = \binom{2r}{r} p^r(1-p)^r$$ $$\boxed{P(Z) = p^{2r} + (1-p)^{2r} + \binom{2r}{r} p^r(1-p)^r}$$
Piege classique
Le rapport du jury signale que les resultats sont souvent donnes sans justification (independance, incompatibilite). Pensez toujours a preciser pourquoi vous additionnez les probabilites (reunion disjointe) ou pourquoi vous multipliez (independance).
Question 6 -- Matrice \(A = UU^T\)

On pose \(U = \begin{pmatrix} X_1 \\ \vdots \\ X_n \end{pmatrix}\) et \(A = UU^T\). (6.1) \(a_{ij}\). (6.2) Loi de \(a_{ij}\). (6.3) \(\text{tr}(A) = \sum X_i\). (6.4) Valeurs de \(\text{rg}(A)\). (6.5) VP de \(A\). (6.6) Loi de \(\text{rg}(A)\).

Pourquoi on fait ca ?
La matrice \(A = UU^T\) est une matrice de rang au plus 1 (c'est le produit d'une colonne par une ligne). C'est un objet fondamental en algebre lineaire et en statistiques (matrice de covariance empirique).

6.1 : \(a_{ij} = X_i \cdot X_j\).

6.2 : Si \(i = j\) : \(a_{ii} = X_i^2 = X_i\) (car \(X_i \in \{0,1\}\)), donc \(a_{ii} \sim \mathcal{B}(p)\).

Si \(i \neq j\) : \(a_{ij} = X_i X_j\), produit de deux Bernoulli independantes. Donc \(a_{ij} \sim \mathcal{B}(p^2)\).

6.3 : \(\text{tr}(A) = \sum_{i=1}^n a_{ii} = \sum_{i=1}^n X_i^2 = \sum_{i=1}^n X_i\) (car \(X_i^2 = X_i\)).

Resultat
$$\boxed{\text{tr}(A) = \sum_{i=1}^n X_i}$$

6.4 : Les colonnes de \(A\) sont toutes de la forme \(X_j \cdot U\), donc colineaires a \(U\). Le rang de \(A\) vaut 0 si \(U = 0\) et 1 sinon.

Resultat
$$\boxed{\text{rg}(A) \in \{0, 1\}}$$

6.5 : Si \(A = 0\), la seule VP est 0. Sinon, \(\text{rg}(A) = 1\) et \(A\) est de taille \(n \times n\). L'unique VP non nulle est \(\text{tr}(A) = \sum X_i\) (car pour une matrice de rang 1, la trace egale la seule VP non nulle). L'autre VP est 0 de multiplicite \(n - 1\).

Resultat
Les valeurs propres de \(A\) sont 0 (multiplicite \(n-1\)) et \(\sum X_i\) (multiplicite 1), si \(A \neq 0\).

6.6 : \(\text{rg}(A) = 0 \iff U = 0 \iff X_1 = \cdots = X_n = 0\). Donc \(P(\text{rg}(A) = 0) = (1-p)^n\) et :

Resultat
$$\boxed{\text{rg}(A) \sim \mathcal{B}(1 - (1-p)^n)}$$
Methode -- Matrice de rang 1
Toute matrice de la forme \(UU^T\) (ou \(U\) est un vecteur colonne) est de rang au plus 1. Sa seule valeur propre non nulle est \(\|U\|^2 = U^T U = \text{tr}(A)\), et le vecteur propre associe est \(U\) lui-meme. C'est un cas tres frequent aux concours.
