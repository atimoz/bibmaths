Endomorphismes et formes bilineaires

Algebre lineaire -- Theme classique E3A
Intermediaire
Theme type Endomorphismes d'un espace euclidien : endomorphismes symetriques, orthogonaux, formes bilineaires symetriques, signature. Lien entre reduction et structure euclidienne.
Methode type -- Endomorphismes symetriques
Rappel de cours
Un endomorphisme \(u\) d'un espace euclidien \(E\) est symetrique (ou auto-adjoint) si \(\langle u(x), y \rangle = \langle x, u(y) \rangle\) pour tous \(x, y\). Equivalemment, sa matrice dans une base orthonormee est symetrique. Le theoreme spectral garantit qu'il est diagonalisable dans une base orthonormee.
Pourquoi on fait ca ?
Les endomorphismes symetriques sont fondamentaux car : (1) ils sont toujours diagonalisables, (2) les sous-espaces propres sont orthogonaux, (3) les valeurs propres sont reelles. Cela simplifie considerablement l'etude par rapport au cas general.

Demarche type. Soit \(\varphi\) une forme bilineaire symetrique sur \(\mathbf{R}^n\). On cherche sa signature \((p, q)\).

  1. Ecrire la matrice \(M\) de \(\varphi\) dans la base canonique
  2. Diagonaliser \(M\) (ou utiliser Gauss pour reduire la forme quadratique)
  3. Compter le nombre de valeurs propres positives (\(p\)) et negatives (\(q\))
  4. La signature est \((p, q)\), le rang est \(p + q\)
Resultat type
Pour \(\varphi(x, y) = x_1 y_1 + x_1 y_2 + x_2 y_1 - x_2 y_2\), la matrice est \(M = \begin{pmatrix} 1 & 1 \\ 1 & -1 \end{pmatrix}\) de valeurs propres \(\sqrt{2}\) et \(-\sqrt{2}\). La signature est \(\boxed{(1, 1)}\).
Piege classique
La matrice d'une forme bilineaire depend de la base. Si on change de base avec la matrice de passage \(P\), la matrice de \(\varphi\) devient \(P^T M P\) (pas \(P^{-1}MP\)). Ne pas confondre avec la formule de changement de base pour les endomorphismes.
Methode type -- Endomorphismes orthogonaux
Rappel de cours
\(u\) est orthogonal si \(\|u(x)\| = \|x\|\) pour tout \(x\), soit \(u^* \circ u = \mathrm{Id}\). En dimension 2, les matrices orthogonales sont les rotations \(\begin{pmatrix} \cos\theta & -\sin\theta \\ \sin\theta & \cos\theta \end{pmatrix}\) et les symetries \(\begin{pmatrix} \cos\theta & \sin\theta \\ \sin\theta & -\cos\theta \end{pmatrix}\). En dimension 3, on decompose en blocs : une rotation plane et un \(\pm 1\).
Methode -- Identifier une isometrie
Pour identifier une isometrie \(u\) de \(\mathbf{R}^3\) : (1) calculer \(\det(u)\) : si \(+1\), c'est une rotation ; si \(-1\), c'est une anti-rotation. (2) Chercher l'axe (espace propre pour la valeur propre 1). (3) Determiner l'angle dans le plan orthogonal.

Suites et series numeriques

Analyse -- Theme classique E3A
Standard
Theme type Suites recurrentes : convergence, rapidite de convergence. Series numeriques : criteres de convergence, series alternees, comparaison serie/integrale, developpements asymptotiques.
Methode type -- Suite recurrente \(u_{n+1} = f(u_n)\)
Methode -- Etude en 4 etapes
1. Points fixes : resoudre \(f(\ell) = \ell\).
2. Monotonie : etudier le signe de \(u_{n+1} - u_n\) (ou montrer par recurrence).
3. Bornitude : montrer que \((u_n)\) reste dans un intervalle \([a, b]\) stable par \(f\).
4. Convergence : suite monotone bornee, donc convergente. La limite est un point fixe.

Exemple type. \(u_0 = 2\), \(u_{n+1} = \frac{1}{2}\left(u_n + \frac{3}{u_n}\right)\). C'est la methode de Newton/Babylone pour \(\sqrt{3}\).

Points fixes : \(\ell = \frac{1}{2}(\ell + 3/\ell)\) donne \(\ell^2 = 3\), soit \(\ell = \sqrt{3}\).

Resultat type
La suite converge vers \(\sqrt{3}\) avec une convergence quadratique : $$u_{n+1} - \sqrt{3} = \frac{(u_n - \sqrt{3})^2}{2u_n}$$ Le nombre de decimales exactes double a chaque etape.
Piege classique
Pour passer a la limite dans \(u_{n+1} = f(u_n)\), il faut d'abord avoir prouve que la suite converge. On ne peut pas ecrire "\(\ell = f(\ell)\)" sans avoir justifie la convergence au prealable.
Methode type -- Convergence de series
Rappel de cours
Comparaison serie/integrale : si \(f\) est positive decroissante, \(\int_1^{N+1} f(t)\,\mathrm{d}t \leq \sum_{n=1}^N f(n) \leq f(1) + \int_1^N f(t)\,\mathrm{d}t\). Cela donne la nature de la serie et un equivalent du reste.
Methode -- Arbre de decision pour la nature
(1) Terme general ne tend pas vers 0 ? Diverge grossierement.
(2) Serie a termes positifs ? Comparaison, equivalent, d'Alembert, Cauchy.
(3) Serie alternee ? Critere de Leibniz.
(4) Convergence absolue ? Si \(\sum |a_n|\) converge, alors \(\sum a_n\) converge.
(5) Transformation d'Abel ? Pour les series \(\sum a_n b_n\) avec \(a_n \to 0\) et \(\sum b_n\) a sommes partielles bornees.

Equations differentielles lineaires

Analyse -- Theme classique E3A
Intermediaire
Theme type Equations differentielles lineaires d'ordre 1 et 2 : resolution, variation de la constante, systemes differentiels, comportement asymptotique, methode d'Euler.
Methode type -- EDL d'ordre 2 a coefficients constants
Rappel de cours
Pour \(ay'' + by' + cy = f(t)\) : (1) resoudre l'equation homogene via l'equation caracteristique \(ar^2 + br + c = 0\), (2) chercher une solution particuliere selon la forme de \(f\), (3) la solution generale est \(y_h + y_p\).
Methode -- Choix de la solution particuliere
Si \(f(t) = P(t)e^{\alpha t}\) : chercher \(y_p = t^m Q(t) e^{\alpha t}\) ou \(m\) est la multiplicite de \(\alpha\) comme racine de l'equation caracteristique et \(\deg Q = \deg P\).
Si \(f(t) = A\cos(\omega t) + B\sin(\omega t)\) : passer en complexes, chercher \(z_p = Ce^{i\omega t}\) (ou \(z_p = Cte^{i\omega t}\) si \(i\omega\) est racine).

Exemple type. \(y'' + 4y = \cos(2t)\). L'equation caracteristique \(r^2 + 4 = 0\) a pour racines \(\pm 2i\). Comme \(2i\) est racine simple, on cherche \(y_p = t(a\cos 2t + b\sin 2t)\).

Resultat type
$$y(t) = C_1\cos(2t) + C_2\sin(2t) + \frac{t}{4}\sin(2t)$$ Le terme \(t\sin(2t)\) traduit le phenomene de resonance : l'amplitude croit lineairement.
Piege classique
Si la frequence du second membre coincide avec une frequence propre, il y a resonance et la solution particuliere contient un facteur \(t\). Beaucoup de candidats cherchent \(y_p = a\cos(2t) + b\sin(2t)\) et trouvent \(0 = 1\), sans comprendre pourquoi. C'est le signe qu'il faut multiplier par \(t\).
Methode type -- Systemes differentiels
Methode -- Resolution par diagonalisation
Pour \(X' = AX\) : (1) diagonaliser \(A = PDP^{-1}\), (2) poser \(Y = P^{-1}X\), (3) le systeme devient \(Y' = DY\), decoupage en equations scalaires independantes, (4) revenir a \(X = PY\). Si \(A\) n'est pas diagonalisable, trigonaliser et resoudre par remontee.

Probabilites -- Variables a densite

Probabilites -- Theme classique E3A
Standard
Theme type Variables aleatoires a densite : loi exponentielle, loi normale, fonction de repartition, densite d'une fonction de variable aleatoire, couple de variables, independance.
Methode type -- Loi d'une fonction de VA
Rappel de cours
Pour trouver la loi de \(Y = g(X)\) quand \(X\) a une densite \(f_X\) : on calcule la fonction de repartition \(F_Y(y) = P(Y \leq y) = P(g(X) \leq y)\), puis on derive pour obtenir la densite \(f_Y = F_Y'\).
Methode -- Cas \(g\) monotone (formule directe)
Si \(g\) est un \(\mathcal{C}^1\)-diffeomorphisme sur le support de \(X\), alors : $$f_Y(y) = f_X(g^{-1}(y)) \cdot |{(g^{-1})}'(y)|$$ C'est la formule de changement de variable pour les densites. Mais attention : si \(g\) n'est pas monotone (par exemple \(Y = X^2\)), on revient a la methode par la fonction de repartition.

Exemple type. \(X \sim \mathcal{E}(\lambda)\), \(Y = \sqrt{X}\). On a \(g^{-1}(y) = y^2\) et \(|(g^{-1})'(y)| = 2y\).

Resultat type
$$f_Y(y) = 2\lambda y \, e^{-\lambda y^2} \cdot \mathbf{1}_{y > 0}$$ On reconnait une loi de Rayleigh (a un parametre pres).
Piege classique
Ne pas oublier la valeur absolue de la derivee dans la formule de changement de variable. Si \(g\) est decroissante, \((g^{-1})' < 0\) et il faut prendre la valeur absolue. Erreur frequente qui donne une densite negative.
Methode type -- Couple de variables et loi conditionnelle
Pourquoi on fait ca ?
Les couples de variables a densite interviennent dans les problemes de file d'attente, de fiabilite, et de statistiques. La formule de Bayes pour les densites permet de passer de la loi conjointe aux lois conditionnelles et vice-versa.
Methode -- Du couple aux marginales
Si \((X, Y)\) a pour densite \(f_{X,Y}(x, y)\) :
- Marginale de \(X\) : \(f_X(x) = \int_{-\infty}^{+\infty} f_{X,Y}(x, y)\,\mathrm{d}y\)
- Marginale de \(Y\) : \(f_Y(y) = \int_{-\infty}^{+\infty} f_{X,Y}(x, y)\,\mathrm{d}x\)
- Independance : \(f_{X,Y}(x,y) = f_X(x) \cdot f_Y(y)\) pour presque tout \((x, y)\).
