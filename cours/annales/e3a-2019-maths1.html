Exercice 1 -- Serie trigonometrique

On rappelle les formules : \(2\cos(p)\cos(q) = \cos(p+q) + \cos(p-q)\) et \(2\sin(p)\cos(q) = \sin(p+q) + \sin(p-q)\). On considere la suite \((\alpha_n)\) et la serie entiere \(\displaystyle\sum_{n \geqslant 0} u_n(x)\) ou \(u_n(x) = \dfrac{\alpha^n \cos(nx)}{n!}\), de rayon de convergence \(R\).

1.

Determiner le rayon de convergence \(R\) et le domaine de definition \(D\) de la somme \(C(x) = \displaystyle\sum_{n \geqslant 0} u_n(x)\).

2.

Etudier la convergence normale de la serie \(\displaystyle\sum_{n \geqslant 0} u_n\) sur \(D\).

3.

Pour \(x \in D\), calculer \(C(x)\).

4.

Pour \(n \in \mathbf{N}\), calculer les integrales :

\(J_n = \displaystyle\int_{-\pi}^{\pi} \sin(nx)\,C(x)\,\mathrm{d}x\) et \(I_n = \displaystyle\int_{-\pi}^{\pi} \cos(nx)\,C(x)\,\mathrm{d}x\).

5.

Calculer \(J_n\).

6.

Calculer \(I_n\).

7.

Determiner \(\displaystyle\lim_{n \to +\infty} J_n\) et \(\displaystyle\lim_{n \to +\infty} I_n\).

8.

On pose \(S(x) = \displaystyle\sum_{n \geqslant 0} \frac{\alpha^n \cos^2(nx)}{n!}\). Determiner le domaine de definition de \(S\) et calculer \(S(x)\).

Exercice 2 -- Endomorphisme de matrices

Soit \(\mathbb{K} = \mathbf{R}\) ou \(\mathbf{C}\). On note \(E_n = \mathcal{M}_n(\mathbb{K})\). Pour \(A \in E_n\), on note \(A_j\) la \(j\)-ieme colonne de \(A\). On definit l'endomorphisme \(u\) de \(E_n\) par : pour \(B \in E_n\), \(B_j = S - A_j\) ou \(S = \displaystyle\sum_{k=1}^{n} A_k\).

1.

Pour \(n = 2\), determiner une base \(\mathcal{B} = (K_1, K_2, K_3, K_4)\) de \(E_2\) avec :

\(K_1 = \begin{pmatrix} 1 & 0 \\ 0 & 0 \end{pmatrix}\), \(K_2 = \begin{pmatrix} 0 & 0 \\ 1 & 0 \end{pmatrix}\), \(K_3 = \begin{pmatrix} 0 & 1 \\ 0 & 0 \end{pmatrix}\), \(K_4 = \begin{pmatrix} 0 & 0 \\ 0 & 1 \end{pmatrix}\).

2.

Ecrire la matrice de \(u\) dans la base \(\mathcal{B}\).

3.

Determiner les valeurs propres de \(u\) pour \(n = 2\).

4.

Calculer \(\det(u(A))\) en fonction de \(\det(A)\) pour \(n = 2\).

5.

Meme question pour \(n = 3\).

6.

Pour \(n\) quelconque et \(S\) la matrice dont toutes les colonnes sont egales a la somme des colonnes de \(A\), montrer que \(\det(u(A)) = (-1)^{n-1}(n-1)\det(A)\).

7.

Determiner les valeurs propres de \(u\) dans le cas general.

8.

Soit \(J_n\) la matrice de \(E_n\) dont tous les coefficients valent 1 et \(U_n = J_n - I_n\). Montrer que \(AU_n\) est en lien avec l'image de \(A\) par \(u\).

Exercice 3 -- Equation integrale et fonctionnelle

On cherche les fonctions \(f\) continues sur \(\mathbf{R}\) verifiant l'equation integrale :

$$ \forall x \in \mathbf{R}, \quad f(x) = 1 - \int_0^x (t + x)\,f(x - t)\,\mathrm{d}t \qquad (E_1)$$

1.

Montrer que \(f\) est continue sur \(\mathbf{R}\).

2.

On pose \(F(x) = \displaystyle\int_0^x f(t)\,\mathrm{d}t\). Montrer que \(f\) est derivable sur \(\mathbf{R}\) et que \(F\) est de classe \(\mathcal{C}^1\) sur \(\mathbf{R}\).

3.

Montrer que \(f\) verifie le probleme \((P_1)\) :

$$\begin{cases} f'(x) + x\,f(x) + 2\displaystyle\int_0^x f(u)\,\mathrm{d}u = 0 \\ f(0) = 1 \end{cases}$$

4.

Montrer que \(f\) verifie le probleme \((P_2)\) :

$$\begin{cases} F''(x) + x\,F'(x) + 2\,F(x) = 0 \\ F'(0) = 1 \end{cases}$$

5.

On cherche \(H\) sous forme de serie entiere : \(H(x) = \displaystyle\sum_{n=0}^{+\infty} a_n x^n\) verifiant \(H''(x) + x\,H'(x) + 2\,H(x) = 0\), \(H'(0) = 1\), \(H(0) = 0\).

6.

Montrer que \(a_0 = 0\), \(a_1 = 1\) et \(\forall n \in \mathbf{N}\), \(a_{n+2} = -\dfrac{a_n}{n+1}\).

7.

Exprimer \(H(x)\) en fonction de \(x\).

Exercice 4 -- Espaces euclidiens de polynomes

Soit \(E = \mathbf{R}_n[X]\) muni du produit scalaire \(\langle P | Q \rangle = \displaystyle\int_0^1 P(t)\,Q(t)\,\mathrm{d}t\). On note \(\mathcal{B}_n = (1, X, \ldots, X^n)\) la base canonique de \(E\). On definit l'endomorphisme \(u\) de \(E\) par :

$$\forall P \in E, \quad u(P)(x) = \int_0^1 (x+t)^n\,P(t)\,\mathrm{d}t$$

1.

Montrer que \(u \in \mathcal{L}(E)\).

2.

Pour \(p, q \in \{0, \ldots, n\}\), calculer \(\langle u(X^p) | X^q \rangle\) et montrer que :

$$\langle u(X^p) | X^q \rangle = \sum_{k=0}^{n} \frac{\binom{n}{k}}{(k+q+1)(n+p+1-k)}$$

3.

Montrer que \(u\) est un endomorphisme symetrique positif.

4.

Soit \((P_j)_{0 \leqslant j \leqslant n}\) une base de vecteurs propres de \(u\) et \((\lambda_j)_{0 \leqslant j \leqslant n}\) les valeurs propres associees. Montrer que :

$$\forall (x, y) \in \mathbf{R}^2, \quad (x+y)^n = \sum_{j=0}^{n} \lambda_j\,P_j(x)\,P_j(y)$$

5.

Pour \(y \in \mathbf{R}\), on pose \(Q_y = (X+y)^n\). Decomposer \(Q_y\) dans la base \((P_j)_{0 \leqslant j \leqslant n}\).

6.

Calculer \(u(Q_y)(y) = \dfrac{1}{2n+1}\left[(y+1)^{2n+1} - y^{2n+1}\right]\).

7.

Calculer \(\displaystyle\int_0^1 u(Q_y)(y)\,\mathrm{d}y\) et en deduire \(\mathrm{tr}(u^2)\).
