I

Equation differentielle \(y'' + y = 0\)

Resolution, suites extraites, limite a l'infini -- 3 questions
Standard
Question 1

Montrer que l'ensemble des solutions de \((\mathcal{E}_0)\) sur \(I\) est \(\{x \to A\cos x + B\sin x \mid (A,B) \in \mathbb{R}^2\}\).

Rappel de cours
L'equation \(y'' + y = 0\) est une EDL2 a coefficients constants. L'equation caracteristique est \(r^2 + 1 = 0\), de racines \(r = \pm i\). Les solutions reelles sont donc \(x \mapsto A\cos x + B\sin x\) avec \((A,B) \in \mathbb{R}^2\).

L'equation caracteristique associee a \(y'' + y = 0\) est \(r^2 + 1 = 0\), dont les racines complexes sont \(r = i\) et \(r = -i\).

D'apres le theoreme general sur les EDL2 a coefficients constants, les solutions reelles sont les fonctions de la forme :

$$g(x) = A\cos x + B\sin x, \quad (A, B) \in \mathbb{R}^2.$$

Cet ensemble est bien un espace vectoriel de dimension 2, de base \((\cos, \sin)\).

Resultat
$$\boxed{\mathrm{Sol}(\mathcal{E}_0) = \{x \mapsto A\cos x + B\sin x \mid (A,B) \in \mathbb{R}^2\}}$$
Question 2

Soit \(g\) une solution de \((\mathcal{E}_0)\) sur \(I\). Que peut-on dire des suites \((g(n\pi))_{n \in \mathbb{N}}\) et \((g(\frac{2n+1}{2}\pi))_{n \in \mathbb{N}}\) ?

Soit \(g(x) = A\cos x + B\sin x\). On calcule les valeurs aux points demandes :

  • \(g(n\pi) = A\cos(n\pi) + B\sin(n\pi) = A(-1)^n\) car \(\sin(n\pi) = 0\).
  • \(g\bigl(\frac{2n+1}{2}\pi\bigr) = A\cos\bigl(\frac{2n+1}{2}\pi\bigr) + B\sin\bigl(\frac{2n+1}{2}\pi\bigr) = B(-1)^n\) car \(\cos\bigl(\frac{2n+1}{2}\pi\bigr) = 0\) et \(\sin\bigl(\frac{2n+1}{2}\pi\bigr) = (-1)^n\).
Resultat
$$\boxed{g(n\pi) = A(-1)^n \quad \text{et} \quad g\!\left(\tfrac{2n+1}{2}\pi\right) = B(-1)^n}$$Les deux suites sont des suites geometriques de raison \(-1\) (constantes si \(A = 0\) ou \(B = 0\)).
Question 3

Soit \(g\) une solution de \((\mathcal{E}_0)\). On suppose que \(g(x)\) tend vers une limite finie lorsque \(x\) tend vers \(+\infty\). Montrer que \(g\) est la fonction nulle.

Methode -- Suites extraites et unicite de la limite
Si \(g(x) \to \ell\), alors toute suite extraite \(g(x_n)\) converge vers \(\ell\). On exploite les deux suites de la question 2 pour forcer \(A = B = 0\).

Supposons que \(g(x) \to \ell \in \mathbb{R}\) quand \(x \to +\infty\). Alors en particulier :

  • \(g(n\pi) = A(-1)^n \to \ell\). Or la suite \((A(-1)^n)\) ne converge que si \(A = 0\), et dans ce cas \(\ell = 0\).
  • \(g\bigl(\frac{2n+1}{2}\pi\bigr) = B(-1)^n \to \ell = 0\). De meme, ceci impose \(B = 0\).

Donc \(A = B = 0\) et \(g\) est la fonction nulle.

Resultat
$$\boxed{g = 0}$$Toute solution de \(y'' + y = 0\) admettant une limite finie en \(+\infty\) est identiquement nulle.
Pourquoi ce resultat est important
Ce resultat sera crucial en partie V : il permet d'identifier deux solutions de \(y'' + y = 1/x\) dont la difference tend vers 0, donc qui sont egales.

II

Espace vectoriel \(V\) et endomorphisme \(\psi\)

Isomorphisme, noyau, image, resolution de \(y'' + y = \cos x\) -- 4 questions
Intermediaire
Question 1 -- \(V\) sous-espace vectoriel

Montrer que \(V\) est un sous-espace vectoriel de \(\mathcal{C}^\infty(\mathbb{R})\).

\(V = \{x \mapsto (ax+b)\cos x + (cx+d)\sin x \mid (a,b,c,d) \in \mathbb{R}^4\}\).

  • Non vide : \(h_{(0,0,0,0)} = 0 \in V\).
  • Stabilite : Si \(h_v, h_w \in V\) et \(\lambda \in \mathbb{R}\), alors \(h_v + \lambda h_w = h_{v + \lambda w} \in V\) car \(v + \lambda w \in \mathbb{R}^4\).

Donc \(V\) est bien un sous-espace vectoriel de \(\mathcal{C}^\infty(\mathbb{R})\). Chaque \(h_v\) est \(\mathcal{C}^\infty\) comme produit de polynomes et de fonctions trigonometriques.

Question 2 -- Isomorphisme \(\mathbb{R}^4 \to V\)

Demontrer que l'application qui envoie le vecteur \(v\) sur \(h_v\) definit un isomorphisme entre \(\mathbb{R}^4\) et \(V\). En deduire que \(\mathcal{B} = \{h_{e_1}, h_{e_2}, h_{e_3}, h_{e_4}\}\) est une base de \(V\).

Methode -- Montrer qu'une application lineaire est un isomorphisme
On montre que \(\Phi : v \mapsto h_v\) est lineaire (immediat) puis injective (noyau trivial). Comme les espaces de depart et d'arrivee ont meme dimension, c'est un isomorphisme.

L'application \(\Phi : v = (a,b,c,d) \mapsto h_v\) est clairement lineaire. Montrons qu'elle est injective : si \(h_v = 0\), alors pour tout \(x \in \mathbb{R}\) :

$$(ax + b)\cos x + (cx + d)\sin x = 0.$$

En evaluant en \(x = 0\) : \(b = 0\). En evaluant en \(x = \pi/2\) : \(d + c\pi/2 = 0\). En evaluant en \(x = \pi\) : \(-a\pi = 0\), donc \(a = 0\). En evaluant en \(x = 3\pi/2\) : \(-d - 3c\pi/2 = 0\). Avec \(d = -c\pi/2\), on obtient \(-(-c\pi/2) - 3c\pi/2 = c\pi/2 - 3c\pi/2 = -c\pi = 0\), donc \(c = 0\) puis \(d = 0\).

Donc \(\ker \Phi = \{0\}\) : \(\Phi\) est injective. Comme \(\Phi\) est surjective par definition de \(V\), c'est un isomorphisme. En particulier, \(\mathcal{B} = \{\Phi(e_1), \Phi(e_2), \Phi(e_3), \Phi(e_4)\}\) est une base de \(V\).

Resultat
\(\Phi : \mathbb{R}^4 \xrightarrow{\sim} V\) est un isomorphisme et \(\mathcal{B} = (x\cos x,\; \cos x,\; x\sin x,\; \sin x)\) est une base de \(V\), de dimension 4.
Question 3 -- Endomorphisme \(\psi\)

Exprimer l'application \(x \mapsto h_v''(x) + h_v(x)\). Demontrer que \(\psi\) est un endomorphisme de \(V\). Determiner le noyau, le rang, la matrice et une base de l'image de \(\psi\).

Calculons \(h_v''(x) + h_v(x)\) pour \(h_v(x) = (ax+b)\cos x + (cx+d)\sin x\).

On a \(h_v'(x) = a\cos x - (ax+b)\sin x + c\sin x + (cx+d)\cos x\), soit :

$$h_v'(x) = (a + cx + d)\cos x + (c - ax - b)\sin x.$$

Puis \(h_v''(x) = (c - a - bx + 2c - ax - d + cx)\)... Calculons plus methodiquement. On utilise les derivees connues :

  • \((x\cos x)'' = -x\cos x - 2\sin x\), donc \((x\cos x)'' + x\cos x = -2\sin x\).
  • \((\cos x)'' = -\cos x\), donc \((\cos x)'' + \cos x = 0\).
  • \((x\sin x)'' = -x\sin x + 2\cos x\), donc \((x\sin x)'' + x\sin x = 2\cos x\).
  • \((\sin x)'' = -\sin x\), donc \((\sin x)'' + \sin x = 0\).

Donc par linearite :

$$h_v'' + h_v = -2a\sin x + 2c\cos x.$$

C'est bien un element de \(V\) (avec les composantes \((0, 2c, 0, -2a)\) dans la base \(\mathcal{B}\)).

(i) \(\psi(h_v) = h_v'' + h_v \in V\) pour tout \(h_v \in V\), et \(\psi\) est lineaire (comme composition d'operations lineaires). Donc \(\psi\) est un endomorphisme de \(V\).

(ii) \(\ker \psi = \{h_v \mid h_v'' + h_v = 0\} = \{h_v \mid a = c = 0\}\). Les solutions de \(y'' + y = 0\) dans \(V\) sont les fonctions \(b\cos x + d\sin x\), soit \(\ker \psi = \mathrm{Vect}(\cos x, \sin x)\). Ainsi \(\dim \ker \psi = 2\) et par le theoreme du rang, \(\mathrm{rg}(\psi) = 4 - 2 = 2\).

(iii) La matrice de \(\psi\) dans la base \(\mathcal{B} = (x\cos x, \cos x, x\sin x, \sin x)\) est :

$$\boxed{M_\psi = \begin{pmatrix} 0 & 0 & 0 & 0 \\ 0 & 0 & 2 & 0 \\ 0 & 0 & 0 & 0 \\ -2 & 0 & 0 & 0 \end{pmatrix}}$$

Les colonnes non nulles donnent une base de \(\mathrm{Im}(\psi)\) :

Resultat
$$\ker\psi = \mathrm{Vect}(\cos x, \sin x), \quad \mathrm{rg}(\psi) = 2, \quad \mathrm{Im}(\psi) = \mathrm{Vect}(\cos x, \sin x) = \ker\psi$$Remarque : \(\mathrm{Im}(\psi) = \ker(\psi)\), ce qui signifie que \(\psi^2 = 0\) (endomorphisme nilpotent d'indice 2).
Question 4 -- Resolution de \(y'' + y = \cos x\)

Resoudre l'equation differentielle \((\mathcal{E}_1) : y'' + y = \cos x\) sur \(\mathbb{R}\).

Methode -- Variation de la constante ou lecture directe
On cherche une solution particuliere dans \(V\). Il suffit de trouver \(h_v\) tel que \(\psi(h_v) = \cos x\), c'est-a-dire \(2c = 1\) et \(-2a = 0\). On prend \(a = 0, c = 1/2\).

On cherche \(h_v \in V\) tel que \(h_v'' + h_v = \cos x\). D'apres le calcul precedent, \(h_v'' + h_v = 2c\cos x - 2a\sin x\). Donc on a besoin de \(2c = 1\) et \(-2a = 0\), soit \(a = 0\) et \(c = 1/2\).

Les valeurs de \(b\) et \(d\) sont libres (elles correspondent au noyau). On prend \(b = d = 0\), ce qui donne la solution particuliere :

$$y_0(x) = \frac{x}{2}\sin x.$$

La solution generale de \((\mathcal{E}_1)\) est donc :

Resultat
$$\boxed{y(x) = \frac{x}{2}\sin x + A\cos x + B\sin x, \quad (A,B) \in \mathbb{R}^2}$$
Piege classique
Ici \(\cos x\) est solution de l'equation homogene : on est en cas de resonance. C'est pourquoi la solution particuliere fait apparaitre un facteur \(x\) (phenomene typique). Ne pas chercher une solution de la forme \(C\cos x + D\sin x\) !

III

Fonction \(G\) et integrale a parametre

Convergence, continuite, derivation sous le signe integrale -- 6 questions
Intermediaire
Question 1 -- Convergence de l'integrale

Demontrer que \(\forall t \in \mathbb{R}_+,\; F(x,t) \leqslant \frac{1}{1+t^2}\). En deduire que \(\int_0^{+\infty} F(x,t)\,dt\) converge.

1a. Pour \(x \geqslant 0\) et \(t \geqslant 0\), on a \(e^{-tx} \leqslant 1\) (car \(tx \geqslant 0\)). Donc :

$$F(x,t) = \frac{e^{-tx}}{1+t^2} \leqslant \frac{1}{1+t^2}.$$

1b. L'integrale \(\int_0^{+\infty} \frac{dt}{1+t^2} = \frac{\pi}{2}\) est convergente (integrale de reference). Par comparaison de fonctions positives, \(\int_0^{+\infty} F(x,t)\,dt\) converge.

Resultat
$$\boxed{G(x) = \int_0^{+\infty} \frac{e^{-tx}}{1+t^2}\,dt \quad \text{est bien definie sur } \mathbb{R}_+}$$En particulier, \(G(0) = \int_0^{+\infty} \frac{dt}{1+t^2} = \frac{\pi}{2}\).
Question 2 -- Continuite de \(G\)

En utilisant l'inegalite demontree en 1a, justifier que la fonction \(G\) est continue sur \(\mathbb{R}_+\). On enoncera avec precision le theoreme utilise.

Rappel -- Theoreme de continuite des integrales a parametre
Si \(F(x,t)\) est continue en \(x\) pour presque tout \(t\), continue en \(t\) pour tout \(x\), et dominee par une fonction integrable \(\varphi(t)\) independante de \(x\), alors \(G(x) = \int F(x,t)\,dt\) est continue.

On verifie les hypotheses du theoreme de continuite sous le signe integrale :

  • Pour tout \(x \geqslant 0\), \(t \mapsto F(x,t)\) est continue sur \(\mathbb{R}_+\).
  • Pour tout \(t \geqslant 0\), \(x \mapsto F(x,t)\) est continue sur \(\mathbb{R}_+\).
  • Domination : \(|F(x,t)| \leqslant \frac{1}{1+t^2} = \varphi(t)\), avec \(\varphi\) integrable sur \(\mathbb{R}_+\).

Par le theoreme de continuite des integrales a parametre (convergence dominee), \(G\) est continue sur \(\mathbb{R}_+\).

Question 3 -- Derivabilite de \(G\)

Demontrer que \(G\) est derivable sur \(\mathbb{R}_+^*\) et que \(G'(x) = -\int_0^{+\infty} \frac{te^{-tx}}{1+t^2}\,dt\).

3a. \(F\) est de classe \(\mathcal{C}^\infty\) sur \(\mathbb{R}^2\) comme quotient de fonctions \(\mathcal{C}^\infty\) (le denominateur ne s'annule pas). La derivee partielle est :

$$\frac{\partial F}{\partial x}(x,t) = \frac{-te^{-tx}}{1+t^2}.$$

3b. Soit \(\epsilon > 0\). Pour \(x \geqslant \epsilon\) et \(t \geqslant 0\) :

$$\left|\frac{\partial F}{\partial x}(x,t)\right| = \frac{te^{-tx}}{1+t^2} \leqslant te^{-tx} \leqslant te^{-t\epsilon} \leqslant e^{-t\epsilon}$$

(la derniere inegalite vient de \(ue^{-u} \leqslant e^{-1}\) pour \(u \geqslant 0\), mais on utilise plus simplement que \(\frac{t}{1+t^2} \leqslant 1\) et \(e^{-tx} \leqslant e^{-t\epsilon}\)).

(i) La fonction \(t \mapsto e^{-t\epsilon}\) est integrable sur \(\mathbb{R}_+\) (integrale = \(1/\epsilon\)). Par comparaison, l'integrale \(\int_0^{+\infty} \frac{\partial F}{\partial x}(x,t)\,dt\) converge absolument pour \(x \geqslant \epsilon\).

(ii) Par le theoreme de derivation sous le signe integrale (hypothese de domination par \(e^{-t\epsilon}\) integrable, uniformement en \(x \in ]\epsilon, +\infty[\)), \(G\) est derivable sur \(]\epsilon, +\infty[\) et :

$$G'(x) = \int_0^{+\infty} \frac{\partial F}{\partial x}(x,t)\,dt = -\int_0^{+\infty} \frac{te^{-tx}}{1+t^2}\,dt.$$

3c. Ceci etant valable pour tout \(\epsilon > 0\), on conclut que \(G\) est derivable sur \(\mathbb{R}_+^* = \bigcup_{\epsilon > 0} ]\epsilon, +\infty[\), avec la formule ci-dessus.

Resultat
$$\boxed{G'(x) = -\int_0^{+\infty} \frac{te^{-tx}}{1+t^2}\,dt, \quad \forall x > 0}$$
Question 4 -- Derivee seconde

Demontrer que \(G\) est deux fois derivable sur \(\mathbb{R}_+^*\) et que \(G''(x) = \int_0^{+\infty} \frac{t^2 e^{-tx}}{1+t^2}\,dt\).

On applique a nouveau le theoreme de derivation sous le signe integrale. La derivee partielle seconde est :

$$\frac{\partial^2 F}{\partial x^2}(x,t) = \frac{t^2 e^{-tx}}{1+t^2}.$$

Pour \(x \geqslant \epsilon > 0\), on a la domination \(\frac{t^2 e^{-tx}}{1+t^2} \leqslant t^2 e^{-t\epsilon}\), et \(t \mapsto t^2 e^{-t\epsilon}\) est integrable sur \(\mathbb{R}_+\). Donc \(G\) est deux fois derivable sur \(]\epsilon, +\infty[\) pour tout \(\epsilon > 0\), d'ou la derivabilite sur \(\mathbb{R}_+^*\).

Resultat
$$\boxed{G''(x) = \int_0^{+\infty} \frac{t^2 e^{-tx}}{1+t^2}\,dt, \quad \forall x > 0}$$
Question 5 -- \(G\) solution de \((\mathcal{E})\)

Montrer que \(G\) est une solution de l'equation differentielle \(y'' + y = 1/x\).

Methode -- Astuce algebrique dans l'integrande
On additionne \(G''(x)\) et \(G(x)\) pour faire apparaitre une simplification dans le denominateur \(1+t^2\).

Pour tout \(x > 0\) :

$$G''(x) + G(x) = \int_0^{+\infty} \frac{t^2 e^{-tx}}{1+t^2}\,dt + \int_0^{+\infty} \frac{e^{-tx}}{1+t^2}\,dt = \int_0^{+\infty} \frac{(t^2+1)e^{-tx}}{1+t^2}\,dt = \int_0^{+\infty} e^{-tx}\,dt.$$

Or \(\int_0^{+\infty} e^{-tx}\,dt = \left[-\frac{e^{-tx}}{x}\right]_0^{+\infty} = \frac{1}{x}\) pour \(x > 0\).

Resultat
$$\boxed{G''(x) + G(x) = \frac{1}{x}, \quad \forall x > 0}$$\(G\) est une solution particuliere de \((\mathcal{E})\).
Questions 6a-6b -- Monotonie et limite de \(G\)

Demontrer que \(G\) est decroissante sur \(\mathbb{R}_+\). En deduire que \(G(x)\) admet une limite lorsque \(x \to +\infty\). Determiner cette limite.

6a. Pour tout \(x > 0\), \(G'(x) = -\int_0^{+\infty} \frac{te^{-tx}}{1+t^2}\,dt < 0\) car l'integrande est strictement positive (sauf en \(t = 0\)). Donc \(G\) est strictement decroissante sur \(\mathbb{R}_+^*\).

6b. \(G\) est decroissante et minoree par 0 (car \(F(x,t) \geqslant 0\), donc \(G(x) \geqslant 0\)). Par le theoreme de la limite monotone, \(G(x)\) converge vers une limite \(\ell \geqslant 0\).

Pour determiner \(\ell\), on utilise le theoreme de convergence dominee. Pour tout \(t > 0\), \(\frac{e^{-tx}}{1+t^2} \to 0\) quand \(x \to +\infty\), et \(|F(x,t)| \leqslant \frac{1}{1+t^2}\) integrable. Donc :

$$\lim_{x \to +\infty} G(x) = \int_0^{+\infty} \lim_{x \to +\infty} \frac{e^{-tx}}{1+t^2}\,dt = 0.$$
Resultat
$$\boxed{G \text{ est strictement decroissante sur } \mathbb{R}_+, \quad \lim_{x \to +\infty} G(x) = 0}$$

IV

Series alternees et integrales

Convergence par paquets, integrales semi-convergentes -- 6 questions
Avance
Question 1 -- Critere des series alternees

Soit \(\{u_n\}\) strictement croissante, positive, tendant vers \(+\infty\). Montrer que la serie \(\sum (-1)^n f(u_n)\) converge.

Rappel -- Critere special des series alternees (CSSA)
Si \((a_n)\) est une suite decroissante de reels positifs tendant vers 0, alors la serie \(\sum (-1)^n a_n\) converge.

\(f\) est positive et decroissante, et \((u_n)\) est strictement croissante, donc \((f(u_n))\) est decroissante. De plus, \(u_n \to +\infty\) et \(f(t) \to 0\) quand \(t \to +\infty\), donc \(f(u_n) \to 0\).

Par le critere special des series alternees, la serie \(\sum (-1)^n f(u_n)\) converge.

Question 2 -- Integrabilite de \(|\sin(t)|f(t)\)

Montrer que \(\sin(t)f(t)\) admet une limite quand \(t \to 0^+\). En deduire que \(|\sin(t)|f(t)\) est integrable sur \([0,x]\).

On a \(\sin(t)f(t) = \frac{\sin t}{t} \cdot tf(t)\). Or \(\frac{\sin t}{t} \to 1\) quand \(t \to 0^+\), et \(tf(t) \to L\) (limite finie par hypothese d)). Donc \(\sin(t)f(t) \to L\).

La fonction \(|\sin(t)|f(t)\) est donc continue sur \(]0, x]\) et admet une limite finie en \(0^+\). Elle se prolonge par continuite en 0, donc est integrable sur \([0, x]\).

Question 3 -- Etude des \(w_n\)

Justifier l'encadrement \(2f((n+1)\pi) \leqslant w_n \leqslant 2f(n\pi)\). En deduire l'existence de \(u_n\). Montrer que \(w_n = (-1)^n \int_{n\pi}^{(n+1)\pi} \sin(t) f(t)\,dt\).

3a. Sur \([n\pi, (n+1)\pi]\), \(f\) est decroissante, donc \(f((n+1)\pi) \leqslant f(t) \leqslant f(n\pi)\). Or \(|\sin t| \geqslant 0\) et \(\int_{n\pi}^{(n+1)\pi} |\sin t|\,dt = 2\). Donc :

$$2f((n+1)\pi) = f((n+1)\pi)\int_{n\pi}^{(n+1)\pi}|\sin t|\,dt \leqslant w_n \leqslant f(n\pi)\int_{n\pi}^{(n+1)\pi}|\sin t|\,dt = 2f(n\pi).$$

3b. La fonction \(t \mapsto f(t)\) est continue sur \([n\pi, (n+1)\pi]\) et prend les valeurs \(f(n\pi)\) et \(f((n+1)\pi)\). Or \(w_n/2\) est compris entre ces deux valeurs. Par le theoreme des valeurs intermediaires, il existe \(u_n \in [n\pi, (n+1)\pi]\) tel que \(f(u_n) = w_n/2\), soit \(w_n = 2f(u_n)\).

3c. Sur \([n\pi, (n+1)\pi]\), le signe de \(\sin t\) est constant : \(\sin t\) a le signe de \((-1)^n\) (positif si \(n\) pair, negatif si \(n\) impair). Donc \(|\sin t| = (-1)^n \sin t\) sur cet intervalle, et :

$$w_n = \int_{n\pi}^{(n+1)\pi} |\sin t| f(t)\,dt = (-1)^n \int_{n\pi}^{(n+1)\pi} \sin(t) f(t)\,dt.$$
Question 4 -- Convergence des suites d'integrales

Montrer que \(\int_0^{2n\pi} \sin(t)f(t)\,dt\) est croissante, \(\int_0^{(2n+1)\pi} \sin(t)f(t)\,dt\) est decroissante, et qu'elles convergent vers une meme limite.

4a. On a \(\int_0^{2(n+1)\pi} - \int_0^{2n\pi} = \int_{2n\pi}^{(2n+1)\pi} \sin(t)f(t)\,dt + \int_{(2n+1)\pi}^{(2n+2)\pi} \sin(t)f(t)\,dt = w_{2n} - w_{2n+1}\).

Or \(w_{2n} = 2f(u_{2n}) \geqslant 2f(u_{2n+1}) = w_{2n+1}\) car \(u_{2n} \leqslant u_{2n+1}\) et \(f\) est decroissante. Donc la suite est croissante.

4b. De meme, \(\int_0^{(2n+3)\pi} - \int_0^{(2n+1)\pi} = w_{2n+1} - w_{2n+2} \leqslant 0\), donc la suite est decroissante.

4c. La suite croissante est majoree par le premier terme de la suite decroissante (et reciproquement, la suite decroissante est minoree). Les deux suites convergent. L'ecart entre les deux est \(w_{2n} = 2f(u_{2n}) \to 0\) (car \(u_{2n} \to +\infty\) et \(f(t) \to 0\)). Donc elles ont la meme limite \(l\).

Resultat
Les deux suites sont adjacentes et convergent vers une meme limite \(l\).
Question 5 -- Convergence de l'integrale impropre

Deduire que \(I_f(x,y) = \int_x^y \sin(t) f(t)\,dt\) admet une limite finie quand \(y \to +\infty\).

Pour \(y\) quelconque, on encadre \(y\) entre \(n\pi\) et \((n+1)\pi\). Les integrales \(\int_0^{n\pi}\) convergent vers \(l\), et le reste \(\int_{n\pi}^y |\sin(t)|f(t)\,dt \leqslant w_n \to 0\). Donc \(\int_0^y \sin(t)f(t)\,dt \to l\), et par translation \(\int_x^{+\infty} \sin(t)f(t)\,dt\) existe.

Question 6 -- Application a \(f(t) = 1/t\)

Justifier l'existence de \(I_x = \int_x^{+\infty} \frac{\sin t}{t}\,dt\).

La fonction \(f(t) = 1/t\) verifie les quatre hypotheses (a-d) de la partie IV sur \(\mathbb{R}_+^*\) :

  • (a) \(f\) est positive, (b) \(f\) est decroissante, (c) \(f(t) \to 0\), (d) \(tf(t) = 1 \to 1\).

Par la question 5, l'integrale \(\int_x^{+\infty} \frac{\sin t}{t}\,dt\) converge pour tout \(x > 0\).

Resultat
$$\boxed{I_x = \int_x^{+\infty} \frac{\sin t}{t}\,dt \text{ existe pour tout } x > 0}$$C'est une integrale semi-convergente (convergente mais pas absolument convergente).

Fonction \(H\) et identification \(H = G\)

Changement de variable, derivation, identification -- 5 questions
Avance
Question 1 -- Hypotheses de la partie IV pour \(h_x\)

Montrer que \(h_x(t) = \frac{1}{x+t}\) verifie les hypotheses de la partie IV.

Soit \(x > 0\). La fonction \(h_x : t \mapsto \frac{1}{x+t}\) est definie et continue sur \(\mathbb{R}_+^*\).

  • (a) \(h_x(t) = \frac{1}{x+t} > 0\) : positive.
  • (b) \(t \mapsto \frac{1}{x+t}\) est decroissante.
  • (c) \(h_x(t) \to 0\) quand \(t \to +\infty\).
  • (d) \(th_x(t) = \frac{t}{x+t} \to 1\) quand \(t \to 0^+\).

Par la partie IV, l'integrale \(H(x) = \int_0^{+\infty} \frac{\sin t}{t+x}\,dt\) existe pour tout \(x > 0\).

Question 2 -- Changement de variable

Demontrer que \(H(x) = \int_x^{+\infty} \frac{\sin(t-x)}{t}\,dt\).

Dans l'integrale \(H(x) = \int_0^{+\infty} \frac{\sin t}{t+x}\,dt\), on effectue le changement de variable \(u = t + x\) (donc \(t = u - x\), \(dt = du\)). Quand \(t = 0\), \(u = x\) ; quand \(t \to +\infty\), \(u \to +\infty\). Donc :

$$H(x) = \int_x^{+\infty} \frac{\sin(u - x)}{u}\,du.$$
Resultat
$$\boxed{H(x) = \int_x^{+\infty} \frac{\sin(t - x)}{t}\,dt}$$
Question 3 -- \(H\) solution de \((\mathcal{E})\)

En developpant \(\sin(t-x)\), demontrer que \(H\) est deux fois derivable sur \(\mathbb{R}_+^*\) et que \(H''(x) + H(x) = \frac{1}{x}\).

Methode -- Decomposer puis deriver
On utilise \(\sin(t-x) = \sin t \cos x - \cos t \sin x\) pour ecrire \(H(x)\) comme combinaison lineaire de deux integrales ne dependant de \(x\) qu'a travers des facteurs trigonometriques.

En developpant \(\sin(t - x) = \sin t \cos x - \cos t \sin x\) :

$$H(x) = \cos x \int_x^{+\infty} \frac{\sin t}{t}\,dt - \sin x \int_x^{+\infty} \frac{\cos t}{t}\,dt.$$

Posons \(S(x) = \int_x^{+\infty} \frac{\sin t}{t}\,dt\) et \(C(x) = \int_x^{+\infty} \frac{\cos t}{t}\,dt\). Ces fonctions sont derivables sur \(\mathbb{R}_+^*\) avec \(S'(x) = -\frac{\sin x}{x}\) et \(C'(x) = -\frac{\cos x}{x}\).

Donc \(H(x) = S(x)\cos x - C(x)\sin x\). En derivant :

$$H'(x) = S'(x)\cos x - S(x)\sin x - C'(x)\sin x - C(x)\cos x$$
$$= -\frac{\sin x\cos x}{x} - S(x)\sin x + \frac{\cos x\sin x}{x} - C(x)\cos x$$
$$= -S(x)\sin x - C(x)\cos x.$$

En derivant une seconde fois :

$$H''(x) = -S'(x)\sin x - S(x)\cos x - C'(x)\cos x + C(x)\sin x$$
$$= \frac{\sin^2 x}{x} - S(x)\cos x + \frac{\cos^2 x}{x} + C(x)\sin x$$
$$= \frac{1}{x} - S(x)\cos x + C(x)\sin x = \frac{1}{x} - H(x).$$
Resultat
$$\boxed{H''(x) + H(x) = \frac{1}{x}, \quad \forall x > 0}$$\(H\) est aussi solution de l'equation differentielle \((\mathcal{E})\).
Question 4 -- Limite de \(H\)

Quelle est la limite de \(H(x)\) lorsque \(x \to +\infty\) ?

Pour \(x > 0\), on a \(|H(x)| \leqslant \int_0^{+\infty} \frac{|\sin t|}{t+x}\,dt \leqslant \int_0^{+\infty} \frac{dt}{t+x} \cdot \mathbb{1}_{[0,M]}(t) + \int_M^{+\infty} \frac{dt}{t+x}\).

Plus simplement, par decoupage : \(|H(x)| \leqslant \frac{1}{x}\int_0^{+\infty} |\sin t| \cdot \frac{x}{t+x}\,dt\). En fait, on montre directement que pour tout \(t \geqslant 0\), \(\frac{|\sin t|}{t+x} \leqslant \frac{1}{t+x} \leqslant \frac{1}{x}\), mais cela ne donne pas la convergence de l'integrale.

Utilisons plutot les suites de la partie IV : \(H(x) = \int_0^{+\infty} \frac{\sin t}{t+x}\,dt\), avec \(f = h_x\) qui verifie les hypotheses. L'encadrement \(0 \leqslant w_n \leqslant 2h_x(n\pi) = \frac{2}{x + n\pi}\) montre que la somme de la serie \(\sum w_n\) est majoree par \(\frac{2}{x}\sum \frac{1}{1 + n\pi/x}\), qui tend vers 0 quand \(x \to +\infty\) (chaque terme tend vers 0). Donc \(H(x) \to 0\).

Resultat
$$\boxed{\lim_{x \to +\infty} H(x) = 0}$$
Question 5 -- Identification \(H = G\)

En deduire que \(H(x) = G(x)\) pour tout \(x > 0\).

Methode -- Difference de deux solutions
Si \(G\) et \(H\) sont toutes deux solutions de \(y'' + y = 1/x\), alors \(G - H\) est solution de \(y'' + y = 0\). On utilise ensuite le resultat de la partie I.

Posons \(\varphi = G - H\). Alors \(\varphi'' + \varphi = (G'' + G) - (H'' + H) = \frac{1}{x} - \frac{1}{x} = 0\).

Donc \(\varphi\) est solution de \(y'' + y = 0\), c'est-a-dire \(\varphi(x) = A\cos x + B\sin x\).

Or \(\lim_{x \to +\infty} G(x) = 0\) (partie III, question 6b) et \(\lim_{x \to +\infty} H(x) = 0\) (question 4). Donc \(\varphi(x) \to 0\) quand \(x \to +\infty\).

Par le resultat de la partie I (question 3), une solution de \(y'' + y = 0\) admettant une limite finie en \(+\infty\) est necessairement nulle. Donc \(\varphi = 0\), c'est-a-dire \(G = H\).

Resultat
$$\boxed{\forall x > 0, \quad \int_0^{+\infty} \frac{e^{-tx}}{1+t^2}\,dt = \int_0^{+\infty} \frac{\sin t}{t+x}\,dt}$$Deux representations integrales differentes de la meme fonction : l'une par transformee de Laplace, l'autre par integrale de type sinus integral.
Pourquoi ce resultat est remarquable
Ce probleme illustre une technique puissante : pour montrer l'egalite de deux integrales, on montre que les fonctions qu'elles definissent verifient la meme equation differentielle, puis on utilise l'unicite (a solution de l'homogene pres) et le comportement a l'infini pour conclure. C'est une methode qui generalise bien au-dela de cet exercice.
