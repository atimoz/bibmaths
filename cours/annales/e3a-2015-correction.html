Matrice \(A = I + J\), diagonalisation et espace \(\mathcal{E}\)

Algebre bilineaire, reduction, produits scalaires -- 4 questions
Intermediaire
EnonceOn etudie la matrice \(A = I_n + J\) ou \(J\) est la matrice dont tous les coefficients valent 1. On la diagonalise, puis on etudie l'espace \(\mathcal{E}\) des matrices \(U\) verifiant \({}^tUA = AU\), et le produit scalaire \(\varphi(x,y) = {}^tXAY\).
Question 1(a) -- \(J^2\) en fonction de \(J\)

Exprimer la matrice \(J^2\) en fonction de la matrice \(J\).

Le coefficient \((i,j)\) de \(J^2\) est \(\sum_{k=1}^n 1 \cdot 1 = n\). Donc chaque coefficient de \(J^2\) vaut \(n\).

Resultat
$$\boxed{J^2 = nJ}$$
Question 1(b) -- Polynome annulateur de \(A\)

En deduire un polynome annulateur du second degre de la matrice \(A\).

Comme \(A = I + J\), on a \(A^2 = I + 2J + J^2 = I + 2J + nJ = I + (n+2)J\). D'autre part \((n+1)A = (n+1)I + (n+1)J\). Donc \(A^2 - (n+1)A = I + (n+2)J - (n+1)I - (n+1)J = -nI + J\). Or \(A - (n+1)I = J - nI\), ce n'est pas commode. Reprenons : \(A = I + J\) et \(J^2 = nJ\) impliquent \(A(A - (n+1)I) = A^2 - (n+1)A = (I+J)^2 - (n+1)(I+J) = I + 2J + nJ - (n+1)I - (n+1)J = -nI + (n+1)J - (n+1)J = -nI\)... Non. Recalculons : \(A^2 = I + 2J + nJ = I + (n+2)J\). Et \((n+2)A = (n+2)I + (n+2)J\). Donc \(A^2 - (n+2)A + (n+1)I = I + (n+2)J - (n+2)I - (n+2)J + (n+1)I = 0\).

Resultat
$$\boxed{A^2 - (n+2)A + (n+1)I = 0, \quad\text{i.e. } P(X) = X^2 - (n+2)X + (n+1) = (X-1)(X-(n+1))}$$
Question 1(c) -- Inversibilite de \(A\)

Montrer que la matrice \(A\) est inversible et calculer son inverse \(A^{-1}\).

Pourquoi on fait ca ?
Le polynome annulateur \((X-1)(X-(n+1)) = 0\) a 0 comme non-racine, donc \(A\) est inversible. De \(A^2 - (n+2)A + (n+1)I = 0\), on tire \(A((n+2)I - A) = (n+1)I\).
Resultat
$$\boxed{A^{-1} = \frac{1}{n+1}\left((n+2)I - A\right) = \frac{1}{n+1}(n+1)I - J) = I - \frac{1}{n+1}J}$$
Question 2 -- Diagonalisation de \(A\)

Montrer \(A = QDQ^{-1}\) avec \(Q \in O_n(\mathbb{R})\). Determiner valeurs propres et sous-espaces propres.

Rappel de cours
\(A = I + J\) est reelle symetrique, donc diagonalisable dans une base orthonormee (theoreme spectral). Ses valeurs propres sont les racines du polynome annulateur : \(1\) et \(n+1\).

\(E_1 = \ker(A - I) = \ker J = \{X \in \mathbb{R}^n : \sum x_i = 0\}\), de dimension \(n-1\).

\(E_{n+1} = \ker(A - (n+1)I) = \ker(J - nI)\). On verifie que \(J \cdot (1,\ldots,1)^T = n(1,\ldots,1)^T\), donc \(E_{n+1} = \mathrm{Vect}((1,\ldots,1)^T)\), de dimension 1.

Resultat
$$\boxed{D = \mathrm{diag}(\underbrace{1,\ldots,1}_{n-1}, n+1), \quad \dim E_1 = n-1, \quad \dim E_{n+1} = 1}$$
Questions 3-4 -- Espace \(\mathcal{E}\) et produit scalaire \(\varphi\)

Montrer que \(\mathcal{E}\) est un sev. Calculer \(\dim \mathcal{E}\). Montrer que \(\varphi\) est un produit scalaire et que les elements de \(\mathcal{E}\) sont \(\varphi\)-symetriques.

Methode -- Changement de base pour \(\mathcal{E}\)
On pose \(V = {}^tQUQ\). La condition \({}^tUA = AU\) se transforme en \({}^tVD = DV\). Comme \(D = \mathrm{diag}(1,\ldots,1,n+1)\), cette condition signifie que \(V\) commute avec les blocs de \(D\). L'ensemble \(\mathcal{F}\) des matrices \(V\) telles que \({}^tVD = DV\) est isomorphe a \(\mathcal{E}\).

La condition \({}^tVD = DV\) avec \(D = \mathrm{diag}(1,\ldots,1,n+1)\) impose que \(V\) est de la forme bloc \(\begin{pmatrix} S & 0 \\ 0 & \lambda \end{pmatrix}\) ou \(S\) est symetrique \((n-1) \times (n-1)\) et \(\lambda \in \mathbb{R}\). Donc \(\dim \mathcal{F} = \frac{(n-1)n}{2} + 1\).

Pour Q4(a) : \(\varphi(x,y) = {}^tXAY\). Symetrie : \(\varphi(y,x) = {}^tYAX = {}^t({}^tXAY) = \varphi(x,y)\) car \(A\) est symetrique. Positivite : \(\varphi(x,x) = {}^tXAX = \|x\|^2 + ({}^tXJ X) = \|x\|^2 + (\sum x_i)^2 \geqslant \|x\|^2 > 0\) si \(x \neq 0\).

Resultat
$$\boxed{\dim \mathcal{E} = \frac{n(n-1)}{2} + 1}$$\(\varphi\) est un produit scalaire car \(A\) est symetrique definie positive. Les elements de \(\mathcal{E}\) sont exactement les endomorphismes \(\varphi\)-symetriques, donc diagonalisables dans une base \(\varphi\)-orthonormale.

Matrices tridiagonales et entrelacement

Recurrence, determinants, wronskien discret -- 4 questions
Avance
EnonceOn etudie les matrices tridiagonales symetriques \(A_n\) de diagonale \((a_1,\ldots,a_n)\) et sur-diagonale \((b_1,\ldots,b_{n-1})\). On montre que les valeurs propres sont simples et que les zeros de \(P_{n+1}\) entrelacent ceux de \(P_n\).
Question 1 -- Recurrence sur \(P_n\)

Determiner une relation de recurrence entre \(P_{n+1}(X)\), \(P_n(X)\) et \(P_{n-1}(X)\).

En developpant \(\det(XI_{n+1} - A_{n+1})\) par rapport a la derniere ligne :

Resultat
$$\boxed{P_{n+1}(X) = (X - a_{n+1})P_n(X) - b_n^2 P_{n-1}(X)}$$
Questions 2 -- Valeurs propres simples

Montrer que \(A_n\) est diagonalisable, puis que ses valeurs propres sont simples.

Rappel de cours
\(A_n\) est reelle symetrique, donc diagonalisable. Pour montrer que les valeurs propres sont simples, on montre que chaque sous-espace propre est de dimension 1.

Q2(b) : Le determinant de la sous-matrice extraite vaut \((-1)^{n-1} b_1 b_2 \cdots b_{n-1} \neq 0\) (car tous les \(b_i\) sont non nuls). Q2(c) : Donc \(\mathrm{rg}(\lambda I - A_n) \geqslant n-1\), soit \(\dim \ker(\lambda I - A_n) \leqslant 1\). Comme c'est un sous-espace propre, il est de dimension exactement 1.

Resultat
$$\boxed{P_n(X) \text{ admet } n \text{ racines reelles distinctes}}$$
Questions 3-4 -- Wronskien discret et entrelacement

Montrer que \(\Delta_n(x) > 0\) pour tout \(x\) reel. En deduire que \(P_{n+1}\) s'annule entre deux zeros consecutifs de \(P_n\).

Methode -- Entrelacement par le wronskien
Le wronskien \(\Delta_n(x) = P'_{n+1}(x)P_n(x) - P_{n+1}(x)P'_n(x)\) est strictement positif. Si \(\alpha < \beta\) sont deux zeros consecutifs de \(P_n\), alors \(P_n\) ne change pas de signe entre eux, mais \(\Delta_n(\alpha) > 0\) et \(\Delta_n(\beta) > 0\) forcent \(P_{n+1}\) a s'annuler entre \(\alpha\) et \(\beta\).

Q3(a) : Par la recurrence \(P_{n+1} = (X-a_{n+1})P_n - b_n^2 P_{n-1}\), on derive et on calcule \(\Delta_n = P_n^2 + b_n^2 \Delta_{n-1}\).

Q3(b) : \(\Delta_1(x) = P'_2(x)P_1(x) - P_2(x)P'_1(x)\). Avec \(P_1(x) = x - a_1\) et \(P_2(x) = (x-a_1)(x-a_2) - b_1^2\), on obtient \(\Delta_1(x) = (x-a_2)^2 \cdot 1 - \ldots = P_1^2(x) + b_1^2 > 0\).

Q4 : Si \(\alpha, \beta\) sont zeros consecutifs de \(P_n\), on considere \(x \mapsto P_{n+1}(x)/P_n(x)\). Sa derivee est \(-\Delta_n(x)/P_n(x)^2 < 0\), donc elle est strictement decroissante entre les poles. Aux voisinages de \(\alpha^+\) et \(\beta^-\), cette fonction passe de \(+\infty\) a \(-\infty\) (ou l'inverse), donc s'annule.

Resultat
$$\boxed{\text{Les zeros de } P_{n+1} \text{ entrelacent ceux de } P_n}$$C'est le theoreme d'entrelacement de Cauchy pour les matrices tridiagonales.

Integrales impropres et transformee de Fourier

Integrale de Dirichlet, IPP, calcul de \(I\) -- 3 questions
Intermediaire
Question 1 -- Convergence de l'integrale de Dirichlet

Montrer que l'integrale \(\int_0^{+\infty} \frac{\sin x}{x}\,\mathrm{d}x\) existe.

Methode -- Convergence semi-convergente
L'integrale n'est pas absolument convergente, mais converge par le critere d'Abel : \(1/x\) decroit vers 0 et les sommes partielles \(\int_0^A \sin x\,\mathrm{d}x\) sont bornees.

En 0 : \(\sin x / x \to 1\), prolongeable par continuite. En \(+\infty\) : IPP avec \(u = 1/x\), \(v' = \sin x\) donne \(\int_1^A \frac{\sin x}{x}\,\mathrm{d}x = \left[-\frac{\cos x}{x}\right]_1^A - \int_1^A \frac{\cos x}{x^2}\,\mathrm{d}x\). Le dernier terme converge absolument car \(|\cos x/x^2| \leqslant 1/x^2\).

Resultat
L'integrale converge (semi-convergence). On admet \(\int_0^{+\infty} \frac{\sin x}{x}\,\mathrm{d}x = \frac{\pi}{2}\).
Questions 2-3 -- Calcul de l'integrale \(I\)

Montrer l'integrabilite et calculer \(I = \int_{-\infty}^{+\infty} \frac{1 - \cos \alpha t}{t^2}\,e^{-itx}\,\mathrm{d}t\).

Q2(a) : En 0, \(\frac{1 - \cos \alpha t}{t^2} \to \frac{\alpha^2}{2}\) (DL), et \(e^{-itx}\) est bornee, donc prolongeable par continuite.

Q2(b) : \(\left|\frac{1 - \cos \alpha t}{t^2} e^{-itx}\right| \leqslant \frac{2}{t^2}\) pour \(|t| \geqslant 1\), integrable. Pres de 0, la fonction est bornee. Donc integrable sur \(\mathbb{R}\).

Q3(a) : \(I\) est reelle car en posant \(t \to -t\), la partie imaginaire s'annule par parite.

Q3(b) : IPP avec \(u = \cos Bx / x^2\) et adaptation donne le resultat.

Q3(c) : \(\int_0^{+\infty} \frac{1-\cos Bx}{x^2}\,\mathrm{d}x = \frac{\pi |B|}{2}\) (par passage a la limite \(A \to 0^+\)).

Q3(d) : En ecrivant \(1 - \cos \alpha t = \frac{1}{2}(e^{i\alpha t} + e^{-i\alpha t}) - 1\), on decompose \(I\) en integrales du type \(\int \frac{1 - \cos Bt}{t^2}\,\mathrm{d}t\) avec \(B = \alpha \pm x\).

Resultat
$$\boxed{I = \pi\alpha - \pi|x| \quad\text{si } |x| \leqslant \alpha, \qquad I = 0 \quad\text{si } |x| > \alpha}$$C'est la transformee de Fourier de \(\frac{1-\cos \alpha t}{t^2}\), qui vaut \(\pi(\alpha - |x|)^+\).

Canal binaire bruite et probabilites

Probabilites conditionnelles, loi binomiale, Python -- 8 questions
Standard
Questions 1-3 -- Transmission d'un bit

Exprimer les probabilites conditionnelles. Probabilite de recevoir un 1. Probabilite a posteriori.

Q1 : \(1-p = P(b'=0|b=1)\), \(q = P(b'=0|b=0)\), \(1-q = P(b'=1|b=0)\).

Q2 : \(P(b'=1) = P(b'=1|b=1)P(b=1) + P(b'=1|b=0)P(b=0) = pa + (1-q)(1-a)\).

Q3 : Par Bayes, \(P(b=1|b'=1) = \frac{P(b'=1|b=1)P(b=1)}{P(b'=1)} = \frac{pa}{pa + (1-q)(1-a)}\).

Resultat
$$\boxed{P(b'=1) = pa + (1-q)(1-a), \quad P(b=1|b'=1) = \frac{pa}{pa + (1-q)(1-a)}}$$
Questions 4-6 -- Repetition du bit et loi de \(X\)

Exprimer \(P(X=k)\) et \(\mathbb{E}(X)\). Probabilite conditionnelle sachant \(X=k\).

Q4 : Conditionnellement a \(b=1\), \(X \sim \mathcal{B}(n,p)\). Conditionnellement a \(b=0\), \(X \sim \mathcal{B}(n,1-q)\). Donc par la formule des probabilites totales :

$$P(X=k) = a\binom{n}{k}p^k(1-p)^{n-k} + (1-a)\binom{n}{k}(1-q)^k q^{n-k}$$

Q5 : \(\mathbb{E}(X) = a \cdot np + (1-a) \cdot n(1-q) = n(ap + (1-a)(1-q))\).

Q6 : \(P(b=1|X=k) = \frac{a \binom{n}{k} p^k(1-p)^{n-k}}{P(X=k)}\) par Bayes.

Resultat
$$\boxed{\mathbb{E}(X) = n\big(ap + (1-a)(1-q)\big)}$$
Questions 7-8 -- Canal symetrique et estimation d'erreur

Canal symetrique (\(q = p\)). Determiner les valeurs de \(k\) pour lesquelles il est plus probable qu'un 1 ait ete envoye. Calculer \(f(n)\).

Pourquoi on fait ca ?
Avec un canal symetrique, on decide par vote majoritaire : si plus de la moitie des bits recus sont des 1, on decide que le bit envoye etait un 1. La question est de quantifier le taux d'erreur.

Q7 : Avec \(q = p\), \(P(b=1|X=k) > P(b=0|X=k)\) ssi \(a \cdot p^k(1-p)^{n-k} > (1-a)(1-p)^k p^{n-k}\), soit \(\frac{a}{1-a} > \left(\frac{1-p}{p}\right)^{2k-n}\). Avec \(a = 1/2\) : \(k > n/2\).

Q8(b) : Avec \(a = 1/2\), \(f(n) = \sum_{k=0}^{\lfloor n/2 \rfloor} \binom{n}{k} p^k(1-p)^{n-k}\) (pour \(n\) impair). C'est la queue de la loi binomiale.

Q8(c)-(d) : Python :

def binom(N, k):
    if k < 0 or k > N:
        return 0
    result = 1
    for i in range(min(k, N - k)):
        result = result * (N - i) // (i + 1)
    return result

def f(n, p=0.95):
    s = 0
    for k in range(n // 2 + 1):
        s += binom(n, k) * p**k * (1 - p)**(n - k)
    return s
Resultat
La probabilite d'erreur \(f(n)\) decroit exponentiellement vers 0 quand \(n\) augmente, ce qui montre l'interet de la repetition pour fiabiliser la transmission.
Piege classique
Attention au cas \(n\) pair : quand \(k = n/2\), il y a egalite et on ne peut pas decider. Il faut traiter ce cas a part dans \(f(n)\).
