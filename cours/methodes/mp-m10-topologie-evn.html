I. Methodes a retenir

M1. Montrer qu'une application est une norme

Methode

Pour montrer que \(N : E \to \mathbb{R}\) est une norme sur l'espace vectoriel \(E\), on verifie les trois axiomes :

  1. Separation : \(N(x) = 0 \Longleftrightarrow x = 0_E\).
  2. Homogeneite : pour tout \(\lambda \in \mathbb{K}\) et tout \(x \in E\), \(N(\lambda x) = |\lambda| \, N(x)\).
  3. Inegalite triangulaire : pour tous \(x, y \in E\), \(N(x + y) \leq N(x) + N(y)\).

Remarque : la positivite \(N(x) \geq 0\) decoule des axiomes (prendre \(y = -x\) dans l'inegalite triangulaire).

Astuce frequente : pour l'inegalite triangulaire, on utilise souvent la croissance de l'integrale, l'inegalite de Cauchy-Schwarz ou l'inegalite triangulaire dans \(\mathbb{R}\).

Exemple resolu

Montrer que \(N : \mathcal{C}^1([0,1], \mathbb{R}) \to \mathbb{R}\) definie par \(N(f) = |f(0)| + \int_0^1 |f'(t)|\,\mathrm{d}t\) est une norme.

Resolution.

  1. Separation : si \(N(f) = 0\), alors \(|f(0)| = 0\) et \(\int_0^1 |f'(t)|\,\mathrm{d}t = 0\). La premiere condition donne \(f(0) = 0\). La seconde, comme \(|f'|\) est continue et positive, implique \(f' = 0\) sur \([0,1]\). Donc \(f\) est constante egale a \(f(0) = 0\). Reciproquement, \(N(0) = 0\).
  2. Homogeneite : \(N(\lambda f) = |\lambda f(0)| + \int_0^1 |\lambda f'(t)|\,\mathrm{d}t = |\lambda|\,|f(0)| + |\lambda|\int_0^1 |f'(t)|\,\mathrm{d}t = |\lambda|\,N(f)\).
  3. Inegalite triangulaire : \(N(f+g) = |f(0)+g(0)| + \int_0^1 |f'(t)+g'(t)|\,\mathrm{d}t \leq |f(0)| + |g(0)| + \int_0^1 |f'(t)|\,\mathrm{d}t + \int_0^1 |g'(t)|\,\mathrm{d}t = N(f) + N(g)\).

Donc \(N\) est bien une norme sur \(\mathcal{C}^1([0,1], \mathbb{R})\).

M2. Equivalence de normes

Methode
  1. Deux normes \(N_1\) et \(N_2\) sont equivalentes s'il existe \(\alpha, \beta > 0\) tels que pour tout \(x \in E\) : \(\alpha \, N_1(x) \leq N_2(x) \leq \beta \, N_1(x)\).
  2. En dimension finie : toutes les normes sont equivalentes (theoreme fondamental). On peut donc choisir la norme la plus adaptee au probleme.
  3. En dimension infinie : il faut exhiber les constantes \(\alpha, \beta\) explicitement, ou montrer que l'une des inegalites est impossible (pour prouver la non-equivalence).
  4. Pour la non-equivalence : exhiber une suite \((x_n)\) telle que \(N_1(x_n) \to 0\) mais \(N_2(x_n) \not\to 0\) (ou inversement).
Exemple resolu

Montrer que \(\|\cdot\|_1\) et \(\|\cdot\|_\infty\) ne sont pas equivalentes sur \(\mathcal{C}^1([0,1], \mathbb{R})\), ou \(\|f\|_1 = \int_0^1 |f(t)|\,\mathrm{d}t\) et \(\|f\|_\infty = \sup_{t \in [0,1]} |f(t)|\).

Resolution. On a toujours \(\|f\|_1 = \int_0^1 |f(t)|\,\mathrm{d}t \leq \int_0^1 \|f\|_\infty \,\mathrm{d}t = \|f\|_\infty\), donc l'inegalite \(\|f\|_1 \leq \|f\|_\infty\) est verifiee avec \(\beta = 1\).

Pour montrer qu'il n'existe pas de \(\alpha > 0\) tel que \(\alpha \|f\|_\infty \leq \|f\|_1\) pour tout \(f\), on considere la suite \(f_n(t) = t^n\). Alors :

$$\|f_n\|_\infty = \sup_{[0,1]} t^n = 1, \quad \|f_n\|_1 = \int_0^1 t^n\,\mathrm{d}t = \frac{1}{n+1} \xrightarrow[n \to +\infty]{} 0$$

Si l'on avait \(\alpha \|f_n\|_\infty \leq \|f_n\|_1\), on aurait \(\alpha \leq \frac{1}{n+1}\) pour tout \(n\), ce qui est absurde. Donc les deux normes ne sont pas equivalentes.

M3. Ouverts et fermes dans un EVN

Methode
  1. Ouvert : montrer que pour tout \(x \in A\), il existe \(r > 0\) tel que \(B(x, r) \subset A\). Souvent, on exhibe \(r\) en fonction de \(x\).
  2. Ferme : montrer que le complementaire est ouvert, ou que \(A\) contient toutes ses valeurs d'adherence (toute suite convergente d'elements de \(A\) a sa limite dans \(A\)).
  3. Image reciproque : si \(f\) est continue, \(f^{-1}(O)\) est ouvert pour \(O\) ouvert, et \(f^{-1}(F)\) est ferme pour \(F\) ferme. C'est souvent la methode la plus elegante.
  4. Ni ouvert ni ferme : exhiber un point de \(A\) qui n'est pas interieur (pas ouvert) et une suite convergente vers un point hors de \(A\) (pas ferme).
Exemple resolu

Montrer que \(\mathcal{S}_n^{++}(\mathbb{R})\) (matrices symetriques definies positives) est un ouvert de \(\mathcal{S}_n(\mathbb{R})\), et que \(\{M \in \mathcal{M}_n(\mathbb{R}) \mid \mathrm{rg}(M) = r\}\) n'est ni ouvert ni ferme pour \(0 < r < n\).

Resolution.

Partie 1 : Soit \(A \in \mathcal{S}_n^{++}(\mathbb{R})\). Notons \(\lambda_1 \leq \cdots \leq \lambda_n\) ses valeurs propres, toutes strictement positives. Posons \(\alpha = \lambda_1 > 0\). Pour toute \(H \in \mathcal{S}_n(\mathbb{R})\) avec \(\|H\| < \alpha\) (norme d'operateur), et tout \(x \neq 0\) :

$$\langle (A+H)x, x \rangle = \langle Ax, x \rangle + \langle Hx, x \rangle \geq \alpha \|x\|^2 - \|H\| \|x\|^2 > 0$$

Donc \(A + H \in \mathcal{S}_n^{++}(\mathbb{R})\), et la boule ouverte \(B(A, \alpha)\) est incluse dans \(\mathcal{S}_n^{++}(\mathbb{R})\).

Partie 2 : Posons \(E_r = \{M \in \mathcal{M}_n(\mathbb{R}) \mid \mathrm{rg}(M) = r\}\) avec \(0 < r < n\).

  • Pas ouvert : soit \(M = \mathrm{diag}(1, \ldots, 1, 0, \ldots, 0)\) de rang \(r\). La suite \(M_k = M + \frac{1}{k}I_n\) verifie \(\mathrm{rg}(M_k) = n\) pour tout \(k\), et \(M_k \to M\). Toute boule autour de \(M\) contient des matrices de rang \(n\).
  • Pas ferme : la suite \(N_k = \mathrm{diag}(1, \ldots, 1, \frac{1}{k}, 0, \ldots, 0)\) avec les \(r-1\) premiers termes egaux a \(1\), le \(r\)-ieme egal a \(1/k\), les autres nuls, est de rang \(r\) pour tout \(k\), mais converge vers une matrice de rang \(r-1\).

M4. Continuite d'une application lineaire

Methode

Soit \(f : (E, \|\cdot\|_E) \to (F, \|\cdot\|_F)\) une application lineaire entre EVN. Les assertions suivantes sont equivalentes :

  1. \(f\) est continue sur \(E\).
  2. \(f\) est continue en \(0\).
  3. \(f\) est lipschitzienne.
  4. \(f\) est bornee sur la boule unite : \(\exists\, C \geq 0,\; \forall x \in E,\; \|f(x)\|_F \leq C \|x\|_E\).

En pratique : on majore \(\|f(x)\|_F\) par \(C \|x\|_E\). La plus petite constante est la norme d'operateur \(\|f\| = \sup_{\|x\|_E = 1} \|f(x)\|_F\).

En dimension finie : toute application lineaire est continue (consequence de l'equivalence des normes).

Exemple resolu

Montrer que la forme lineaire \(\varphi : (\mathcal{C}([0,1], \mathbb{R}), \|\cdot\|_\infty) \to \mathbb{R}\) definie par \(\varphi(f) = \int_0^1 t^2 f(t)\,\mathrm{d}t\) est continue, et calculer \(\|\varphi\|\).

Resolution. Pour tout \(f \in \mathcal{C}([0,1], \mathbb{R})\) :

$$|\varphi(f)| = \left|\int_0^1 t^2 f(t)\,\mathrm{d}t\right| \leq \int_0^1 t^2 |f(t)|\,\mathrm{d}t \leq \|f\|_\infty \int_0^1 t^2\,\mathrm{d}t = \frac{1}{3}\|f\|_\infty$$

Donc \(\varphi\) est continue et \(\|\varphi\| \leq \frac{1}{3}\).

Pour l'egalite, on prend \(f_0 = 1\) (fonction constante). Alors \(\varphi(f_0) = \int_0^1 t^2\,\mathrm{d}t = \frac{1}{3}\) et \(\|f_0\|_\infty = 1\). Donc \(\|\varphi\| \geq \frac{|\varphi(f_0)|}{\|f_0\|_\infty} = \frac{1}{3}\).

$$\boxed{\|\varphi\| = \frac{1}{3}}$$

M5. Compacite dans un EVN

Methode
  1. Definition sequentielle : \(K\) est compact si de toute suite d'elements de \(K\) on peut extraire une sous-suite convergente dans \(K\).
  2. En dimension finie : compact \(\Leftrightarrow\) ferme et borne (Borel-Lebesgue).
  3. En dimension infinie : ferme et borne n'implique PAS compact. La boule unite fermee n'est pas compacte (theoreme de Riesz).
  4. Image d'un compact : si \(f\) est continue et \(K\) compact, alors \(f(K)\) est compact. En particulier, \(f\) est bornee et atteint ses bornes sur \(K\).
  5. Theoreme de Heine : \(f\) continue sur un compact \(K\) est uniformement continue sur \(K\).
Exemple resolu

Soit \(f : K \to \mathbb{R}\) continue, ou \(K\) est une partie compacte non vide d'un EVN. Montrer que \(f\) atteint ses bornes et que \(f\) est uniformement continue sur \(K\).

Resolution.

Bornes atteintes : l'image \(f(K)\) est une partie compacte de \(\mathbb{R}\), donc fermee et bornee. Elle admet une borne superieure \(M = \sup f(K)\). Comme \(f(K)\) est ferme, \(M \in f(K)\), c'est-a-dire qu'il existe \(x_0 \in K\) tel que \(f(x_0) = M\). De meme pour la borne inferieure.

Continuite uniforme (Heine) : raisonnons par l'absurde. Supposons \(f\) non uniformement continue. Il existe \(\varepsilon_0 > 0\) et deux suites \((x_n), (y_n)\) de \(K\) telles que \(\|x_n - y_n\| < 1/n\) et \(|f(x_n) - f(y_n)| \geq \varepsilon_0\). Par compacite de \(K\), on extrait \(x_{\sigma(n)} \to \ell \in K\). Comme \(\|y_{\sigma(n)} - \ell\| \leq \|y_{\sigma(n)} - x_{\sigma(n)}\| + \|x_{\sigma(n)} - \ell\| \to 0\), on a aussi \(y_{\sigma(n)} \to \ell\). Par continuite : \(|f(x_{\sigma(n)}) - f(y_{\sigma(n)})| \to 0\), contradiction.

II. Vrai ou Faux

Pour chaque affirmation, determiner si elle est vraie ou fausse, puis cliquer pour verifier.

Testez vos connaissances 0 / 10

1. Toute partie ouverte d'un EVN est connexe.

Contre-exemple : dans \(\mathbb{R}\), l'ensemble \(]-\infty, 0[ \cup ]0, +\infty[\) est ouvert mais n'est pas connexe (reunion disjointe de deux ouverts non vides).

2. En dimension finie, toute partie fermee et bornee est compacte.

C'est le theoreme de Borel-Lebesgue : dans un EVN de dimension finie, les compacts sont exactement les fermes bornes.

3. La boule unite fermee de \((\mathcal{C}([0,1], \mathbb{R}), \|\cdot\|_\infty)\) est compacte.

\(\mathcal{C}([0,1], \mathbb{R})\) est de dimension infinie. Par le theoreme de Riesz, la boule unite fermee d'un EVN n'est compacte que si l'espace est de dimension finie. On peut exhiber la suite \(f_n(t) = t^n\) qui n'admet aucune sous-suite convergente pour \(\|\cdot\|_\infty\).

4. Si \(f : E \to F\) est lineaire et continue, alors \(\ker f\) est un ferme de \(E\).

\(\{0_F\}\) est ferme dans \(F\), et \(\ker f = f^{-1}(\{0_F\})\). L'image reciproque d'un ferme par une application continue est un ferme.

5. Deux normes equivalentes definissent les memes ouverts.

Si \(\alpha N_1 \leq N_2 \leq \beta N_1\), toute boule ouverte pour \(N_1\) contient une boule ouverte pour \(N_2\) et reciproquement. Les topologies sont identiques.

6. L'intersection d'une famille quelconque d'ouverts est un ouvert.

Seule une intersection finie d'ouverts est garantie ouverte. Contre-exemple : \(\bigcap_{n \geq 1} ]-1/n, 1/n[ = \{0\}\), qui n'est pas ouvert dans \(\mathbb{R}\).

7. Si \(A\) est dense dans \(E\) et \(f : E \to F\) est continue, alors \(f(A)\) est dense dans \(f(E)\).

Soit \(y = f(x) \in f(E)\). Comme \(A\) est dense, il existe \((a_n) \subset A\) avec \(a_n \to x\). Par continuite, \(f(a_n) \to f(x) = y\), et \(f(a_n) \in f(A)\). Donc tout element de \(f(E)\) est adherent a \(f(A)\).

8. La norme d'operateur verifie \(\|\varphi\| = \sup_{\|x\| \leq 1} |\varphi(x)|\).

On a \(\|\varphi\| = \sup_{\|x\| = 1} |\varphi(x)| = \sup_{\|x\| \leq 1} |\varphi(x)|\). L'egalite decoule de l'homogeneite : le sup sur la boule fermee est atteint sur la sphere.

9. L'adherence d'un sous-espace vectoriel d'un EVN est un sous-espace vectoriel.

Soient \(x, y \in \overline{F}\) et \(\lambda \in \mathbb{K}\). Il existe \((x_n), (y_n) \subset F\) avec \(x_n \to x\), \(y_n \to y\). Alors \(\lambda x_n + y_n \in F\) et \(\lambda x_n + y_n \to \lambda x + y\). Donc \(\lambda x + y \in \overline{F}\), et \(0 \in F \subset \overline{F}\).

10. Si \(E\) est de dimension finie et \(f : E \to E\) est lineaire injective, alors \(f\) est un homeomorphisme.

En dimension finie, lineaire et injective implique surjective (meme dimension), donc \(f\) est bijective. De plus \(f\) et \(f^{-1}\) sont lineaires en dimension finie, donc continues. Ainsi \(f\) est un homeomorphisme.

III. Exercices corriges

Monier Ch. 6 Exercice 1 -- Normes sur les polynomes

On considere \(E = \mathbb{R}_n[X]\) muni de la norme \(\|P\|_\infty = \sup_{t \in [0,1]} |P(t)|\). On definit aussi \(N(P) = \sum_{k=0}^{n} |a_k|\) ou \(P = \sum_{k=0}^{n} a_k X^k\).

  1. Montrer que \(N\) est une norme sur \(E\).
  2. Montrer que \(\|P\|_\infty \leq N(P)\) pour tout \(P \in E\).
  3. En deduire que les deux normes sont equivalentes. Pourquoi pouvait-on le prevoir sans calcul ?
Correction

1. Verifions les trois axiomes de norme pour \(N\) :

  • Separation : \(N(P) = 0 \Rightarrow |a_k| = 0\) pour tout \(k\), donc \(P = 0\). Reciproquement, \(N(0) = 0\).
  • Homogeneite : \(N(\lambda P) = \sum_k |\lambda a_k| = |\lambda| \sum_k |a_k| = |\lambda| N(P)\).
  • Inegalite triangulaire : si \(Q = \sum_k b_k X^k\), alors \(N(P+Q) = \sum_k |a_k + b_k| \leq \sum_k (|a_k| + |b_k|) = N(P) + N(Q)\).

2. Pour tout \(t \in [0,1]\) : \(|P(t)| = |\sum_k a_k t^k| \leq \sum_k |a_k| |t|^k \leq \sum_k |a_k| = N(P)\). En passant au sup : \(\|P\|_\infty \leq N(P)\).

3. On a \(\|P\|_\infty \leq N(P)\). Pour l'inegalite inverse, \(E\) est de dimension finie \(n+1\), donc toute norme est continue par rapport a toute autre norme. En particulier \(N\) est continue pour \(\|\cdot\|_\infty\). Par compacite de la sphere unite \(S = \{P : \|P\|_\infty = 1\}\), la restriction de \(N\) a \(S\) atteint son maximum \(\beta\). Donc \(N(P) \leq \beta \|P\|_\infty\) pour tout \(P\). On pouvait le prevoir car \(\dim E = n + 1 < +\infty\) et en dimension finie toutes les normes sont equivalentes.

∎

Monier Ch. 6 Exercice 2 -- Ouverts et fermes dans \(\mathcal{M}_n(\mathbb{R})\)
  1. Montrer que \(\mathrm{GL}_n(\mathbb{R})\) est un ouvert de \(\mathcal{M}_n(\mathbb{R})\).
  2. En deduire que \(\mathrm{SL}_n(\mathbb{R}) = \{M \mid \det(M) = 1\}\) est un ferme.
  3. Montrer que \(\mathcal{O}_n(\mathbb{R}) = \{M \mid M^T M = I_n\}\) est compact.
Correction

1. L'application \(\det : \mathcal{M}_n(\mathbb{R}) \to \mathbb{R}\) est polynomiale en les coefficients, donc continue. Comme \(\mathrm{GL}_n(\mathbb{R}) = \det^{-1}(\mathbb{R}^*)\) et \(\mathbb{R}^*\) est ouvert, l'image reciproque \(\mathrm{GL}_n(\mathbb{R})\) est ouverte.

2. \(\mathrm{SL}_n(\mathbb{R}) = \det^{-1}(\{1\})\). Le singleton \(\{1\}\) est ferme dans \(\mathbb{R}\), et \(\det\) est continue, donc \(\mathrm{SL}_n(\mathbb{R})\) est ferme.

3. Montrons que \(\mathcal{O}_n(\mathbb{R})\) est ferme et borne dans \(\mathcal{M}_n(\mathbb{R})\).

  • Ferme : l'application \(\Phi : M \mapsto M^T M\) est continue (polynomiale). Donc \(\mathcal{O}_n(\mathbb{R}) = \Phi^{-1}(\{I_n\})\) est un ferme.
  • Borne : si \(M \in \mathcal{O}_n(\mathbb{R})\), les colonnes de \(M\) sont de norme euclidienne \(1\), donc chaque coefficient verifie \(|m_{ij}| \leq 1\). L'ensemble est borne.

Comme \(\mathcal{M}_n(\mathbb{R}) \simeq \mathbb{R}^{n^2}\) est de dimension finie, ferme et borne implique compact.

∎

Monier Ch. 6 Exercice 3 -- Continuite de formes lineaires

On munit \(E = \mathcal{C}([0,1], \mathbb{R})\) de la norme \(\|f\|_\infty = \sup_{[0,1]} |f|\). Determiner si chaque forme lineaire est continue et, le cas echeant, calculer sa norme.

  1. \(\varphi_1(f) = f(1/2)\).
  2. \(\varphi_2(f) = \int_0^1 f(t)\,\mathrm{d}t\).
  3. \(\varphi_3(f) = \int_0^1 \sin(\pi t)\,f(t)\,\mathrm{d}t\).
Correction

1. \(|\varphi_1(f)| = |f(1/2)| \leq \|f\|_\infty\), donc \(\varphi_1\) est continue et \(\|\varphi_1\| \leq 1\). Prenant \(f = 1\) : \(\varphi_1(1) = 1\) et \(\|1\|_\infty = 1\), d'ou \(\|\varphi_1\| \geq 1\). Conclusion : \(\boxed{\|\varphi_1\| = 1}\).

2. \(|\varphi_2(f)| = |\int_0^1 f(t)\,\mathrm{d}t| \leq \int_0^1 |f(t)|\,\mathrm{d}t \leq \|f\|_\infty\). Donc \(\|\varphi_2\| \leq 1\). Prenant \(f = 1\) : \(\varphi_2(1) = 1\). Donc \(\boxed{\|\varphi_2\| = 1}\).

3. \(|\varphi_3(f)| \leq \int_0^1 |\sin(\pi t)| |f(t)|\,\mathrm{d}t \leq \|f\|_\infty \int_0^1 \sin(\pi t)\,\mathrm{d}t\). Or \(\int_0^1 \sin(\pi t)\,\mathrm{d}t = \left[-\frac{\cos(\pi t)}{\pi}\right]_0^1 = \frac{1}{\pi}(1+1) = \frac{2}{\pi}\). Donc \(\|\varphi_3\| \leq \frac{2}{\pi}\).

Prenant \(f = 1\) : \(\varphi_3(1) = \frac{2}{\pi}\) et \(\|1\|_\infty = 1\), d'ou \(\|\varphi_3\| \geq \frac{2}{\pi}\). Conclusion : \(\boxed{\|\varphi_3\| = \frac{2}{\pi}}\).

∎

Monier Ch. 7 Exercice 4 -- Compacite et existence d'extremum

Soit \(f : \mathbb{R}^n \to \mathbb{R}\) continue telle que \(\lim_{\|x\| \to +\infty} f(x) = +\infty\).

  1. Montrer qu'il existe \(R > 0\) tel que pour tout \(x\) avec \(\|x\| > R\), on a \(f(x) > f(0)\).
  2. En deduire que \(f\) admet un minimum global sur \(\mathbb{R}^n\).
  3. Application : montrer que \(g(x, y) = x^4 + y^4 - 2x^2 + 4xy - 2y^2 + 1\) admet un minimum global.
Correction

1. Par definition de la limite en \(+\infty\), pour \(M = f(0) + 1\), il existe \(R > 0\) tel que \(\|x\| > R \Rightarrow f(x) > M > f(0)\).

2. La boule fermee \(\overline{B}(0, R)\) est compacte dans \(\mathbb{R}^n\) (fermee et bornee en dimension finie). La restriction de \(f\) a \(\overline{B}(0, R)\) est continue sur un compact, donc elle atteint son minimum : il existe \(x_0 \in \overline{B}(0, R)\) tel que \(f(x_0) = \min_{\overline{B}(0,R)} f\).

Pour \(x \notin \overline{B}(0, R)\), on a \(f(x) > f(0) \geq f(x_0)\). Donc \(f(x_0) = \min_{\mathbb{R}^n} f\).

3. Montrons que \(g(x,y) \to +\infty\) quand \(r = \|(x,y)\| \to +\infty\). Le terme dominant est \(x^4 + y^4 \geq \frac{1}{2}(x^2 + y^2)^2 = \frac{r^4}{2}\). Les termes restants verifient \(|-2x^2 + 4xy - 2y^2| \leq 2x^2 + 4|xy| + 2y^2 \leq 4(x^2+y^2) = 4r^2\). Donc :

$$g(x,y) \geq \frac{r^4}{2} - 4r^2 + 1 \xrightarrow[r \to +\infty]{} +\infty$$

Par la question 2, \(g\) admet un minimum global sur \(\mathbb{R}^2\).

∎

Monier Ch. 7 Exercice 5 -- Prolongement par densite

Soient \((E, \|\cdot\|_E)\) et \((F, \|\cdot\|_F)\) deux EVN, \(F\) complet. Soit \(D\) un sous-espace vectoriel dense dans \(E\) et \(f : D \to F\) lineaire continue.

  1. Montrer que pour tout \(x \in E\), si \((d_n)\) et \((d_n')\) sont deux suites de \(D\) convergeant vers \(x\), alors \((f(d_n))\) et \((f(d_n'))\) convergent vers la meme limite.
  2. On pose \(\tilde{f}(x) = \lim f(d_n)\). Montrer que \(\tilde{f}\) est lineaire et continue.
  3. Montrer que \(\|\tilde{f}\| = \|f\|\).
Correction

1. Comme \(f\) est lineaire continue, \(\exists\, C > 0\) tel que \(\|f(u)\|_F \leq C\|u\|_E\) pour tout \(u \in D\). On a \(\|f(d_n) - f(d_n')\|_F = \|f(d_n - d_n')\|_F \leq C\|d_n - d_n'\|_E \to 0\) car \(d_n - d_n' \to 0\).

Il reste a montrer la convergence. La suite \((f(d_n))\) est de Cauchy : \(\|f(d_m) - f(d_n)\|_F \leq C\|d_m - d_n\|_E \to 0\). Comme \(F\) est complet, elle converge.

2. Linearite : soient \(x, y \in E\), \(\lambda \in \mathbb{K}\), \((d_n) \to x\) et \((e_n) \to y\) dans \(D\). Alors \(\lambda d_n + e_n \to \lambda x + y\) et :

$$\tilde{f}(\lambda x + y) = \lim f(\lambda d_n + e_n) = \lim (\lambda f(d_n) + f(e_n)) = \lambda \tilde{f}(x) + \tilde{f}(y)$$

Continuite : \(\|\tilde{f}(x)\|_F = \lim \|f(d_n)\|_F \leq \lim C\|d_n\|_E = C\|x\|_E\). Donc \(\|\tilde{f}\| \leq \|f\|\).

3. On a deja \(\|\tilde{f}\| \leq \|f\|\). Reciproquement, pour \(d \in D\) : \(\tilde{f}(d) = f(d)\) (suite constante). Donc \(\|f(d)\|_F = \|\tilde{f}(d)\|_F \leq \|\tilde{f}\| \|d\|_E\). Passant au sup sur \(\|d\| = 1\) : \(\|f\| \leq \|\tilde{f}\|\). Conclusion : \(\|\tilde{f}\| = \|f\|\).

∎

Monier Ch. 7 Exercice 6 -- Theoreme du point fixe de Banach

Soit \((E, d)\) un espace metrique complet et \(f : E \to E\) contractante : \(\exists\, k \in [0,1[\), \(\forall\, x,y \in E\), \(d(f(x), f(y)) \leq k\,d(x,y)\).

  1. Montrer que \(f\) est uniformement continue.
  2. Soit \(x_0 \in E\) et \(x_{n+1} = f(x_n)\). Montrer que \(d(x_{n+1}, x_n) \leq k^n d(x_1, x_0)\).
  3. En deduire que \((x_n)\) est de Cauchy et que \(f\) admet un unique point fixe.
  4. Application : montrer que \(x = \frac{1}{2}\cos(x)\) admet une unique solution dans \(\mathbb{R}\).
Correction

1. Soit \(\varepsilon > 0\). Posons \(\delta = \varepsilon\). Si \(d(x,y) < \delta\), alors \(d(f(x), f(y)) \leq k\,d(x,y) < k\delta \leq \varepsilon\). Donc \(f\) est lipschitzienne (rapport \(k\)), a fortiori uniformement continue.

2. Recurrence : \(d(x_2, x_1) = d(f(x_1), f(x_0)) \leq k\,d(x_1, x_0)\). Si \(d(x_{n+1}, x_n) \leq k^n d(x_1, x_0)\), alors \(d(x_{n+2}, x_{n+1}) = d(f(x_{n+1}), f(x_n)) \leq k\,d(x_{n+1}, x_n) \leq k^{n+1} d(x_1, x_0)\).

3. Pour \(m > n\), par l'inegalite triangulaire :

$$d(x_m, x_n) \leq \sum_{j=n}^{m-1} d(x_{j+1}, x_j) \leq d(x_1, x_0)\sum_{j=n}^{m-1} k^j \leq \frac{k^n}{1-k}\,d(x_1, x_0) \xrightarrow[n \to \infty]{} 0$$

Donc \((x_n)\) est de Cauchy. Comme \(E\) est complet, \(x_n \to \ell \in E\). Par continuite de \(f\) : \(\ell = \lim x_{n+1} = \lim f(x_n) = f(\ell)\).

Unicite : si \(\ell'\) est un autre point fixe, \(d(\ell, \ell') = d(f(\ell), f(\ell')) \leq k\,d(\ell, \ell')\), donc \((1-k)d(\ell, \ell') \leq 0\) et \(\ell = \ell'\).

4. Posons \(g(x) = \frac{1}{2}\cos(x)\). On a \(|g'(x)| = \frac{1}{2}|\sin(x)| \leq \frac{1}{2}\). Par le TAF : \(|g(x)-g(y)| \leq \frac{1}{2}|x-y|\). Donc \(g\) est contractante de rapport \(k = 1/2\) sur \(\mathbb{R}\) complet. De plus, \(g\) envoie \([-1/2, 1/2]\) dans lui-meme (cet intervalle est ferme, donc complet). Par le theoreme du point fixe de Banach, l'equation admet une unique solution. En iterant a partir de \(x_0 = 0\) : \(x_1 = 0{,}5\), \(x_2 \approx 0{,}439\), et la suite converge vers \(\ell \approx 0{,}450\).

∎
