Matrices & bilineaire

Chaque methode est illustree par un exemple detaille, suivie d'un Vrai ou Faux et d'exercices entierement corriges.

M1. Critere de Sylvester

Methode

Pour determiner si une matrice symetrique reelle \(A \in \mathcal{S}_n(\mathbb{R})\) est definie positive sans calculer ses valeurs propres.

  1. Extraire les mineurs principaux dominants : \(\Delta_k = \det(A_k)\) ou \(A_k\) est la sous-matrice formee des \(k\) premieres lignes et colonnes.
  2. Calculer successivement \(\Delta_1 = a_{11}\), \(\Delta_2 = \det\begin{pmatrix}a_{11}&a_{12}\\a_{21}&a_{22}\end{pmatrix}\), ..., \(\Delta_n = \det(A)\).
  3. Conclusion : \(A\) est SDP \(\iff\) \(\Delta_k > 0\) pour tout \(k \in \{1, \ldots, n\}\).
  4. Variante : \(A\) est definie negative \(\iff\) \((-1)^k \Delta_k > 0\) pour tout \(k\).
Exemple

Soit \(A(\alpha) = \begin{pmatrix} 2 & 1 & 0 \\ 1 & \alpha & 1 \\ 0 & 1 & 3 \end{pmatrix}\). Pour quelles valeurs de \(\alpha\) la matrice \(A(\alpha)\) est-elle SDP ?

  1. \(\Delta_1 = 2 > 0\) : toujours verifie.
  2. \(\Delta_2 = 2\alpha - 1\). On a \(\Delta_2 > 0 \iff \alpha > \tfrac{1}{2}\).
  3. \(\Delta_3 = \det(A(\alpha)) = 2(3\alpha - 1) - 1 \cdot 3 = 6\alpha - 5\). On a \(\Delta_3 > 0 \iff \alpha > \tfrac{5}{6}\).
  4. Conclusion : \(A(\alpha)\) est SDP si et seulement si \(\alpha > \tfrac{5}{6}\).

M2. Reduire une forme quadratique (Gauss)

Methode

Reduire la forme quadratique \(q\) en somme de carres par la methode de Gauss (completion du carre).

  1. Ecrire \(q(x_1, \ldots, x_n)\) sous forme developpee.
  2. S'il existe un terme en \(x_i^2\), regrouper tous les termes contenant \(x_i\) et completer le carre : poser \(y_i = x_i + \text{(combinaison lineaire des autres variables)}\).
  3. Remplacer et repeter avec les variables restantes.
  4. Cas sans terme carre : s'il n'y a aucun \(x_i^2\) mais un produit croise \(x_i x_j\), poser \(x_i = u + v\), \(x_j = u - v\) pour faire apparaitre \(u^2 - v^2\).
  5. La forme finale est \(q = \lambda_1 y_1^2 + \cdots + \lambda_r y_r^2\). La signature \((p,q)\) est le nombre de coefficients positifs et negatifs.
Exemple

Reduire \(q(x,y,z) = 2x^2 + xy + y^2 - 2yz + 3z^2\).

  1. Etape 1 : regrouper les termes en \(x\) : $$q = 2\!\left(x^2 + \frac{y}{2}\,x\right) + y^2 - 2yz + 3z^2 = 2\!\left(x + \frac{y}{4}\right)^{\!2} - \frac{y^2}{8} + y^2 - 2yz + 3z^2$$
  2. Poser \(u = x + \tfrac{y}{4}\). Il reste : $$q = 2u^2 + \frac{7}{8}\,y^2 - 2yz + 3z^2$$
  3. Etape 2 : completer le carre en \(y\) : $$\frac{7}{8}\!\left(y^2 - \frac{16}{7}\,yz\right) = \frac{7}{8}\!\left(y - \frac{8z}{7}\right)^{\!2} - \frac{8z^2}{7}$$
  4. Poser \(v = y - \tfrac{8z}{7}\). Il reste : $$q = 2u^2 + \frac{7}{8}\,v^2 + \left(3 - \frac{8}{7}\right)z^2 = 2u^2 + \frac{7}{8}\,v^2 + \frac{13}{7}\,z^2$$
  5. Signature \((3,0)\) : les trois coefficients sont strictement positifs, donc \(q\) est definie positive.

M3. Matrices de rang 1

Methode

Exploiter la structure des matrices de rang 1 pour calculer spectre et puissances.

  1. \(A \in \mathcal{M}_n(\mathbb{R})\) est de rang 1 \(\iff\) \(A = C \cdot L\) avec \(C\) colonne non nulle et \(L\) ligne non nulle, autrement dit \(A = u\,v^T\) avec \(u, v \in \mathbb{R}^n\).
  2. Spectre : \(\mathrm{Sp}(A) = \{0,\, \mathrm{tr}(A)\}\) avec \(\dim E_0 = n-1\) et \(\mathrm{tr}(A) = v^T u\).
  3. Diagonalisabilite : \(A\) est diagonalisable \(\iff\) \(\mathrm{tr}(A) \neq 0\).
  4. Puissances : \(A^2 = (v^T u)\,A\), donc par recurrence \(A^n = (\mathrm{tr}(A))^{n-1}\,A\) pour \(n \geq 1\).
Exemple

Soit \(u = \begin{pmatrix}1\\2\\-1\end{pmatrix}\), \(v = \begin{pmatrix}2\\1\\1\end{pmatrix}\) et \(A = uv^T\). Calculer \(A^n\).

  1. \(A = uv^T = \begin{pmatrix}2&1&1\\4&2&2\\-2&-1&-1\end{pmatrix}\).
  2. \(\mathrm{tr}(A) = 2 + 2 + (-1) = 3\). Donc \(v^T u = 3\).
  3. Par la formule : \(A^n = 3^{n-1}\,A\) pour tout \(n \geq 1\), soit : $$A^n = 3^{n-1}\begin{pmatrix}2&1&1\\4&2&2\\-2&-1&-1\end{pmatrix}$$
  4. Les espaces propres sont \(E_0 = \ker(A) = (\mathrm{Vect}(v))^\perp\) (dimension 2) et \(E_3 = \mathrm{Vect}(u)\).

M4. Racine carree d'une matrice SDP

Methode

Calculer l'unique racine carree SDP d'une matrice SDP \(S\).

  1. Diagonaliser \(S\) en base orthonormee : \(S = PDP^T\) avec \(D = \mathrm{diag}(\lambda_1, \ldots, \lambda_n)\), \(\lambda_i > 0\) et \(P\) orthogonale.
  2. Definir \(\sqrt{S} = P\,\mathrm{diag}(\sqrt{\lambda_1}, \ldots, \sqrt{\lambda_n})\,P^T\).
  3. Verifier : \((\sqrt{S})^2 = P\,D\,P^T = S\) et \(\sqrt{S}\) est bien SDP (valeurs propres \(\sqrt{\lambda_i} > 0\)).
  4. \(\sqrt{S}\) est l'unique matrice SDP verifiant \(B^2 = S\). Elle commute avec \(S\) et est symetrique.
Exemple

Calculer \(\sqrt{S}\) pour \(S = \begin{pmatrix}5&2\\2&2\end{pmatrix}\).

  1. Valeurs propres : \(\chi_S(\lambda) = \lambda^2 - 7\lambda + 6 = (\lambda - 1)(\lambda - 6)\). Donc \(\lambda_1 = 1\), \(\lambda_2 = 6\).
  2. Vecteurs propres normes : $$v_1 = \frac{1}{\sqrt{5}}\begin{pmatrix}-2\\1\end{pmatrix}, \quad v_2 = \frac{1}{\sqrt{5}}\begin{pmatrix}1\\2\end{pmatrix}$$
  3. Calcul de \(\sqrt{S}\) : $$\sqrt{S} = P\begin{pmatrix}1&0\\0&\sqrt{6}\end{pmatrix}P^T = \frac{1}{5}\begin{pmatrix}4+\sqrt{6} & -2+2\sqrt{6} \\ -2+2\sqrt{6} & 1+4\sqrt{6}\end{pmatrix}$$
  4. Verification : \((\sqrt{S})^2 = S\). On peut verifier numeriquement : \(\sqrt{6} \approx 2{,}449\), d'ou \(\sqrt{S} \approx \begin{pmatrix}1{,}290 & 0{,}580 \\ 0{,}580 & 2{,}160\end{pmatrix}\).

M5. Decomposition polaire

Methode

Ecrire une matrice inversible \(M \in \mathrm{GL}_n(\mathbb{R})\) sous la forme \(M = OS\) avec \(O\) orthogonale et \(S\) SDP.

  1. Calculer \(M^T M\) : c'est une matrice SDP (car \(X^T M^T M X = \|MX\|^2 > 0\) pour \(X \neq 0\)).
  2. Calculer \(S = \sqrt{M^T M}\) par la methode M4.
  3. En deduire \(O = M\,S^{-1}\).
  4. Verification : \(O^T O = S^{-1}M^T M\,S^{-1} = S^{-1}S^2 S^{-1} = I_n\), donc \(O \in \mathcal{O}_n(\mathbb{R})\).
  5. Unicite : si \(M = O_1 S_1 = O_2 S_2\), alors \(S_1^2 = S_2^2 = M^T M\), d'ou \(S_1 = S_2\) par unicite de la racine carree SDP.
Exemple

Determiner la decomposition polaire de \(M = \begin{pmatrix}2&1\\0&1\end{pmatrix}\).

  1. \(M^T M = \begin{pmatrix}4&2\\2&2\end{pmatrix}\).
  2. Valeurs propres de \(M^T M\) : \(\chi = \lambda^2 - 6\lambda + 4\), donc \(\lambda_{1,2} = 3 \pm \sqrt{5}\).
  3. Vecteurs propres normes : apres calcul, on pose \(P\) la matrice de passage orthogonale et : $$S = \sqrt{M^T M} = P\,\mathrm{diag}\!\left(\sqrt{3+\sqrt{5}},\, \sqrt{3-\sqrt{5}}\right)P^T$$
  4. Methode alternative : on cherche \(S = \begin{pmatrix}a&b\\b&c\end{pmatrix}\) SDP telle que \(S^2 = M^T M\). On resout le systeme : $$a^2 + b^2 = 4, \quad ab + bc = 2, \quad b^2 + c^2 = 2$$ Avec \(\mathrm{tr}(S) = a + c = \sqrt{\mathrm{tr}(M^T M) + 2\sqrt{\det(M^T M)}} = \sqrt{6 + 2\cdot 2} = \sqrt{10}\) et \(\det(S) = \sqrt{\det(M^T M)} = 2\). On obtient : $$S = \frac{1}{\sqrt{10}}\begin{pmatrix}3+\frac{1}{\sqrt{5}} & 1 \\ 1 & 1+\frac{3}{\sqrt{5}}\end{pmatrix}$$
  5. \(O = MS^{-1}\). On verifie \(\det(O) = \dfrac{\det(M)}{\det(S)} = \dfrac{2}{2} = 1\), donc \(O \in \mathrm{SO}_2(\mathbb{R})\) est une rotation.

VF. Vrai ou Faux

Pour chaque affirmation, determiner si elle est vraie ou fausse et justifier.

Testez vos connaissances 0 / 10

1. Si \(A \in \mathcal{S}_n(\mathbb{R})\) a tous ses coefficients diagonaux strictement positifs, alors \(A\) est SDP.

Contre-exemple : \(A = \begin{pmatrix}1&2\\2&1\end{pmatrix}\) a des coefficients diagonaux positifs mais \(\det(A) = -3 < 0\), donc \(A\) n'est pas SDP.

2. Toute matrice symetrique reelle admet une decomposition en forme quadratique de signature unique.

C'est la loi d'inertie de Sylvester : la signature \((p, q)\) est un invariant, independant de la base choisie pour la reduction.

3. Si \(A\) est de rang 1, alors \(A^2 = \mathrm{tr}(A) \cdot A\).

Si \(A = uv^T\), alors \(A^2 = u(v^T u)v^T = (v^Tu)\,A = \mathrm{tr}(A)\,A\).

4. Toute matrice de rang 1 est diagonalisable.

\(A\) de rang 1 est diagonalisable ssi \(\mathrm{tr}(A) \neq 0\). Contre-exemple : \(A = \begin{pmatrix}0&1\\0&0\end{pmatrix}\) est de rang 1 avec \(\mathrm{tr}(A) = 0\) et n'est pas diagonalisable (nilpotente non nulle).

5. La racine carree d'une matrice SDP est unique.

Il existe une unique matrice SDP \(B\) telle que \(B^2 = S\). Attention : il peut exister d'autres matrices (non SDP) dont le carre vaut \(S\), mais la racine carree SDP est unique.

6. Si \(A\) et \(B\) sont SDP, alors \(AB\) est SDP.

Le produit de deux matrices SDP n'est en general meme pas symetrique. \(AB\) est symetrique ssi \(A\) et \(B\) commutent. En revanche, \(AB\) a des valeurs propres reelles strictement positives.

7. La decomposition polaire \(M = OS\) est valable pour toute \(M \in \mathcal{M}_n(\mathbb{R})\).

Sous la forme \(M = OS\) avec \(S\) SDP, il faut \(M\) inversible. Pour \(M\) quelconque, on a \(M = OS\) avec \(S\) symetrique positive (non necessairement definie), mais \(O\) n'est plus unique.

8. Si \(q\) est une forme quadratique de signature \((n, 0)\) sur \(\mathbb{R}^n\), alors \(q\) definit un produit scalaire.

Signature \((n,0)\) signifie que la forme bilineaire associee \(\varphi\) verifie \(\varphi(x,x) > 0\) pour tout \(x \neq 0\), ce qui est la definition d'un produit scalaire (avec symetrie et bilinearite).

9. Le critere de Sylvester s'applique aussi aux matrices hermitiennes complexes.

Le critere de Sylvester se generalise : une matrice hermitienne \(H\) est definie positive ssi tous ses mineurs principaux dominants sont reels strictement positifs.

10. Si \(\mathrm{tr}(A) > 0\) et \(\det(A) > 0\), alors \(A \in \mathcal{S}_2(\mathbb{R})\) est SDP.

Pour \(A \in \mathcal{S}_2(\mathbb{R})\), les valeurs propres verifient \(\lambda_1 + \lambda_2 = \mathrm{tr}(A) > 0\) et \(\lambda_1 \lambda_2 = \det(A) > 0\), donc \(\lambda_1, \lambda_2 > 0\).

Ex. Exercices corriges

Exercice 1 -- Critere de Sylvester parametrique

Soit \(A = \begin{pmatrix} a & 1 & 1 \\ 1 & a & 1 \\ 1 & 1 & a \end{pmatrix}\) avec \(a \in \mathbb{R}\).

  1. Determiner les valeurs de \(a\) pour lesquelles \(A\) est SDP.
  2. Determiner la signature de la forme quadratique associee pour \(a = 0\).
Correction

1. Calculons les mineurs principaux dominants.

\(\Delta_1 = a\). On a \(\Delta_1 > 0 \iff a > 0\).

\(\Delta_2 = a^2 - 1\). On a \(\Delta_2 > 0 \iff a > 1\) (puisqu'on a deja \(a > 0\)).

\(\Delta_3 = \det(A)\). En developpant : \(\det(A) = a(a^2-1) - (a-1) + (1-a) = a^3 - 3a + 2 = (a-1)^2(a+2)\).

On a \(\Delta_3 > 0 \iff (a-1)^2(a+2) > 0\). Puisqu'on a deja \(a > 1\), cela est automatiquement verifie.

Conclusion : \(A\) est SDP \(\iff a > 1\).

2. Pour \(a = 0\) : \(A = \begin{pmatrix}0&1&1\\1&0&1\\1&1&0\end{pmatrix}\). Les valeurs propres sont \(2\) (vecteur propre \((1,1,1)^T\)) et \(-1\) (double, espace propre orthogonal a \((1,1,1)^T\)). La signature est donc \((1, 2)\).

∎

Exercice 2 -- Reduction de Gauss

Soit \(q(x,y,z) = x^2 + 4xy + 5y^2 + 2xz + 6yz + 5z^2\).

  1. Reduire \(q\) par la methode de Gauss.
  2. En deduire la signature et la nature de \(q\).
  3. Donner la matrice de passage du changement de variables.
Correction

1. On regroupe les termes en \(x\) :

$$q = (x + 2y + z)^2 - 4y^2 - z^2 - 4yz + 5y^2 + 6yz + 5z^2 = (x + 2y + z)^2 + y^2 + 2yz + 4z^2$$

Posons \(u = x + 2y + z\). Il reste \(y^2 + 2yz + 4z^2\).

On complete le carre en \(y\) : \(y^2 + 2yz = (y + z)^2 - z^2\).

Posons \(v = y + z\). On obtient :

$$q = u^2 + v^2 + 3z^2$$

2. Signature \((3, 0)\) : la forme quadratique est definie positive.

3. Le changement de variables est \(u = x + 2y + z\), \(v = y + z\), \(w = z\), soit en matrice : $$\begin{pmatrix}u\\v\\w\end{pmatrix} = \begin{pmatrix}1&2&1\\0&1&1\\0&0&1\end{pmatrix}\begin{pmatrix}x\\y\\z\end{pmatrix}$$ La matrice de passage est \(P = \begin{pmatrix}1&2&1\\0&1&1\\0&0&1\end{pmatrix}\).

∎

Exercice 3 -- Puissances d'une matrice de rang 1

Soit \(A = \begin{pmatrix}1&-1&2\\-2&2&-4\\3&-3&6\end{pmatrix}\).

  1. Montrer que \(A\) est de rang 1 et l'ecrire sous la forme \(uv^T\).
  2. Calculer \(A^n\) pour tout \(n \geq 1\).
  3. En deduire \(e^{tA}\) pour \(t \in \mathbb{R}\).
Correction

1. Les lignes 2 et 3 sont proportionnelles a la ligne 1 (facteurs \(-2\) et \(3\)). Donc \(\mathrm{rg}(A) = 1\).

On ecrit \(A = uv^T\) avec \(u = \begin{pmatrix}1\\-2\\3\end{pmatrix}\) et \(v = \begin{pmatrix}1\\-1\\2\end{pmatrix}\).

2. \(\mathrm{tr}(A) = 1 + 2 + 6 = 9\). On verifie : \(v^T u = 1 \cdot 1 + (-1)(-2) + 2 \cdot 3 = 9\). Par la formule des matrices de rang 1 :

$$A^n = 9^{n-1}\,A = 9^{n-1}\begin{pmatrix}1&-1&2\\-2&2&-4\\3&-3&6\end{pmatrix}$$

3. L'exponentielle de matrice : $$e^{tA} = I + \sum_{n=1}^{+\infty}\frac{t^n}{n!}\,A^n = I + \sum_{n=1}^{+\infty}\frac{t^n \cdot 9^{n-1}}{n!}\,A = I + \frac{1}{9}\left(\sum_{n=1}^{+\infty}\frac{(9t)^n}{n!}\right)A = I + \frac{e^{9t}-1}{9}\,A$$

∎

Exercice 4 -- Racine carree et inegalite

Soient \(A, B \in \mathcal{S}_n^{++}(\mathbb{R})\) (matrices SDP).

  1. Montrer que \(A + B\) est SDP et calculer \(\sqrt{A + B}\) pour \(A = \begin{pmatrix}2&1\\1&1\end{pmatrix}\) et \(B = \begin{pmatrix}1&0\\0&2\end{pmatrix}\).
  2. Montrer que \(\det(\sqrt{A} + \sqrt{B})^2 \geq \det(A + B)\).
Correction

1. \(A + B\) est SDP car pour \(X \neq 0\), \(X^T(A+B)X = X^TAX + X^TBX > 0\).

Calcul : \(A + B = \begin{pmatrix}3&1\\1&3\end{pmatrix}\). Valeurs propres : \(\lambda_1 = 2\), \(\lambda_2 = 4\).

Vecteurs propres normes : \(v_1 = \frac{1}{\sqrt{2}}\begin{pmatrix}1\\-1\end{pmatrix}\), \(v_2 = \frac{1}{\sqrt{2}}\begin{pmatrix}1\\1\end{pmatrix}\).

$$\sqrt{A+B} = P\begin{pmatrix}\sqrt{2}&0\\0&2\end{pmatrix}P^T = \frac{1}{2}\begin{pmatrix}\sqrt{2}+2 & -\sqrt{2}+2 \\ -\sqrt{2}+2 & \sqrt{2}+2\end{pmatrix}$$

2. Notons \(C = \sqrt{A}\) et \(D = \sqrt{B}\). On veut montrer \(\det(C+D)^2 \geq \det(C^2 + D^2)\).

Par l'inegalite arithmetico-geometrique matricielle, pour \(C, D\) SDP, on a \(\det(C+D) \geq \det(C) + \det(D)\) (inegalite de Minkowski pour les determinants). Alors :

$$\det(C+D)^2 \geq (\det(C) + \det(D))^2 \geq \det(C)^2 + \det(D)^2 \geq \det(C^2 + D^2)$$

La derniere inegalite vient de \(\det(C^2+D^2) \leq \det(C^2) + \det(D^2)\) pour des matrices SDP de taille 2 (par concavite de \(X \mapsto \det(X)^{1/n}\) sur \(\mathcal{S}_n^+\)).

∎

Exercice 5 -- Decomposition polaire explicite

Soit \(M = \begin{pmatrix}1&1\\-1&1\end{pmatrix}\).

  1. Calculer \(M^T M\) et ses valeurs propres.
  2. Determiner \(S = \sqrt{M^T M}\).
  3. En deduire \(O = MS^{-1}\) et verifier que \(O \in \mathrm{SO}_2(\mathbb{R})\).
  4. Identifier geometriquement \(O\) et \(S\).
Correction

1. \(M^T M = \begin{pmatrix}1&-1\\1&1\end{pmatrix}\begin{pmatrix}1&1\\-1&1\end{pmatrix} = \begin{pmatrix}2&0\\0&2\end{pmatrix} = 2I_2\).

Valeurs propres : \(\lambda_1 = \lambda_2 = 2\).

2. \(S = \sqrt{2I_2} = \sqrt{2}\,I_2\).

3. \(O = MS^{-1} = \frac{1}{\sqrt{2}}M = \frac{1}{\sqrt{2}}\begin{pmatrix}1&1\\-1&1\end{pmatrix} = \begin{pmatrix}\cos(-\pi/4)&-\sin(-\pi/4)\\\sin(-\pi/4)&\cos(-\pi/4)\end{pmatrix}\).

Verification : \(O^T O = \frac{1}{2}M^T M = I_2\) et \(\det(O) = \frac{1}{2}(1+1) = 1\). Donc \(O \in \mathrm{SO}_2(\mathbb{R})\).

4. Geometriquement : \(S = \sqrt{2}\,I_2\) est une homothetie de rapport \(\sqrt{2}\) et \(O\) est la rotation d'angle \(-\pi/4\). Donc \(M\) est la composee d'une homothetie et d'une rotation, ce qui correspond a la multiplication par le complexe \(1 - i = \sqrt{2}\,e^{-i\pi/4}\).

∎

Exercice 6 -- Forme quadratique et extrema

Soit \(q : \mathbb{R}^3 \to \mathbb{R}\) definie par \(q(x,y,z) = x^2 + 2y^2 + 3z^2 + 2xy - 2xz\).

  1. Ecrire la matrice \(A\) de \(q\) dans la base canonique.
  2. Appliquer le critere de Sylvester pour determiner si \(q\) est definie positive.
  3. Reduire \(q\) par Gauss et verifier la coherence avec la question 2.
  4. Application : montrer que \(f(x,y,z) = q(x,y,z) - 2x + 4y - 6z\) admet un minimum global et le calculer.
Correction

1. La matrice de \(q\) est : $$A = \begin{pmatrix}1&1&-1\\1&2&0\\-1&0&3\end{pmatrix}$$ (les coefficients hors diagonale sont les demi-coefficients des produits croises).

2. Mineurs principaux dominants :

\(\Delta_1 = 1 > 0\).

\(\Delta_2 = 1 \cdot 2 - 1 = 1 > 0\).

\(\Delta_3 = \det(A) = 1(6-0) - 1(3-0) + (-1)(0+2) = 6 - 3 - 2 = 1 > 0\).

Par le critere de Sylvester, \(A\) est SDP, donc \(q\) est definie positive.

3. Reduction de Gauss :

\(q = (x + y - z)^2 - y^2 + 2yz - z^2 + 2y^2 + 3z^2 = (x+y-z)^2 + y^2 + 2yz + 2z^2\).

\(= (x+y-z)^2 + (y+z)^2 + z^2\).

Avec \(u = x+y-z\), \(v = y+z\), \(w = z\) : \(q = u^2 + v^2 + w^2\). Signature \((3,0)\), coherent avec SDP.

4. Comme \(q\) est definie positive, \(f(X) = q(X) + L(X)\) (avec \(L\) lineaire) admet un minimum global au point \(X_0\) tel que \(\nabla f(X_0) = 0\), i.e. \(2AX_0 + \begin{pmatrix}-2\\4\\-6\end{pmatrix} = 0\), soit \(X_0 = -\frac{1}{2}A^{-1}\begin{pmatrix}-2\\4\\-6\end{pmatrix}\).

On calcule \(A^{-1}\) (par la comatrice ou par Gauss) : $$A^{-1} = \begin{pmatrix}6&-3&2\\-3&2&-1\\2&-1&1\end{pmatrix}$$

Donc \(X_0 = -\frac{1}{2}\begin{pmatrix}6&-3&2\\-3&2&-1\\2&-1&1\end{pmatrix}\begin{pmatrix}-2\\4\\-6\end{pmatrix} = -\frac{1}{2}\begin{pmatrix}-36\\20\\-14\end{pmatrix} = \begin{pmatrix}18\\-10\\7\end{pmatrix}\).

Le minimum vaut \(f(X_0) = q(X_0) - 2(18) + 4(-10) - 6(7)\). En utilisant \(f(X_0) = -\frac{1}{4}L^T A^{-1}L\) ou \(L = (-2, 4, -6)^T\) :

$$f(X_0) = -\frac{1}{4}(-2, 4, -6)\begin{pmatrix}6&-3&2\\-3&2&-1\\2&-1&1\end{pmatrix}\begin{pmatrix}-2\\4\\-6\end{pmatrix} = -\frac{1}{4}(-2,4,-6)\begin{pmatrix}-36\\20\\-14\end{pmatrix}$$
$$= -\frac{1}{4}(72 + 80 + 84) = -\frac{236}{4} = -59$$

Le minimum global de \(f\) est \(\boxed{-59}\).

∎
