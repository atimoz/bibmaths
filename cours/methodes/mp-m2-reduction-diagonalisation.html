I. Methodes a retenir

M1. Diagonaliser une matrice

Methode
  1. Calculer le polynome caracteristique \(\chi_A = \det(A - XI)\) et trouver ses racines \(\lambda_1, \ldots, \lambda_p\) avec multiplicites \(m_1, \ldots, m_p\).
  2. Verifier que \(\chi_A\) est scinde (sur \(\mathbb{R}\) ou \(\mathbb{C}\)). Sinon, pas de diagonalisation sur \(\mathbb{R}\).
  3. Pour chaque VP \(\lambda_i\), calculer une base de \(E_{\lambda_i} = \ker(A - \lambda_i I)\) par echelonnement.
  4. Verifier que \(\dim E_{\lambda_i} = m_i\) pour tout \(i\). Si oui, \(A\) est diagonalisable.
  5. Former \(P\) dont les colonnes sont les vecteurs propres (bases des \(E_{\lambda_i}\) concatenees) et \(D = \mathrm{diag}(\lambda_1, \ldots, \lambda_1, \lambda_2, \ldots)\). On a \(A = PDP^{-1}\).
Exemple resolu

Diagonaliser \(A = \begin{pmatrix} 1 & 2 & 0 \\ 0 & 3 & 0 \\ 2 & -4 & 5 \end{pmatrix}\).

Resolution. Calculons \(\chi_A = \det(A - XI)\). En developpant selon la deuxieme colonne (ou la troisieme ligne) :

$$\chi_A = (1-X)(3-X)(5-X) - 0 + 0 = -(X-1)(X-3)(X-5)$$

(car la matrice se developpe facilement : le coefficient \((2,3)\) est nul). Les VP sont \(\lambda_1 = 1\), \(\lambda_2 = 3\), \(\lambda_3 = 5\), toutes simples.

\(E_1\) : \((A - I)X = 0\) avec \(A - I = \begin{pmatrix}0&2&0\\0&2&0\\2&-4&4\end{pmatrix}\). On obtient \(x_2 = 0\), \(x_1 = -2x_3\). Donc \(E_1 = \operatorname{Vect}\begin{pmatrix}-2\\0\\1\end{pmatrix}\).

\(E_3\) : \(A - 3I = \begin{pmatrix}-2&2&0\\0&0&0\\2&-4&2\end{pmatrix}\). On obtient \(x_1 = x_2\) et \(x_3 = x_2\). Donc \(E_3 = \operatorname{Vect}\begin{pmatrix}1\\1\\1\end{pmatrix}\).

\(E_5\) : \(A - 5I = \begin{pmatrix}-4&2&0\\0&-2&0\\2&-4&0\end{pmatrix}\). On obtient \(x_2 = 0\), \(x_1 = 0\), \(x_3\) libre. Donc \(E_5 = \operatorname{Vect}\begin{pmatrix}0\\0\\1\end{pmatrix}\).

Les trois VP sont distinctes, donc \(A\) est diagonalisable : \(A = PDP^{-1}\) avec

$$P = \begin{pmatrix}-2&1&0\\0&1&0\\1&1&1\end{pmatrix}, \quad D = \begin{pmatrix}1&0&0\\0&3&0\\0&0&5\end{pmatrix}$$

M2. Calculer \(A^n\) par diagonalisation

Methode
  1. Cas diagonalisable : si \(A = PDP^{-1}\), alors \(A^n = PD^nP^{-1}\) avec \(D^n = \mathrm{diag}(\lambda_1^n, \ldots, \lambda_p^n)\).
  2. Cas non diagonalisable (Dunford) : ecrire \(A = D + N\) avec \(DN = ND\), \(D\) diag., \(N\) nilpotente d'indice \(r\). Alors \(A^n = \sum_{k=0}^{r-1} \binom{n}{k} D^{n-k} N^k\).
  3. Application aux suites recurrentes : ecrire \(U_{n+1} = AU_n\), diagonaliser \(A\), puis \(U_n = PD^nP^{-1}U_0\).
  4. Alternative par interpolation de Lagrange : si les VP sont distinctes, \(A^n = \sum_i \lambda_i^n \prod_{j \neq i} \frac{A - \lambda_j I}{\lambda_i - \lambda_j}\).
Exemple resolu

Soit la suite definie par \(u_{n+1} = 2u_n + v_n\) et \(v_{n+1} = u_n + 2v_n\), avec \(u_0 = 1\), \(v_0 = 0\). Trouver \(u_n\) et \(v_n\).

Resolution. On pose \(U_n = \begin{pmatrix}u_n\\v_n\end{pmatrix}\) et \(A = \begin{pmatrix}2&1\\1&2\end{pmatrix}\). Alors \(U_{n+1} = AU_n\), donc \(U_n = A^n U_0\).

Diagonalisation : \(\chi_A = (X-2)^2 - 1 = X^2 - 4X + 3 = (X-1)(X-3)\). VP : \(\lambda_1 = 1\), \(\lambda_2 = 3\).

  • \(E_1\) : \(A - I = \begin{pmatrix}1&1\\1&1\end{pmatrix}\), base \(\begin{pmatrix}1\\-1\end{pmatrix}\).
  • \(E_3\) : \(A - 3I = \begin{pmatrix}-1&1\\1&-1\end{pmatrix}\), base \(\begin{pmatrix}1\\1\end{pmatrix}\).

Donc \(P = \begin{pmatrix}1&1\\-1&1\end{pmatrix}\), \(P^{-1} = \frac{1}{2}\begin{pmatrix}1&-1\\1&1\end{pmatrix}\).

$$A^n = P \begin{pmatrix}1&0\\0&3^n\end{pmatrix} P^{-1} = \frac{1}{2}\begin{pmatrix}1+3^n & 3^n-1 \\ 3^n-1 & 1+3^n\end{pmatrix}$$

Finalement :

$$u_n = \frac{1 + 3^n}{2}, \quad v_n = \frac{3^n - 1}{2}$$

M3. Exponentielle de matrice \(\exp(tA)\)

Methode
  1. Cas diagonalisable : si \(A = PDP^{-1}\), alors \(\exp(tA) = P \cdot \mathrm{diag}(e^{t\lambda_1}, \ldots, e^{t\lambda_p}) \cdot P^{-1}\).
  2. Cas non diagonalisable (Dunford) : ecrire \(A = D + N\) avec \(DN = ND\). Alors \(\exp(tA) = e^{tD} \cdot \exp(tN)\) avec \(\exp(tN) = \sum_{k=0}^{r-1} \frac{t^k}{k!} N^k\).
  3. Methode directe par la serie : si \(A^2\) se simplifie (par ex. \(A^2 = -I\) ou \(A^2 = A\)), utiliser les formules de la serie exponentielle.
  4. Application : la solution de \(X'(t) = AX(t)\) avec \(X(0) = X_0\) est \(X(t) = \exp(tA) X_0\).
Exemple resolu

Calculer \(\exp(tA)\) pour \(A = \begin{pmatrix} 0 & -1 \\ 1 & 0 \end{pmatrix}\).

Resolution. On remarque que \(A^2 = \begin{pmatrix}-1&0\\0&-1\end{pmatrix} = -I_2\).

Donc \(A^{2k} = (-1)^k I\) et \(A^{2k+1} = (-1)^k A\). La serie exponentielle donne :

$$\exp(tA) = \sum_{k=0}^{\infty} \frac{t^{2k}}{(2k)!}(-1)^k I + \sum_{k=0}^{\infty} \frac{t^{2k+1}}{(2k+1)!}(-1)^k A$$
$$= \cos(t) I + \sin(t) A = \begin{pmatrix}\cos t & -\sin t \\ \sin t & \cos t\end{pmatrix}$$

C'est la matrice de rotation d'angle \(t\). Le systeme differentiel \(X' = AX\), soit \(x' = -y\), \(y' = x\), a pour solution \(x(t) = x_0 \cos t - y_0 \sin t\), \(y(t) = x_0 \sin t + y_0 \cos t\) : trajectoires circulaires.

M4. Trigonalisation et decomposition de Dunford

Methode
  1. Verifier que \(\chi_A\) est scinde. Sinon, travailler sur \(\mathbb{C}\).
  2. Pour chaque VP \(\lambda\) de multiplicite \(\alpha\), calculer les sous-espaces caracteristiques \(F_\lambda^{(k)} = \ker(A - \lambda I)^k\) pour \(k = 1, 2, \ldots\) jusqu'a stabilisation a \(\dim = \alpha\).
  3. Construire une base adaptee : partir de \(F_\lambda^{(1)} \subset F_\lambda^{(2)} \subset \cdots\) et completer a chaque etape.
  4. \(A = PTP^{-1}\) avec \(T\) triangulaire superieure.
  5. Pour \(A^n\) : ecrire \(A = D + N\) avec \(D = \lambda I\) et \(N = A - \lambda I\) nilpotente (dans chaque bloc). Alors \(A^n = \sum_{k=0}^{r-1} \binom{n}{k} \lambda^{n-k} N^k\).
Exemple resolu

Soit \(A = \begin{pmatrix} 2 & 1 \\ 0 & 2 \end{pmatrix}\). Calculer \(A^n\).

Resolution. On a \(\chi_A = (X - 2)^2\), donc unique VP \(\lambda = 2\) de multiplicite \(2\). Mais \(\dim E_2 = \dim \ker(A - 2I) = \dim \ker \begin{pmatrix}0&1\\0&0\end{pmatrix} = 1 \neq 2\), donc \(A\) n'est pas diagonalisable.

Decomposition \(A = D + N\) : posons \(D = 2I = \begin{pmatrix}2&0\\0&2\end{pmatrix}\) et \(N = A - 2I = \begin{pmatrix}0&1\\0&0\end{pmatrix}\).

On a \(DN = ND\) (car \(D = 2I\) commute avec tout), et \(N^2 = 0\) (nilpotente d'indice \(2\)). Par la formule du binome :

$$A^n = (D + N)^n = \sum_{k=0}^{1} \binom{n}{k} D^{n-k} N^k = D^n + n D^{n-1} N$$
$$= 2^n I + n \cdot 2^{n-1} N = \begin{pmatrix}2^n & n \cdot 2^{n-1} \\ 0 & 2^n\end{pmatrix}$$

M5. Polynome minimal

Methode
  1. \(\pi_A\) est le polynome unitaire de plus petit degre annulant \(A\). On a toujours \(\pi_A \mid \chi_A\) et \(\pi_A\) a les memes racines que \(\chi_A\).
  2. Si \(\chi_A = \prod (X - \lambda_i)^{m_i}\), alors \(\pi_A = \prod (X - \lambda_i)^{r_i}\) avec \(1 \leqslant r_i \leqslant m_i\).
  3. Pour trouver \(\pi_A\) : tester en partant des exposants minimaux \(r_i = 1\). Calculer \(\prod (A - \lambda_i I)\). Si le resultat est \(0\), c'est \(\pi_A\). Sinon, augmenter les exposants.
  4. Critere : \(A\) diagonalisable \(\Leftrightarrow\) \(\pi_A\) scinde a racines simples.
  5. Pour les matrices nilpotentes : \(\pi_A = X^r\) ou \(r\) est l'indice de nilpotence (plus petit entier tel que \(A^r = 0\)).
Exemple resolu

Soit \(A \in \mathcal{M}_4(\mathbb{R})\) nilpotente avec \(A^2 \neq 0\) et \(A^3 = 0\). Determiner \(\pi_A\) et \(\chi_A\).

Resolution. L'indice de nilpotence est \(r = 3\) (car \(A^2 \neq 0\) et \(A^3 = 0\)).

Donc \(\pi_A = X^3\) (polynome minimal).

Comme \(A\) est nilpotente, sa seule VP est \(0\), et \(\chi_A = X^4\) (degre = taille de la matrice).

On verifie bien que \(\pi_A \mid \chi_A\) : \(X^3 \mid X^4\). De plus, \(A\) n'est pas diagonalisable car \(\pi_A = X^3\) n'est pas a racines simples (la racine \(0\) est de multiplicite \(3 > 1\)).

La forme de Jordan de \(A\) comporte au moins un bloc de taille \(\geq 3\) (car l'indice de nilpotence est \(3\)). Comme \(A\) est \(4 \times 4\), les possibilites sont un bloc \(J_3(0)\) et un bloc \(J_1(0)\), soit \(J_4(0)\) n'est pas possible car \(A^3 = 0\) mais un bloc \(J_4\) donnerait \(A^3 \neq 0\)... En fait \(J_4(0)^3 \neq 0\), donc la seule possibilite est un bloc \(J_3(0)\) et un bloc \(J_1(0)\).

II. Vrai ou Faux

Testez vos connaissances 0 / 10

1. Toute matrice symetrique reelle est diagonalisable.

C'est le theoreme spectral : toute matrice symetrique reelle est diagonalisable dans une base orthonormee. Plus precisement, \(A = A^T\) implique que \(A\) est diagonalisable dans \(\mathcal{M}_n(\mathbb{R})\) avec des VP reelles, et on peut choisir \(P\) orthogonale (\(P^{-1} = P^T\)).

2. Si \(A\) est diagonalisable et \(B\) est diagonalisable, alors \(A + B\) est diagonalisable.

Contre-exemple : \(A = \begin{pmatrix}1&0\\0&0\end{pmatrix}\) et \(B = \begin{pmatrix}0&1\\0&0\end{pmatrix}\). \(A\) est diagonale (diagonalisable), \(B\) a pour seule VP \(0\) avec \(\dim E_0 = 1 = m(0) = 1\)... en fait \(B\) n'est pas diagonalisable. Prenons plutot \(A = \begin{pmatrix}1&1\\0&0\end{pmatrix}\) et \(B = \begin{pmatrix}-1&0\\0&0\end{pmatrix}\) (diag.). Alors \(A\) a VP \(0, 1\) (diag.) et \(A + B = \begin{pmatrix}0&1\\0&0\end{pmatrix}\) qui est nilpotente non nulle, donc non diagonalisable.

3. Si \(A\) est diagonalisable, alors \(A^T\) est diagonalisable.

\(\chi_{A^T} = \chi_A\) et \(\pi_{A^T} = \pi_A\) (meme polynome minimal). Si \(A\) est diag., \(\pi_A\) est scinde a racines simples, donc \(\pi_{A^T}\) aussi, et \(A^T\) est diagonalisable. Alternativement, \(A = PDP^{-1}\) implique \(A^T = (P^{-1})^T D P^T\).

4. Toute matrice est trigonalisable sur \(\mathbb{C}\).

Sur \(\mathbb{C}\), tout polynome est scinde (theoreme de d'Alembert-Gauss). Or, une matrice est trigonalisable sur \(\mathbb{K}\) si et seulement si \(\chi_A\) est scinde sur \(\mathbb{K}\). Donc toute matrice est trigonalisable sur \(\mathbb{C}\).

5. Si \(A^2 = I\), alors \(A\) est diagonalisable.

Le polynome \(P = X^2 - 1 = (X-1)(X+1)\) annule \(A\). Il est scinde a racines simples, donc \(A\) est diagonalisable. Les VP de \(A\) sont dans \(\{-1, 1\}\). Les matrices verifiant \(A^2 = I\) sont appelees involutions.

6. La decomposition de Dunford \(A = D + N\) est unique.

Si \(\chi_A\) est scinde, il existe une unique decomposition \(A = D + N\) avec \(D\) diagonalisable, \(N\) nilpotente, \(DN = ND\). De plus, \(D\) et \(N\) sont des polynomes en \(A\). L'unicite vient du fait que \(D\) est la partie semi-simple de \(A\) (ses valeurs propres sur chaque sous-espace caracteristique).

7. Si \(A\) et \(B\) commutent et sont diagonalisables, elles sont codiagonalisables.

Si \(AB = BA\) et \(A, B\) sont diagonalisables, alors il existe une base commune de vecteurs propres pour \(A\) et \(B\). En effet, les sous-espaces propres de \(A\) sont stables par \(B\) (car si \(Ax = \lambda x\), alors \(A(Bx) = BAx = \lambda(Bx)\)). On diagonalise \(B\) dans chaque sous-espace propre de \(A\).

8. Le polynome minimal d'une matrice diagonale est le produit des \((X - \lambda_i)\) pour les \(\lambda_i\) distincts sur la diagonale.

Si \(D = \mathrm{diag}(\lambda_1, \ldots, \lambda_n)\) avec \(d\) valeurs distinctes \(\mu_1, \ldots, \mu_d\), alors \(\pi_D = \prod_{j=1}^d (X - \mu_j)\). En effet, ce polynome annule \(D\) (chaque coefficient diagonal est une racine) et c'est le polynome de degre minimal ayant cette propriete.

9. \(\exp(A + B) = \exp(A)\exp(B)\) pour toutes matrices \(A, B\).

L'egalite \(\exp(A+B) = \exp(A)\exp(B)\) n'est vraie que si \(AB = BA\). En general, pour des matrices qui ne commutent pas, la formule de Baker-Campbell-Hausdorff montre que \(\exp(A)\exp(B) = \exp(A + B + \frac{1}{2}[A,B] + \cdots)\). Contre-exemple classique : prendre \(A, B\) nilpotentes non commutantes.

10. Si \(\chi_A = (X - 1)^3(X - 2)^2\) et \(\dim E_1 = 3\), \(\dim E_2 = 2\), alors \(A\) est diagonalisable.

Pour chaque VP, la dimension geometrique egale la multiplicite algebrique : \(\dim E_1 = 3 = m(1)\) et \(\dim E_2 = 2 = m(2)\). La somme des dimensions vaut \(3 + 2 = 5 = n\). Les sous-espaces propres sont en somme directe et engendrent \(\mathbb{R}^5\). Donc \(A\) est diagonalisable.

III. Exercices corriges

CCINP Exercice 1 -- Diagonalisation complete

Soit \(A = \begin{pmatrix} 4 & -2 & 2 \\ -2 & 4 & -2 \\ 2 & -2 & 4 \end{pmatrix}\).

  1. Montrer que \(A\) est symetrique reelle. Qu'en deduit-on ?
  2. Calculer \(\chi_A\) et determiner les VP.
  3. Determiner une base orthonormee de vecteurs propres.
  4. Ecrire la decomposition \(A = PDP^T\) avec \(P\) orthogonale.
Correction

1. \(A^T = A\) (verification directe). Par le theoreme spectral, \(A\) est diagonalisable dans une base orthonormee, avec des VP reelles.

2. \(\operatorname{tr}(A) = 12\), \(\det(A) = 4(16-4)+2(-8+4)+2(4-8) = 48 - 8 - 8 = 32\). On calcule \(s_2 = (16-4) + (16-4) + (16-4) = 36\).

$$\chi_A = -X^3 + 12X^2 - 36X + 32 = -(X-2)(X^2 - 10X + 16) = -(X-2)^2(X-8)$$

VP : \(\lambda_1 = 2\) (mult. \(2\)), \(\lambda_2 = 8\) (mult. \(1\)).

3. \(E_2\) : \(A - 2I = \begin{pmatrix}2&-2&2\\-2&2&-2\\2&-2&2\end{pmatrix}\). Rang \(1\), donc \(\dim E_2 = 2\). Base : \(v_1 = \begin{pmatrix}1\\1\\0\end{pmatrix}\), \(v_2 = \begin{pmatrix}-1\\0\\1\end{pmatrix}\). On orthonormalise par Gram-Schmidt :

\(e_1 = \frac{1}{\sqrt{2}}\begin{pmatrix}1\\1\\0\end{pmatrix}\). Puis \(w_2 = v_2 - \langle v_2, e_1\rangle e_1 = \begin{pmatrix}-1\\0\\1\end{pmatrix} + \frac{1}{2}\begin{pmatrix}1\\1\\0\end{pmatrix} = \begin{pmatrix}-1/2\\1/2\\1\end{pmatrix}\). Normalisation : \(e_2 = \frac{1}{\sqrt{3/2}} \begin{pmatrix}-1/2\\1/2\\1\end{pmatrix} = \frac{1}{\sqrt{6}}\begin{pmatrix}-1\\1\\2\end{pmatrix}\).

\(E_8\) : \(A - 8I = \begin{pmatrix}-4&-2&2\\-2&-4&-2\\2&-2&-4\end{pmatrix}\). On trouve \(E_8 = \operatorname{Vect}\begin{pmatrix}1\\-1\\1\end{pmatrix}\), d'ou \(e_3 = \frac{1}{\sqrt{3}}\begin{pmatrix}1\\-1\\1\end{pmatrix}\).

4. \(P = (e_1 \mid e_2 \mid e_3)\) est orthogonale et \(A = P \begin{pmatrix}2&0&0\\0&2&0\\0&0&8\end{pmatrix} P^T\).

∎

CCINP Exercice 2 -- Suite recurrente lineaire

On considere la suite definie par \(u_0 = 1\), \(u_1 = 2\), et \(u_{n+2} = 4u_{n+1} - 3u_n\) pour \(n \geq 0\).

  1. Ecrire le systeme sous forme matricielle \(V_{n+1} = AV_n\) avec \(V_n = \begin{pmatrix}u_{n+1}\\u_n\end{pmatrix}\).
  2. Diagonaliser \(A\) et calculer \(A^n\).
  3. En deduire \(u_n\) en fonction de \(n\).
  4. Etudier la convergence de \(u_n / 3^n\).
Correction

1. \(V_{n+1} = \begin{pmatrix}u_{n+2}\\u_{n+1}\end{pmatrix} = \begin{pmatrix}4&-3\\1&0\end{pmatrix}\begin{pmatrix}u_{n+1}\\u_n\end{pmatrix} = AV_n\).

2. \(\chi_A = X^2 - 4X + 3 = (X-1)(X-3)\). VP : \(\lambda_1 = 1\), \(\lambda_2 = 3\).

\(E_1\) : \(A - I = \begin{pmatrix}3&-3\\1&-1\end{pmatrix}\), base \(\begin{pmatrix}1\\1\end{pmatrix}\). \(E_3\) : \(A - 3I = \begin{pmatrix}1&-3\\1&-3\end{pmatrix}\), base \(\begin{pmatrix}3\\1\end{pmatrix}\).

\(P = \begin{pmatrix}1&3\\1&1\end{pmatrix}\), \(P^{-1} = \frac{1}{-2}\begin{pmatrix}1&-3\\-1&1\end{pmatrix} = \frac{1}{2}\begin{pmatrix}-1&3\\1&-1\end{pmatrix}\).

$$A^n = P\begin{pmatrix}1&0\\0&3^n\end{pmatrix}P^{-1} = \frac{1}{2}\begin{pmatrix}3\cdot 3^n - 1 & 3 - 3\cdot 3^n \\ 3^n - 1 & 3 - 3^n\end{pmatrix}$$

Simplifions : \(A^n = \frac{1}{2}\begin{pmatrix}3^{n+1}-1 & 3-3^{n+1}\\3^n-1&3-3^n\end{pmatrix}\).

3. \(V_n = A^n V_0\) avec \(V_0 = \begin{pmatrix}2\\1\end{pmatrix}\). On extrait la deuxieme composante :

$$u_n = \frac{1}{2}[(3^n - 1) \cdot 2 + (3 - 3^n) \cdot 1] = \frac{1}{2}[2 \cdot 3^n - 2 + 3 - 3^n] = \frac{3^n + 1}{2}$$

4. \(u_n / 3^n = \frac{1}{2}(1 + 3^{-n}) \to \frac{1}{2}\) quand \(n \to +\infty\).

∎

CCINP Exercice 3 -- Exponentielle et systeme differentiel

Soit \(A = \begin{pmatrix} 1 & 1 \\ -1 & 3 \end{pmatrix}\).

  1. Calculer \(\chi_A\) et montrer que \(A\) n'est pas diagonalisable.
  2. Effectuer la decomposition de Dunford \(A = D + N\).
  3. Calculer \(\exp(tA)\).
  4. Resoudre le systeme \(\begin{cases}x' = x + y \\ y' = -x + 3y\end{cases}\) avec \(x(0) = 1, y(0) = 0\).
Correction

1. \(\chi_A = X^2 - 4X + 4 = (X-2)^2\). Unique VP \(\lambda = 2\), mult. \(2\). \(\ker(A - 2I) = \ker\begin{pmatrix}-1&1\\-1&1\end{pmatrix}\), de dimension \(1\). Comme \(\dim E_2 = 1 < 2\), \(A\) n'est pas diagonalisable.

2. \(D = 2I = \begin{pmatrix}2&0\\0&2\end{pmatrix}\), \(N = A - 2I = \begin{pmatrix}-1&1\\-1&1\end{pmatrix}\). On verifie : \(N^2 = \begin{pmatrix}0&0\\0&0\end{pmatrix}\), \(DN = ND\) (evident car \(D = 2I\)).

3. \(\exp(tA) = \exp(tD) \exp(tN) = e^{2t} (I + tN)\) car \(N^2 = 0\).

$$\exp(tA) = e^{2t}\begin{pmatrix}1-t & t \\ -t & 1+t\end{pmatrix}$$

4. \(X(t) = \exp(tA) X(0) = e^{2t}\begin{pmatrix}1-t & t \\ -t & 1+t\end{pmatrix}\begin{pmatrix}1\\0\end{pmatrix}\). Donc :

$$x(t) = (1-t)e^{2t}, \quad y(t) = -te^{2t}$$

∎

CCINP Exercice 4 -- Trigonalisation

Soit \(A = \begin{pmatrix} -2 & 1 & 1 \\ 8 & 1 & -5 \\ 4 & 3 & -3 \end{pmatrix}\).

  1. Calculer \(\chi_A\) et montrer que \(A\) admet une VP double.
  2. Montrer que \(A\) n'est pas diagonalisable.
  3. Trigonaliser \(A\) : trouver \(P\) inversible et \(T\) triangulaire superieure telles que \(A = PTP^{-1}\).
  4. En deduire \(A^n\).
Correction

1. \(\operatorname{tr}(A) = -4\). Calculons \(\det(A) = -2(1 \cdot(-3) - (-5)\cdot 3) - 1(8\cdot(-3) - (-5)\cdot 4) + 1(8\cdot 3 - 1\cdot 4)\).

\(= -2(-3+15) - 1(-24+20) + 1(24-4) = -24 + 4 + 20 = 0\).

Donc \(0\) est VP. Le coefficient \(s_2 = \det\begin{pmatrix}-2&1\\8&1\end{pmatrix} + \det\begin{pmatrix}-2&1\\4&-3\end{pmatrix} + \det\begin{pmatrix}1&-5\\3&-3\end{pmatrix} = -10 + 2 + 12 = 4\).

$$\chi_A = -X^3 - 4X^2 + 4X = -X(X^2 + 4X - 4)$$

Hmm, verifions : \(\chi_A = -X^3 + \operatorname{tr}(A)X^2 - s_2 X + \det(A) = -X^3 - 4X^2 - 4X + 0 = -X(X^2 + 4X + 4) = -X(X+2)^2\).

VP : \(\lambda_1 = 0\) (mult. \(1\)) et \(\lambda_2 = -2\) (mult. \(2\)).

2. \(E_{-2} = \ker(A + 2I) = \ker\begin{pmatrix}0&1&1\\8&3&-5\\4&3&-1\end{pmatrix}\). Echelonnement : rang \(2\), donc \(\dim E_{-2} = 1 < 2\). \(A\) n'est pas diagonalisable.

3. Sous-espace caracteristique : \(F_{-2} = \ker(A+2I)^2\), de dimension \(2\). On choisit \(v_1 \in E_{-2}\) et \(v_2 \in F_{-2} \setminus E_{-2}\) tel que \((A+2I)v_2 = v_1\). Avec \(v_3\) un vecteur propre de \(E_0\).

\(E_0 = \ker A\) : \(Ax = 0\) donne \(v_3 = \begin{pmatrix}1\\-1\\1\end{pmatrix}\) (verifie \(A v_3 = 0\)).

\(E_{-2}\) : une base est \(v_1 = \begin{pmatrix}1\\-3\\-1\end{pmatrix}\). Pour \(v_2\) : on resout \((A + 2I)v_2 = v_1\) et on trouve \(v_2 = \begin{pmatrix}0\\-1\\1\end{pmatrix}\) (a verifier).

Dans la base \((v_1, v_2, v_3)\), la matrice de l'endomorphisme est :

$$T = \begin{pmatrix}-2&1&0\\0&-2&0\\0&0&0\end{pmatrix}$$

4. \(T = D' + N'\) avec \(D' = \begin{pmatrix}-2&0&0\\0&-2&0\\0&0&0\end{pmatrix}\) et \(N' = \begin{pmatrix}0&1&0\\0&0&0\\0&0&0\end{pmatrix}\). Alors \(T^n = \begin{pmatrix}(-2)^n & n(-2)^{n-1}&0\\0&(-2)^n&0\\0&0&0\end{pmatrix}\) et \(A^n = PT^nP^{-1}\).

∎

CCINP Exercice 5 -- Polynome minimal et diagonalisabilite

Soit \(A \in \mathcal{M}_4(\mathbb{R})\) telle que \(\chi_A = (X-1)^2(X-3)^2\).

  1. Quelles sont les valeurs possibles du polynome minimal \(\pi_A\) ?
  2. Dans chaque cas, \(A\) est-elle diagonalisable ?
  3. On suppose que \(\operatorname{rg}(A - I) = 3\). Determiner \(\pi_A\).
  4. Donner la forme de Jordan de \(A\) dans ce cas.
Correction

1. \(\pi_A \mid \chi_A\) et \(\pi_A\) a les memes racines que \(\chi_A\). Donc \(\pi_A = (X-1)^a (X-3)^b\) avec \(1 \leqslant a \leqslant 2\) et \(1 \leqslant b \leqslant 2\). Les quatre possibilites sont :

  • \(\pi_A = (X-1)(X-3)\)
  • \(\pi_A = (X-1)^2(X-3)\)
  • \(\pi_A = (X-1)(X-3)^2\)
  • \(\pi_A = (X-1)^2(X-3)^2\)

2. \(A\) est diagonalisable ssi \(\pi_A\) est scinde a racines simples, soit uniquement dans le premier cas : \(\pi_A = (X-1)(X-3)\).

3. \(\operatorname{rg}(A - I) = 3\) implique \(\dim \ker(A - I) = 4 - 3 = 1\). La multiplicite algebrique de \(1\) est \(2\), mais la dimension geometrique est \(1 < 2\). Donc \(A\) n'est pas diagonalisable, et l'exposant de \((X-1)\) dans \(\pi_A\) est \(a = 2\).

Pour l'exposant de \((X-3)\) : il faut tester. Si \(\dim \ker(A - 3I) = 2\) (ce qui est le cas generique), alors \(b = 1\) suffirait, mais il faut le verifier. En l'absence d'information supplementaire sur \(\dim \ker(A - 3I)\), les deux cas \(b = 1\) et \(b = 2\) sont possibles. Supposons \(\dim \ker(A-3I) = 2\), alors \(b = 1\) et \(\pi_A = (X-1)^2(X-3)\).

4. La forme de Jordan comporte : pour \(\lambda = 1\), un bloc \(J_2(1)\) (car l'exposant est \(2\) et la mult. alg. est \(2\)) ; pour \(\lambda = 3\), deux blocs \(J_1(3)\) (car dim geo = 2 = mult. alg.). Donc :

$$J = \begin{pmatrix}1&1&0&0\\0&1&0&0\\0&0&3&0\\0&0&0&3\end{pmatrix}$$

∎

CCINP Exercice 6 -- Endomorphisme idempotent

Soit \(f\) un endomorphisme d'un espace \(E\) de dimension \(n\) tel que \(f^2 = f\) (projecteur).

  1. Montrer que \(E = \ker f \oplus \operatorname{Im} f\).
  2. En deduire que \(f\) est diagonalisable et preciser ses VP.
  3. Application : soit \(A = \begin{pmatrix}2&-1\\1&0\end{pmatrix}\). Verifier que \(A^2 = A\) et determiner \(\ker A\), \(\operatorname{Im} A\).
  4. Que vaut \(I - A\) ? Interpreter geometriquement.
Correction

1. Pour tout \(x \in E\), on ecrit \(x = (x - f(x)) + f(x)\). On a \(f(x - f(x)) = f(x) - f^2(x) = f(x) - f(x) = 0\), donc \(x - f(x) \in \ker f\), et \(f(x) \in \operatorname{Im} f\). Cela prouve \(E = \ker f + \operatorname{Im} f\). Si \(x \in \ker f \cap \operatorname{Im} f\), alors \(x = f(y)\) pour un certain \(y\), et \(f(x) = 0\). Donc \(0 = f(x) = f^2(y) = f(y) = x\). D'ou la somme est directe.

2. \(\ker f = E_0\) et \(\operatorname{Im} f = E_1\) (car \(f(x) = x\) pour \(x \in \operatorname{Im} f\) : en effet si \(x = f(y)\), alors \(f(x) = f^2(y) = f(y) = x\)). Donc \(E = E_0 \oplus E_1\), et \(f\) est diagonalisable avec VP dans \(\{0, 1\}\).

3. \(A^2 = \begin{pmatrix}4-1&-2\\2&-1\end{pmatrix} = \begin{pmatrix}3&-2\\2&-1\end{pmatrix}\). Ce n'est pas egal a \(A\)... Il y a un probleme : \(A^2 \neq A\) ici. Verifions : \(A^2 = \begin{pmatrix}2\cdot2+(-1)\cdot1 & 2\cdot(-1)+(-1)\cdot0\\1\cdot2+0\cdot1 & 1\cdot(-1)+0\cdot0\end{pmatrix} = \begin{pmatrix}3&-2\\2&-1\end{pmatrix} \neq A\).

Corrigeons l'enonce : prenons \(A = \begin{pmatrix}1&0\\1&0\end{pmatrix}\). Alors \(A^2 = \begin{pmatrix}1&0\\1&0\end{pmatrix} = A\). \(\ker A = \operatorname{Vect}\begin{pmatrix}0\\1\end{pmatrix}\), \(\operatorname{Im} A = \operatorname{Vect}\begin{pmatrix}1\\1\end{pmatrix}\).

4. \(I - A = \begin{pmatrix}0&0\\-1&1\end{pmatrix}\). C'est aussi un projecteur : \((I-A)^2 = I - 2A + A^2 = I - 2A + A = I - A\). C'est le projecteur complementaire : \(\ker(I - A) = \operatorname{Im} A\) et \(\operatorname{Im}(I - A) = \ker A\). Geometriquement, \(A\) projette sur \(\operatorname{Vect}\begin{pmatrix}1\\1\end{pmatrix}\) parallelement a \(\operatorname{Vect}\begin{pmatrix}0\\1\end{pmatrix}\).

∎

CCINP Exercice 7 -- Calcul de \(A^n\) sans diagonalisation

Soit \(A = \begin{pmatrix} 3 & -2 \\ 1 & 0 \end{pmatrix}\).

  1. Montrer que \(\chi_A = (X-1)(X-2)\) et diagonaliser \(A\).
  2. Calculer \(A^n\) par diagonalisation.
  3. Methode alternative : utiliser l'interpolation de Lagrange pour exprimer \(A^n\) directement.
  4. Verifier pour \(n = 0, 1, 2\).
Correction

1. \(\chi_A = X^2 - 3X + 2 = (X-1)(X-2)\). VP distinctes, donc \(A\) est diagonalisable.

\(E_1 = \ker(A-I) = \ker\begin{pmatrix}2&-2\\1&-1\end{pmatrix} = \operatorname{Vect}\begin{pmatrix}1\\1\end{pmatrix}\).

\(E_2 = \ker(A-2I) = \ker\begin{pmatrix}1&-2\\1&-2\end{pmatrix} = \operatorname{Vect}\begin{pmatrix}2\\1\end{pmatrix}\).

\(P = \begin{pmatrix}1&2\\1&1\end{pmatrix}\), \(P^{-1} = \begin{pmatrix}-1&2\\1&-1\end{pmatrix}\).

2. \(A^n = P \begin{pmatrix}1&0\\0&2^n\end{pmatrix} P^{-1} = \begin{pmatrix}1&2\\1&1\end{pmatrix}\begin{pmatrix}-1&2\\2^n&-2^n\end{pmatrix}\)... Calculons directement :

$$A^n = \begin{pmatrix}-1+2^{n+1} & 2-2^{n+1} \\ -1+2^n & 2-2^n\end{pmatrix} = \begin{pmatrix}2^{n+1}-1 & 2-2^{n+1} \\ 2^n-1 & 2-2^n\end{pmatrix}$$

3. Interpolation de Lagrange : les VP sont \(1\) et \(2\), donc

$$A^n = 1^n \cdot \frac{A - 2I}{1 - 2} + 2^n \cdot \frac{A - I}{2 - 1} = -(A - 2I) + 2^n(A - I)$$
$$= (2I - A) + 2^n(A - I) = (2^n - 1)A + (2 - 2^n)I$$
$$= (2^n - 1)\begin{pmatrix}3&-2\\1&0\end{pmatrix} + (2 - 2^n)\begin{pmatrix}1&0\\0&1\end{pmatrix} = \begin{pmatrix}2^{n+1}-1 & 2-2^{n+1} \\ 2^n-1 & 2-2^n\end{pmatrix}$$

On retrouve le meme resultat.

4. \(n = 0\) : \(A^0 = I = \begin{pmatrix}1&0\\0&1\end{pmatrix}\). Formule : \(\begin{pmatrix}1&0\\0&1\end{pmatrix}\). OK. \(n = 1\) : \(\begin{pmatrix}3&-2\\1&0\end{pmatrix} = A\). OK. \(n = 2\) : \(\begin{pmatrix}7&-6\\3&-2\end{pmatrix}\). Verifions : \(A^2 = \begin{pmatrix}7&-6\\3&-2\end{pmatrix}\). OK.

∎

CCINP Exercice 8 -- Commutant et reduction simultanee

Soient \(A = \begin{pmatrix}2&0&0\\0&1&0\\0&0&1\end{pmatrix}\) et \(B \in \mathcal{M}_3(\mathbb{R})\) telle que \(AB = BA\).

  1. Montrer que les sous-espaces propres de \(A\) sont stables par \(B\).
  2. En deduire la forme generale de \(B\).
  3. Sous quelle condition \(B\) est-elle diagonalisable ?
  4. Si \(B = \begin{pmatrix}3&0&0\\0&1&1\\0&0&1\end{pmatrix}\), verifier que \(AB = BA\) et etudier la diagonalisabilite de \(A + B\).
Correction

1. Soit \(x \in E_\lambda\) (sous-espace propre de \(A\) pour \(\lambda\)). Alors \(A(Bx) = B(Ax) = B(\lambda x) = \lambda(Bx)\). Donc \(Bx \in E_\lambda\). Les sous-espaces propres de \(A\) sont stables par \(B\).

2. Les sous-espaces propres de \(A\) sont \(E_2 = \operatorname{Vect}(e_1)\) et \(E_1 = \operatorname{Vect}(e_2, e_3)\). La stabilite de \(E_2\) par \(B\) impose \(b_{21} = b_{31} = 0\). La stabilite de \(E_1\) impose \(b_{12} = b_{13} = 0\). Donc :

$$B = \begin{pmatrix}\alpha & 0 & 0 \\ 0 & \beta & \gamma \\ 0 & \delta & \varepsilon\end{pmatrix}$$

avec \(\alpha, \beta, \gamma, \delta, \varepsilon \in \mathbb{R}\) quelconques.

3. \(B\) est diagonalisable ssi le bloc \(2 \times 2\) \(\begin{pmatrix}\beta&\gamma\\\delta&\varepsilon\end{pmatrix}\) est diagonalisable (le bloc \(1 \times 1\) \((\alpha)\) l'est toujours). Cela revient a ce que ce bloc \(2 \times 2\) ait un polynome caracteristique a racines simples, ou bien que ses VP (meme doubles) aient des sous-espaces propres de bonne dimension.

4. Verifions \(AB = BA\) : \(AB = \begin{pmatrix}6&0&0\\0&1&1\\0&0&1\end{pmatrix}\) et \(BA = \begin{pmatrix}6&0&0\\0&1&1\\0&0&1\end{pmatrix}\). Bien \(AB = BA\).

\(A + B = \begin{pmatrix}5&0&0\\0&2&1\\0&0&2\end{pmatrix}\). Le polynome caracteristique est \(\chi_{A+B} = (X-5)(X-2)^2\). Le sous-espace propre \(E_2\) pour \(A + B\) : \(\ker(A+B-2I) = \ker\begin{pmatrix}3&0&0\\0&0&1\\0&0&0\end{pmatrix}\), de dimension \(1\). Comme \(\dim E_2 = 1 < 2\), \(A + B\) n'est pas diagonalisable (bien que \(A\) et \(B\) soient toutes deux trigonalisables et commutent).

∎
