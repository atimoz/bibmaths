I. Methodes a retenir

M1. Calculer les coefficients de Fourier

Methode

Soit \(f\) une fonction \(2\pi\)-periodique et integrable sur \([-\pi, \pi]\). Ses coefficients de Fourier sont :

$$a_0(f) = \frac{1}{2\pi}\int_{-\pi}^{\pi} f(t)\,dt, \quad a_n(f) = \frac{1}{\pi}\int_{-\pi}^{\pi} f(t)\cos(nt)\,dt, \quad b_n(f) = \frac{1}{\pi}\int_{-\pi}^{\pi} f(t)\sin(nt)\,dt$$

Ou, en forme exponentielle : \(c_n(f) = \frac{1}{2\pi}\int_{-\pi}^{\pi} f(t)e^{-int}\,dt\).

  1. Parite : si \(f\) est paire, alors \(b_n = 0\) pour tout \(n\) et \(a_n = \frac{2}{\pi}\int_0^{\pi} f(t)\cos(nt)\,dt\).
  2. Imparite : si \(f\) est impaire, alors \(a_n = 0\) pour tout \(n\) et \(b_n = \frac{2}{\pi}\int_0^{\pi} f(t)\sin(nt)\,dt\).
  3. Periodicite supplementaire : si \(f\) est \(\pi\)-periodique, les coefficients d'indice impair sont nuls.
  4. Utiliser la linearite et les tables de primitives standard.
Exemple resolu

Calculer les coefficients de Fourier de \(f(t) = |\sin(t)|\), \(2\pi\)-periodique.

Resolution. La fonction \(f\) est paire et \(\pi\)-periodique. Donc \(b_n = 0\) pour tout \(n\), et les \(a_n\) d'indice impair sont nuls. On calcule :

$$a_0 = \frac{1}{2\pi}\int_{-\pi}^{\pi}|\sin t|\,dt = \frac{1}{2\pi}\cdot 2\int_0^{\pi}\sin t\,dt = \frac{1}{\pi}\left[-\cos t\right]_0^{\pi} = \frac{2}{\pi}$$

Pour \(n \geq 1\), comme \(f\) est paire et \(\pi\)-periodique, seuls les \(a_{2k}\) sont non nuls. On a sur \([0, \pi]\), \(|\sin t| = \sin t\), donc :

$$a_{2k} = \frac{2}{\pi}\int_0^{\pi}\sin(t)\cos(2kt)\,dt = \frac{1}{\pi}\int_0^{\pi}\left[\sin((2k+1)t) - \sin((2k-1)t)\right]dt$$

En integrant :

$$a_{2k} = \frac{1}{\pi}\left[\frac{-\cos((2k+1)t)}{2k+1} + \frac{\cos((2k-1)t)}{2k-1}\right]_0^{\pi}$$

Pour \(t = \pi\) : \(\cos((2k+1)\pi) = -1\) et \(\cos((2k-1)\pi) = -1\). Pour \(t = 0\) : les cosinus valent \(1\). D'ou :

$$a_{2k} = \frac{1}{\pi}\left(\frac{2}{2k+1} - \frac{2}{2k-1}\right) = \frac{-4}{\pi(4k^2 - 1)}$$

Finalement : \(f(t) \sim \dfrac{2}{\pi} - \dfrac{4}{\pi}\displaystyle\sum_{k=1}^{+\infty}\dfrac{\cos(2kt)}{4k^2 - 1}\).

M2. Convergence ponctuelle par le theoreme de Dirichlet

Methode

Le theoreme de Dirichlet affirme : si \(f\) est \(2\pi\)-periodique et de classe \(C^1\) par morceaux, alors en tout point \(t\) :

$$S_f(t) = \frac{f(t^+) + f(t^-)}{2}$$
  1. Verifier que \(f\) est bien \(C^1\) par morceaux (continue par morceaux, derivee continue par morceaux).
  2. Aux points de continuite : \(S_f(t) = f(t)\).
  3. Aux points de discontinuite : \(S_f(t)\) est la demi-somme des limites a gauche et a droite.
  4. Choisir des valeurs de \(t\) judicieuses pour calculer des sommes de series numeriques.
Exemple resolu

Soit \(f(t) = t\) sur \(]-\pi, \pi[\), prolongee par \(2\pi\)-periodicite. Determiner la serie de Fourier et sa somme en \(t = \pi/2\).

Resolution. La fonction \(f\) est impaire, donc \(a_n = 0\) pour tout \(n \geq 0\). On calcule :

$$b_n = \frac{2}{\pi}\int_0^{\pi}t\sin(nt)\,dt = \frac{2}{\pi}\left[\frac{-t\cos(nt)}{n} + \frac{\sin(nt)}{n^2}\right]_0^{\pi} = \frac{2}{\pi}\cdot\frac{-\pi\cos(n\pi)}{n} = \frac{2(-1)^{n+1}}{n}$$

La serie de Fourier est \(S_f(t) = 2\sum_{n=1}^{+\infty}\frac{(-1)^{n+1}}{n}\sin(nt)\).

Par Dirichlet, aux points de continuite (\(t \notin \pi + 2\pi\mathbb{Z}\)), \(S_f(t) = t\). En \(t = \pi/2\) :

$$\frac{\pi}{2} = 2\sum_{n=1}^{+\infty}\frac{(-1)^{n+1}}{n}\sin\!\left(\frac{n\pi}{2}\right) = 2\left(1 - \frac{1}{3} + \frac{1}{5} - \frac{1}{7} + \cdots\right)$$

D'ou la formule de Leibniz : \(\displaystyle\sum_{k=0}^{+\infty}\frac{(-1)^k}{2k+1} = \frac{\pi}{4}\).

M3. Egalite de Parseval et sommes de series

Methode

Si \(f\) est \(2\pi\)-periodique et de carre integrable, l'egalite de Parseval donne :

$$\frac{1}{2\pi}\int_{-\pi}^{\pi}|f(t)|^2\,dt = |a_0|^2 + \frac{1}{2}\sum_{n=1}^{+\infty}(a_n^2 + b_n^2) = \sum_{n=-\infty}^{+\infty}|c_n|^2$$
  1. Calculer le membre de gauche : \(\frac{1}{2\pi}\int_{-\pi}^{\pi}|f(t)|^2\,dt\).
  2. Calculer les coefficients de Fourier de \(f\).
  3. Identifier le membre de droite comme une serie numerique a sommer.
  4. Par egalite, en deduire la valeur de la somme cherchee.
Exemple resolu

Calculer \(\sum_{n=1}^{+\infty}\frac{1}{n^2}\) et \(\sum_{n=1}^{+\infty}\frac{1}{n^4}\) par Parseval.

Pour \(\pi^2/6\). On utilise \(f(t) = t\) sur \(]-\pi, \pi[\). On a \(a_0 = 0\), \(a_n = 0\), \(b_n = \frac{2(-1)^{n+1}}{n}\). Le membre de gauche :

$$\frac{1}{2\pi}\int_{-\pi}^{\pi}t^2\,dt = \frac{1}{2\pi}\cdot\frac{2\pi^3}{3} = \frac{\pi^2}{3}$$

L'egalite de Parseval donne :

$$\frac{\pi^2}{3} = \frac{1}{2}\sum_{n=1}^{+\infty}\frac{4}{n^2} = 2\sum_{n=1}^{+\infty}\frac{1}{n^2}$$
$$\boxed{\sum_{n=1}^{+\infty}\frac{1}{n^2} = \frac{\pi^2}{6}}$$

Pour \(\pi^4/90\). On utilise \(g(t) = t^2\) sur \(]-\pi, \pi[\). La fonction \(g\) est paire, donc \(b_n = 0\). On calcule :

$$a_0 = \frac{1}{2\pi}\int_{-\pi}^{\pi}t^2\,dt = \frac{\pi^2}{3}, \qquad a_n = \frac{2}{\pi}\int_0^{\pi}t^2\cos(nt)\,dt = \frac{4(-1)^n}{n^2} \quad (n \geq 1)$$

Le membre de gauche de Parseval :

$$\frac{1}{2\pi}\int_{-\pi}^{\pi}t^4\,dt = \frac{\pi^4}{5}$$

L'egalite de Parseval donne :

$$\frac{\pi^4}{5} = \frac{\pi^4}{9} + \frac{1}{2}\sum_{n=1}^{+\infty}\frac{16}{n^4} = \frac{\pi^4}{9} + 8\sum_{n=1}^{+\infty}\frac{1}{n^4}$$
$$\sum_{n=1}^{+\infty}\frac{1}{n^4} = \frac{1}{8}\left(\frac{\pi^4}{5} - \frac{\pi^4}{9}\right) = \frac{1}{8}\cdot\frac{4\pi^4}{45} = \boxed{\frac{\pi^4}{90}}$$

M4. Convergence normale par regularite

Methode

Pour montrer que la serie de Fourier de \(f\) converge normalement :

  1. Regularite \(\Rightarrow\) decroissance : si \(f\) est \(C^k\) et \(2\pi\)-periodique, alors \(c_n(f^{(k)}) = (in)^k c_n(f)\), d'ou \(c_n(f) = o(1/|n|^k)\) par Riemann-Lebesgue.
  2. Critere de convergence normale : si \(|c_n(f)| \leq C/|n|^\alpha\) avec \(\alpha > 1\), alors la serie de Fourier converge normalement (et donc uniformement) car \(\sum 1/|n|^\alpha\) converge.
  3. En pratique : si \(f\) est \(C^1\) et \(2\pi\)-periodique, \(c_n = O(1/n^2)\) (par integration par parties), donc la serie converge normalement.
  4. Regle pratique : \(f \in C^1_{2\pi}\) suffit pour la convergence normale de la serie de Fourier vers \(f\).
Exemple resolu

Montrer que la serie de Fourier de \(f(t) = t^2\) sur \(]-\pi, \pi[\), prolongee par periodicite, converge normalement sur \(\mathbb{R}\).

Resolution. Le prolongement periodique de \(t^2\) est continu (car \(f(-\pi) = f(\pi) = \pi^2\)) et \(C^1\) par morceaux. Les coefficients sont \(a_0 = \pi^2/3\) et \(a_n = 4(-1)^n / n^2\).

On a \(|a_n| \leq 4/n^2\) et \(\sum 1/n^2\) converge, donc la serie \(\sum a_n \cos(nt)\) converge normalement :

$$\sum_{n=1}^{+\infty}|a_n| \leq 4\sum_{n=1}^{+\infty}\frac{1}{n^2} = \frac{2\pi^2}{3} < +\infty$$

La convergence normale est assuree. Par Dirichlet, la somme vaut \(f(t)\) en tout point de continuite, et comme \(f\) est continue, la serie converge normalement vers \(f\) sur \(\mathbb{R}\) tout entier.

M5. Resolution d'EDP par separation des variables

Methode

Pour resoudre une EDP lineaire sur \([0, L]\) (chaleur, ondes, Laplace) avec conditions aux limites :

  1. Separation des variables : chercher \(u(x, t) = X(x)\,T(t)\). Injecter dans l'EDP pour obtenir deux EDO separees.
  2. Probleme spectral : resoudre l'equation en \(X\) avec les conditions aux limites. Trouver les valeurs propres \(\lambda_n\) et les fonctions propres \(X_n\).
  3. Equation temporelle : pour chaque \(\lambda_n\), resoudre l'EDO en \(T_n(t)\).
  4. Superposition : la solution generale est \(u(x,t) = \sum_n c_n\,X_n(x)\,T_n(t)\).
  5. Condition initiale : determiner les coefficients \(c_n\) par developpement en serie de Fourier de la condition initiale.
Exemple resolu

Resoudre l'equation de la chaleur \(\frac{\partial u}{\partial t} = \frac{\partial^2 u}{\partial x^2}\) sur \([0, \pi]\) avec \(u(0,t) = u(\pi, t) = 0\) et \(u(x, 0) = \sin(x) + 3\sin(2x)\).

Resolution. On pose \(u(x,t) = X(x)T(t)\). L'EDP devient \(XT' = X''T\), soit \(\frac{T'}{T} = \frac{X''}{X} = -\lambda\) (constante).

Equation en \(X\) : \(X'' + \lambda X = 0\) avec \(X(0) = X(\pi) = 0\). Les solutions non triviales existent pour \(\lambda_n = n^2\) (\(n \geq 1\)), avec \(X_n(x) = \sin(nx)\).

Equation en \(T\) : \(T_n'(t) = -n^2 T_n(t)\), d'ou \(T_n(t) = e^{-n^2 t}\).

La solution generale est \(u(x,t) = \sum_{n=1}^{+\infty}c_n\sin(nx)\,e^{-n^2 t}\). La condition initiale donne :

$$u(x, 0) = \sum_{n=1}^{+\infty}c_n\sin(nx) = \sin(x) + 3\sin(2x)$$

Par identification : \(c_1 = 1\), \(c_2 = 3\), \(c_n = 0\) pour \(n \geq 3\). La solution est :

$$\boxed{u(x, t) = e^{-t}\sin(x) + 3e^{-4t}\sin(2x)}$$

II. Vrai ou Faux

Testez vos connaissances 0 / 10

1. Si \(f\) est continue et \(2\pi\)-periodique, sa serie de Fourier converge uniformement vers \(f\).

La continuite seule ne suffit pas pour la convergence uniforme. Il existe des fonctions continues \(2\pi\)-periodiques dont la serie de Fourier diverge en certains points (du Bois-Reymond, 1876). Il faut au moins \(f \in C^1_{2\pi}\) ou une condition de type Holder pour assurer la convergence uniforme.

2. Les coefficients de Fourier d'une fonction reelle paire sont tous reels.

Si \(f\) est reelle et paire, \(b_n = 0\) pour tout \(n\), donc la serie ne contient que des cosinus. En forme exponentielle, \(c_n = c_{-n}\) et \(c_n = \overline{c_{-n}}\) (car \(f\) est reelle), donc \(c_n = \overline{c_n}\) : les \(c_n\) sont reels.

3. Si \(f\) est \(C^1\) par morceaux et \(2\pi\)-periodique, alors \(\sum |c_n(f)|^2\) converge.

Une fonction \(C^1\) par morceaux est continue par morceaux, donc de carre integrable. L'inegalite de Bessel (et l'egalite de Parseval) assurent \(\sum_{n \in \mathbb{Z}}|c_n|^2 = \frac{1}{2\pi}\int_{-\pi}^{\pi}|f(t)|^2\,dt < +\infty\).

4. L'egalite de Parseval est valable pour toute fonction continue par morceaux.

L'egalite de Parseval est valable pour toute fonction \(f \in L^2_{2\pi}\). Toute fonction continue par morceaux sur \([-\pi, \pi]\) est bornee donc de carre integrable, et l'egalite s'applique.

5. Si \(f\) est \(C^2\) et \(2\pi\)-periodique, alors \(c_n(f) = O(1/n^2)\).

Si \(f \in C^2_{2\pi}\), on a \(c_n(f'') = (in)^2 c_n(f) = -n^2 c_n(f)\). Comme \(f''\) est continue et \(2\pi\)-periodique, Riemann-Lebesgue donne \(c_n(f'') \to 0\), donc \(n^2 c_n(f) \to 0\) et \(c_n(f) = o(1/n^2)\), a fortiori \(O(1/n^2)\).

6. Le theoreme de Dirichlet s'applique a toute fonction continue \(2\pi\)-periodique.

Le theoreme de Dirichlet exige que \(f\) soit \(C^1\) par morceaux, pas seulement continue. L'hypothese cle est que \(f\) et \(f'\) soient continues par morceaux. Il existe des fonctions continues dont la serie de Fourier diverge en certains points.

7. Pour toute fonction \(f\) de classe \(C^1\) par morceaux, on a \(c_n(f') = in\,c_n(f)\) pour tout \(n\).

La formule \(c_n(f') = in\,c_n(f)\) n'est valable que si \(f\) est continue et \(C^1\) par morceaux. Si \(f\) a des discontinuites, l'integration par parties produit des termes de saut : \(c_n(f') = in\,c_n(f) - \frac{1}{2\pi}\sum_k [f]_{t_k}\,e^{-int_k}\) ou \([f]_{t_k}\) est le saut de \(f\) en \(t_k\).

8. Si \(f\) et \(g\) sont \(2\pi\)-periodiques et continues par morceaux, alors \(c_n(f * g) = 2\pi\,c_n(f)\,c_n(g)\).

Avec la convention \((f * g)(t) = \int_{-\pi}^{\pi}f(t - s)g(s)\,ds\), on a \(c_n(f * g) = 2\pi\,c_n(f)\,c_n(g)\). C'est la propriete fondamentale : la convolution periodique transforme le produit en produit des coefficients (a un facteur \(2\pi\) pres).

9. Le phenomene de Gibbs disparait lorsqu'on augmente le nombre de termes dans la somme partielle.

Le phenomene de Gibbs persiste quel que soit le nombre de termes. Au voisinage d'un point de discontinuite, les sommes partielles presentent un depassement d'environ \(9\%\) du saut. Le pic se rapproche de la discontinuite quand \(N \to \infty\), mais son amplitude relative reste constante (environ \(8{,}95\%\)).

10. Si tous les coefficients de Fourier de \(f\) sont nuls, alors \(f = 0\) presque partout.

C'est le theoreme d'unicite. Si \(c_n(f) = 0\) pour tout \(n \in \mathbb{Z}\), l'egalite de Parseval donne \(\int |f|^2 = 0\), donc \(f = 0\) presque partout. Si de plus \(f\) est continue, alors \(f = 0\) partout.

III. Exercices corriges

Monier Ch. 18 Exercice 1 -- Coefficients de Fourier et somme de Dirichlet

Soit \(f\) la fonction \(2\pi\)-periodique definie par \(f(t) = |t|\) sur \([-\pi, \pi]\).

  1. Calculer les coefficients de Fourier de \(f\).
  2. En deduire la valeur de \(\sum_{k=0}^{+\infty}\frac{1}{(2k+1)^2}\).
  3. Retrouver la valeur de \(\sum_{n=1}^{+\infty}\frac{1}{n^2}\).
Correction

1. La fonction \(f(t) = |t|\) est paire et continue, donc \(b_n = 0\) pour tout \(n\).

$$a_0 = \frac{1}{2\pi}\int_{-\pi}^{\pi}|t|\,dt = \frac{1}{2\pi}\cdot 2\int_0^{\pi}t\,dt = \frac{1}{2\pi}\cdot\pi^2 = \frac{\pi}{2}$$

Pour \(n \geq 1\), par parite :

$$a_n = \frac{2}{\pi}\int_0^{\pi}t\cos(nt)\,dt = \frac{2}{\pi}\left[\frac{t\sin(nt)}{n} + \frac{\cos(nt)}{n^2}\right]_0^{\pi} = \frac{2}{\pi}\cdot\frac{\cos(n\pi) - 1}{n^2} = \frac{2((-1)^n - 1)}{\pi n^2}$$

Donc \(a_n = 0\) si \(n\) est pair, et \(a_n = \frac{-4}{\pi n^2}\) si \(n\) est impair. La serie de Fourier est :

$$f(t) \sim \frac{\pi}{2} - \frac{4}{\pi}\sum_{k=0}^{+\infty}\frac{\cos((2k+1)t)}{(2k+1)^2}$$

2. La fonction \(f\) est continue et \(C^1\) par morceaux, donc par Dirichlet la serie converge vers \(f(t)\) en tout point. En \(t = 0\) : \(f(0) = 0\), d'ou :

$$0 = \frac{\pi}{2} - \frac{4}{\pi}\sum_{k=0}^{+\infty}\frac{1}{(2k+1)^2}$$
$$\boxed{\sum_{k=0}^{+\infty}\frac{1}{(2k+1)^2} = \frac{\pi^2}{8}}$$

3. On decompose :

$$\sum_{n=1}^{+\infty}\frac{1}{n^2} = \sum_{k=0}^{+\infty}\frac{1}{(2k+1)^2} + \sum_{k=1}^{+\infty}\frac{1}{(2k)^2} = \frac{\pi^2}{8} + \frac{1}{4}\sum_{n=1}^{+\infty}\frac{1}{n^2}$$

Donc \(\frac{3}{4}\sum_{n=1}^{+\infty}\frac{1}{n^2} = \frac{\pi^2}{8}\), d'ou :

$$\boxed{\sum_{n=1}^{+\infty}\frac{1}{n^2} = \frac{\pi^2}{6}}$$

∎

Monier Ch. 18 Exercice 2 -- Parseval et somme \(\sum 1/n^6\)

Soit \(f(t) = t^2\) sur \(]-\pi, \pi[\), prolongee par \(2\pi\)-periodicite.

  1. Calculer les coefficients de Fourier de \(f\).
  2. Appliquer l'egalite de Parseval a \(f\) pour retrouver \(\sum_{n=1}^{+\infty}\frac{1}{n^4}\).
  3. En appliquant Parseval a \(g(t) = t^3\) sur \(]-\pi, \pi[\), calculer \(\sum_{n=1}^{+\infty}\frac{1}{n^6}\).
Correction

1. La fonction \(f(t) = t^2\) est paire, donc \(b_n = 0\). On calcule :

$$a_0 = \frac{1}{2\pi}\int_{-\pi}^{\pi}t^2\,dt = \frac{\pi^2}{3}$$

Pour \(n \geq 1\), deux integrations par parties :

$$a_n = \frac{2}{\pi}\int_0^{\pi}t^2\cos(nt)\,dt = \frac{2}{\pi}\left[\frac{t^2\sin(nt)}{n}\right]_0^{\pi} - \frac{4}{\pi n}\int_0^{\pi}t\sin(nt)\,dt$$

Le premier terme est nul. Pour le second :

$$a_n = -\frac{4}{\pi n}\cdot\frac{\pi(-1)^{n+1}}{n} = \frac{4(-1)^n}{n^2}$$

2. Parseval applique a \(f(t) = t^2\) :

$$\frac{1}{2\pi}\int_{-\pi}^{\pi}t^4\,dt = \frac{\pi^4}{9} + \frac{1}{2}\sum_{n=1}^{+\infty}\frac{16}{n^4}$$

Le membre de gauche : \(\frac{1}{2\pi}\cdot\frac{2\pi^5}{5} = \frac{\pi^4}{5}\). Donc :

$$\frac{\pi^4}{5} = \frac{\pi^4}{9} + 8\sum_{n=1}^{+\infty}\frac{1}{n^4} \implies \sum_{n=1}^{+\infty}\frac{1}{n^4} = \frac{1}{8}\cdot\frac{4\pi^4}{45} = \boxed{\frac{\pi^4}{90}}$$

3. La fonction \(g(t) = t^3\) est impaire sur \(]-\pi, \pi[\), donc \(a_n = 0\). Par integration par parties iteree :

$$b_n = \frac{2}{\pi}\int_0^{\pi}t^3\sin(nt)\,dt$$

Trois integrations par parties donnent :

$$b_n = \frac{2(-1)^{n+1}}{n}\left(\pi^2 - \frac{6}{n^2}\right)$$

Parseval applique a \(g\) :

$$\frac{1}{2\pi}\int_{-\pi}^{\pi}t^6\,dt = \frac{\pi^6}{7} = \frac{1}{2}\sum_{n=1}^{+\infty}b_n^2 = \frac{1}{2}\sum_{n=1}^{+\infty}\frac{4}{n^2}\left(\pi^2 - \frac{6}{n^2}\right)^2$$

En developpant \(\left(\pi^2 - \frac{6}{n^2}\right)^2 = \pi^4 - \frac{12\pi^2}{n^2} + \frac{36}{n^4}\) :

$$\frac{\pi^6}{7} = 2\pi^4\cdot\frac{\pi^2}{6} - 24\pi^2\cdot\frac{\pi^4}{90} + 72\sum_{n=1}^{+\infty}\frac{1}{n^6} = \frac{\pi^6}{3} - \frac{4\pi^6}{15} + 72\sum\frac{1}{n^6}$$
$$72\sum\frac{1}{n^6} = \frac{\pi^6}{7} - \frac{\pi^6}{3} + \frac{4\pi^6}{15} = \pi^6\cdot\frac{15 - 35 + 28}{105} = \frac{8\pi^6}{105}$$
$$\boxed{\sum_{n=1}^{+\infty}\frac{1}{n^6} = \frac{\pi^6}{945}}$$

∎

Monier Ch. 18 Exercice 3 -- Signal creneau et formule de Leibniz

Soit \(f\) la fonction \(2\pi\)-periodique definie par \(f(t) = 1\) sur \(]0, \pi[\) et \(f(t) = -1\) sur \(]-\pi, 0[\), avec \(f(0) = f(\pi) = 0\).

  1. Calculer les coefficients de Fourier de \(f\).
  2. La serie de Fourier converge-t-elle normalement ?
  3. En deduire : \(\frac{\pi}{4} = 1 - \frac{1}{3} + \frac{1}{5} - \frac{1}{7} + \cdots\)
  4. Appliquer Parseval pour calculer \(\sum_{k=0}^{+\infty}\frac{1}{(2k+1)^2}\).
Correction

1. La fonction \(f\) est impaire, donc \(a_n = 0\) pour tout \(n \geq 0\). On calcule :

$$b_n = \frac{2}{\pi}\int_0^{\pi}\sin(nt)\,dt = \frac{2}{\pi}\left[\frac{-\cos(nt)}{n}\right]_0^{\pi} = \frac{2(1 - (-1)^n)}{\pi n}$$

Donc \(b_n = 0\) si \(n\) est pair, et \(b_{2k+1} = \frac{4}{\pi(2k+1)}\).

2. La serie ne converge pas normalement car \(|b_n| \sim \frac{4}{\pi n}\) et \(\sum 1/n\) diverge. Les coefficients decroissent seulement en \(O(1/n)\) (car \(f\) a des discontinuites). On observe le phenomene de Gibbs aux points de discontinuite.

3. Par Dirichlet, en \(t = \pi/2\) (point de continuite, \(f(\pi/2) = 1\)) :

$$1 = \frac{4}{\pi}\sum_{k=0}^{+\infty}\frac{\sin((2k+1)\pi/2)}{2k+1} = \frac{4}{\pi}\sum_{k=0}^{+\infty}\frac{(-1)^k}{2k+1}$$

car \(\sin((2k+1)\pi/2) = (-1)^k\). D'ou :

$$\boxed{\frac{\pi}{4} = \sum_{k=0}^{+\infty}\frac{(-1)^k}{2k+1} = 1 - \frac{1}{3} + \frac{1}{5} - \frac{1}{7} + \cdots}$$

4. Parseval : \(\frac{1}{2\pi}\int_{-\pi}^{\pi}|f(t)|^2\,dt = \frac{1}{2\pi}\cdot 2\pi = 1\). Donc :

$$1 = \frac{1}{2}\sum_{k=0}^{+\infty}\frac{16}{\pi^2(2k+1)^2} = \frac{8}{\pi^2}\sum_{k=0}^{+\infty}\frac{1}{(2k+1)^2}$$
$$\boxed{\sum_{k=0}^{+\infty}\frac{1}{(2k+1)^2} = \frac{\pi^2}{8}}$$

∎

Monier Ch. 18 Exercice 4 -- Noyau de Fejer et moyennes de Cesaro

On definit les sommes de Fejer \(\sigma_N(f)(t) = \frac{1}{N+1}\sum_{k=0}^{N}S_k(f)(t)\).

  1. Montrer que \(\sigma_N(f)(t) = \sum_{n=-N}^{N}\left(1 - \frac{|n|}{N+1}\right)c_n(f)\,e^{int}\).
  2. Calculer \(\sigma_2(f)(t)\) pour le signal creneau de l'exercice 3.
  3. Montrer que si \(f\) est continue et \(2\pi\)-periodique, \(\sigma_N(f) \to f\) uniformement.
Correction

1. On a \(S_k(f)(t) = \sum_{n=-k}^{k}c_n\,e^{int}\). Donc :

$$\sigma_N(f)(t) = \frac{1}{N+1}\sum_{k=0}^{N}\sum_{n=-k}^{k}c_n\,e^{int}$$

Le terme \(c_n\,e^{int}\) apparait dans \(S_k\) pour \(k \geq |n|\), soit pour \(k = |n|, |n|+1, \ldots, N\), ce qui fait \(N + 1 - |n|\) fois. Donc :

$$\sigma_N(f)(t) = \sum_{n=-N}^{N}\frac{N + 1 - |n|}{N+1}\,c_n\,e^{int} = \sum_{n=-N}^{N}\left(1 - \frac{|n|}{N+1}\right)c_n\,e^{int}$$

2. Pour le creneau, \(c_0 = 0\), et les coefficients non nuls sont \(c_n = \frac{-2i}{\pi n}\) pour \(n\) impair (avec \(c_{-n} = \overline{c_n}\) car \(f\) est reelle). Pour \(N = 2\), les termes sont \(n \in \{-2, -1, 0, 1, 2\}\). Comme \(c_0 = c_2 = c_{-2} = 0\), seuls \(n = \pm 1\) contribuent :

$$\sigma_2(f)(t) = \left(1 - \frac{1}{3}\right)\left(c_1 e^{it} + c_{-1}e^{-it}\right) = \frac{2}{3}\cdot\frac{4}{\pi}\sin(t) = \frac{8}{3\pi}\sin(t)$$

3. C'est le theoreme de Fejer. On ecrit \(\sigma_N(f)(t) = (f * F_N)(t)\) ou \(F_N\) est le noyau de Fejer :

$$F_N(t) = \frac{1}{N+1}\left(\frac{\sin\left(\frac{(N+1)t}{2}\right)}{\sin\left(\frac{t}{2}\right)}\right)^2$$

Le noyau \(F_N\) est positif, d'integrale \(2\pi\), et se concentre autour de \(0\) quand \(N \to \infty\). Si \(f\) est continue et \(2\pi\)-periodique, elle est uniformement continue, et on montre par un argument standard d'approximation de l'identite que \(\|\sigma_N(f) - f\|_\infty \to 0\).

∎

Monier Ch. 18 Exercice 5 -- Equation de la chaleur sur un cercle

On considere \(\frac{\partial u}{\partial t} = \frac{\partial^2 u}{\partial x^2}\) pour \(x \in [-\pi, \pi]\), \(t > 0\), avec conditions periodiques et \(u(x, 0) = \cos^2(x)\).

  1. Chercher des solutions sous la forme \(u(x,t) = e^{-\lambda t}(A\cos(nx) + B\sin(nx))\) et determiner les valeurs possibles de \(\lambda\).
  2. Ecrire la solution generale sous forme de serie de Fourier.
  3. Determiner la solution avec la condition initiale \(u(x,0) = \cos^2(x)\).
  4. Etudier le comportement de la solution quand \(t \to +\infty\).
Correction

1. On pose \(u(x,t) = T(t)X(x)\). L'EDP donne \(T'X = TX''\), soit \(\frac{T'}{T} = \frac{X''}{X} = -\lambda\).

Equation en \(X\) : \(X'' + \lambda X = 0\), \(X\) \(2\pi\)-periodique. Les solutions periodiques sont \(X_n(x) = A_n\cos(nx) + B_n\sin(nx)\) pour \(\lambda_n = n^2\), \(n \in \mathbb{N}\). Et \(T_n(t) = e^{-n^2 t}\).

2. La solution generale est :

$$u(x,t) = \frac{a_0}{2} + \sum_{n=1}^{+\infty}(a_n\cos(nx) + b_n\sin(nx))\,e^{-n^2 t}$$

3. La condition initiale est \(u(x,0) = \cos^2(x) = \frac{1 + \cos(2x)}{2}\). Par identification : \(a_0/2 = 1/2\) (donc \(a_0 = 1\)), \(a_2 = 1/2\), et tous les autres coefficients sont nuls. La solution est :

$$\boxed{u(x,t) = \frac{1}{2} + \frac{1}{2}\cos(2x)\,e^{-4t}}$$

4. Quand \(t \to +\infty\), \(e^{-4t} \to 0\), donc \(u(x,t) \to \frac{1}{2}\) uniformement en \(x\). La temperature converge vers la moyenne de la donnee initiale :

$$\frac{1}{2\pi}\int_{-\pi}^{\pi}\cos^2(x)\,dx = \frac{1}{2}$$

C'est un resultat general : pour l'equation de la chaleur sur le cercle, la solution converge vers le coefficient \(a_0\), c'est-a-dire la valeur moyenne de la condition initiale.

∎

Monier Ch. 18 Exercice 6 -- Corde vibrante et equation des ondes

On considere \(\frac{\partial^2 u}{\partial t^2} = \frac{\partial^2 u}{\partial x^2}\) sur \([0, \pi]\) avec \(u(0,t) = u(\pi,t) = 0\), condition initiale \(u(x,0) = \sin(x) - \frac{1}{2}\sin(3x)\) et vitesse initiale nulle.

  1. Par separation des variables, determiner les modes propres.
  2. Ecrire la solution generale.
  3. Determiner la solution avec les conditions initiales donnees.
  4. Montrer que la solution est periodique en temps et donner la periode.
Correction

1. On pose \(u(x,t) = X(x)T(t)\). L'EDP donne \(XT'' = X''T\), soit \(\frac{T''}{T} = \frac{X''}{X} = -\lambda\).

Equation en \(X\) : \(X'' + \lambda X = 0\), \(X(0) = X(\pi) = 0\). Solutions non triviales : \(\lambda_n = n^2\), \(X_n(x) = \sin(nx)\) pour \(n \geq 1\).

Equation en \(T\) : \(T'' + n^2 T = 0\), d'ou \(T_n(t) = \alpha_n\cos(nt) + \beta_n\sin(nt)\).

2. La solution generale est :

$$u(x,t) = \sum_{n=1}^{+\infty}\left(\alpha_n\cos(nt) + \beta_n\sin(nt)\right)\sin(nx)$$

3. La condition \(u(x,0) = \sin(x) - \frac{1}{2}\sin(3x)\) donne \(\alpha_1 = 1\), \(\alpha_3 = -1/2\), et \(\alpha_n = 0\) pour \(n \neq 1, 3\).

La condition \(\frac{\partial u}{\partial t}(x,0) = 0\) donne \(\sum n\beta_n\sin(nx) = 0\), donc \(\beta_n = 0\) pour tout \(n\). La solution est :

$$\boxed{u(x,t) = \cos(t)\sin(x) - \frac{1}{2}\cos(3t)\sin(3x)}$$

4. Le premier terme a pour periode temporelle \(2\pi\), le second \(2\pi/3\). La periode de la solution est le PPCM de \(2\pi\) et \(2\pi/3\), soit \(T = 2\pi\).

Verification : \(u(x, t + 2\pi) = \cos(t + 2\pi)\sin(x) - \frac{1}{2}\cos(3t + 6\pi)\sin(3x) = \cos(t)\sin(x) - \frac{1}{2}\cos(3t)\sin(3x) = u(x,t)\).

La corde vibrante revient a sa position initiale apres un temps \(T = 2\pi\). Physiquement, la frequence fondamentale est \(\omega_1 = 1\) et les harmoniques presentes sont les modes \(n = 1\) et \(n = 3\).

∎
