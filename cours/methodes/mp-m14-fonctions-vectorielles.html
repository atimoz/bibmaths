I. Methodes a retenir

M1. Derivee n-ieme (Leibniz, DES)

Methode
  1. Formule de Leibniz : si \(f = uv\), alors \(\displaystyle f^{(n)} = \sum_{k=0}^n \binom{n}{k} u^{(k)} v^{(n-k)}\). Privilegier cette formule quand l'un des facteurs est un polynome (ses derivees s'annulent a un rang fini).
  2. Decomposition en elements simples (DES) : pour une fraction rationnelle, decomposer puis deriver chaque element. Rappel : \(\left(\dfrac{1}{(x-a)^m}\right)^{(n)} = \dfrac{(-1)^n (m+n-1)!}{(m-1)!} \cdot \dfrac{1}{(x-a)^{m+n}}\).
  3. Via les series entieres : si \(f(x) = \sum a_n x^n\), alors \(f^{(n)}(0) = n!\, a_n\).
  4. Verifier sur les premieres valeurs de \(n\) pour conjecturer, puis demontrer par recurrence si necessaire.
Exemple resolu

Calculer la derivee n-ieme de \(f(x) = x^2 e^{3x}\).

Resolution. On pose \(u(x) = x^2\) et \(v(x) = e^{3x}\). On a \(u' = 2x\), \(u'' = 2\), \(u^{(k)} = 0\) pour \(k \geq 3\), et \(v^{(k)} = 3^k e^{3x}\). Par Leibniz :

$$f^{(n)}(x) = \sum_{k=0}^{\min(n,2)} \binom{n}{k} u^{(k)}(x) \cdot v^{(n-k)}(x)$$

Seuls trois termes subsistent (\(k = 0, 1, 2\)) :

$$f^{(n)}(x) = x^2 \cdot 3^n e^{3x} + n \cdot 2x \cdot 3^{n-1} e^{3x} + \frac{n(n-1)}{2} \cdot 2 \cdot 3^{n-2} e^{3x}$$

En factorisant par \(3^{n-2} e^{3x}\) :

$$f^{(n)}(x) = 3^{n-2} e^{3x} \left(9x^2 + 6nx + n(n-1)\right)$$

M2. Inegalite par etude de variations

Methode
  1. Ramener l'inegalite \(f(x) \leq g(x)\) a l'etude du signe de \(h(x) = g(x) - f(x)\).
  2. Etudier les variations de \(h\) : calculer \(h'\), determiner son signe, en deduire le tableau de variations de \(h\).
  3. Conclure en montrant que \(h(x) \geq 0\) pour tout \(x\) du domaine (en utilisant le minimum).
  4. Inegalite de convexite : si \(f\) est convexe, alors \(f\left(\frac{a+b}{2}\right) \leq \frac{f(a)+f(b)}{2}\). On peut aussi utiliser la position relative de la courbe et de ses tangentes/secantes.
Exemple resolu

Montrer que \(\displaystyle \ln(x) \leq \frac{x^2 - 1}{2x}\) pour tout \(x > 0\).

Resolution. Posons \(h(x) = \frac{x^2 - 1}{2x} - \ln(x) = \frac{x}{2} - \frac{1}{2x} - \ln(x)\) pour \(x > 0\).

Derivons : \(h'(x) = \frac{1}{2} + \frac{1}{2x^2} - \frac{1}{x} = \frac{x^2 - 2x + 1}{2x^2} = \frac{(x-1)^2}{2x^2} \geq 0\).

Donc \(h\) est croissante sur \(]0, +\infty[\), avec egalite de la derivee uniquement en \(x = 1\). On a \(h(1) = 0 - 0 = 0\).

Comme \(h' \geq 0\) et \(h(1) = 0\), on a \(h(x) \leq 0\) pour \(x \leq 1\) et \(h(x) \geq 0\) pour \(x \geq 1\). Mais \(h\) est decroissante puis croissante... Reprenons : \(h'(x) = \frac{(x-1)^2}{2x^2} \geq 0\) pour tout \(x > 0\). Donc \(h\) est croissante sur \(]0, +\infty[\). Comme \(h(1) = 0\) est un minimum global (unique point ou \(h' = 0\), et \(h\) est strictement croissante ailleurs), on a \(h(x) \geq h(1) = 0\) pour tout \(x > 0\).

En fait, \(h\) est croissante : pour \(x < 1\), \(h(x) < h(1) = 0\)... Non, \(h' \geq 0\) partout signifie \(h\) croissante. Donc pour \(x < 1\), \(h(x) \leq h(1) = 0\), ce qui donne \(\frac{x^2-1}{2x} \leq \ln(x)\) pour \(x < 1\). Pour \(x > 1\), \(h(x) \geq h(1) = 0\), ce qui donne \(\ln(x) \leq \frac{x^2-1}{2x}\).

L'inegalite \(\ln(x) \leq \frac{x^2-1}{2x}\) est donc valable pour \(x \geq 1\), avec egalite en \(x = 1\). Pour \(x \in ]0,1[\), c'est l'inegalite inverse.

M3. Inequation differentielle

Methode
  1. Lemme de Gronwall : si \(f\) est continue, \(f' \leq \alpha f\) sur \([a, b]\) et \(f(a) = c\), alors \(f(x) \leq c \, e^{\alpha(x-a)}\).
  2. Technique de la multiplication : pour \(f' \leq \alpha f\), poser \(g(x) = f(x) e^{-\alpha x}\). Alors \(g'(x) = (f'(x) - \alpha f(x))e^{-\alpha x} \leq 0\), donc \(g\) est decroissante.
  3. Conclure : \(g(x) \leq g(a)\), soit \(f(x)e^{-\alpha x} \leq f(a) e^{-\alpha a}\), d'ou \(f(x) \leq f(a) e^{\alpha(x-a)}\).
  4. Pour \(f' \geq \alpha f\), on obtient de meme \(f(x) \geq f(a) e^{\alpha(x-a)}\).
Exemple resolu

Soit \(f : [0, +\infty[ \to \mathbb{R}\) de classe \(\mathcal{C}^1\) verifiant \(f'(x) \leq 3f(x)\) pour tout \(x \geq 0\). Montrer que \(f(x) \leq f(0) e^{3x}\).

Resolution. Posons \(g(x) = f(x) e^{-3x}\). Alors \(g\) est de classe \(\mathcal{C}^1\) et :

$$g'(x) = f'(x) e^{-3x} - 3f(x) e^{-3x} = (f'(x) - 3f(x)) e^{-3x} \leq 0$$

car \(f'(x) - 3f(x) \leq 0\) par hypothese et \(e^{-3x} > 0\). Donc \(g\) est decroissante sur \([0, +\infty[\).

Pour tout \(x \geq 0\) : \(g(x) \leq g(0) = f(0)\), soit \(f(x) e^{-3x} \leq f(0)\), d'ou :

$$f(x) \leq f(0) e^{3x}$$

M4. Calcul d'integrale (recurrence, astuces)

Methode
  1. Relations de recurrence : pour les integrales \(I_n = \int_a^b f(t, n) \, dt\), chercher un lien entre \(I_n\) et \(I_{n-1}\) (ou \(I_{n-2}\)) par integration par parties ou formule trigonometrique.
  2. Integration par parties (IPP) : choisir \(u\) et \(dv\) de sorte que \(v \, du\) soit plus simple. Cas typique : \(\int P(t) e^{\alpha t} dt\), \(\int t^n \ln(t) dt\).
  3. Changement de variable : substitutions classiques \(t = \tan(\theta/2)\) (Weierstrass), \(t = a \sin\theta\), etc.
  4. Symetrie : si l'integrande est paire/impaire sur un intervalle symetrique, simplifier.
  5. Pour les integrales de Wallis : \(W_n = \int_0^{\pi/2} \sin^n(t) dt = \int_0^{\pi/2} \cos^n(t) dt\), utiliser \((n+1)W_{n+1} = n W_{n-1}\) et le fait que \(W_n \sim \sqrt{\frac{\pi}{2n}}\).
Exemple resolu

Soit \(\displaystyle I_n = \int_0^{\pi/2} \tan^n(t) \, dt\) pour \(n \in \mathbb{N}\). Etablir une relation de recurrence et calculer \(I_0\), \(I_1\), \(I_2\).

Resolution. On ecrit \(\tan^n(t) = \tan^{n-2}(t) \cdot \tan^2(t) = \tan^{n-2}(t)(\sec^2(t) - 1)\). Donc :

$$I_n = \int_0^{\pi/2} \tan^{n-2}(t) \sec^2(t) \, dt - I_{n-2}$$

Or \(\int_0^{\pi/2} \tan^{n-2}(t) \sec^2(t) \, dt = \left[\frac{\tan^{n-1}(t)}{n-1}\right]_0^{\pi/2}\). Attention, \(\tan(\pi/2) = +\infty\). L'integrale est impropre ; verifions : pour \(n \geq 2\), on pose \(u = \tan(t)\) et \(du = \sec^2(t) dt\), donc \(\int_0^{\pi/2} \tan^{n-2}(t) \sec^2(t) dt = \int_0^{+\infty} u^{n-2} du\) qui diverge pour \(n \geq 2\).

Correction : la bonne relation est obtenue autrement. On ecrit pour \(n \geq 2\) :

$$I_n + I_{n-2} = \int_0^{\pi/2} \tan^{n-2}(t)(1 + \tan^2(t)) dt = \int_0^{\pi/2} \frac{\tan^{n-2}(t)}{\cos^2(t)} dt = \left[\frac{\tan^{n-1}(t)}{n-1}\right]_0^{\pi/2}$$

Cette expression diverge. En fait, \(I_n\) converge ssi \(n < 1\) en tant qu'integrale impropre... Non, verifions : pres de \(\pi/2\), \(\tan(t) \sim \frac{1}{\pi/2 - t}\), donc \(\tan^n(t) \sim (\pi/2-t)^{-n}\), qui est integrable ssi \(n < 1\). Donc \(I_n\) diverge pour \(n \geq 1\).

Remplacons par l'integrale convergente : \(\displaystyle J_n = \int_0^{\pi/4} \tan^n(t) \, dt\). Alors \(J_n + J_{n-2} = \int_0^{\pi/4} \tan^{n-2}(t) \sec^2(t) \, dt = \left[\frac{\tan^{n-1}(t)}{n-1}\right]_0^{\pi/4} = \frac{1}{n-1}\).

Ainsi \(J_n = \frac{1}{n-1} - J_{n-2}\). Et \(J_0 = \frac{\pi}{4}\), \(J_1 = \frac{1}{2}\ln 2\), \(J_2 = 1 - \frac{\pi}{4}\).

M5. Integrale dependant d'un parametre aux bornes

Methode
  1. Soit \(\displaystyle G(x) = \int_{a(x)}^{b(x)} f(t) \, dt\) ou \(a, b\) sont derivables et \(f\) continue. Par le theoreme fondamental et la regle de la chaine :
  2. \(\displaystyle G'(x) = f(b(x)) \cdot b'(x) - f(a(x)) \cdot a'(x)\).
  3. Si en plus le parametre intervient dans l'integrande : \(\displaystyle G(x) = \int_{a(x)}^{b(x)} f(x, t) \, dt\), deriver sous le signe integral (si les conditions de regularite sont verifiees) :
  4. \(\displaystyle G'(x) = f(x, b(x)) \cdot b'(x) - f(x, a(x)) \cdot a'(x) + \int_{a(x)}^{b(x)} \frac{\partial f}{\partial x}(x, t) \, dt\).
Exemple resolu

Soit \(\displaystyle G(x) = \int_x^{x^2} \frac{e^{-t}}{t} \, dt\) pour \(x > 0\). Calculer \(G'(x)\) et etudier \(\lim_{x \to 0^+} G(x)\).

Resolution. Ici \(f(t) = \frac{e^{-t}}{t}\) est continue sur \(]0, +\infty[\), \(a(x) = x\), \(b(x) = x^2\). Donc :

$$G'(x) = \frac{e^{-x^2}}{x^2} \cdot 2x - \frac{e^{-x}}{x} \cdot 1 = \frac{2e^{-x^2}}{x} - \frac{e^{-x}}{x} = \frac{2e^{-x^2} - e^{-x}}{x}$$

Pour la limite en \(0^+\), faisons le changement de variable \(t = xu\) dans l'integrale :

$$G(x) = \int_x^{x^2} \frac{e^{-t}}{t} dt = \int_1^{x} \frac{e^{-xu}}{xu} \cdot x \, du = \int_1^{x} \frac{e^{-xu}}{u} du$$

Quand \(x \to 0^+\), \(e^{-xu} \to 1\) uniformement sur les compacts et la borne superieure \(x \to 0\). Donc \(G(x) \to \int_1^{0} \frac{1}{u} du = -\int_0^1 \frac{1}{u} du\) qui diverge. Examinons plus directement : pour \(0 < x < 1\), on a \(x^2 < x\), donc \(G(x) = \int_x^{x^2} \frac{e^{-t}}{t} dt = -\int_{x^2}^{x} \frac{e^{-t}}{t} dt < 0\). Et \(\frac{e^{-t}}{t} \geq \frac{e^{-1}}{t}\) sur \([x^2, x] \subset ]0, 1]\), donc \(|G(x)| \geq e^{-1} \int_{x^2}^{x} \frac{dt}{t} = e^{-1} |\ln x|\). Comme \(\ln x \to -\infty\), on a \(G(x) \to -\infty\).

II. Vrai ou Faux

Testez vos connaissances 0 / 10

1. La formule de Leibniz s'applique au produit de trois fonctions.

La formule de Leibniz classique concerne le produit de deux fonctions : \((uv)^{(n)} = \sum \binom{n}{k} u^{(k)} v^{(n-k)}\). Pour trois fonctions \(uvw\), on peut appliquer Leibniz deux fois : d'abord a \((uv) \cdot w\), ou utiliser la formule multinomiale \((uvw)^{(n)} = \sum_{i+j+k=n} \frac{n!}{i!j!k!} u^{(i)} v^{(j)} w^{(k)}\), mais ce n'est pas la formule de Leibniz standard.

2. Si \(f\) est convexe sur \(\mathbb{R}\), alors pour tous \(a, b\) : \(f\left(\frac{a+b}{2}\right) \leq \frac{f(a)+f(b)}{2}\).

C'est la definition meme de la convexite appliquee avec \(\lambda = 1/2\) : \(f(\lambda a + (1-\lambda)b) \leq \lambda f(a) + (1-\lambda)f(b)\). Avec \(\lambda = 1/2\), on obtient bien l'inegalite de Jensen au milieu.

3. Si \(f' \leq g'\) sur \([a, b]\) et \(f(a) = g(a)\), alors \(f \leq g\) sur \([a, b]\).

Posons \(h = g - f\). Alors \(h' = g' - f' \geq 0\) sur \([a, b]\) et \(h(a) = 0\). Donc \(h\) est croissante et \(h(x) \geq h(a) = 0\) pour tout \(x \in [a, b]\), soit \(f(x) \leq g(x)\).

4. L'inegalite \(e^x \geq 1 + x\) est valable pour tout \(x \in \mathbb{R}\).

La fonction \(h(x) = e^x - 1 - x\) verifie \(h(0) = 0\) et \(h'(x) = e^x - 1\). On a \(h'(x) < 0\) pour \(x < 0\) et \(h'(x) > 0\) pour \(x > 0\). Donc \(h\) admet un minimum global en \(x = 0\), ou \(h(0) = 0\). Ainsi \(h(x) \geq 0\) pour tout \(x\). C'est aussi l'inegalite de convexite de \(e^x\) (tangente en \(0\)).

5. Le lemme de Gronwall donne \(f' \leq \alpha f \Rightarrow f(x) \leq f(0) e^{\alpha x}\) meme si \(f(0) < 0\).

Oui, le lemme de Gronwall ne fait pas d'hypothese de signe sur \(f(0)\). La demonstration par la fonction \(g(x) = f(x)e^{-\alpha x}\) donne \(g' \leq 0\), donc \(g\) decroissante, donc \(g(x) \leq g(0) = f(0)\), d'ou \(f(x) \leq f(0) e^{\alpha x}\). Si \(f(0) < 0\), l'inegalite est bien valide (et meme plus forte : \(f(x) \leq f(0) e^{\alpha x} < 0\) pour \(\alpha x > 0\)).

6. Les integrales de Wallis verifient \(W_n \to 0\) quand \(n \to +\infty\).

\(W_n = \int_0^{\pi/2} \cos^n(t) dt\). Sur \(]0, \pi/2[\), \(0 \leq \cos(t) < 1\), donc \(\cos^n(t) \to 0\) simplement. Par convergence dominee (dominee par \(1\), integrable sur \([0, \pi/2]\)), \(W_n \to 0\). Plus precisement, \(W_n \sim \sqrt{\frac{\pi}{2n}}\).

7. Si \(F(x) = \int_0^x f(t) dt\) avec \(f\) continue, alors \(F'(x) = f(x)\) pour tout \(x\).

C'est exactement le theoreme fondamental de l'analyse (premiere partie) : si \(f\) est continue sur un intervalle contenant \(0\) et \(x\), alors \(F(x) = \int_0^x f(t) dt\) est derivable et \(F'(x) = f(x)\).

8. \(\displaystyle \frac{d}{dx} \int_0^{x^2} e^{-t^2} dt = 2x e^{-x^4}\).

On pose \(F(u) = \int_0^u e^{-t^2} dt\) (la fonction d'erreur a un facteur pres). Par le theoreme fondamental, \(F'(u) = e^{-u^2}\). Par la regle de la chaine avec \(u = x^2\) : \(\frac{d}{dx} F(x^2) = F'(x^2) \cdot 2x = e^{-x^4} \cdot 2x = 2x e^{-x^4}\).

9. L'integration par parties \(\int_a^b u \, dv = [uv]_a^b - \int_a^b v \, du\) necessite que \(u\) et \(v\) soient de classe \(\mathcal{C}^1\).

Dans la version classique (programme de prepa), on demande que \(u\) et \(v\) soient de classe \(\mathcal{C}^1\) sur \([a, b]\). La formule repose sur la derivee de \(uv\) : \((uv)' = u'v + uv'\), qu'on integre sur \([a, b]\). Il existe des generalisations avec des hypotheses plus faibles, mais dans le cadre du cours, \(\mathcal{C}^1\) est requis.

10. Si \(G(x) = \int_x^{2x} f(t) dt\) avec \(f\) continue, alors \(G'(x) = 2f(2x) - f(x)\).

Par la formule de derivation avec parametre aux bornes : \(G(x) = \int_x^{2x} f(t) dt\), les bornes sont \(a(x) = x\) (donc \(a'(x) = 1\)) et \(b(x) = 2x\) (donc \(b'(x) = 2\)). Ainsi \(G'(x) = f(2x) \cdot 2 - f(x) \cdot 1 = 2f(2x) - f(x)\).

III. Exercices corriges

Monier Ch. 12 Exercice 1 -- Derivees n-iemes
  1. Calculer la derivee n-ieme de \(f(x) = (x^2 + 1)e^x\) par la formule de Leibniz.
  2. Calculer la derivee n-ieme de \(g(x) = \frac{1}{x^2 - 1}\) par DES.
  3. En deduire \(g^{(n)}(0)\).
Correction

1. Posons \(u(x) = x^2 + 1\) et \(v(x) = e^x\). On a \(u' = 2x\), \(u'' = 2\), \(u^{(k)} = 0\) pour \(k \geq 3\), et \(v^{(k)} = e^x\). Par Leibniz :

$$f^{(n)}(x) = (x^2+1)e^x + n \cdot 2x \cdot e^x + \frac{n(n-1)}{2} \cdot 2 \cdot e^x$$
$$= e^x \left(x^2 + 2nx + n(n-1) + 1\right) = e^x(x^2 + 2nx + n^2 - n + 1)$$

2. DES : \(\frac{1}{x^2-1} = \frac{1}{(x-1)(x+1)} = \frac{1/2}{x-1} - \frac{1/2}{x+1}\). Alors :

$$g^{(n)}(x) = \frac{1}{2} \cdot \frac{(-1)^n n!}{(x-1)^{n+1}} - \frac{1}{2} \cdot \frac{(-1)^n n!}{(x+1)^{n+1}}$$
$$= \frac{(-1)^n n!}{2}\left(\frac{1}{(x-1)^{n+1}} - \frac{1}{(x+1)^{n+1}}\right)$$

3. En \(x = 0\) :

$$g^{(n)}(0) = \frac{(-1)^n n!}{2}\left(\frac{1}{(-1)^{n+1}} - 1\right) = \frac{(-1)^n n!}{2}\left((-1)^{n+1} - 1\right)$$

Si \(n\) est pair : \((-1)^{n+1} = -1\), donc \(g^{(n)}(0) = \frac{n!}{2}(-1-1) = -n!\).

Si \(n\) est impair : \((-1)^{n+1} = 1\), donc \(g^{(n)}(0) = \frac{-n!}{2}(1-1) = 0\).

Bilan : \(g^{(n)}(0) = \begin{cases} -n! & \text{si } n \text{ pair} \\ 0 & \text{si } n \text{ impair} \end{cases}\)

C'est coherent avec le DSE : \(\frac{1}{x^2-1} = -\frac{1}{1-x^2} = -\sum x^{2k}\) pour \(|x| < 1\), donc \(g^{(n)}(0)/n! = 0\) si \(n\) impair, et \(g^{(2k)}(0)/(2k)! = -1\), soit \(g^{(2k)}(0) = -(2k)!\).

∎

Monier Ch. 12 Exercice 2 -- Inegalites classiques
  1. Montrer que pour tout \(x > 0\) : \(\dfrac{x}{1+x} \leq \ln(1+x) \leq x\).
  2. Montrer que pour tout \(x \geq 0\) : \(\sin(x) \leq x\) et \(\cos(x) \geq 1 - \frac{x^2}{2}\).
  3. En deduire que pour tout \(x \geq 0\) : \(\sin(x) \geq x - \frac{x^3}{6}\).
Correction

1. Inegalite \(\ln(1+x) \leq x\) : Posons \(h(x) = x - \ln(1+x)\). Alors \(h(0) = 0\) et \(h'(x) = 1 - \frac{1}{1+x} = \frac{x}{1+x} \geq 0\) pour \(x > 0\). Donc \(h\) est croissante sur \([0, +\infty[\) avec \(h(0) = 0\), d'ou \(h(x) \geq 0\), soit \(\ln(1+x) \leq x\).

Inegalite \(\frac{x}{1+x} \leq \ln(1+x)\) : Posons \(\varphi(x) = \ln(1+x) - \frac{x}{1+x}\). Alors \(\varphi(0) = 0\) et \(\varphi'(x) = \frac{1}{1+x} - \frac{1}{(1+x)^2} = \frac{x}{(1+x)^2} \geq 0\) pour \(x \geq 0\). Donc \(\varphi(x) \geq \varphi(0) = 0\).

2. Inegalite \(\sin(x) \leq x\) : Posons \(h(x) = x - \sin(x)\). \(h(0) = 0\), \(h'(x) = 1 - \cos(x) \geq 0\). Donc \(h\) croissante et \(h(x) \geq 0\).

Inegalite \(\cos(x) \geq 1 - x^2/2\) : Posons \(\psi(x) = \cos(x) - 1 + \frac{x^2}{2}\). \(\psi(0) = 0\), \(\psi'(x) = -\sin(x) + x \geq 0\) (d'apres l'inegalite precedente). Donc \(\psi\) croissante et \(\psi(x) \geq 0\).

3. Posons \(\theta(x) = \sin(x) - x + \frac{x^3}{6}\). \(\theta(0) = 0\), \(\theta'(x) = \cos(x) - 1 + \frac{x^2}{2} \geq 0\) (d'apres la question 2). Donc \(\theta\) croissante et \(\theta(x) \geq 0\), soit \(\sin(x) \geq x - \frac{x^3}{6}\).

On retrouve les encadrements du sinus par les sommes partielles de son developpement de Taylor, la methode generalisant a tout ordre.

∎

Monier Ch. 12 Exercice 3 -- Inequation differentielle et applications

Soit \(f : [0, +\infty[ \to \mathbb{R}\) de classe \(\mathcal{C}^1\).

  1. On suppose \(f' + f \geq 0\) et \(f(0) = 1\). Montrer que \(f(x) \geq e^{-x}\) pour tout \(x \geq 0\).
  2. On suppose \(|f'(x)| \leq M|f(x)|\) pour tout \(x \geq 0\), avec \(M > 0\), et \(f(0) = 0\). Montrer que \(f = 0\) sur \([0, +\infty[\).
  3. Soit \(g : [0,1] \to \mathbb{R}\) de classe \(\mathcal{C}^1\) verifiant \(g(0) = 0\) et \(|g'(x)| \leq 2|g(x)|\) pour tout \(x \in [0,1]\). Montrer que \(g = 0\) sur \([0, 1]\).
Correction

1. Posons \(h(x) = f(x) e^x\). Alors \(h'(x) = (f'(x) + f(x))e^x \geq 0\) car \(f' + f \geq 0\) et \(e^x > 0\). Donc \(h\) est croissante. Pour \(x \geq 0\) : \(h(x) \geq h(0) = f(0) = 1\), soit \(f(x) e^x \geq 1\), d'ou \(f(x) \geq e^{-x}\).

2. L'hypothese \(|f'| \leq M|f|\) donne \(-M f \leq f' \leq Mf\) quand \(f \geq 0\), et de meme quand \(f \leq 0\). Posons \(\varphi(x) = f(x)^2\). Alors \(\varphi'(x) = 2f(x)f'(x)\) et \(|\varphi'(x)| = 2|f(x)||f'(x)| \leq 2M f(x)^2 = 2M\varphi(x)\).

Donc \(\varphi' \leq 2M\varphi\). Par Gronwall, \(\varphi(x) \leq \varphi(0) e^{2Mx} = 0\) car \(\varphi(0) = f(0)^2 = 0\).

Comme \(\varphi(x) = f(x)^2 \geq 0\) et \(\varphi(x) \leq 0\), on a \(\varphi(x) = 0\), soit \(f(x) = 0\) pour tout \(x \geq 0\).

3. C'est un cas particulier de la question 2 avec \(M = 2\) sur l'intervalle \([0, 1]\). On pose \(\varphi(x) = g(x)^2\). L'hypothese \(|g'| \leq 2|g|\) donne \(\varphi' \leq 4\varphi\). Par Gronwall : \(\varphi(x) \leq \varphi(0)e^{4x} = 0\). Donc \(g = 0\) sur \([0, 1]\).

∎

Monier Ch. 12 Exercice 4 -- Integrales de Wallis et equivalents
  1. Montrer la relation de recurrence \(W_n = \frac{n-1}{n} W_{n-2}\) pour \(n \geq 2\), ou \(W_n = \int_0^{\pi/2} \sin^n(t) \, dt\).
  2. Calculer \(W_0, W_1, W_2, W_3, W_4, W_5\).
  3. Montrer que \(W_n W_{n+1} = \frac{\pi}{2(n+1)}\).
  4. En deduire que \(W_n \sim \sqrt{\frac{\pi}{2n}}\) quand \(n \to +\infty\).
Correction

1. IPP avec \(u = \sin^{n-1}(t)\) et \(dv = \sin(t) dt\), d'ou \(du = (n-1)\sin^{n-2}(t)\cos(t) dt\) et \(v = -\cos(t)\) :

$$W_n = \left[-\sin^{n-1}(t)\cos(t)\right]_0^{\pi/2} + (n-1)\int_0^{\pi/2} \sin^{n-2}(t)\cos^2(t) dt$$

Le crochet s'annule aux deux bornes. Et \(\cos^2(t) = 1 - \sin^2(t)\), donc :

$$W_n = (n-1)(W_{n-2} - W_n), \quad \text{soit} \quad nW_n = (n-1)W_{n-2}$$

2. \(W_0 = \frac{\pi}{2}\), \(W_1 = 1\). Puis :

  • \(W_2 = \frac{1}{2} W_0 = \frac{\pi}{4}\)
  • \(W_3 = \frac{2}{3} W_1 = \frac{2}{3}\)
  • \(W_4 = \frac{3}{4} W_2 = \frac{3\pi}{16}\)
  • \(W_5 = \frac{4}{5} W_3 = \frac{8}{15}\)

3. Par la recurrence \(W_{n+1} = \frac{n}{n+1} W_{n-1}\), on a \(W_n W_{n+1} = W_n \cdot \frac{n}{n+1} W_{n-1}\). Par ailleurs, \(W_n = \frac{n-1}{n} W_{n-2}\), donc \(W_{n-1} W_n = W_{n-1} \cdot \frac{n-1}{n} W_{n-2}\). En iterant et en utilisant \(W_0 W_1 = \frac{\pi}{2}\), on montre par recurrence que :

$$W_n W_{n+1} = \frac{\pi}{2(n+1)}$$

4. La suite \((W_n)\) est decroissante (car \(0 \leq \sin(t) \leq 1\) sur \([0, \pi/2]\) implique \(\sin^{n+1}(t) \leq \sin^n(t)\)). De \(W_{n+1} \leq W_n \leq W_{n-1}\) et \(W_{n+1} = \frac{n}{n+1} W_{n-1}\), on tire \(\frac{n}{n+1} \leq \frac{W_n}{W_{n-1}} \leq 1\), d'ou \(\frac{W_n}{W_{n+1}} \to 1\), soit \(W_n \sim W_{n+1}\). Alors \(W_n^2 \sim W_n W_{n+1} = \frac{\pi}{2(n+1)} \sim \frac{\pi}{2n}\), d'ou :

$$W_n \sim \sqrt{\frac{\pi}{2n}}$$

∎

Monier Ch. 12 Exercice 5 -- Integrales a parametre

On pose pour \(x > 0\) :

$$F(x) = \int_0^1 \frac{t^x - 1}{\ln t} \, dt$$
  1. Montrer que l'integrale est bien definie pour \(x > 0\).
  2. Calculer \(F'(x)\) en derivant sous le signe integral (justifier).
  3. En deduire la valeur de \(F(x)\) pour tout \(x > 0\).
  4. Application : calculer \(\displaystyle \int_0^1 \frac{t^2 - 1}{\ln t} dt\).
Correction

1. Posons \(f(t, x) = \frac{t^x - 1}{\ln t}\) pour \(t \in ]0, 1[\). Pres de \(t = 1\), on ecrit \(t^x = e^{x \ln t}\) et \(\frac{t^x - 1}{\ln t} = \frac{e^{x\ln t} - 1}{\ln t} \to x\) quand \(t \to 1^-\) (par le taux d'accroissement de l'exponentielle). Donc \(f\) se prolonge par continuite en \(t = 1\) avec \(f(1, x) = x\).

Pres de \(t = 0^+\) : \(t^x \to 0\), \(\ln t \to -\infty\), donc \(\frac{t^x - 1}{\ln t} \to \frac{-1}{-\infty} = 0\). Plus precisement, \(\left|\frac{t^x - 1}{\ln t}\right| \leq \frac{1}{|\ln t|}\) pour \(t\) petit, qui est integrable au voisinage de \(0\). L'integrale est bien definie.

2. On a \(\frac{\partial f}{\partial x}(t, x) = \frac{t^x \ln t}{\ln t} = t^x\) pour \(t \in ]0, 1[\). La fonction \(t \mapsto t^x\) est continue et bornee par \(1\) sur \([0, 1]\), donc integrable. Les hypotheses du theoreme de derivation sous le signe integral (domination uniforme sur tout compact en \(x\)) sont satisfaites. Donc :

$$F'(x) = \int_0^1 t^x \, dt = \left[\frac{t^{x+1}}{x+1}\right]_0^1 = \frac{1}{x+1}$$

3. On a \(F'(x) = \frac{1}{x+1}\), donc \(F(x) = \ln(x+1) + C\). Or \(F(0) = \int_0^1 \frac{t^0 - 1}{\ln t} dt = \int_0^1 0 \, dt = 0\). Donc \(C = 0\) et :

$$F(x) = \ln(x + 1)$$

4. \(\displaystyle \int_0^1 \frac{t^2 - 1}{\ln t} dt = F(2) = \ln 3\).

∎

Monier Ch. 12 Exercice 6 -- Derivation d'une integrale a bornes variables

Pour \(x > 0\), on definit :

$$H(x) = \int_x^{x^2} \frac{\sin(t)}{t} \, dt$$
  1. Montrer que \(H\) est bien definie et de classe \(\mathcal{C}^1\) sur \(]0, +\infty[\).
  2. Calculer \(H'(x)\).
  3. Montrer que \(\lim_{x \to 0^+} H(x) = 0\).
  4. Etudier le signe de \(H(x)\) pour \(x\) au voisinage de \(0^+\) et pour \(x\) au voisinage de \(+\infty\).
Correction

1. La fonction \(t \mapsto \frac{\sin t}{t}\) est continue sur \(]0, +\infty[\) et se prolonge par continuite en \(0\) (avec la valeur \(1\)). Pour \(x > 0\), les bornes \(x\) et \(x^2\) sont dans \(]0, +\infty[\), donc l'integrale est bien definie. Comme les bornes \(x \mapsto x\) et \(x \mapsto x^2\) sont \(\mathcal{C}^1\) et l'integrande est continue, \(H\) est de classe \(\mathcal{C}^1\).

2. Par la formule de derivation avec bornes variables :

$$H'(x) = \frac{\sin(x^2)}{x^2} \cdot 2x - \frac{\sin(x)}{x} \cdot 1 = \frac{2\sin(x^2)}{x} - \frac{\sin(x)}{x}$$

3. Pour \(0 < x < 1\), on a \(x^2 < x\), donc \(H(x) = -\int_{x^2}^x \frac{\sin t}{t} dt\). Or pres de \(0\), \(\frac{\sin t}{t} \to 1\), donc pour \(t \in [x^2, x]\) avec \(x\) petit, \(\frac{\sin t}{t} \approx 1\). Ainsi :

$$|H(x)| \leq \int_{x^2}^x \left|\frac{\sin t}{t}\right| dt \leq \int_{x^2}^x 1 \, dt = x - x^2 \to 0$$

Par encadrement, \(H(x) \to 0\) quand \(x \to 0^+\).

4. Au voisinage de \(0^+\) : pour \(0 < x < 1\) petit, \(x^2 < x\) et \(\sin(t)/t > 0\) pour \(t \in ]0, \pi[\). Donc \(\int_{x^2}^x \frac{\sin t}{t} dt > 0\) et \(H(x) = -\int_{x^2}^x \frac{\sin t}{t} dt < 0\).

Au voisinage de \(+\infty\) : pour \(x > 1\), \(x^2 > x\), donc \(H(x) = \int_x^{x^2} \frac{\sin t}{t} dt\). Par integration par parties (ou le lemme de Riemann-Lebesgue) : \(\int_x^{x^2} \frac{\sin t}{t} dt = \left[-\frac{\cos t}{t}\right]_x^{x^2} - \int_x^{x^2} \frac{\cos t}{t^2} dt\). Le premier terme est \(\frac{\cos x}{x} - \frac{\cos(x^2)}{x^2}\), qui tend vers \(0\), et l'integrale restante est majoree par \(\int_x^{+\infty} \frac{1}{t^2} dt = \frac{1}{x} \to 0\). Donc \(H(x) \to 0\) quand \(x \to +\infty\). Le signe oscille avec les zeros de sin.

∎
