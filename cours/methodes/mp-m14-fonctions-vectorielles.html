Fonctions vectorielles

Fiche de methodes sur les fonctions d'une variable reelle : derivee n-ieme, inegalites, inequations differentielles, calcul d'integrales et integrales a parametre aux bornes. Chaque methode est accompagnee d'un exercice de style concours.

Sommaire
  1. M1. Calculer la derivee n-ieme
  2. M2. Etablir une inegalite via l'etude de variations
  3. M3. Resoudre une inequation differentielle
  4. M4. Calculer ou evaluer une integrale
  5. M5. Integrale dependant d'un parametre aux bornes

M1. Calculer la derivee n-ieme

Methode
  1. Formule de Leibniz : si \(f = uv\), alors \(f^{(n)} = \sum_{k=0}^n \binom{n}{k} u^{(k)} v^{(n-k)}\). Utile quand l'un des deux facteurs a ses derivees successives qui s'annulent (polynome).
  2. Decomposition en elements simples (DES) : pour une fraction rationnelle \(f(x) = \frac{P(x)}{Q(x)}\), decomposer en elements simples, puis deriver chaque terme. Rappel : \(\left(\frac{1}{(x-a)^m}\right)^{(n)} = \frac{(-1)^n (m+n-1)!}{(m-1)!} \cdot \frac{1}{(x-a)^{m+n}}\).
  3. Reconnaitre une serie entiere : si \(f(x) = \sum a_n x^n\), alors \(f^{(n)}(0) = n!\, a_n\).
  4. Verifier le resultat sur les premieres valeurs de \(n\) (conjecturer puis demontrer par recurrence si necessaire).
Astuce

Pour \(f(x) = x^2 e^{3x}\), Leibniz donne trois termes non nuls (car \((x^2)^{(k)} = 0\) pour \(k \geq 3\)). C'est beaucoup plus rapide qu'une recurrence.

Style concours Exercice -- Style concours

Soit \(f(x) = \dfrac{x^2 + 1}{(x-1)(x+2)}\).

  1. Decomposer \(f\) en elements simples.
  2. En deduire \(f^{(n)}(x)\) pour tout \(n \geq 1\).
  3. Calculer \(f^{(n)}(0)\).

Indication : la DES donne \(f(x) = 1 + \frac{2/3}{x-1} + \frac{5/3}{x+2}\). Puis deriver chaque fraction elementaire n fois.

M2. Etablir une inegalite via l'etude de variations

Methode

Pour montrer que \(f(x) \geq g(x)\) sur un intervalle \(I\) :

  1. Poser \(h(x) = f(x) - g(x)\).
  2. Calculer \(h'(x)\) et etudier son signe sur \(I\).
  3. Dresser le tableau de variations de \(h\).
  4. Montrer que le minimum de \(h\) sur \(I\) est \(\geq 0\). Souvent \(h\) atteint son minimum en un point explicite \(x_0\) et \(h(x_0) = 0\).
Variante : convexite

Si \(f\) est convexe, alors \(f(x) \geq f(a) + f'(a)(x-a)\) pour tout \(x\). Cela donne des inegalites classiques comme \(e^x \geq 1 + x\) ou \(\ln(1+x) \leq x\).

Style concours Exercice -- Style concours

Montrer que pour tout \(x > 0\) :

$$\ln(x) \leq \frac{x^2 - 1}{2x}$$
  1. Poser \(h(x) = \frac{x^2 - 1}{2x} - \ln(x)\) et montrer que \(h\) est definie et derivable sur \(]0, +\infty[\).
  2. Etudier le signe de \(h'(x)\) et en deduire les variations de \(h\).
  3. Conclure.

Indication : \(h'(x) = \frac{1}{2}\left(1 + \frac{1}{x^2}\right) - \frac{1}{x} = \frac{(x-1)^2}{2x^2} \geq 0\). Donc \(h\) est croissante sur \(]0,+\infty[\)... non, verifier : \(h(1) = 0\) et \(h' \geq 0\) partout, donc \(h(x) \geq 0\) pour \(x \geq 1\). Pour \(x < 1\), \(h\) decroit puis... recalculer soigneusement.

M3. Resoudre une inequation differentielle

Methode

Pour montrer que \(f(x) \leq Ce^{\alpha x}\) a partir de \(f'(x) \leq \alpha f(x) + \beta\) :

  1. Introduire la fonction auxiliaire \(g(x) = e^{-\alpha x} f(x)\).
  2. Calculer \(g'(x) = e^{-\alpha x}(f'(x) - \alpha f(x))\).
  3. Utiliser l'hypothese sur \(f'\) pour majorer \(g'(x)\).
  4. Integrer l'inegalite \(g'(x) \leq \ldots\) entre \(0\) et \(x\) pour obtenir \(g(x) - g(0) \leq \ldots\)
  5. Revenir a \(f(x) = e^{\alpha x} g(x)\) pour conclure.
Lemme de Gronwall

Si \(f\) est continue positive et \(f(t) \leq C + \int_0^t \alpha f(s)\,ds\), alors \(f(t) \leq C e^{\alpha t}\). C'est la version integrale de la methode ci-dessus.

Style concours Exercice -- Style concours

Soit \(f : [0,+\infty[ \to \mathbb{R}\) de classe \(\mathcal{C}^1\) telle que \(f(0) = 1\) et pour tout \(x \geq 0\) : $$f'(x) \leq 3f(x) + 2$$

  1. Poser \(g(x) = e^{-3x}(f(x) + \frac{2}{3})\) et calculer \(g'(x)\).
  2. Montrer que \(g\) est decroissante sur \([0,+\infty[\).
  3. En deduire que \(f(x) \leq \frac{5}{3}e^{3x} - \frac{2}{3}\) pour tout \(x \geq 0\).

Indication : on pose \(h = f + 2/3\) pour se ramener a \(h' \leq 3h\), puis la fonction \(g = e^{-3x}h\) verifie \(g' \leq 0\).

M4. Calculer ou evaluer une integrale

Methode
  1. Integration par parties (IPP) : \(\int_a^b u'v = [uv]_a^b - \int_a^b uv'\). Choisir \(v\) dont la primitive est simple, et \(u'\) tel que \(u\) se simplifie en derivant.
  2. Changement de variable : poser \(t = \varphi(u)\) avec \(\varphi\) bijective \(\mathcal{C}^1\). Ne pas oublier de changer les bornes et l'element differentiel \(dt = \varphi'(u)\,du\).
  3. Relation de Chasles : decouper l'intervalle en sous-intervalles adaptes (par exemple pour lever des valeurs absolues ou exploiter une symetrie).
  4. Parite / periodicite : si \(f\) est paire et l'intervalle symetrique, \(\int_{-a}^a f = 2\int_0^a f\). Si \(f\) est impaire, \(\int_{-a}^a f = 0\).
Integrales de Wallis

Les integrales \(W_n = \int_0^{\pi/2} \sin^n(t)\,dt\) verifient la relation \(W_n = \frac{n-1}{n}W_{n-2}\) et \(W_n \sim \sqrt{\frac{\pi}{2n}}\). Elles apparaissent souvent dans les exercices de concours.

Style concours Exercice -- Style concours

Soit \(I_n = \displaystyle\int_0^{\pi/4} \tan^n(t)\,dt\) pour \(n \in \mathbb{N}\).

  1. Montrer que \(I_n + I_{n+2} = \frac{1}{n+1}\) pour tout \(n \geq 0\).
  2. En deduire que \(I_n \to 0\) quand \(n \to +\infty\).
  3. Calculer \(I_0\), \(I_1\), \(I_2\), \(I_3\).

Indication : ecrire \(\tan^{n+2}(t) = \tan^n(t)(\sec^2(t) - 1)\) et integrer. On obtient \(I_{n+2} = \frac{1}{n+1} - I_n\). De plus \(0 \leq I_n \leq (\tan(\pi/4))^n \cdot \pi/4 \to 0\) ne marche pas car \(\tan(\pi/4) = 1\) ; utiliser l'encadrement \(0 \leq I_{n+1} \leq I_n\) et la relation.

M5. Integrale dependant d'un parametre aux bornes

Methode

Soit \(G(x) = \displaystyle\int_{u(x)}^{v(x)} f(x,t)\,dt\) ou \(u, v\) sont derivables.

  1. Si \(f\) ne depend pas de \(x\) : \(G(x) = \int_{u(x)}^{v(x)} f(t)\,dt\) et par le theoreme fondamental : $$G'(x) = f(v(x))\,v'(x) - f(u(x))\,u'(x)$$
  2. Si \(f\) depend aussi de \(x\) : combiner la regle de Leibniz avec la derivation des bornes : $$G'(x) = \int_{u(x)}^{v(x)} \frac{\partial f}{\partial x}(x,t)\,dt + f(x,v(x))\,v'(x) - f(x,u(x))\,u'(x)$$
  3. Justifier la derivation sous l'integrale (domination de \(\frac{\partial f}{\partial x}\)) avant d'appliquer la formule.
  4. Calculer explicitement chaque terme.
Cas frequent

Quand une seule borne depend de \(x\) et \(f\) ne depend pas de \(x\), on se ramene simplement a la derivation d'une primitive : \(\frac{d}{dx}\int_a^{v(x)} f(t)\,dt = f(v(x))\,v'(x)\).

Style concours Exercice -- Style concours

Soit \(H(x) = \displaystyle\int_x^{x^2} \frac{e^{-t}}{t}\,dt\) pour \(x > 0\).

  1. Montrer que \(H\) est bien definie et de classe \(\mathcal{C}^1\) sur \(]0, +\infty[\).
  2. Calculer \(H'(x)\).
  3. Etudier \(\lim_{x \to 0^+} H(x)\) et \(\lim_{x \to +\infty} H(x)\).

Indication : \(H'(x) = \frac{e^{-x^2}}{x^2}\cdot 2x - \frac{e^{-x}}{x} = \frac{2e^{-x^2}}{x} - \frac{e^{-x}}{x}\). Pour la limite en \(0^+\), remarquer que \(\frac{e^{-t}}{t} \sim \frac{1}{t}\) donc \(H(x) \sim \int_x^{x^2} \frac{dt}{t} = \ln(x)\) quand \(x \to 0^+\).
