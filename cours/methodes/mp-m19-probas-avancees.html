I. Methodes a retenir

M1. Loi conjointe et lois marginales

Methode

Soient \(X\) et \(Y\) deux VA discretes definies sur le meme espace \((\Omega, \mathcal{A}, P)\).

  1. Loi conjointe : c'est la donnee de \(P(X = x_i, Y = y_j)\) pour tous \((x_i, y_j) \in X(\Omega) \times Y(\Omega)\). On la presente souvent sous forme de tableau a double entree.
  2. Lois marginales : on les obtient en sommant la loi conjointe : $$P(X = x_i) = \sum_j P(X = x_i, Y = y_j), \quad P(Y = y_j) = \sum_i P(X = x_i, Y = y_j).$$
  3. Ajouter une ligne et une colonne de totaux pour verifier les sommes.
  4. Attention : les lois marginales ne determinent pas la loi conjointe (sauf si independance).
Exemple resolu

Couple de VA. On lance deux des equilibres. Soit \(X = \min(d_1, d_2)\) et \(Y = \max(d_1, d_2)\). Determiner \(P(X = 1, Y = 4)\) et la loi marginale de \(X\) restreinte a \(X = 1\).

Resolution. L'univers a \(36\) issues equiprobables. L'evenement \(\{X = 1, Y = 4\}\) correspond aux cas ou l'un des des vaut \(1\) et l'autre \(4\) : ce sont \((1,4)\) et \((4,1)\), soit \(2\) issues.

$$P(X = 1, Y = 4) = \frac{2}{36} = \frac{1}{18}$$

Loi marginale de \(X\) en \(X = 1\) : \(\{X = 1\}\) signifie qu'au moins un de vaut \(1\). Pour \(Y = j\) avec \(j \in \{1, \ldots, 6\}\) :

  • \(P(X = 1, Y = 1) = 1/36\) (les deux des valent \(1\))
  • \(P(X = 1, Y = j) = 2/36\) pour \(j \geq 2\) (un de vaut \(1\), l'autre vaut \(j\))

Donc \(P(X = 1) = \frac{1}{36} + 5 \times \frac{2}{36} = \frac{11}{36}\). On verifie : c'est \(1 - P(\text{aucun des ne vaut } 1) = 1 - (5/6)^2 = 11/36\). Correct.

M2. Verifier l'independance de deux VA

Methode
  1. Definition : \(X\) et \(Y\) sont independantes si et seulement si pour tous \((x_i, y_j)\) : $$P(X = x_i, Y = y_j) = P(X = x_i) \cdot P(Y = y_j).$$
  2. Pour montrer l'independance : verifier la factorisation pour tous les couples, ou utiliser les fonctions generatrices (\(G_{(X,Y)}(s,t) = G_X(s) \cdot G_Y(t)\)).
  3. Pour montrer la non-independance : il suffit de trouver un seul couple \((x_i, y_j)\) pour lequel la factorisation echoue.
  4. Consequence : si \(X \perp Y\), alors \(E(XY) = E(X)\,E(Y)\) et \(V(X + Y) = V(X) + V(Y)\).
  5. Piege : \(E(XY) = E(X)E(Y)\) n'implique pas l'independance (non-correlation n'implique pas independance).
Exemple resolu

Non-correlation sans independance. Soit \(X\) uniforme sur \(\{-1, 0, 1\}\) et \(Y = X^2\). Montrer que \(\text{Cov}(X, Y) = 0\) mais que \(X\) et \(Y\) ne sont pas independantes.

Resolution. On a \(P(X = k) = 1/3\) pour \(k \in \{-1, 0, 1\}\). Alors \(E(X) = 0\) (par symetrie) et :

$$E(XY) = E(X \cdot X^2) = E(X^3) = \frac{(-1)^3 + 0^3 + 1^3}{3} = 0$$

Donc \(\text{Cov}(X, Y) = E(XY) - E(X)E(Y) = 0 - 0 = 0\). Les VA sont non correlees.

Cependant, \(P(X = 0, Y = 1) = 0\) (car \(X = 0 \Rightarrow Y = 0\)), tandis que \(P(X = 0) \cdot P(Y = 1) = \frac{1}{3} \times \frac{2}{3} = \frac{2}{9} \neq 0\). Donc \(X\) et \(Y\) ne sont pas independantes.

M3. Calculer la covariance

Methode
  1. Definition : \(\text{Cov}(X, Y) = E\bigl((X - E(X))(Y - E(Y))\bigr) = E(XY) - E(X)\,E(Y)\).
  2. Calcul pratique : utiliser la formule developpee. Calculer d'abord \(E(X)\), \(E(Y)\), puis \(E(XY)\) par la loi conjointe : $$E(XY) = \sum_{i,j} x_i\, y_j\, P(X = x_i, Y = y_j).$$
  3. Proprietes : $$V(X + Y) = V(X) + V(Y) + 2\,\text{Cov}(X, Y)$$ $$V(aX + bY) = a^2 V(X) + b^2 V(Y) + 2ab\,\text{Cov}(X, Y)$$
  4. Coefficient de correlation : \(\rho(X, Y) = \frac{\text{Cov}(X, Y)}{\sigma(X)\,\sigma(Y)} \in [-1, 1]\).
  5. Bilinearite : \(\text{Cov}(aX + bY, Z) = a\,\text{Cov}(X, Z) + b\,\text{Cov}(Y, Z)\).
Exemple resolu

Variance d'une somme. Soient \(X\) et \(Y\) de loi conjointe : \(P(X=0,Y=0) = 1/4\), \(P(X=0,Y=1) = 1/4\), \(P(X=1,Y=0) = 1/4\), \(P(X=1,Y=1) = 1/4\). Calculer \(\text{Cov}(X,Y)\) et \(V(X+Y)\).

Resolution. Marginales : \(P(X = 0) = P(X = 1) = 1/2\) et \(P(Y = 0) = P(Y = 1) = 1/2\).

Donc \(E(X) = 1/2\), \(E(Y) = 1/2\), \(V(X) = 1/4\), \(V(Y) = 1/4\).

$$E(XY) = 0 \cdot 0 \cdot \frac{1}{4} + 0 \cdot 1 \cdot \frac{1}{4} + 1 \cdot 0 \cdot \frac{1}{4} + 1 \cdot 1 \cdot \frac{1}{4} = \frac{1}{4}$$
$$\text{Cov}(X,Y) = E(XY) - E(X)E(Y) = \frac{1}{4} - \frac{1}{2} \cdot \frac{1}{2} = 0$$

Ici on verifie aussi que \(P(X=x, Y=y) = P(X=x)P(Y=y)\) pour tout couple : \(X\) et \(Y\) sont independantes.

$$V(X + Y) = V(X) + V(Y) + 2\,\text{Cov}(X,Y) = \frac{1}{4} + \frac{1}{4} + 0 = \frac{1}{2}$$

M4. Lois usuelles (tableau B, Bin, P, G)

Methode

Tableau recapitulatif des lois discretes fondamentales :

  • Bernoulli \(\mathcal{B}(1, p)\) : \(X(\Omega) = \{0, 1\}\), \(P(X=1) = p\). \(E = p\), \(V = p(1-p)\), \(G_X(t) = 1 - p + pt\).
  • Binomiale \(\mathcal{B}(n, p)\) : \(P(X=k) = \binom{n}{k}p^k(1-p)^{n-k}\). \(E = np\), \(V = np(1-p)\), \(G_X(t) = (1-p+pt)^n\).
  • Poisson \(\mathcal{P}(\lambda)\) : \(P(X=k) = e^{-\lambda}\frac{\lambda^k}{k!}\). \(E = V = \lambda\), \(G_X(t) = e^{\lambda(t-1)}\).
  • Geometrique \(\mathcal{G}(p)\) : \(P(X=k) = p(1-p)^{k-1}\), \(k \geq 1\). \(E = 1/p\), \(V = (1-p)/p^2\), \(G_X(t) = \frac{pt}{1-(1-p)t}\).

Strategie de reconnaissance :

  1. Nombre de succes en \(n\) epreuves independantes avec remise : binomiale.
  2. Temps d'attente du premier succes : geometrique.
  3. Nombre d'evenements rares sur un grand nombre d'essais : Poisson.
  4. Verifier \(E = V\) pour suspecter une Poisson.
Exemple resolu

Reconnaissance de loi. Un standard telephonique recoit en moyenne \(4\) appels par heure. Quel est le nombre d'appels en une heure ? En deduire \(P(X \geq 2)\).

Resolution. Les appels sont des evenements rares, independants, de taux constant : \(X \sim \mathcal{P}(4)\).

$$P(X \geq 2) = 1 - P(X = 0) - P(X = 1) = 1 - e^{-4} - 4e^{-4} = 1 - 5e^{-4}$$

Numeriquement : \(e^{-4} \approx 0{,}0183\), donc \(P(X \geq 2) \approx 1 - 5 \times 0{,}0183 = 1 - 0{,}0916 \approx 0{,}908\).

Verification : \(E(X) = 4\), \(V(X) = 4\). L'egalite \(E = V\) confirme la loi de Poisson.

M5. Approximation de Poisson

Methode

Lorsque \(n\) est grand, \(p\) est petit, et \(\lambda = np\) est modere, la loi binomiale \(\mathcal{B}(n, p)\) est bien approchee par la loi de Poisson \(\mathcal{P}(\lambda)\) :

$$\binom{n}{k} p^k (1-p)^{n-k} \approx e^{-\lambda} \frac{\lambda^k}{k!}, \quad \lambda = np$$
  1. Regle pratique : l'approximation est bonne si \(n \geq 30\) et \(p \leq 0{,}1\) (ou \(np \leq 10\)).
  2. Justification : si \(X_n \sim \mathcal{B}(n, \lambda/n)\), alors pour tout \(k\) fixe, \(P(X_n = k) \to e^{-\lambda}\frac{\lambda^k}{k!}\) quand \(n \to +\infty\).
  3. Par les FG : \(G_{X_n}(t) = (1 - \lambda/n + \lambda t/n)^n \to e^{\lambda(t-1)} = G_{\mathcal{P}(\lambda)}(t)\).
Exemple resolu

Defauts sur une chaine. Une chaine de production fabrique \(n = 1000\) composants par jour. Chaque composant a une probabilite \(p = 0{,}002\) d'etre defectueux (independamment). Soit \(X\) le nombre de composants defectueux. Approcher \(P(X \geq 3)\).

Resolution. On a \(X \sim \mathcal{B}(1000,\, 0{,}002)\). Comme \(n\) est grand et \(p\) petit, on approche par \(\mathcal{P}(\lambda)\) avec \(\lambda = np = 2\).

$$P(X \geq 3) \approx 1 - P(X = 0) - P(X = 1) - P(X = 2)$$
$$= 1 - e^{-2}\left(1 + 2 + \frac{4}{2}\right) = 1 - 5e^{-2} \approx 1 - 0{,}6767 = 0{,}3233$$

Avec la loi binomiale exacte, on trouverait \(P(X \geq 3) \approx 0{,}3222\). L'approximation de Poisson est tres bonne (erreur relative inferieure a \(0{,}4\%\)).

II. Vrai ou Faux

Testez vos connaissances 0 / 10

1. La loi conjointe de \((X, Y)\) est entierement determinee par les lois marginales de \(X\) et \(Y\).

Les lois marginales ne suffisent pas a determiner la loi conjointe. Deux couples \((X, Y)\) peuvent avoir les memes marginales mais des lois conjointes differentes. Ce n'est que sous l'hypothese d'independance que la loi conjointe est le produit des marginales. Contre-exemple classique : \(X \sim \mathcal{B}(1, 1/2)\), \(Y = X\) versus \(Y = 1 - X\) donnent les memes marginales mais des lois conjointes differentes.

2. Si \(\text{Cov}(X, Y) = 0\), alors \(X\) et \(Y\) sont independantes.

La non-correlation (\(\text{Cov}(X,Y) = 0\)) n'implique pas l'independance. Exemple classique : \(X\) uniforme sur \(\{-1, 0, 1\}\) et \(Y = X^2\). Alors \(\text{Cov}(X, Y) = E(X^3) - E(X)E(X^2) = 0\), mais \(Y\) est fonction deterministe de \(X\), donc ils ne sont pas independants. La reciproque est vraie : l'independance implique la non-correlation.

3. La covariance verifie \(\text{Cov}(X, X) = V(X)\).

Par definition : \(\text{Cov}(X, X) = E(X \cdot X) - E(X) \cdot E(X) = E(X^2) - E(X)^2 = V(X)\). La covariance generalise la variance au cas de deux variables.

4. Si \(X \sim \mathcal{B}(100,\, 0{,}01)\), on peut approcher \(X\) par une loi \(\mathcal{P}(1)\).

On a \(n = 100\) (grand), \(p = 0{,}01\) (petit), et \(\lambda = np = 1\) (modere). Les conditions de l'approximation de Poisson sont satisfaites. Par exemple, \(P(X = 0) = 0{,}99^{100} \approx 0{,}366\) a comparer avec \(e^{-1} \approx 0{,}368\).

5. Le coefficient de correlation \(\rho(X, Y)\) peut valoir \(2\).

L'inegalite de Cauchy-Schwarz donne \(|\text{Cov}(X,Y)| \leq \sigma(X)\,\sigma(Y)\), d'ou \(\rho(X,Y) \in [-1, 1]\). L'egalite \(|\rho| = 1\) n'est atteinte que lorsque \(Y\) est une fonction affine de \(X\) (\(Y = aX + b\) avec \(a \neq 0\)).

6. Pour deux VA independantes \(X \sim \mathcal{B}(n_1, p)\) et \(Y \sim \mathcal{B}(n_2, p)\), on a \(X + Y \sim \mathcal{B}(n_1 + n_2, p)\).

Par les fonctions generatrices : \(G_{X+Y}(t) = G_X(t) \cdot G_Y(t) = (1-p+pt)^{n_1}(1-p+pt)^{n_2} = (1-p+pt)^{n_1+n_2}\). On reconnait la FG d'une \(\mathcal{B}(n_1 + n_2, p)\). Attention : il faut le meme parametre \(p\) et l'independance.

7. La variance est toujours additive : \(V(X + Y) = V(X) + V(Y)\).

En general, \(V(X+Y) = V(X) + V(Y) + 2\,\text{Cov}(X,Y)\). L'additivite n'est valable que si \(\text{Cov}(X,Y) = 0\), ce qui est garanti par l'independance mais peut aussi arriver sans independance (non-correlation). Exemple : si \(Y = -X\), \(V(X+Y) = V(0) = 0 \neq V(X) + V(Y) = 2V(X)\).

8. Si \(X\) et \(Y\) sont independantes, alors \(f(X)\) et \(g(Y)\) sont independantes pour toutes fonctions \(f, g\).

C'est un resultat fondamental. Si \(X\) et \(Y\) sont independantes, alors pour tous \(a, b\) : \(P(f(X) = a, g(Y) = b) = P(X \in f^{-1}(\{a\}), Y \in g^{-1}(\{b\})) = P(X \in f^{-1}(\{a\})) \cdot P(Y \in g^{-1}(\{b\})) = P(f(X) = a) \cdot P(g(Y) = b)\).

9. Pour \(X \sim \mathcal{P}(\lambda)\) et \(Y \sim \mathcal{P}(\mu)\) independantes, \(\text{Cov}(X + Y, X - Y) = \lambda - \mu\).

Par bilinearite de la covariance et independance de \(X\) et \(Y\) :

\(\text{Cov}(X+Y, X-Y) = \text{Cov}(X,X) - \text{Cov}(X,Y) + \text{Cov}(Y,X) - \text{Cov}(Y,Y)\).

Comme \(X \perp Y\), \(\text{Cov}(X,Y) = 0\), donc \(\text{Cov}(X+Y, X-Y) = V(X) - V(Y) = \lambda - \mu\).

10. Trois VA \(X, Y, Z\) sont mutuellement independantes des que toute paire est independante.

L'independance deux a deux n'implique pas l'independance mutuelle. L'independance mutuelle exige que \(P(X = x, Y = y, Z = z) = P(X = x)\,P(Y = y)\,P(Z = z)\) pour tous \(x, y, z\), ce qui est strictement plus fort. Contre-exemple classique : on lance deux pieces. \(X = \mathbf{1}_{\text{pile piece 1}}\), \(Y = \mathbf{1}_{\text{pile piece 2}}\), \(Z = \mathbf{1}_{X \neq Y}\). Chaque paire est independante mais le triplet ne l'est pas car \(Z\) est determine par \(X\) et \(Y\).

III. Exercices corriges

Monier Ch. 17 Exercice 1 -- Loi conjointe et marginales

Le couple \((X, Y)\) a pour loi conjointe :

\(P(X=0,Y=0) = 0{,}1\), \(P(X=0,Y=1) = 0{,}2\), \(P(X=0,Y=2) = 0{,}1\),
\(P(X=1,Y=0) = 0{,}15\), \(P(X=1,Y=1) = 0{,}25\), \(P(X=1,Y=2) = 0{,}2\).

  1. Determiner les lois marginales de \(X\) et de \(Y\).
  2. Calculer \(E(X)\), \(E(Y)\), \(V(X)\) et \(V(Y)\).
  3. \(X\) et \(Y\) sont-elles independantes ?
  4. Calculer \(\text{Cov}(X, Y)\) et \(\rho(X, Y)\).
Correction

1. Marginale de \(X\) :

  • \(P(X = 0) = 0{,}1 + 0{,}2 + 0{,}1 = 0{,}4\)
  • \(P(X = 1) = 0{,}15 + 0{,}25 + 0{,}2 = 0{,}6\)

Marginale de \(Y\) :

  • \(P(Y = 0) = 0{,}1 + 0{,}15 = 0{,}25\)
  • \(P(Y = 1) = 0{,}2 + 0{,}25 = 0{,}45\)
  • \(P(Y = 2) = 0{,}1 + 0{,}2 = 0{,}3\)

Verification : \(0{,}4 + 0{,}6 = 1\) et \(0{,}25 + 0{,}45 + 0{,}3 = 1\). Correct.

2. \(E(X) = 0 \times 0{,}4 + 1 \times 0{,}6 = 0{,}6\). \(E(X^2) = 0{,}6\), donc \(V(X) = 0{,}6 - 0{,}36 = 0{,}24\).

\(E(Y) = 0 \times 0{,}25 + 1 \times 0{,}45 + 2 \times 0{,}3 = 1{,}05\). \(E(Y^2) = 0 + 0{,}45 + 4 \times 0{,}3 = 1{,}65\), donc \(V(Y) = 1{,}65 - 1{,}1025 = 0{,}5475\).

3. Verifions : \(P(X=0)\,P(Y=0) = 0{,}4 \times 0{,}25 = 0{,}1 = P(X=0,Y=0)\). Continuons : \(P(X=0)\,P(Y=1) = 0{,}4 \times 0{,}45 = 0{,}18 \neq 0{,}2 = P(X=0,Y=1)\).

Donc \(X\) et \(Y\) ne sont pas independantes.

4. Calculons \(E(XY)\) par la loi conjointe :

$$E(XY) = \sum_{i,j} x_i\,y_j\,P(X=x_i, Y=y_j)$$

Les seuls termes non nuls sont ceux avec \(x_i = 1\) et \(y_j \geq 1\) :

$$E(XY) = 1 \times 1 \times 0{,}25 + 1 \times 2 \times 0{,}2 = 0{,}25 + 0{,}4 = 0{,}65$$
$$\text{Cov}(X,Y) = E(XY) - E(X)E(Y) = 0{,}65 - 0{,}6 \times 1{,}05 = 0{,}65 - 0{,}63 = 0{,}02$$

Coefficient de correlation : \(\sigma(X) = \sqrt{0{,}24} \approx 0{,}4899\) et \(\sigma(Y) = \sqrt{0{,}5475} \approx 0{,}7399\).

$$\rho(X,Y) = \frac{0{,}02}{0{,}4899 \times 0{,}7399} \approx \frac{0{,}02}{0{,}3625} \approx 0{,}0552$$

La correlation est tres faible et positive malgre la non-independance.

∎

Monier Ch. 17 Exercice 2 -- Independance et loi du couple

On tire une carte au hasard dans un jeu de \(32\) cartes. Soient \(X = \mathbf{1}_{\text{coeur}}\) et \(Y = \mathbf{1}_{\text{figure}}\).

  1. Determiner la loi conjointe de \((X, Y)\).
  2. \(X\) et \(Y\) sont-elles independantes ? Justifier.
  3. Calculer \(\text{Cov}(X, Y)\).
  4. Calculer \(E(X + Y)\) et \(V(X + Y)\).
Correction

1. Dans un jeu de 32 cartes : 8 coeurs (dont 3 figures : valet, dame, roi), 4 couleurs, 12 figures au total (3 par couleur).

  • \(P(X=1, Y=1) = 3/32\) (figure de coeur)
  • \(P(X=1, Y=0) = 5/32\) (coeur non-figure : 7, 8, 9, 10, as)
  • \(P(X=0, Y=1) = 9/32\) (figure non-coeur)
  • \(P(X=0, Y=0) = 15/32\) (ni coeur ni figure)

Verification : \(3 + 5 + 9 + 15 = 32\). Correct.

2. Marginales : \(P(X=1) = 8/32 = 1/4\), \(P(Y=1) = 12/32 = 3/8\).

Verifions : \(P(X=1)\,P(Y=1) = 1/4 \times 3/8 = 3/32 = P(X=1, Y=1)\). Verifions les autres :

  • \(P(X=1)\,P(Y=0) = 1/4 \times 5/8 = 5/32 = P(X=1, Y=0)\). Correct.
  • \(P(X=0)\,P(Y=1) = 3/4 \times 3/8 = 9/32 = P(X=0, Y=1)\). Correct.
  • \(P(X=0)\,P(Y=0) = 3/4 \times 5/8 = 15/32 = P(X=0, Y=0)\). Correct.

Toutes les factorisations sont verifiees : \(X\) et \(Y\) sont independantes.

3. Par independance, \(\text{Cov}(X,Y) = 0\). Verification directe : \(E(XY) = P(X=1,Y=1) = 3/32\), \(E(X)\,E(Y) = 1/4 \times 3/8 = 3/32\). Donc \(\text{Cov}(X,Y) = 0\).

4. \(E(X+Y) = E(X) + E(Y) = 1/4 + 3/8 = 5/8\).

\(V(X) = 1/4 \times 3/4 = 3/16\) et \(V(Y) = 3/8 \times 5/8 = 15/64\).

Par independance : \(V(X+Y) = V(X) + V(Y) = 3/16 + 15/64 = 12/64 + 15/64 = 27/64\).

∎

Monier Ch. 17 Exercice 3 -- Covariance et combinaisons lineaires

Soient \(X\) et \(Y\) deux VA de meme loi \(\mathcal{B}(1, 1/2)\) avec \(\text{Cov}(X, Y) = -1/4\). On pose \(U = X + Y\) et \(V = X - Y\).

  1. Calculer \(E(U)\), \(E(V)\), \(V(U)\) et \(V(V)\).
  2. Calculer \(\text{Cov}(U, V)\). Que peut-on en deduire ?
  3. Determiner la loi de \(U\) et la loi de \(V\).
  4. Montrer que \(Y = 1 - X\) est la seule possibilite pour obtenir \(\text{Cov}(X,Y) = -1/4\).
Correction

1. \(E(X) = E(Y) = 1/2\), \(V(X) = V(Y) = 1/4\).

$$E(U) = E(X) + E(Y) = 1, \quad E(V) = E(X) - E(Y) = 0$$
$$V(U) = V(X) + V(Y) + 2\,\text{Cov}(X,Y) = \frac{1}{4} + \frac{1}{4} - \frac{1}{2} = 0$$
$$V(V) = V(X) + V(Y) - 2\,\text{Cov}(X,Y) = \frac{1}{4} + \frac{1}{4} + \frac{1}{2} = 1$$

2. Par bilinearite :

$$\text{Cov}(U,V) = \text{Cov}(X+Y, X-Y) = V(X) - \text{Cov}(X,Y) + \text{Cov}(Y,X) - V(Y)$$
$$= V(X) - V(Y) = \frac{1}{4} - \frac{1}{4} = 0$$

\(U\) et \(V\) sont non correlees. Mais attention, cela n'implique pas l'independance.

3. Comme \(V(U) = 0\), \(U\) est presque surement constante : \(U = E(U) = 1\) p.s. Donc \(X + Y = 1\) p.s., ce qui signifie \(Y = 1 - X\) p.s.

Pour \(V = X - Y = X - (1-X) = 2X - 1\) : comme \(X \sim \mathcal{B}(1, 1/2)\), on a \(V \in \{-1, 1\}\) avec \(P(V = -1) = P(V = 1) = 1/2\). C'est une loi de Rademacher.

4. Si \(X, Y \sim \mathcal{B}(1, 1/2)\), alors \(\text{Cov}(X,Y) = E(XY) - 1/4\). Comme \(XY \in \{0,1\}\), on a \(E(XY) = P(X=1, Y=1)\). Donc \(\text{Cov}(X,Y) = P(X=1,Y=1) - 1/4 = -1/4\) implique \(P(X=1,Y=1) = 0\). Comme de plus \(P(X=0,Y=0) = 1 - P(X=1) - P(Y=1) + P(X=1,Y=1) = 0\), le seul cas est \(Y = 1-X\) p.s. La covariance \(-1/4\) est le minimum possible (correlation \(\rho = -1\)).

∎

Monier Ch. 18 Exercice 4 -- Approximation de Poisson

Dans un lot de \(n = 500\) composants electroniques, chacun a une probabilite \(p = 0{,}004\) d'etre defectueux (independamment). Soit \(X\) le nombre de defectueux.

  1. Donner la loi exacte de \(X\). Pourquoi est-il difficile de calculer directement \(P(X = k)\) ?
  2. Justifier l'approximation de Poisson et donner le parametre \(\lambda\).
  3. Calculer (par l'approximation) \(P(X = 0)\), \(P(X = 1)\), \(P(X = 2)\), \(P(X \geq 3)\).
  4. On accepte le lot si \(X \leq 3\). Quelle est la probabilite d'acceptation ?
Correction

1. \(X \sim \mathcal{B}(500,\, 0{,}004)\). Le calcul direct de \(P(X = k) = \binom{500}{k}(0{,}004)^k(0{,}996)^{500-k}\) est numeriquement instable car \(\binom{500}{k}\) est immense et \(p^k\) est minuscule.

2. On a \(n = 500 \geq 30\), \(p = 0{,}004 \leq 0{,}1\), et \(\lambda = np = 2\). L'approximation de Poisson est justifiee : \(X \approx \mathcal{P}(2)\).

3. Avec \(\lambda = 2\) et \(e^{-2} \approx 0{,}13534\) :

  • \(P(X = 0) \approx e^{-2} \approx 0{,}1353\)
  • \(P(X = 1) \approx 2e^{-2} \approx 0{,}2707\)
  • \(P(X = 2) \approx 2e^{-2} \approx 0{,}2707\)
$$P(X \geq 3) \approx 1 - e^{-2}(1 + 2 + 2) = 1 - 5e^{-2} \approx 0{,}3233$$

4. Probabilite d'acceptation (\(X \leq 3\)) :

$$P(X \leq 3) = P(X \leq 2) + P(X = 3) \approx 5e^{-2} + \frac{8}{6}\,e^{-2} = e^{-2}\left(5 + \frac{4}{3}\right) = \frac{19}{3}\,e^{-2} \approx 0{,}8571$$

Il y a environ \(85{,}7\%\) de chances que le lot soit accepte.

∎

Monier Ch. 17 Exercice 5 -- Loi conjointe et esperance conditionnelle

On dispose de deux urnes. L'urne \(A\) contient \(3\) boules blanches et \(2\) noires. L'urne \(B\) contient \(1\) blanche et \(4\) noires. On lance un de equilibre : si on obtient \(1\) ou \(2\), on tire dans l'urne \(A\) ; sinon dans l'urne \(B\). Soit \(X = \mathbf{1}_{\text{urne } A}\) et \(Y = \mathbf{1}_{\text{boule blanche}}\).

  1. Determiner la loi conjointe de \((X, Y)\).
  2. \(X\) et \(Y\) sont-elles independantes ?
  3. Calculer \(P(Y = 1)\) par les probabilites totales.
  4. Calculer \(\text{Cov}(X, Y)\) et interpreter le signe.
Correction

1. \(P(X = 1) = 2/6 = 1/3\) et \(P(X = 0) = 4/6 = 2/3\).

  • \(P(X=1, Y=1) = P(X=1)\,P(Y=1 \mid X=1) = \frac{1}{3} \times \frac{3}{5} = \frac{1}{5}\)
  • \(P(X=1, Y=0) = \frac{1}{3} \times \frac{2}{5} = \frac{2}{15}\)
  • \(P(X=0, Y=1) = \frac{2}{3} \times \frac{1}{5} = \frac{2}{15}\)
  • \(P(X=0, Y=0) = \frac{2}{3} \times \frac{4}{5} = \frac{8}{15}\)

Verification : \(\frac{1}{5} + \frac{2}{15} + \frac{2}{15} + \frac{8}{15} = \frac{3 + 2 + 2 + 8}{15} = 1\). Correct.

2. \(P(Y = 1) = 1/5 + 2/15 = 3/15 + 2/15 = 1/3\). Verifions la factorisation : \(P(X=1)\,P(Y=1) = 1/3 \times 1/3 = 1/9\), mais \(P(X=1, Y=1) = 1/5 \neq 1/9\). Donc \(X\) et \(Y\) ne sont pas independantes.

3. Par les probabilites totales :

$$P(Y=1) = P(X=1)\,P(Y=1 \mid X=1) + P(X=0)\,P(Y=1 \mid X=0) = \frac{1}{3} \times \frac{3}{5} + \frac{2}{3} \times \frac{1}{5} = \frac{1}{5} + \frac{2}{15} = \frac{1}{3}$$

4. \(E(XY) = P(X=1, Y=1) = 1/5\). \(E(X) = 1/3\), \(E(Y) = 1/3\).

$$\text{Cov}(X,Y) = \frac{1}{5} - \frac{1}{3} \times \frac{1}{3} = \frac{1}{5} - \frac{1}{9} = \frac{9 - 5}{45} = \frac{4}{45} > 0$$

La covariance est positive : quand on tire dans l'urne \(A\) (plus riche en boules blanches), on a plus de chances d'obtenir une blanche. Il y a une dependance positive entre le choix de l'urne et la couleur.

∎

Monier Ch. 18 Exercice 6 -- Somme de VA independantes et stabilite

Soient \(X_1, \ldots, X_n\) des VA independantes avec \(X_i \sim \mathcal{P}(\lambda_i)\). On pose \(S = X_1 + \cdots + X_n\).

  1. Determiner la loi de \(S\) par les fonctions generatrices.
  2. Application : \(X \sim \mathcal{P}(3)\) et \(Y \sim \mathcal{P}(5)\) independantes. Calculer \(P(X + Y = 2)\) et \(E(XY)\).
  3. Montrer que si \(S = X + Y \sim \mathcal{P}(\lambda + \mu)\) et \(X \sim \mathcal{P}(\lambda)\), \(Y \sim \mathcal{P}(\mu)\) independantes, alors \(P(X = k \mid S = n) = \binom{n}{k}\left(\frac{\lambda}{\lambda+\mu}\right)^k\left(\frac{\mu}{\lambda+\mu}\right)^{n-k}\).
  4. Reconnaitre la loi conditionnelle de \(X\) sachant \(S = n\).
Correction

1. Par independance et propriete multiplicative de la FG :

$$G_S(t) = \prod_{i=1}^{n} G_{X_i}(t) = \prod_{i=1}^{n} e^{\lambda_i(t-1)} = e^{(\lambda_1 + \cdots + \lambda_n)(t-1)}$$

On reconnait la FG d'une loi \(\mathcal{P}(\Lambda)\) avec \(\Lambda = \sum_{i=1}^n \lambda_i\). Par unicite :

$$\boxed{S \sim \mathcal{P}\left(\sum_{i=1}^n \lambda_i\right)}$$

2. \(X + Y \sim \mathcal{P}(8)\). Donc :

$$P(X+Y = 2) = e^{-8}\frac{8^2}{2!} = 32e^{-8} \approx 32 \times 0{,}000\,3355 \approx 0{,}01073$$

Pour \(E(XY)\) : par independance, \(E(XY) = E(X)\,E(Y) = 3 \times 5 = 15\).

3. Par definition de la probabilite conditionnelle :

$$P(X = k \mid S = n) = \frac{P(X = k, Y = n-k)}{P(S = n)} = \frac{P(X=k)\,P(Y=n-k)}{P(S=n)}$$

En remplacant par les formules de Poisson :

$$= \frac{e^{-\lambda}\frac{\lambda^k}{k!} \cdot e^{-\mu}\frac{\mu^{n-k}}{(n-k)!}}{e^{-(\lambda+\mu)}\frac{(\lambda+\mu)^n}{n!}} = \frac{n!}{k!(n-k)!} \cdot \frac{\lambda^k \mu^{n-k}}{(\lambda+\mu)^n}$$
$$= \binom{n}{k}\left(\frac{\lambda}{\lambda+\mu}\right)^k\left(\frac{\mu}{\lambda+\mu}\right)^{n-k}$$

4. On reconnait une loi binomiale : \((X \mid S = n) \sim \mathcal{B}\left(n, \frac{\lambda}{\lambda + \mu}\right)\). Sachant que la somme vaut \(n\), le nombre d'evenements de type \(X\) se repartit selon une binomiale avec parametre de succes egal a la proportion des intensites.

∎

Monier Ch. 18 Exercice 7 -- Variance et covariance dans un modele d'urne

On tire \(n\) boules avec remise dans une urne contenant \(N\) boules dont \(R\) rouges. Pour \(i = 1, \ldots, n\), on pose \(X_i = \mathbf{1}_{\text{la }i\text{-eme boule est rouge}}\) et \(S_n = \sum_{i=1}^n X_i\).

  1. Donner la loi de \(X_i\) et calculer \(E(X_i)\), \(V(X_i)\).
  2. Les \(X_i\) sont-elles independantes ? En deduire la loi de \(S_n\), \(E(S_n)\) et \(V(S_n)\).
  3. On tire maintenant sans remise (\(n \leq N\)). Calculer \(\text{Cov}(X_i, X_j)\) pour \(i \neq j\).
  4. En deduire \(V(S_n)\) dans le cas sans remise et comparer avec le cas avec remise.
Correction

1. Avec remise : \(X_i \sim \mathcal{B}(1, p)\) avec \(p = R/N\). Donc \(E(X_i) = p\) et \(V(X_i) = p(1-p)\).

2. Avec remise, les tirages sont independants, donc les \(X_i\) sont independantes. Par consequent :

$$S_n \sim \mathcal{B}(n, p), \quad E(S_n) = np, \quad V(S_n) = np(1-p)$$

3. Sans remise, pour \(i \neq j\) :

$$\text{Cov}(X_i, X_j) = E(X_i X_j) - E(X_i)E(X_j)$$

\(E(X_i X_j) = P(X_i = 1, X_j = 1)\) = probabilite que les \(i\)-eme et \(j\)-eme boules soient rouges :

$$P(X_i = 1, X_j = 1) = \frac{R}{N} \cdot \frac{R-1}{N-1}$$

Donc :

$$\text{Cov}(X_i, X_j) = \frac{R(R-1)}{N(N-1)} - \frac{R^2}{N^2} = \frac{R^2(N) - R \cdot N - R^2(N-1)}{N^2(N-1)}$$

Simplifions : \(\frac{R(R-1)}{N(N-1)} - p^2 = p \cdot \frac{R-1}{N-1} - p^2 = p\left(\frac{R-1}{N-1} - p\right)\).

Or \(\frac{R-1}{N-1} - \frac{R}{N} = \frac{N(R-1) - R(N-1)}{N(N-1)} = \frac{-N+R}{N(N-1)} = \frac{-(N-R)}{N(N-1)}\). Donc :

$$\text{Cov}(X_i, X_j) = -\frac{p(1-p)}{N-1}$$

La covariance est negative : si une boule est rouge, la proportion de rouges restantes diminue.

4. Par bilinearite de la variance :

$$V(S_n) = \sum_{i=1}^n V(X_i) + 2\sum_{i < j} \text{Cov}(X_i, X_j) = np(1-p) + n(n-1) \times \left(-\frac{p(1-p)}{N-1}\right)$$
$$= np(1-p)\left(1 - \frac{n-1}{N-1}\right) = np(1-p) \cdot \frac{N-n}{N-1}$$

Le facteur \(\frac{N-n}{N-1}\) est le facteur de correction de population finie. Comme \(\frac{N-n}{N-1} \leq 1\), la variance sans remise est toujours inferieure ou egale a celle avec remise. Quand \(N \gg n\), le facteur est proche de \(1\) et l'on retrouve la variance binomiale.

∎
