Series entieres & EDL

Fiche de methodes avec exercices types CCINP : rayon de convergence, developpement en serie entiere, resolution d'EDL par series entieres, somme de series et theoreme d'Abel radial. Chaque methode est accompagnee d'un exercice d'annale ou de style concours.

Sommaire
  1. M1. Calculer le rayon de convergence \(R\)
  2. M2. Obtenir un DSE par operations
  3. M3. Resoudre une EDL par series entieres
  4. M4. Somme d'une serie entiere
  5. M5. Theoreme d'Abel radial

M1. Calculer le rayon de convergence \(R\)

Methode
  1. Regle de d'Alembert : calculer \(\displaystyle L = \lim_{n \to +\infty} \left| \frac{a_{n+1}}{a_n} \right|\). Alors \(R = \dfrac{1}{L}\) (avec \(R = +\infty\) si \(L = 0\) et \(R = 0\) si \(L = +\infty\)).
  2. Regle de Hadamard : \(\displaystyle \frac{1}{R} = \limsup_{n \to +\infty} |a_n|^{1/n}\).
  3. Par comparaison : si \(|a_n| \sim |b_n|\) ou \(a_n = O(b_n)\), comparer avec une serie de rayon connu.
  4. Etudier la convergence aux bornes \(x = \pm R\) separement (series numeriques).
Astuce

La regle de d'Alembert est la plus rapide quand \(a_n\) fait intervenir des factorielles ou des puissances. La regle de Hadamard est utile quand \(a_n = f(n)^n\).

Style CCINP Exercice -- Style CCINP

Determiner le rayon de convergence de la serie entiere \(\displaystyle \sum_{n \geq 1} \frac{n! \, x^n}{n^n}\).

Indication : appliquer la regle de d'Alembert. On calcule \(\left|\dfrac{a_{n+1}}{a_n}\right| = \dfrac{(n+1)!}{(n+1)^{n+1}} \cdot \dfrac{n^n}{n!} = \left(\dfrac{n}{n+1}\right)^n \to e^{-1}\). Donc \(R = e\).

M2. Obtenir un DSE par operations

Methode
  1. Derivation terme a terme : si \(f(x) = \sum a_n x^n\) pour \(|x| < R\), alors \(f'(x) = \sum n a_n x^{n-1}\) avec le meme rayon \(R\).
  2. Integration terme a terme : \(\displaystyle \int_0^x f(t)\,dt = \sum \frac{a_n}{n+1} x^{n+1}\) avec le meme rayon \(R\).
  3. Substitution : remplacer \(x\) par \(g(x)\) dans un DSE connu (ex : \(x^2\), \(-x\), etc.) en verifiant que \(|g(x)| < R\).
  4. Produit de Cauchy : si \(f(x) = \sum a_n x^n\) et \(g(x) = \sum b_n x^n\), alors \(f(x)g(x) = \sum c_n x^n\) avec \(c_n = \sum_{k=0}^n a_k b_{n-k}\).
DSE de reference

Retenir : \(\dfrac{1}{1-x} = \sum_{n=0}^{+\infty} x^n\), \(e^x = \sum \dfrac{x^n}{n!}\), \(\ln(1+x) = \sum \dfrac{(-1)^{n+1}}{n} x^n\), \((1+x)^\alpha = \sum \binom{\alpha}{n} x^n\).

Style CCINP Exercice -- Style CCINP

Trouver le developpement en serie entiere de \(f(x) = \dfrac{x}{(1-x)^2}\) pour \(|x| < 1\).

Indication : on part de \(\dfrac{1}{1-x} = \sum_{n=0}^{+\infty} x^n\). En derivant : \(\dfrac{1}{(1-x)^2} = \sum_{n=1}^{+\infty} n x^{n-1} = \sum_{n=0}^{+\infty} (n+1) x^n\). Donc \(\dfrac{x}{(1-x)^2} = \sum_{n=0}^{+\infty} (n+1) x^{n+1} = \sum_{n=1}^{+\infty} n x^n\).

M3. Resoudre une EDL par series entieres

Methode
  1. Poser \(y(x) = \displaystyle\sum_{n=0}^{+\infty} a_n x^n\) avec rayon de convergence \(R > 0\) a determiner.
  2. Calculer \(y'(x) = \sum_{n=1}^{+\infty} n a_n x^{n-1}\) et eventuellement \(y''(x)\).
  3. Injecter dans l'EDL et reindexer les sommes pour que toutes portent sur \(x^n\).
  4. Identifier les coefficients : pour tout \(n\), le coefficient de \(x^n\) est nul. Cela donne une relation de recurrence sur les \((a_n)\).
  5. Resoudre la recurrence, determiner \(R\) et reconnaitre la somme si possible.
Astuce

Quand l'EDL a des coefficients polynomiaux en \(x\) (comme \(y' - 2xy = 0\)), la methode des series entieres est particulierement adaptee car la multiplication par \(x\) decale simplement les indices.

Style CCINP 2024 Exercice -- Style CCINP 2024 Maths 1

Resoudre l'equation differentielle \(y' - 2xy = 0\) par developpement en serie entiere.

  1. Poser \(y = \sum a_n x^n\). Injecter dans l'EDL.
  2. Montrer que la relation de recurrence donne \((n+1)a_{n+1} = 2a_{n-1}\) pour \(n \geq 1\) et \(a_1 = 0\).
  3. En deduire que les coefficients d'indice impair sont nuls et que \(a_{2p} = \dfrac{a_0}{p!}\).
  4. Conclure que \(y(x) = a_0 \displaystyle\sum_{p=0}^{+\infty} \frac{x^{2p}}{p!} = a_0 \, e^{x^2}\), de rayon \(R = +\infty\).

M4. Somme d'une serie entiere

Methode
  1. Poser \(S(x) = \sum_{n=0}^{+\infty} a_n x^n\) et determiner le rayon de convergence \(R\).
  2. Si \(a_n\) contient un facteur \(n\) : reconnaitre que \(\sum n x^n = x \left(\sum x^n\right)'\). Deriver ou multiplier par \(x\) pour se ramener a une serie connue.
  3. Si \(a_n\) contient un facteur \(\frac{1}{n}\) : reconnaitre une primitive. Integrer la serie pour se ramener a une forme connue.
  4. Combiner derivation et integration si necessaire (ex : \(n^2 = n(n-1) + n\)).
  5. Evaluer la somme en forme close.
Decomposition utile

Pour \(n^2 x^n\), ecrire \(n^2 = n(n-1) + n\). Alors \(\sum n^2 x^n = \sum n(n-1)x^n + \sum n x^n = x^2 S''(x) + x S'(x)\) avec \(S(x) = \dfrac{1}{1-x}\).

Style CCINP Exercice -- Style CCINP

Calculer \(\displaystyle S(x) = \sum_{n=1}^{+\infty} n^2 x^n\) pour \(|x| < 1\).

Indication : on pose \(f(x) = \sum x^n = \dfrac{1}{1-x}\). Alors \(f'(x) = \dfrac{1}{(1-x)^2}\) et \(xf'(x) = \sum n x^n = \dfrac{x}{(1-x)^2}\). En derivant a nouveau : \((xf'(x))' = \sum n^2 x^{n-1}\), donc \(\sum n^2 x^n = x(xf'(x))' = x \cdot \dfrac{1+x}{(1-x)^3} = \dfrac{x(1+x)}{(1-x)^3}\).

M5. Theoreme d'Abel radial

Methode
  1. Theoreme d'Abel radial : si la serie numerique \(\sum a_n\) converge, alors la serie entiere \(\sum a_n x^n\) (de rayon \(R \geq 1\)) verifie \(\displaystyle \lim_{x \to 1^-} \sum_{n=0}^{+\infty} a_n x^n = \sum_{n=0}^{+\infty} a_n\).
  2. On peut donc calculer la somme d'une serie numerique en l'interpretant comme la limite d'une serie entiere dont on connait la somme.
  3. Identifier la serie entiere \(f(x) = \sum a_n x^n\) et calculer \(f(x)\) en forme close sur \(]-R, R[\).
  4. Passer a la limite \(x \to 1^-\) : par Abel, \(\sum a_n = \lim_{x \to 1^-} f(x)\).
Attention

La reciproque est fausse : la convergence de \(f(x)\) quand \(x \to 1^-\) n'implique pas la convergence de \(\sum a_n\). Le theoreme d'Abel ne fonctionne que dans un sens.

Style CCINP Exercice -- Style CCINP

Montrer que \(\displaystyle \ln 2 = \sum_{n=1}^{+\infty} \frac{(-1)^{n+1}}{n}\) en utilisant le theoreme d'Abel radial.

  1. On sait que \(\ln(1+x) = \sum_{n=1}^{+\infty} \dfrac{(-1)^{n+1}}{n} x^n\) pour \(|x| < 1\).
  2. La serie numerique \(\sum \dfrac{(-1)^{n+1}}{n}\) est une serie alternee convergente (critere de Leibniz).
  3. Par le theoreme d'Abel radial : \(\displaystyle \lim_{x \to 1^-} \ln(1+x) = \sum_{n=1}^{+\infty} \frac{(-1)^{n+1}}{n}\).
  4. Or \(\lim_{x \to 1^-} \ln(1+x) = \ln 2\). Conclusion : \(\ln 2 = \sum_{n=1}^{+\infty} \dfrac{(-1)^{n+1}}{n}\).
