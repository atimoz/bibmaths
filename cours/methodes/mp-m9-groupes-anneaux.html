Groupes & Anneaux

Fiche de methodes avec exercices types : structures de groupes, morphismes, congruences (theoreme chinois), indicateur d'Euler et elements remarquables des anneaux. Chaque methode est accompagnee d'un exercice d'annale ou de style concours.

Sommaire
  1. M1. Montrer qu'un ensemble est un groupe
  2. M2. Montrer qu'une application est un morphisme de groupes
  3. M3. Resoudre un systeme de congruences (theoreme chinois)
  4. M4. Manipuler l'indicateur d'Euler
  5. M5. Etudier les elements d'un anneau

M1. Montrer qu'un ensemble est un groupe

Methode
  1. Verifier que la loi \(\star\) est une loi de composition interne : pour tous \(x, y \in G\), \(x \star y \in G\).
  2. Verifier l'associativite : pour tous \(x, y, z \in G\), \((x \star y) \star z = x \star (y \star z)\).
  3. Verifier l'existence d'un element neutre \(e \in G\) tel que pour tout \(x \in G\), \(e \star x = x \star e = x\).
  4. Verifier l'existence d'un symetrique : pour tout \(x \in G\), il existe \(x' \in G\) tel que \(x \star x' = x' \star x = e\).
  5. Alternative (sous-groupe) : montrer que \(H \subset G\) est un sous-groupe en verifiant : \(H \neq \emptyset\), et pour tous \(x, y \in H\), \(x \star y^{-1} \in H\).
Astuce

Pour montrer qu'un ensemble fini muni d'une loi interne et associative est un groupe, il suffit souvent de montrer la simplifiabilite (a gauche et a droite), ce qui entraine l'existence du neutre et des inverses.

Style CCINP Exercice -- Style CCINP

On considere l'ensemble \(G = \left\{ \begin{pmatrix} a & b \\ 0 & 1 \end{pmatrix} \mid a \in \mathbb{R}^*, \, b \in \mathbb{R} \right\}\) muni du produit matriciel.

  1. Montrer que \(G\) est un sous-groupe de \(\mathrm{GL}_2(\mathbb{R})\).
  2. \(G\) est-il abelien ?
  3. Determiner le centre de \(G\).

Indication : verifier que le produit de deux matrices de cette forme reste dans \(G\), et calculer l'inverse explicitement.

M2. Montrer qu'une application est un morphisme de groupes

Methode
  1. Identifier clairement les groupes de depart \((G, \star)\) et d'arrivee \((G', \diamond)\).
  2. Montrer que pour tous \(x, y \in G\), on a \(f(x \star y) = f(x) \diamond f(y)\).
  3. Pour un isomorphisme, montrer en plus que \(f\) est bijective.
  4. Exploiter les consequences : \(f(e_G) = e_{G'}\), \(f(x^{-1}) = f(x)^{-1}\), \(\ker f\) est un sous-groupe distingue de \(G\).
  5. Si \(\ker f = \{e_G\}\), alors \(f\) est injectif.
Astuce

Pour montrer qu'un morphisme est un isomorphisme entre groupes finis de meme cardinal, il suffit de montrer l'injectivite (ou la surjectivite).

Style CCINP Exercice -- Style CCINP

On definit \(\varphi : (\mathbb{R}^*, \times) \to (\mathbb{R}^*, \times)\) par \(\varphi(x) = x^3\).

  1. Montrer que \(\varphi\) est un morphisme de groupes.
  2. Determiner \(\ker \varphi\) et \(\mathrm{Im}(\varphi)\).
  3. \(\varphi\) est-il un isomorphisme ?
  4. Meme questions avec \(\psi : (\mathbb{C}^*, \times) \to (\mathbb{C}^*, \times)\) definie par \(\psi(z) = z^3\).

Indication : sur \(\mathbb{R}^*\), tout reel admet une unique racine cubique reelle, donc \(\varphi\) est bijectif. Sur \(\mathbb{C}^*\), le noyau contient les racines cubiemes de l'unite.

M3. Resoudre un systeme de congruences (theoreme chinois)

Methode
  1. Ecrire le systeme sous la forme \(\begin{cases} x \equiv a_1 \pmod{n_1} \\ x \equiv a_2 \pmod{n_2} \\ \vdots \end{cases}\)
  2. Verifier que les moduli \(n_1, n_2, \ldots\) sont deux a deux premiers entre eux.
  3. Appliquer le theoreme chinois : il existe une unique solution modulo \(N = n_1 \cdot n_2 \cdots n_k\).
  4. Pour chaque \(i\), poser \(N_i = N / n_i\) et trouver \(u_i\) tel que \(N_i u_i \equiv 1 \pmod{n_i}\) (Bezout ou Euclide etendu).
  5. La solution est \(x \equiv \sum_{i=1}^{k} a_i N_i u_i \pmod{N}\).
Astuce

Pour deux congruences, on peut aussi proceder par substitution directe : ecrire \(x = a_1 + n_1 t\) et injecter dans la seconde equation pour trouver \(t\).

Style CCINP Exercice -- Style CCINP

Resoudre le systeme de congruences :

$$\begin{cases} x \equiv 2 \pmod{5} \\ x \equiv 4 \pmod{7} \\ x \equiv 1 \pmod{3} \end{cases}$$

  1. Verifier que les moduli sont deux a deux premiers entre eux.
  2. Appliquer la methode du theoreme chinois pour trouver l'unique solution modulo \(105\).
  3. Determiner tous les entiers \(x \in \{1, \ldots, 500\}\) verifiant ce systeme.

Indication : \(N = 105\). Calculer \(N_1 = 21\), \(N_2 = 15\), \(N_3 = 35\) et les inverses modulaires correspondants. La solution est \(x \equiv 67 \pmod{105}\).

M4. Manipuler l'indicateur d'Euler

Methode
  1. Rappel : \(\varphi(n)\) est le nombre d'entiers \(k \in \{1, \ldots, n\}\) tels que \(\mathrm{pgcd}(k, n) = 1\).
  2. Formules cles : \(\varphi(p) = p - 1\) pour \(p\) premier, \(\varphi(p^\alpha) = p^{\alpha-1}(p-1)\).
  3. Multiplicativite : si \(\mathrm{pgcd}(m, n) = 1\), alors \(\varphi(mn) = \varphi(m)\varphi(n)\).
  4. Theoreme d'Euler : si \(\mathrm{pgcd}(a, n) = 1\), alors \(a^{\varphi(n)} \equiv 1 \pmod{n}\).
  5. Application : pour calculer \(a^k \pmod{n}\), reduire \(k\) modulo \(\varphi(n)\).
Cas particulier : petit Fermat

Si \(p\) est premier et \(p \nmid a\), alors \(a^{p-1} \equiv 1 \pmod{p}\). C'est le cas particulier du theoreme d'Euler avec \(n = p\).

Style CCINP Exercice -- Style CCINP

Soit \(n = 60\).

  1. Calculer \(\varphi(60)\) en utilisant la decomposition \(60 = 2^2 \times 3 \times 5\).
  2. En deduire le reste de la division euclidienne de \(7^{100}\) par \(60\).
  3. Determiner le dernier chiffre de \(3^{1000}\) (c'est-a-dire \(3^{1000} \pmod{10}\)).
  4. Montrer que pour tout \(a\) premier avec \(60\), on a \(a^4 \equiv 1 \pmod{60}\) ou donner un contre-exemple.

Indication : \(\varphi(60) = \varphi(4)\varphi(3)\varphi(5) = 2 \times 2 \times 4 = 16\). Donc \(7^{16} \equiv 1 \pmod{60}\) et \(100 = 6 \times 16 + 4\).

M5. Etudier les elements d'un anneau

Methode
  1. Inversibles : \(a\) est inversible dans \(A\) s'il existe \(b \in A\) tel que \(ab = ba = 1_A\). Dans \(\mathbb{Z}/n\mathbb{Z}\), \(\bar{a}\) est inversible ssi \(\mathrm{pgcd}(a, n) = 1\).
  2. Nilpotents : \(a\) est nilpotent s'il existe \(k \in \mathbb{N}^*\) tel que \(a^k = 0\). Chercher le plus petit tel \(k\) (indice de nilpotence).
  3. Idempotents : \(a\) est idempotent si \(a^2 = a\). Les seuls idempotents triviaux sont \(0\) et \(1\).
  4. Diviseurs de zero : \(a \neq 0\) est diviseur de zero s'il existe \(b \neq 0\) tel que \(ab = 0\).
  5. Un element nilpotent non nul est toujours diviseur de zero. Un inversible n'est jamais diviseur de zero.
Astuce

Dans \(\mathbb{Z}/n\mathbb{Z}\), on peut trouver tous les nilpotents, idempotents et inversibles en testant systematiquement. Les idempotents non triviaux n'existent que si \(n\) n'est pas une puissance de nombre premier.

Style CCINP Exercice -- Style CCINP

On travaille dans l'anneau \(\mathbb{Z}/12\mathbb{Z}\).

  1. Determiner l'ensemble des inversibles de \(\mathbb{Z}/12\mathbb{Z}\). Verifier que c'est un groupe pour la multiplication.
  2. Trouver tous les diviseurs de zero de \(\mathbb{Z}/12\mathbb{Z}\).
  3. Trouver tous les idempotents (elements \(\bar{a}\) tels que \(\bar{a}^2 = \bar{a}\)).
  4. Trouver tous les nilpotents et leur indice de nilpotence.

Indication : les inversibles sont les \(\bar{a}\) avec \(\mathrm{pgcd}(a, 12) = 1\), soit \(\{\bar{1}, \bar{5}, \bar{7}, \bar{11}\}\). Pour les idempotents, resoudre \(a^2 \equiv a \pmod{12}\), soit \(a(a-1) \equiv 0 \pmod{12}\).
