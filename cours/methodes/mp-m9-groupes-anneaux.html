I. Methodes a retenir

M1. Montrer qu'un ensemble est un groupe

Methode

Pour montrer que \((G, \star)\) est un groupe, on verifie les quatre axiomes :

  1. Loi interne : pour tous \(x, y \in G\), \(x \star y \in G\).
  2. Associativite : pour tous \(x, y, z \in G\), \((x \star y) \star z = x \star (y \star z)\).
  3. Element neutre : il existe \(e \in G\) tel que pour tout \(x \in G\), \(e \star x = x \star e = x\).
  4. Symetrique : pour tout \(x \in G\), il existe \(x' \in G\) tel que \(x \star x' = x' \star x = e\).

Critere de sous-groupe : pour montrer que \(H \subset G\) est un sous-groupe, il suffit de verifier :

  1. \(H \neq \emptyset\) (en general \(e \in H\)).
  2. Pour tous \(x, y \in H\), \(x \star y^{-1} \in H\).
Exemple resolu

Montrer que \((\mathbb{Z}/5\mathbb{Z})^* = \{\bar{1}, \bar{2}, \bar{3}, \bar{4}\}\) est un groupe pour la multiplication.

Resolution. L'ensemble \((\mathbb{Z}/5\mathbb{Z})^*\) est forme des classes inversibles modulo \(5\). Comme \(5\) est premier, tout \(\bar{a}\) avec \(a \not\equiv 0\) est inversible, d'ou \((\mathbb{Z}/5\mathbb{Z})^* = \{\bar{1}, \bar{2}, \bar{3}, \bar{4}\}\).

  1. Loi interne : le produit de deux elements non nuls modulo un nombre premier est non nul (car \(\mathbb{Z}/5\mathbb{Z}\) est integre).
  2. Associativite : heritee de la multiplication dans \(\mathbb{Z}/5\mathbb{Z}\).
  3. Neutre : \(\bar{1}\).
  4. Inverses : \(\bar{1}^{-1} = \bar{1}\), \(\bar{2}^{-1} = \bar{3}\) (car \(2 \times 3 = 6 \equiv 1\)), \(\bar{4}^{-1} = \bar{4}\) (car \(4 \times 4 = 16 \equiv 1\)).

Le groupe est abelien et de cardinal \(4 = \varphi(5)\). C'est un groupe cyclique engendre par \(\bar{2}\) : on a \(\bar{2}^1 = \bar{2}\), \(\bar{2}^2 = \bar{4}\), \(\bar{2}^3 = \bar{3}\), \(\bar{2}^4 = \bar{1}\).

M2. Montrer qu'une application est un morphisme

Methode
  1. Identifier les groupes \((G, \star)\) et \((G', \diamond)\).
  2. Montrer que pour tous \(x, y \in G\) : \(f(x \star y) = f(x) \diamond f(y)\).
  3. Consequences automatiques : \(f(e_G) = e_{G'}\) et \(f(x^{-1}) = f(x)^{-1}\).
  4. Pour l'injectivite : montrer que \(\ker f = \{e_G\}\).
  5. Pour un isomorphisme : bijectivite (entre groupes finis de meme cardinal, l'injectivite suffit).
Exemple resolu

Montrer que \(\det : (\mathrm{GL}_n(\mathbb{K}), \times) \to (\mathbb{K}^*, \times)\) est un morphisme de groupes.

Resolution. Pour toutes matrices \(A, B \in \mathrm{GL}_n(\mathbb{K})\) :

$$\det(AB) = \det(A) \cdot \det(B)$$

C'est la multiplicativite du determinant. Donc \(\det\) est un morphisme de groupes.

  • Noyau : \(\ker(\det) = \{A \in \mathrm{GL}_n(\mathbb{K}) \mid \det(A) = 1\} = \mathrm{SL}_n(\mathbb{K})\) (groupe special lineaire).
  • Surjectivite : pour tout \(\lambda \in \mathbb{K}^*\), la matrice \(\mathrm{diag}(\lambda, 1, \ldots, 1)\) a pour determinant \(\lambda\). Donc \(\det\) est surjectif.
  • Par le premier theoreme d'isomorphisme : \(\mathrm{GL}_n(\mathbb{K}) / \mathrm{SL}_n(\mathbb{K}) \simeq \mathbb{K}^*\).

M3. Resoudre un systeme de congruences (theoreme chinois)

Methode
  1. Ecrire le systeme : \(\begin{cases} x \equiv a_1 \pmod{n_1} \\ x \equiv a_2 \pmod{n_2} \end{cases}\)
  2. Verifier que \(\mathrm{pgcd}(n_1, n_2) = 1\) (condition du theoreme chinois).
  3. Methode par substitution : ecrire \(x = a_1 + n_1 t\) et injecter dans la seconde congruence pour trouver \(t\).
  4. Methode systematique : poser \(N = n_1 n_2\), \(N_1 = N/n_1\), \(N_2 = N/n_2\). Trouver \(u_1, u_2\) tels que \(N_1 u_1 \equiv 1 \pmod{n_1}\) et \(N_2 u_2 \equiv 1 \pmod{n_2}\).
  5. Solution : \(x \equiv a_1 N_1 u_1 + a_2 N_2 u_2 \pmod{N}\).
Exemple resolu

Resoudre : \(\begin{cases} x \equiv 2 \pmod{5} \\ x \equiv 3 \pmod{7} \end{cases}\)

Resolution par substitution. On ecrit \(x = 2 + 5t\) avec \(t \in \mathbb{Z}\). On injecte dans la seconde : \(2 + 5t \equiv 3 \pmod{7}\), soit \(5t \equiv 1 \pmod{7}\).

On cherche l'inverse de \(5\) modulo \(7\) : \(5 \times 3 = 15 \equiv 1 \pmod{7}\), donc \(t \equiv 3 \pmod{7}\).

D'ou \(x = 2 + 5 \times 3 = 17\) et l'ensemble des solutions est :

$$x \equiv 17 \pmod{35}$$

M4. Indicateur d'Euler et theoreme d'Euler

Methode
  1. Calculer \(\varphi(n)\) : decomposer \(n = p_1^{\alpha_1} \cdots p_r^{\alpha_r}\) puis \(\varphi(n) = n \prod_{i=1}^{r}\left(1 - \frac{1}{p_i}\right)\).
  2. Theoreme d'Euler : si \(\mathrm{pgcd}(a, n) = 1\), alors \(a^{\varphi(n)} \equiv 1 \pmod{n}\).
  3. Pour calculer \(a^k \pmod{n}\) : reduire l'exposant \(k\) modulo \(\varphi(n)\).
  4. Petit Fermat : si \(p\) est premier et \(p \nmid a\), alors \(a^{p-1} \equiv 1 \pmod{p}\).
Exemple resolu

Calculer le reste de \(3^{100}\) dans la division par \(40\).

Resolution. On a \(40 = 2^3 \times 5\), donc :

$$\varphi(40) = 40 \times \left(1 - \frac{1}{2}\right)\left(1 - \frac{1}{5}\right) = 40 \times \frac{1}{2} \times \frac{4}{5} = 16$$

Comme \(\mathrm{pgcd}(3, 40) = 1\), le theoreme d'Euler donne \(3^{16} \equiv 1 \pmod{40}\).

Division euclidienne : \(100 = 6 \times 16 + 4\), donc :

$$3^{100} = (3^{16})^6 \times 3^4 \equiv 1^6 \times 81 \equiv 81 \pmod{40}$$

Or \(81 = 2 \times 40 + 1\), donc \(3^{100} \equiv 1 \pmod{40}\).

M5. Elements d'un anneau (nilpotents, idempotents, inversibles)

Methode
  1. Inversibles de \(\mathbb{Z}/n\mathbb{Z}\) : \(\bar{a}\) inversible ssi \(\mathrm{pgcd}(a, n) = 1\). L'ensemble \((\mathbb{Z}/n\mathbb{Z})^*\) forme un groupe de cardinal \(\varphi(n)\).
  2. Nilpotents : \(\bar{a}\) nilpotent ssi \(a^k \equiv 0 \pmod{n}\) pour un certain \(k\). Cela revient a : tout facteur premier de \(n\) divise \(a\).
  3. Idempotents : resoudre \(a^2 \equiv a \pmod{n}\), soit \(a(a-1) \equiv 0 \pmod{n}\). On utilise le theoreme chinois si \(n\) n'est pas une puissance de premier.
  4. Diviseurs de zero : \(\bar{a} \neq \bar{0}\) tel qu'il existe \(\bar{b} \neq \bar{0}\) avec \(\bar{a} \cdot \bar{b} = \bar{0}\). Ce sont exactement les non-nuls non inversibles.
Exemple resolu

Etudier les elements de \(\mathbb{Z}/12\mathbb{Z}\).

Resolution. On a \(12 = 2^2 \times 3\).

  • Inversibles : \(\bar{a}\) avec \(\mathrm{pgcd}(a, 12) = 1\) : \((\mathbb{Z}/12\mathbb{Z})^* = \{\bar{1}, \bar{5}, \bar{7}, \bar{11}\}\). On a \(\varphi(12) = 4\). C'est le groupe de Klein \(\mathbb{Z}/2\mathbb{Z} \times \mathbb{Z}/2\mathbb{Z}\) car chaque element est d'ordre \(2\) sauf \(\bar{1}\).
  • Nilpotents : il faut que \(2\) et \(3\) divisent \(a\), donc \(6 \mid a\). Les nilpotents sont \(\{\bar{0}, \bar{6}\}\). En effet \(\bar{6}^2 = \overline{36} = \bar{0}\), d'indice \(2\).
  • Idempotents : resoudre \(a^2 \equiv a \pmod{12}\). On trouve \(\bar{0}, \bar{1}, \bar{4}, \bar{9}\). Verification : \(4^2 = 16 \equiv 4\) et \(9^2 = 81 \equiv 9\).
  • Diviseurs de zero : tous les elements non nuls et non inversibles, soit \(\{\bar{2}, \bar{3}, \bar{4}, \bar{6}, \bar{8}, \bar{9}, \bar{10}\}\).

II. Vrai ou Faux

Pour chaque affirmation, determiner si elle est vraie ou fausse, puis cliquer pour verifier.

Testez vos connaissances 0 / 10

1. Tout groupe d'ordre \(4\) est cyclique.

Le groupe de Klein \(V_4 = \mathbb{Z}/2\mathbb{Z} \times \mathbb{Z}/2\mathbb{Z}\) est d'ordre \(4\) mais n'est pas cyclique (tous ses elements non neutres sont d'ordre \(2\)). Il y a exactement deux groupes d'ordre \(4\) a isomorphisme pres : \(\mathbb{Z}/4\mathbb{Z}\) (cyclique) et \(V_4\).

2. Si \(f : G \to G'\) est un morphisme de groupes et \(\ker f = \{e\}\), alors \(f\) est un isomorphisme.

\(\ker f = \{e\}\) implique que \(f\) est injectif, mais pas necessairement surjectif. Par exemple, \(f : \mathbb{Z} \to \mathbb{Z}\), \(n \mapsto 2n\) est un morphisme injectif de \((\mathbb{Z}, +)\) dans lui-meme, mais pas surjectif.

3. Dans un groupe fini, l'ordre de tout element divise l'ordre du groupe.

C'est le theoreme de Lagrange : l'ordre d'un element \(g\) est egal au cardinal du sous-groupe \(\langle g \rangle\) engendre par \(g\), et le cardinal d'un sous-groupe divise celui du groupe.

4. L'anneau \(\mathbb{Z}/6\mathbb{Z}\) est integre.

\(\bar{2} \times \bar{3} = \bar{6} = \bar{0}\) dans \(\mathbb{Z}/6\mathbb{Z}\), avec \(\bar{2} \neq \bar{0}\) et \(\bar{3} \neq \bar{0}\). Donc \(\mathbb{Z}/6\mathbb{Z}\) a des diviseurs de zero et n'est pas integre. Rappel : \(\mathbb{Z}/n\mathbb{Z}\) est integre si et seulement si \(n\) est premier.

5. \(\varphi(15) = \varphi(3) \times \varphi(5) = 2 \times 4 = 8\).

Comme \(\mathrm{pgcd}(3, 5) = 1\), la multiplicativite de l'indicateur d'Euler donne \(\varphi(15) = \varphi(3) \cdot \varphi(5) = 2 \times 4 = 8\). On peut verifier : les entiers entre \(1\) et \(15\) premiers avec \(15\) sont \(1, 2, 4, 7, 8, 11, 13, 14\).

6. Tout sous-groupe d'un groupe abelien est distingue.

Si \(G\) est abelien, pour tout sous-groupe \(H\) et tout \(g \in G\), on a \(gH = Hg\) car la loi est commutative. Donc tout sous-groupe est distingue.

7. L'image d'un morphisme de groupes est un sous-groupe du groupe d'arrivee.

\(\mathrm{Im}(f)\) contient \(e_{G'} = f(e_G)\) donc est non vide. Si \(y_1 = f(x_1)\) et \(y_2 = f(x_2)\) sont dans \(\mathrm{Im}(f)\), alors \(y_1 y_2^{-1} = f(x_1) f(x_2)^{-1} = f(x_1 x_2^{-1}) \in \mathrm{Im}(f)\).

8. Dans \(\mathbb{Z}/15\mathbb{Z}\), l'element \(\bar{6}\) est nilpotent.

Les nilpotents de \(\mathbb{Z}/15\mathbb{Z}\) sont les \(\bar{a}\) tels que tout facteur premier de \(15\) (c'est-a-dire \(3\) et \(5\)) divise \(a\). Il faut \(15 \mid a\), donc le seul nilpotent est \(\bar{0}\). En effet, \(6^2 = 36 \equiv 6 \pmod{15}\), donc \(\bar{6}\) est idempotent, pas nilpotent.

9. Le systeme \(\begin{cases} x \equiv 1 \pmod{4} \\ x \equiv 2 \pmod{6} \end{cases}\) admet une solution.

\(\mathrm{pgcd}(4, 6) = 2\). La condition necessaire \(a_1 \equiv a_2 \pmod{\mathrm{pgcd}(n_1, n_2)}\) donne \(1 \equiv 2 \pmod{2}\), soit \(1 \equiv 0 \pmod{2}\), ce qui est faux. Le systeme est incompatible. Attention : le theoreme chinois dans sa forme classique exige des moduli premiers entre eux.

10. Tout groupe d'ordre premier est cyclique.

Soit \(G\) d'ordre \(p\) premier. Soit \(g \in G \setminus \{e\}\). Le sous-groupe \(\langle g \rangle\) a un cardinal qui divise \(p\) (Lagrange), et ce cardinal est \(\geq 2\) car \(g \neq e\). Comme \(p\) est premier, \(|\langle g \rangle| = p = |G|\), donc \(G = \langle g \rangle\) est cyclique.

III. Exercices corriges

Monier Ch. 1 Exercice 1 -- Calculs dans un groupe

Soit \((G, \cdot)\) un groupe. On suppose que deux elements \(a, b \in G\) verifient la relation \(ab = ba^3\).

  1. Exprimer \(ab^n\) en fonction de \(a\) et \(b\) pour tout \(n \in \mathbb{N}\).
  2. En deduire que \(a^{2^n - 1} = e\) pour tout \(n \geq 1\) tel que \(b^n = e\).
  3. Application : si \(G\) est fini et \(b\) est d'ordre \(3\), montrer que \(a^7 = e\).
Correction

1. On montre par recurrence que \(ab^n = b^n a^{3^n}\).

Initialisation : pour \(n = 1\), \(ab = ba^3\), c'est l'hypothese.

Heredite : supposons \(ab^n = b^n a^{3^n}\). Alors :

$$ab^{n+1} = (ab^n) b = b^n a^{3^n} b$$

Il faut commuter \(a^{3^n}\) et \(b\). De la relation \(ab = ba^3\), on deduit \(a^k b = b a^{3k}\) par recurrence sur \(k\) (en effet \(a^{k+1}b = a \cdot a^k b = a \cdot b a^{3k} = ba^3 \cdot a^{3k} = ba^{3(k+1)}\)). Donc :

$$ab^{n+1} = b^n \cdot b \cdot a^{3 \cdot 3^n} = b^{n+1} a^{3^{n+1}}$$

2. Si \(b^n = e\), alors \(ab^n = a \cdot e = a\) et par la formule ci-dessus \(ab^n = b^n a^{3^n} = e \cdot a^{3^n} = a^{3^n}\). Donc \(a = a^{3^n}\), soit \(a^{3^n - 1} = e\).

3. Si \(b\) est d'ordre \(3\), on a \(b^3 = e\), donc \(a^{3^3 - 1} = a^{26} = e\). De plus, \(b^1 \neq e\) et \(b^2 \neq e\), mais on peut aussi utiliser \(n = 2\) : \(b^2 \neq e\) ne permet pas de conclure directement. Cependant \(a^{26} = e\) signifie que l'ordre de \(a\) divise \(26 = 2 \times 13\). En utilisant \(n = 1\) : \(ab = ba^3\) et \(b^3 = e\) donnent \(a^{3^3 - 1} = a^{26} = e\). De plus, avec la relation \(a = a^{3^3}\) et les ordres possibles divisant \(26\), si \(G\) a la bonne structure, on montre que \(a^7 = e\). En fait, reprenons : \(b^3 = e\) donne \(a^{26} = e\). Mais on veut \(a^7 = e\). Utilisons aussi \(n=1\) : \(ab = ba^3\) implique \(b^{-1}ab = a^3\), donc par conjugaison iteree \(b^{-k}ab^k = a^{3^k}\). Pour \(k = 3\) : \(b^{-3}ab^3 = a^{27}\), et comme \(b^3 = e\), on obtient \(a = a^{27}\), soit \(a^{26} = e\). Finalement, en utilisant \(k = 1, 2\) on peut affiner : l'ordre de \(a\) divise a la fois \(3^3 - 1 = 26\) et \(3^1 - 1 = 2\)... Non, pour \(n=1\) il faut \(b^1 = e\) ce qui n'est pas le cas. Corrigeons l'enonce : montrons \(a^{26} = e\), et si en plus l'ordre de \(a\) divise \(7\), alors \(a^7 = e\). En realite, \(\mathrm{pgcd}(26, 3^2 - 1) = \mathrm{pgcd}(26, 8) = 2\). Et \(\mathrm{pgcd}(26, 3^1 - 1) = \mathrm{pgcd}(26, 2) = 2\). Donc l'ordre de \(a\) divise \(26\). Les ordres possibles de \(a\) sont \(1, 2, 13, 26\). Si le groupe est fini, \(a^{26} = e\) est le meilleur resultat general. L'affirmation \(a^7 = e\) necessite une hypothese supplementaire.

∎

Monier Ch. 1 Exercice 2 -- Sous-groupes de \((\mathbb{Z}, +)\) et \((\mathbb{Z}/n\mathbb{Z}, +)\)
  1. Montrer que tout sous-groupe de \((\mathbb{Z}, +)\) est de la forme \(n\mathbb{Z}\) pour un certain \(n \in \mathbb{N}\).
  2. Determiner tous les sous-groupes de \((\mathbb{Z}/18\mathbb{Z}, +)\).
  3. Parmi ces sous-groupes, lesquels sont cycliques ? Donner un generateur pour chacun.
Correction

1. Soit \(H\) un sous-groupe de \((\mathbb{Z}, +)\). Si \(H = \{0\}\), c'est \(0\mathbb{Z}\). Sinon, \(H\) contient un element non nul. Comme \(H\) est stable par passage a l'oppose, \(H\) contient un element strictement positif. Soit \(n = \min(H \cap \mathbb{N}^*)\).

Montrons que \(H = n\mathbb{Z}\). L'inclusion \(n\mathbb{Z} \subset H\) vient du fait que \(n \in H\) et \(H\) est stable par addition et oppose. Pour l'inclusion inverse, soit \(h \in H\). La division euclidienne donne \(h = nq + r\) avec \(0 \leq r < n\). Alors \(r = h - nq \in H\) et \(0 \leq r < n\). Par minimalite de \(n\), on a \(r = 0\), donc \(h \in n\mathbb{Z}\).

2. Les sous-groupes de \((\mathbb{Z}/18\mathbb{Z}, +)\) sont en bijection avec les diviseurs de \(18\). Comme \(18 = 2 \times 3^2\), les diviseurs sont \(1, 2, 3, 6, 9, 18\). Pour chaque diviseur \(d\) de \(18\), le sous-groupe correspondant est \(\langle \overline{18/d} \rangle\), de cardinal \(d\) :

  • \(\{\bar{0}\}\) (cardinal \(1\)),
  • \(\langle \bar{9} \rangle = \{\bar{0}, \bar{9}\}\) (cardinal \(2\)),
  • \(\langle \bar{6} \rangle = \{\bar{0}, \bar{6}, \overline{12}\}\) (cardinal \(3\)),
  • \(\langle \bar{3} \rangle = \{\bar{0}, \bar{3}, \bar{6}, \bar{9}, \overline{12}, \overline{15}\}\) (cardinal \(6\)),
  • \(\langle \bar{2} \rangle = \{\bar{0}, \bar{2}, \bar{4}, \bar{6}, \bar{8}, \overline{10}, \overline{12}, \overline{14}, \overline{16}\}\) (cardinal \(9\)),
  • \(\mathbb{Z}/18\mathbb{Z}\) tout entier (cardinal \(18\)).

3. Tous ces sous-groupes sont cycliques, car tout sous-groupe d'un groupe cyclique est cyclique. Les generateurs sont indiques ci-dessus : \(\bar{9}, \bar{6}, \bar{3}, \bar{2}, \bar{1}\) respectivement (et d'autres generateurs existent : par exemple \(\langle \bar{3} \rangle = \langle \overline{15} \rangle\)).

∎

Monier Ch. 1 Exercice 3 -- Morphismes de groupes et noyau

On considere l'application \(f : (\mathbb{Z}, +) \to (\mathbb{Z}/7\mathbb{Z}, +)\) definie par \(f(n) = \bar{n}\) (la surjection canonique).

  1. Montrer que \(f\) est un morphisme de groupes surjectif.
  2. Determiner \(\ker f\).
  3. En deduire un isomorphisme via le premier theoreme d'isomorphisme.
  4. Soit \(g : (\mathbb{Z}/21\mathbb{Z}, +) \to (\mathbb{Z}/7\mathbb{Z}, +)\) defini par \(g(\bar{n}) = \overline{n \bmod 7}\). Montrer que \(g\) est bien defini et est un morphisme. Determiner \(\ker g\) et \(\mathrm{Im}(g)\).
Correction

1. Pour tous \(m, n \in \mathbb{Z}\) : \(f(m + n) = \overline{m + n} = \bar{m} + \bar{n} = f(m) + f(n)\). Donc \(f\) est un morphisme. Il est surjectif car pour tout \(\bar{k} \in \mathbb{Z}/7\mathbb{Z}\), on a \(f(k) = \bar{k}\).

2. \(\ker f = \{n \in \mathbb{Z} \mid \bar{n} = \bar{0}\} = \{n \in \mathbb{Z} \mid 7 \mid n\} = 7\mathbb{Z}\).

3. Par le premier theoreme d'isomorphisme : \(\mathbb{Z} / \ker f \simeq \mathrm{Im}(f)\), soit \(\mathbb{Z} / 7\mathbb{Z} \simeq \mathbb{Z}/7\mathbb{Z}\). C'est coherent puisque \(f\) est la surjection canonique qui realise cet isomorphisme.

4. L'application \(g\) est la reduction modulo \(7\). Elle est bien definie car si \(n \equiv n' \pmod{21}\), alors \(21 \mid (n - n')\), donc \(7 \mid (n - n')\), d'ou \(n \equiv n' \pmod{7}\). C'est un morphisme car \(g(\bar{m} + \bar{n}) = g(\overline{m+n}) = \overline{(m+n) \bmod 7} = \overline{m \bmod 7} + \overline{n \bmod 7} = g(\bar{m}) + g(\bar{n})\).

\(\ker g = \{\bar{n} \in \mathbb{Z}/21\mathbb{Z} \mid 7 \mid n\} = \{\bar{0}, \bar{7}, \overline{14}\} = \langle \bar{7} \rangle\), sous-groupe d'ordre \(3\).

\(\mathrm{Im}(g) = \mathbb{Z}/7\mathbb{Z}\) (surjectif). On retrouve : \(|\mathbb{Z}/21\mathbb{Z}| = |\ker g| \times |\mathrm{Im}(g)| = 3 \times 7 = 21\).

∎

Monier Ch. 2 Exercice 4 -- Resolution de congruences dans \(\mathbb{Z}\)

Resoudre les systemes de congruences suivants :

  1. \(\begin{cases} x \equiv 3 \pmod{7} \\ x \equiv 5 \pmod{11} \end{cases}\)
  2. \(\begin{cases} x \equiv 1 \pmod{3} \\ x \equiv 2 \pmod{5} \\ x \equiv 4 \pmod{7} \end{cases}\)
  3. Determiner le plus petit entier naturel \(x > 0\) verifiant \(x \equiv 2 \pmod{3}\), \(x \equiv 3 \pmod{5}\) et \(x \equiv 2 \pmod{7}\).
Correction

1. Les moduli \(7\) et \(11\) sont premiers entre eux. On ecrit \(x = 3 + 7t\) et on injecte : \(3 + 7t \equiv 5 \pmod{11}\), soit \(7t \equiv 2 \pmod{11}\). L'inverse de \(7\) modulo \(11\) : \(7 \times 8 = 56 = 5 \times 11 + 1\), donc \(7^{-1} \equiv 8 \pmod{11}\). Ainsi \(t \equiv 16 \equiv 5 \pmod{11}\), d'ou \(x = 3 + 35 = 38\).

$$\boxed{x \equiv 38 \pmod{77}}$$

2. Les moduli \(3, 5, 7\) sont deux a deux premiers entre eux. \(N = 105\).

Premiere etape : resoudre les deux premieres congruences. \(x = 1 + 3t\), puis \(1 + 3t \equiv 2 \pmod{5}\), soit \(3t \equiv 1 \pmod{5}\). Inverse de \(3\) modulo \(5\) : \(3 \times 2 = 6 \equiv 1\), donc \(t \equiv 2 \pmod{5}\), et \(x = 1 + 6 = 7\). Donc \(x \equiv 7 \pmod{15}\).

Seconde etape : \(x = 7 + 15s\), puis \(7 + 15s \equiv 4 \pmod{7}\), soit \(15s \equiv -3 \equiv 4 \pmod{7}\). Or \(15 \equiv 1 \pmod{7}\), donc \(s \equiv 4 \pmod{7}\), et \(x = 7 + 60 = 67\).

$$\boxed{x \equiv 67 \pmod{105}}$$

3. Moduli \(3, 5, 7\) premiers entre eux, \(N = 105\). On ecrit \(x = 2 + 3t\), puis \(2 + 3t \equiv 3 \pmod{5}\), soit \(3t \equiv 1 \pmod{5}\), d'ou \(t \equiv 2 \pmod{5}\) et \(x = 2 + 6 = 8\), soit \(x \equiv 8 \pmod{15}\).

Puis \(x = 8 + 15s\), et \(8 + 15s \equiv 2 \pmod{7}\), soit \(15s \equiv -6 \equiv 1 \pmod{7}\). Comme \(15 \equiv 1 \pmod{7}\), on a \(s \equiv 1 \pmod{7}\) et \(x = 8 + 15 = 23\).

$$\boxed{x = 23}$$

Verification : \(23 = 7 \times 3 + 2\), \(23 = 4 \times 5 + 3\), \(23 = 3 \times 7 + 2\). C'est correct.

∎

Monier Ch. 2 Exercice 5 -- Elements inversibles de \(\mathbb{Z}/n\mathbb{Z}\)
  1. Dresser la table de multiplication de \((\mathbb{Z}/8\mathbb{Z})^*\). En deduire la structure de ce groupe.
  2. Montrer que \((\mathbb{Z}/10\mathbb{Z})^*\) est cyclique et trouver un generateur.
  3. Calculer \(3^{200} \pmod{20}\) en utilisant l'indicateur d'Euler.
  4. Determiner l'inverse de \(\bar{7}\) dans \(\mathbb{Z}/18\mathbb{Z}\).
Correction

1. \((\mathbb{Z}/8\mathbb{Z})^* = \{\bar{1}, \bar{3}, \bar{5}, \bar{7}\}\) (les impairs). Table de multiplication :

$$\bar{3}^2 = \bar{9} = \bar{1}, \quad \bar{5}^2 = \overline{25} = \bar{1}, \quad \bar{7}^2 = \overline{49} = \bar{1}$$

Chaque element non neutre est d'ordre \(2\). Le groupe n'est pas cyclique (un groupe cyclique d'ordre \(4\) possede un element d'ordre \(4\)). C'est le groupe de Klein : \((\mathbb{Z}/8\mathbb{Z})^* \simeq \mathbb{Z}/2\mathbb{Z} \times \mathbb{Z}/2\mathbb{Z}\).

2. \((\mathbb{Z}/10\mathbb{Z})^* = \{\bar{1}, \bar{3}, \bar{7}, \bar{9}\}\), de cardinal \(\varphi(10) = 4\). Calculons les puissances de \(\bar{3}\) : \(\bar{3}^1 = \bar{3}\), \(\bar{3}^2 = \bar{9}\), \(\bar{3}^3 = \overline{27} = \bar{7}\), \(\bar{3}^4 = \overline{21} = \bar{1}\). Donc \(\bar{3}\) est d'ordre \(4\) et engendre le groupe : \((\mathbb{Z}/10\mathbb{Z})^*\) est cyclique de generateur \(\bar{3}\).

3. \(20 = 2^2 \times 5\), donc \(\varphi(20) = 20(1 - 1/2)(1 - 1/5) = 8\). Comme \(\mathrm{pgcd}(3, 20) = 1\), le theoreme d'Euler donne \(3^8 \equiv 1 \pmod{20}\). Or \(200 = 25 \times 8\), donc :

$$3^{200} = (3^8)^{25} \equiv 1^{25} = 1 \pmod{20}$$

4. On cherche \(x\) tel que \(7x \equiv 1 \pmod{18}\). Par l'algorithme d'Euclide etendu : \(18 = 2 \times 7 + 4\), \(7 = 1 \times 4 + 3\), \(4 = 1 \times 3 + 1\). Remontee : \(1 = 4 - 3 = 4 - (7 - 4) = 2 \times 4 - 7 = 2(18 - 2 \times 7) - 7 = 2 \times 18 - 5 \times 7\). Donc \(7 \times (-5) \equiv 1 \pmod{18}\), soit \(\bar{7}^{-1} = \overline{-5} = \overline{13}\). Verification : \(7 \times 13 = 91 = 5 \times 18 + 1\).

∎

Monier Ch. 2 Exercice 6 -- Anneaux quotients et ideaux

On travaille dans l'anneau \(\mathbb{Z}/30\mathbb{Z}\).

  1. Determiner tous les inversibles de \(\mathbb{Z}/30\mathbb{Z}\) et calculer \(\varphi(30)\).
  2. Trouver tous les idempotents de \(\mathbb{Z}/30\mathbb{Z}\) (elements \(\bar{a}\) tels que \(\bar{a}^2 = \bar{a}\)).
  3. Trouver tous les nilpotents de \(\mathbb{Z}/30\mathbb{Z}\).
  4. Ecrire l'isomorphisme du theoreme chinois : \(\mathbb{Z}/30\mathbb{Z} \simeq \mathbb{Z}/2\mathbb{Z} \times \mathbb{Z}/3\mathbb{Z} \times \mathbb{Z}/5\mathbb{Z}\), et retrouver les idempotents via cet isomorphisme.
Correction

1. \(30 = 2 \times 3 \times 5\). Donc \(\varphi(30) = 30(1-1/2)(1-1/3)(1-1/5) = 30 \times 1/2 \times 2/3 \times 4/5 = 8\). Les inversibles sont les \(\bar{a}\) avec \(\mathrm{pgcd}(a, 30) = 1\) :

$$(\mathbb{Z}/30\mathbb{Z})^* = \{\bar{1}, \bar{7}, \overline{11}, \overline{13}, \overline{17}, \overline{19}, \overline{23}, \overline{29}\}$$

2. On cherche \(a(a-1) \equiv 0 \pmod{30}\). Par le theoreme chinois, un idempotent de \(\mathbb{Z}/30\mathbb{Z}\) correspond a un triplet d'idempotents dans \(\mathbb{Z}/2\mathbb{Z} \times \mathbb{Z}/3\mathbb{Z} \times \mathbb{Z}/5\mathbb{Z}\). Les idempotents de \(\mathbb{Z}/p\mathbb{Z}\) (premier) sont \(\bar{0}\) et \(\bar{1}\). Il y a donc \(2^3 = 8\) idempotents dans \(\mathbb{Z}/30\mathbb{Z}\).

Calculons-les en resolvant les systemes :

  • \((0,0,0) \to \bar{0}\)
  • \((1,1,1) \to \bar{1}\)
  • \((1,0,0) \to \overline{15}\) (car \(15 \equiv 1 \pmod{2}\), \(15 \equiv 0 \pmod{3}\), \(15 \equiv 0 \pmod{5}\))
  • \((0,1,0) \to \overline{10}\) (car \(10 \equiv 0 \pmod{2}\), \(10 \equiv 1 \pmod{3}\), \(10 \equiv 0 \pmod{5}\))
  • \((0,0,1) \to \bar{6}\) (car \(6 \equiv 0 \pmod{2}\), \(6 \equiv 0 \pmod{3}\), \(6 \equiv 1 \pmod{5}\))
  • \((1,1,0) \to \overline{25}\) (car \(25 \equiv 1 \pmod{2}\), \(25 \equiv 1 \pmod{3}\), \(25 \equiv 0 \pmod{5}\))
  • \((1,0,1) \to \overline{21}\) (car \(21 \equiv 1 \pmod{2}\), \(21 \equiv 0 \pmod{3}\), \(21 \equiv 1 \pmod{5}\))
  • \((0,1,1) \to \overline{16}\) (car \(16 \equiv 0 \pmod{2}\), \(16 \equiv 1 \pmod{3}\), \(16 \equiv 1 \pmod{5}\))

Les \(8\) idempotents sont : \(\bar{0}, \bar{1}, \bar{6}, \overline{10}, \overline{15}, \overline{16}, \overline{21}, \overline{25}\).

3. \(\bar{a}\) est nilpotent dans \(\mathbb{Z}/30\mathbb{Z}\) ssi tous les facteurs premiers de \(30\) divisent \(a\), c'est-a-dire \(30 \mid a\). Donc le seul nilpotent est \(\bar{0}\).

4. L'isomorphisme est \(\phi : \mathbb{Z}/30\mathbb{Z} \to \mathbb{Z}/2\mathbb{Z} \times \mathbb{Z}/3\mathbb{Z} \times \mathbb{Z}/5\mathbb{Z}\), \(\phi(\bar{n}) = (\overline{n \bmod 2}, \overline{n \bmod 3}, \overline{n \bmod 5})\). C'est un isomorphisme d'anneaux. Les idempotents dans le produit sont les triplets \((e_1, e_2, e_3)\) ou chaque \(e_i\) est idempotent dans le facteur correspondant (soit \(\bar{0}\) soit \(\bar{1}\)). On retrouve les \(8\) idempotents ci-dessus.

∎
