I. Structures cristallines

Définitions — Vocabulaire cristallographique
  • Maille : volume élémentaire dont la répétition par translation reconstitue tout le cristal.
  • Nœud : point du réseau cristallin, position géométrique où se trouve un atome ou un motif.
  • Motif : ensemble d'atomes associé à chaque nœud du réseau.
  • Coordinence : nombre de plus proches voisins d'un atome donné.
  • Compacité : fraction du volume de la maille effectivement occupée par les atomes.
Propriété — Structure CFC (Cubique à Faces Centrées)
  • Coordinence : 12
  • Nombre d'atomes par maille : \(Z = 4\)
  • Compacité : \(\displaystyle C = \frac{\pi}{3\sqrt{2}} \approx 74\,\%\)
  • Relation paramètre de maille / rayon : \(a = 2\sqrt{2}\,r\)
  • Sites interstitiels : octaédriques et tétraédriques
Propriété — Structure CC (Cubique Centré)
  • Coordinence : 8
  • Nombre d'atomes par maille : \(Z = 2\)
  • Compacité : \(\displaystyle C = \frac{\pi\sqrt{3}}{8} \approx 68\,\%\)
  • Relation paramètre de maille / rayon : \(a = \frac{4r}{\sqrt{3}}\)
Remarque — Hexagonal compact

La structure hexagonale compacte a la même compacité que la CFC : 74 %. Ce sont les deux empilements compacts (ABCABC pour CFC, ABAB pour HC).

Méthode — Calculer la masse volumique
$$\rho = \frac{Z \cdot M}{N_A \cdot V_{\text{maille}}}$$
  1. Déterminer \(Z\), nombre d'atomes par maille (CFC : \(Z=4\), CC : \(Z=2\)).
  2. Calculer le paramètre de maille \(a\) via les rayons atomiques (contact le long de la face pour CFC, le long de la diagonale pour CC).
  3. Volume de la maille : \(V = a^3\).
  4. Application numérique avec \(M\) en kg/mol et \(N_A = 6{,}022 \times 10^{23}\;\text{mol}^{-1}\).

II. Diffraction de Bragg

Loi de Bragg
$$2\,d_{hkl}\,\sin\theta = n\lambda$$
  • \(d_{hkl}\) : distance inter-réticulaire entre les plans \((hkl)\).
  • \(\theta\) : angle d'incidence rasante (angle de Bragg).
  • \(n\) : ordre de diffraction (entier positif).
  • \(\lambda\) : longueur d'onde du rayonnement.
Distance inter-réticulaire — Cas cubique
$$d_{hkl} = \frac{a}{\sqrt{h^2 + k^2 + l^2}}$$

où \(a\) est le paramètre de maille et \((h,k,l)\) les indices de Miller.

Extinctions systématiques

CFC : seuls les pics avec \(h, k, l\) tous pairs ou tous impairs sont observés. Les pics avec des indices de parité mixte s'éteignent.

CC : seuls les pics avec \(h + k + l\) pair sont observés.

Méthode — Identifier une structure par diffraction
  1. Mesurer les angles \(\theta\) des pics de diffraction.
  2. Calculer \(\sin^2\theta\) pour chaque pic.
  3. Former les rapports des \(\sin^2\theta\) successifs pour identifier \(h^2 + k^2 + l^2\).
  4. CFC : rapports \(3, 4, 8, 11, 12, 16, \dots\)
    CC : rapports \(1, 2, 3, 4, 5, 6, \dots\)

III. Thermochimie

Enthalpie standard de réaction
$$\Delta_r H° = \sum_i \nu_i \,\Delta_f H°(\text{produits}) - \sum_i \nu_i \,\Delta_f H°(\text{réactifs})$$
Enthalpie libre standard de réaction
$$\Delta_r G° = \Delta_r H° - T\,\Delta_r S°$$

Une réaction est thermodynamiquement favorisée dans le sens direct si \(\Delta_r G° < 0\).

Loi de Hess

L'enthalpie de réaction ne dépend que de l'état initial et de l'état final. On peut donc combiner des réactions connues dans un cycle thermochimique pour déterminer un \(\Delta_r H°\) inconnu.

Loi de Kirchhoff
$$\Delta_r H°(T_2) = \Delta_r H°(T_1) + \int_{T_1}^{T_2} \Delta_r C_p \,\mathrm{d}T$$

Permet de calculer l'enthalpie de réaction à une température différente de 298 K.

IV. Équilibre chimique

Constante d'équilibre
$$K° = \exp\!\left(-\frac{\Delta_r G°}{RT}\right)$$

Lien fondamental entre thermodynamique et équilibre : \(\Delta_r G° = -RT \ln K°\).

Loi de Van't Hoff
$$\frac{\mathrm{d}(\ln K°)}{\mathrm{d}T} = \frac{\Delta_r H°}{RT^2}$$
  • Si \(\Delta_r H° > 0\) (endothermique) : \(K°\) augmente avec \(T\).
  • Si \(\Delta_r H° < 0\) (exothermique) : \(K°\) diminue avec \(T\).
Quotient réactionnel \(Q\)
  • \(Q < K°\) : le système évolue dans le sens direct (formation de produits).
  • \(Q = K°\) : équilibre chimique.
  • \(Q > K°\) : le système évolue dans le sens indirect (retour vers les réactifs).

V. Électrochimie

Équation de Nernst
$$E = E° + \frac{RT}{nF}\,\ln Q_{\text{ox/red}}$$
  • \(E°\) : potentiel standard du couple ox/red.
  • \(n\) : nombre d'électrons échangés dans la demi-réaction.
  • \(F = 96\,485\;\text{C/mol}\) : constante de Faraday.
  • À 25 °C : \(\displaystyle E = E° + \frac{0{,}059}{n}\,\log Q_{\text{ox/red}}\)
Piles et accumulateurs
  • Force électromotrice : \(\text{fem} = E(\text{cathode}) - E(\text{anode})\)
  • Cathode : électrode où a lieu la réduction (gain d'électrons).
  • Anode : électrode où a lieu l'oxydation (perte d'électrons).
  • Lien avec l'enthalpie libre : \(\Delta_r G° = -nFE°\)
Méthode — Calculer la fem d'une pile
  1. Identifier les deux demi-réactions et les couples ox/red.
  2. Calculer \(E°\) pour chaque couple à partir des tables.
  3. \(\text{fem} = E°(\text{cathode}) - E°(\text{anode})\).
  4. Si les concentrations sont différentes de 1 mol/L, appliquer Nernst à chaque électrode.
Piège classique

Le potentiel de Nernst est intensif : il ne dépend pas de la stœchiométrie de la demi-réaction mais uniquement du quotient réactionnel. Ne pas multiplier \(E°\) quand on équilibre une équation !

VI. Lois d'absorption

Loi de Beer-Lambert
$$A = \varepsilon \cdot l \cdot c$$
  • \(A\) : absorbance (sans unité).
  • \(\varepsilon\) : coefficient d'extinction molaire (L·mol\(^{-1}\)·cm\(^{-1}\)).
  • \(l\) : longueur de traversée (cm).
  • \(c\) : concentration molaire (mol/L).

Forme exponentielle équivalente :

$$I = I_0 \,e^{-\mu x}$$
Loi d'Arrhenius
$$k(T) = A \cdot \exp\!\left(-\frac{E_a}{RT}\right)$$
  • \(k(T)\) : constante de vitesse à la température \(T\).
  • \(A\) : facteur pré-exponentiel (facteur de fréquence).
  • \(E_a\) : énergie d'activation (J/mol).
  • Forme linéarisée : \(\displaystyle \ln k = \ln A - \frac{E_a}{RT}\)

VII. Formules résumé

Cristallographie
$$\rho = \frac{Z \cdot M}{N_A \cdot a^3} \qquad 2d_{hkl}\sin\theta = n\lambda \qquad d_{hkl} = \frac{a}{\sqrt{h^2+k^2+l^2}}$$
Thermochimie & équilibre
$$\Delta_r G° = \Delta_r H° - T\Delta_r S° = -RT\ln K°$$
$$\frac{\mathrm{d}(\ln K°)}{\mathrm{d}T} = \frac{\Delta_r H°}{RT^2} \qquad \Delta_r H°(T_2) = \Delta_r H°(T_1) + \int_{T_1}^{T_2}\Delta_r C_p\,\mathrm{d}T$$
Électrochimie
$$E = E° + \frac{RT}{nF}\ln Q \qquad \Delta_r G° = -nFE° \qquad \text{fem} = E_{\text{cathode}} - E_{\text{anode}}$$
Absorption & cinétique
$$A = \varepsilon \cdot l \cdot c \qquad k(T) = A\,\exp\!\left(-\frac{E_a}{RT}\right)$$
