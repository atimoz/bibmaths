I. Polynômes d'endomorphismes

Définition — Algèbre \(\mathbb{K}[u]\)

Pour \(u \in \mathcal{L}(E)\), on note \(\mathbb{K}[u] = \{P(u) \mid P \in \mathbb{K}[X]\}\). C'est une sous-algèbre commutative de \(\mathcal{L}(E)\).

Définition — Polynôme minimal

Le polynôme minimal \(\pi_u\) de \(u\) est le polynôme unitaire de plus petit degré tel que \(\pi_u(u) = 0\).

  • \(\pi_u\) divise tout polynôme annulateur de \(u\).
  • \(\deg(\pi_u) \leq n = \dim E\).
  • \(\pi_u\) et \(\chi_u\) ont les mêmes racines (même spectre).
Proposition — Inversibilité

\(u\) est inversible \(\iff \pi_u(0) \neq 0\), et dans ce cas \(u^{-1} \in \mathbb{K}[u]\).

II. Polynôme caractéristique

Définition

\(\chi_A = \det(XI_n - A)\), polynôme unitaire de degré \(n\).

  • Racines de \(\chi_A\) = valeurs propres de \(A\).
  • \(\operatorname{tr}(A) = \sum \lambda_i\), \(\det(A) = \prod \lambda_i\).
Théorème de Cayley-Hamilton

\(\chi_A(A) = 0\). En conséquence : \(\pi_A \mid \chi_A\).

Conséquence immédiate

Pour \(n = 2\) : \(A^2 - \operatorname{tr}(A)\,A + \det(A)\,I_2 = 0\), ce qui donne directement \(A^{-1}\) si \(\det(A) \neq 0\).

III. Diagonalisation

CNS de diagonalisabilité

\(u\) diagonalisable \(\iff\) les deux conditions suivantes sont réunies :

  • \(\chi_u\) est scindé sur \(\mathbb{K}\).
  • Pour chaque valeur propre \(\lambda\) : \(\dim E_\lambda = \) multiplicité algébrique de \(\lambda\).
Critères suffisants
  • \(n\) valeurs propres distinctes \(\Rightarrow\) diagonalisable.
  • \(\exists\) polynôme annulateur scindé à racines simples \(\Rightarrow\) diagonalisable.
Caractérisation par le polynôme minimal

\(u\) diagonalisable \(\iff \pi_u\) est scindé à racines simples.

IV. Trigonalisation

Critère de trigonalisabilité

\(u\) trigonalisable \(\iff \chi_u\) scindé sur \(\mathbb{K}\). Sur \(\mathbb{C}\), toute matrice est trigonalisable.

Sous-espaces caractéristiques

Pour \(\lambda\) valeur propre de multiplicité \(\alpha\) :

$$F_\lambda = \ker(u - \lambda\,\mathrm{Id})^\alpha$$
  • \(E_\lambda \subset F_\lambda\) et \(\dim F_\lambda = \alpha\).
  • Si \(\chi_u\) scindé : \(E = \bigoplus_\lambda F_\lambda\).
Lemme de décomposition des noyaux

Si \(P = P_1 \cdots P_r\) avec les \(P_i\) deux à deux premiers entre eux :

$$\ker P(u) = \bigoplus_{i=1}^r \ker P_i(u)$$

V. Formulaire & calculs

Puissances par diagonalisation

Si \(A = PDP^{-1}\) avec \(D = \operatorname{diag}(\lambda_1, \dots, \lambda_n)\) :

$$A^k = P\,D^k\,P^{-1} = P\,\operatorname{diag}(\lambda_1^k, \dots, \lambda_n^k)\,P^{-1}$$
Exponentielle de matrice

Si \(A = PDP^{-1}\) :

$$e^{tA} = P\,\operatorname{diag}(e^{t\lambda_1}, \dots, e^{t\lambda_n})\,P^{-1}$$

Si \(A\) non diagonalisable, utiliser la décomposition de Dunford :

$$e^{tA} = e^{t(D+N)} = e^{tD}\,e^{tN} \quad\text{(car } DN = ND\text{)}$$
Décomposition de Dunford

Si \(\chi_u\) scindé : \(u = d + n\) avec :

  • \(d\) diagonalisable, \(n\) nilpotent,
  • \(dn = nd\),
  • \(d, n \in \mathbb{K}[u]\).

Cette décomposition est unique.

Projecteurs spectraux

Si \(u\) diagonalisable avec \(\operatorname{Sp}(u) = \{\lambda_1, \dots, \lambda_p\}\), les projecteurs spectraux \(\pi_i\) vérifient :

$$\pi_i = \prod_{j \neq i} \frac{u - \lambda_j\,\mathrm{Id}}{\lambda_i - \lambda_j}, \qquad u = \sum_{i=1}^p \lambda_i\,\pi_i, \qquad \sum_{i=1}^p \pi_i = \mathrm{Id}$$

VI. Méthodes types CCINP

Méthode — Diagonaliser une matrice 3×3
  1. Calculer \(\chi_A = \det(XI_3 - A)\) (développement par une ligne ou Sarrus).
  2. Trouver les racines (valeur évidente \(\lambda_0\), puis factoriser).
  3. Pour chaque \(\lambda_i\) : résoudre \((A - \lambda_i I)X = 0\) pour obtenir une base de \(E_{\lambda_i}\).
  4. Vérifier : \(\sum \dim E_{\lambda_i} = 3\). Si oui, \(A\) est diagonalisable.
  5. Former \(P\) (colonnes = vecteurs propres), \(D = \operatorname{diag}(\lambda_1, \lambda_2, \lambda_3)\).
Méthode — Calculer \(A^n\) ou \(e^{tA}\)
  1. Diagonaliser \(A = PDP^{-1}\) (ou trigonaliser si non diagonalisable).
  2. \(A^n = PD^nP^{-1}\), \(e^{tA} = Pe^{tD}P^{-1}\).
  3. Alternative sans \(P^{-1}\) : chercher \(A^n = a_n I + b_n A + c_n A^2\) via Cayley-Hamilton.
Pièges classiques
  • Oublier de vérifier que \(\chi\) est scindé sur \(\mathbb{R}\) avant de conclure.
  • Confondre multiplicité algébrique (ordre de la racine dans \(\chi\)) et multiplicité géométrique (\(\dim E_\lambda\)).
  • Oublier que sur \(\mathbb{C}\) toute matrice est trigonalisable mais pas forcément diagonalisable.
  • Se tromper dans le sens de \(\chi_A = \det(XI - A)\) vs \(\det(A - XI)\) (signe).
