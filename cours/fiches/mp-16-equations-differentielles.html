Equations differentielles lineaires

Fiche de revision condensee : EDL d'ordre 1 et 2, equations caracteristiques, seconds membres speciaux, systemes differentiels \(X'=AX\), exponentielle de matrice, methodes de resolution CCINP. Pas de demonstrations, uniquement les resultats et techniques operatoires.

I. EDL d'ordre 1

Forme generale

Une EDL d'ordre 1 s'ecrit :

$$y'(x) + a(x)\,y(x) = b(x)$$

ou \(a, b : I \to \mathbb{K}\) sont continues sur un intervalle \(I\).

Theoreme de Cauchy-Lipschitz lineaire

Pour tout \((x_0, y_0) \in I \times \mathbb{K}\), il existe une unique solution maximale \(y\) definie sur \(I\) tout entier, verifiant \(y(x_0) = y_0\).

Solutions de l'equation homogene \(y' + a(x)y = 0\)

L'espace des solutions est une droite vectorielle :

$$\mathcal{S}_0 = \left\{\lambda\, e^{-A(x)},\; \lambda \in \mathbb{K}\right\} \quad \text{ou} \quad A(x) = \int_{x_0}^{x} a(t)\,dt$$
Variation de la constante

On cherche une solution particuliere sous la forme \(y_p(x) = \lambda(x)\,e^{-A(x)}\). En injectant dans l'equation complete :

$$\lambda'(x) = b(x)\,e^{A(x)} \quad \Longrightarrow \quad y_p(x) = e^{-A(x)}\int_{x_0}^{x} b(t)\,e^{A(t)}\,dt$$

La solution generale est alors \(y = y_h + y_p\) :

$$y(x) = e^{-A(x)}\left(C + \int_{x_0}^{x} b(t)\,e^{A(t)}\,dt\right), \quad C \in \mathbb{K}$$
Remarque

La constante \(C\) est determinee par la condition initiale : \(y(x_0) = y_0\) donne \(C = y_0\).

II. EDL d'ordre 2 a coefficients constants

Forme generale

On considere l'equation a coefficients constants \(a, b \in \mathbb{R}\) :

$$y''(x) + a\,y'(x) + b\,y(x) = f(x)$$
Equation caracteristique

L'equation homogene \(y'' + ay' + by = 0\) admet pour equation caracteristique :

$$r^2 + ar + b = 0 \qquad \Delta = a^2 - 4b$$
Les trois cas pour la solution homogene

Cas 1 : \(\Delta > 0\) -- Deux racines reelles distinctes \(r_1, r_2\) :

$$y_h(x) = C_1\,e^{r_1 x} + C_2\,e^{r_2 x}$$

Cas 2 : \(\Delta = 0\) -- Une racine double \(r_0 = -a/2\) :

$$y_h(x) = (C_1 + C_2\,x)\,e^{r_0 x}$$

Cas 3 : \(\Delta < 0\) -- Deux racines complexes conjuguees \(r = \alpha \pm i\beta\) :

$$y_h(x) = e^{\alpha x}\bigl(C_1\cos(\beta x) + C_2\sin(\beta x)\bigr)$$

avec \(\alpha = -a/2\) et \(\beta = \sqrt{4b - a^2}/2\).

Structure de la solution generale

L'espace des solutions de l'equation complete est un espace affine :

$$\mathcal{S} = y_p + \mathcal{S}_0$$

ou \(y_p\) est une solution particuliere quelconque et \(\mathcal{S}_0\) l'espace vectoriel (de dimension 2) des solutions de l'homogene.

III. Second membre special

Second membre \(f(x) = P(x)\,e^{\alpha x}\)

Pour \(y'' + ay' + by = P(x)\,e^{\alpha x}\) avec \(P\) polynome de degre \(n\), on cherche une solution particuliere de la forme :

$$y_p(x) = x^s\,Q(x)\,e^{\alpha x}$$

ou \(Q\) est un polynome de degre \(n\) a determiner, et \(s\) est l'ordre de multiplicite de \(\alpha\) comme racine de l'equation caracteristique :

  • \(s = 0\) si \(\alpha\) n'est pas racine
  • \(s = 1\) si \(\alpha\) est racine simple
  • \(s = 2\) si \(\alpha\) est racine double
Principe de superposition

Si \(y_1\) est solution de \(Ly = f_1\) et \(y_2\) est solution de \(Ly = f_2\), alors pour tous \(\lambda_1, \lambda_2 \in \mathbb{K}\) :

$$\lambda_1 y_1 + \lambda_2 y_2 \quad \text{est solution de} \quad Ly = \lambda_1 f_1 + \lambda_2 f_2$$
Methode de complexification pour \(\cos\) et \(\sin\)

Pour resoudre \(y'' + ay' + by = P(x)\,e^{\alpha x}\cos(\beta x)\) :

  • Poser \(\tilde{f}(x) = P(x)\,e^{(\alpha + i\beta)x}\) (on remplace \(\cos\) par l'exponentielle complexe).
  • Resoudre \(y'' + ay' + by = \tilde{f}(x)\) avec la methode du second membre exponentiel.
  • La solution particuliere reelle est \(y_p = \operatorname{Re}(\tilde{y}_p)\).
  • Pour \(\sin(\beta x)\), prendre \(\operatorname{Im}(\tilde{y}_p)\).
Remarque

La complexification evite les calculs lourds d'identification. C'est la methode a privilegier en concours.

IV. Systemes differentiels

Systeme differentiel lineaire

Soit \(A \in \mathcal{M}_n(\mathbb{R})\). Le systeme differentiel homogene s'ecrit :

$$X'(t) = A\,X(t), \qquad X(t) \in \mathbb{R}^n$$

L'ensemble des solutions est un espace vectoriel de dimension \(n\).

Solution par l'exponentielle de matrice

La solution du probleme de Cauchy \(X' = AX\), \(X(0) = X_0\) est :

$$X(t) = e^{tA}\,X_0$$

ou \(e^{tA}\) est l'exponentielle de matrice, definie par :

$$e^{tA} = \sum_{k=0}^{+\infty} \frac{(tA)^k}{k!} = I_n + tA + \frac{t^2 A^2}{2!} + \cdots$$

Cette serie converge pour toute matrice \(A\) et tout \(t \in \mathbb{R}\).

Proprietes de \(e^{tA}\)
  • \(e^{0 \cdot A} = I_n\)
  • \(\dfrac{d}{dt}\,e^{tA} = A\,e^{tA} = e^{tA}\,A\)
  • \(e^{(s+t)A} = e^{sA}\,e^{tA}\)
  • \(\bigl(e^{tA}\bigr)^{-1} = e^{-tA}\)
  • Si \(AB = BA\), alors \(e^{A+B} = e^A\,e^B\)
Resolvante

La matrice \(R(t) = e^{tA}\) est l'unique solution du systeme matriciel :

$$R'(t) = A\,R(t), \qquad R(0) = I_n$$

On l'appelle la resolvante du systeme. Toute solution s'ecrit \(X(t) = R(t)\,X_0\).

Attention

La formule \(e^{A+B} = e^A\,e^B\) est fausse en general si \(AB \neq BA\). C'est un piege classique.

V. Formulaire

Exponentielle de matrice diagonalisable

Si \(A = P\,D\,P^{-1}\) avec \(D = \operatorname{diag}(\lambda_1, \ldots, \lambda_n)\), alors :

$$e^{tA} = P\,e^{tD}\,P^{-1} = P\,\operatorname{diag}\bigl(e^{\lambda_1 t}, \ldots, e^{\lambda_n t}\bigr)\,P^{-1}$$
Exponentielle de matrice trigonalisable

Si \(A = D + N\) avec \(D\) diagonale, \(N\) nilpotente et \(DN = ND\), alors :

$$e^{tA} = e^{t(D+N)} = e^{tD}\,e^{tN}$$

avec \(e^{tN}\) un polynome en \(t\) (puisque \(N\) est nilpotente, la serie est finie) :

$$e^{tN} = I + tN + \frac{t^2}{2!}N^2 + \cdots + \frac{t^{p-1}}{(p-1)!}N^{p-1} \quad \text{si } N^p = 0$$
Cas \(2 \times 2\) a connaitre

Pour \(A = \begin{pmatrix} 0 & -\omega \\ \omega & 0 \end{pmatrix}\) :

$$e^{tA} = \begin{pmatrix} \cos(\omega t) & -\sin(\omega t) \\ \sin(\omega t) & \cos(\omega t) \end{pmatrix}$$

Pour \(A = \begin{pmatrix} \lambda & 1 \\ 0 & \lambda \end{pmatrix}\) :

$$e^{tA} = e^{\lambda t}\begin{pmatrix} 1 & t \\ 0 & 1 \end{pmatrix}$$
Wronskien

Pour un systeme \(X' = AX\), si \((X_1, \ldots, X_n)\) est un systeme de solutions, le Wronskien est :

$$W(t) = \det\bigl(X_1(t), \ldots, X_n(t)\bigr)$$

Il verifie la formule de Liouville :

$$W(t) = W(0)\,e^{t\,\operatorname{tr}(A)}$$

En particulier, \(W\) est soit identiquement nul, soit ne s'annule jamais.

Variation de la constante (systemes)

Pour le systeme complet \(X' = AX + B(t)\), on pose \(X(t) = e^{tA}\,C(t)\) :

$$C'(t) = e^{-tA}\,B(t) \quad \Longrightarrow \quad X(t) = e^{tA}\left(X_0 + \int_0^t e^{-sA}\,B(s)\,ds\right)$$

VI. Methodes CCINP

Methode -- Resoudre une EDL2 complete
  • Etape 1 : Ecrire l'equation caracteristique \(r^2 + ar + b = 0\) et calculer \(\Delta\).
  • Etape 2 : En deduire la solution homogene \(y_h\) (selon les trois cas).
  • Etape 3 : Determiner la forme de la solution particuliere \(y_p\) selon le type de second membre (exponentiel-polynome, complexification si cos/sin).
  • Etape 4 : Injecter \(y_p\) dans l'equation, identifier les coefficients.
  • Etape 5 : Ecrire \(y = y_h + y_p\) et determiner les constantes avec les conditions initiales.
Methode -- Resoudre \(X' = AX\) par diagonalisation
  • Etape 1 : Calculer les valeurs propres de \(A\) (polynome caracteristique).
  • Etape 2 : Trouver les vecteurs propres associes, former \(P\).
  • Etape 3 : Ecrire \(A = PDP^{-1}\), d'ou \(e^{tA} = P\,\operatorname{diag}(e^{\lambda_i t})\,P^{-1}\).
  • Etape 4 : La solution est \(X(t) = e^{tA}\,X_0\).
  • Si \(A\) n'est pas diagonalisable : trigonaliser \(A = P\,T\,P^{-1}\), decomposer \(T = D + N\) et utiliser \(e^{tT} = e^{tD}\,e^{tN}\).
Pieges classiques a eviter
  • Oublier le facteur \(x^s\) dans la solution particuliere quand \(\alpha\) est racine de l'equation caracteristique (surtout en racine double : \(s = 2\)).
  • Confondre solution generale et solution particuliere : la solution generale contient les constantes d'integration, la solution particuliere n'en a pas.
  • Erreur sur l'exponentielle de matrice : \(e^{A+B} \neq e^A e^B\) sauf si \(AB = BA\).
  • Oublier d'inverser \(P\) : dans \(e^{tA} = P\,e^{tD}\,P^{-1}\), ne pas oublier le facteur \(P^{-1}\) a droite.
  • Mauvaise identification en complexification : bien prendre \(\operatorname{Re}\) pour \(\cos\) et \(\operatorname{Im}\) pour \(\sin\).
  • Cauchy-Lipschitz : en lineaire, les solutions sont globales (definies sur \(I\) entier). Ne pas chercher un intervalle maximal reduit.
